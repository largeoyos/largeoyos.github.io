# 更新日志 (Changelog)

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/) (语义化版本) 规范 (`x.x.x`)。

## [1.5.0] - 2026-04-28

### 新增 (Added)
- 首页新增“布局设置”按钮，支持用户在“日历优先”与“项目优先”两种布局顺序间切换。
- 布局偏好支持本地持久化存储（LocalStorage）。
- 全站版本号同步更新至 v1.5.0。

## [1.4.0] - 2026-04-28

### 新增 (Added)
- 首页日历模块支持节气显示，并对有节气的日期进行视觉强化。
- 优化日历单元格标签显示逻辑，移除冗余的“日历”占位符，突出“今天”与“节气”。

## [1.3.1] - 2026-04-28

### 变更 (Changed)
- 首页每日签到日历的农历与干支计算切换为 `lunar-javascript`（Solar-Calculator）实现。
- 新增本地静态依赖脚本 `js/vendor/lunar.js`，避免运行时依赖外部 CDN。

## [1.3.0] - 2026-04-28

### 新增 (Added)
- 首页新增每日签到日历，支持公历、农历、干支、星期与周次信息展示。
- 新增古今时钟显示，支持“时辰 / 初正 / 刻”格式的当前时间展示。
- 点击日历日期后可查看对应运势值。

## [1.2.0] - 2026-04-28

### 新增 (Added)
- 新增博客文章《C语言中结构体自引用报错原因分析》。

## [1.1.0] - 2026-04-28

### 新增 (Added)
- 新增 `auto_split_docs.js` 脚本，支持基于文件修改时间戳（mtime）进行全自动增量 Markdown 文档拆解。
- 引入项目版本号控制约定，并在主页页脚展示当前版本号。
- 建立根目录 `.github/copilot-instructions.md` 进行 AI 辅助开发标准化提示。

### 移除 (Removed)
- 废弃并删除了旧版本的冗余文件：`ai-tutorials/split.js`、`process_new_docs.js` 和 `test_hljs.js`。

## [1.0.0] - 2026-04-28 及之前

### 初始化 (Initial Release)
- 初始建立并部署个人静态博客系统。
- 包含个人简介、AI 教程文库页面（长文分发）与游戏大厅导航。
- 集成经典的 Web 游戏（如 2048 与 并购 Acquire）。
- 配置 `.nojekyll` 用以直接托管 `_assets` 等前端框架编译产出。
