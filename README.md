# largeoyos.github.io

这是一个部署在 GitHub Pages 上的个人网站/知识库仓库。包含了个人博客、计算机科学相关的学习笔记（AI教程），以及一些经典的网页小游戏。

## 目录结构与功能模块

- **博客 (Blog)**: 位于 [`blog/`](blog/) 目录下，包含个人相关的文章记录，使用 Markdown 进行编写与展示。
- **AI 教程与笔记 (AI Tutorials)**: 位于 [`ai-tutorials/`](ai-tutorials/) 目录下，包含了对计算机基础课程的深度学习笔记。目前涵盖：
  - 编译原理 (Compilers)
  - 计算机网络 (Computer Networks)
  - 数据结构 (Data Structures)
- **网页游戏 (Games)**:
  - **并购 (Acquire)**: 经典桌游的网页版实现。
  - **2048 (flxg-2048)**: 经典的 2048 网页小游戏。

## 技术栈

- 原生 HTML / CSS / JavaScript 作为主站架构
- **Vite & React & TypeScript**: 用于部分复杂项目（如桌游《并购》的实现）
- **Markdown**: 作为博客与知识库笔记的主要内容格式
