# 信息论入门课程轮廓

很高兴带你入门信息论！在给你轮廓之前，我想先了解一下你的背景，这样课程能更贴合你。不过我先把大致框架给你看看，让你有个预期。

### 课程大纲（8 个模块）

**模块 1：什么是"信息"？** 从直觉出发——为什么"明天太阳升起"没信息，而"明天下雪"有信息？引出"意外程度"和概率的联系。

**模块 2：熵（Entropy）** 信息论的核心概念。香农熵 H(X) 的定义、含义、性质。这是整门课的地基。

**模块 3：联合熵、条件熵、互信息** 两个随机变量之间的信息关系。互信息 I(X;Y) 是个非常漂亮的量。

**模块 4：相对熵（KL 散度）** 衡量两个分布的"距离"。在机器学习里无处不在。

**模块 5：信源编码（数据压缩）** 香农第一定理。为什么熵是压缩的极限？Huffman 编码。

**模块 6：信道与信道容量** 有噪声的通信。信道容量 C 的定义。

**模块 7：信道编码定理** 香农第二定理——信息论最震撼的结果：只要传输速率低于 C，就能做到几乎无错传输。

**模块 8：应用与延伸** 在机器学习、密码学、生物学中的应用。



很好，背景清楚了。电子信息工程专业学信息论是**本专业的核心课**——香农当年就是为了解决通信问题才创立这门学科的。你以后会看到它贯穿通信原理、编码理论、数字信号处理。

### 开始模块 1 之前，先做个小热身

我想让你自己先思考，而不是直接灌输定义。请回答下面两个问题，不用查资料，凭直觉就行：

**问题 1**：下面这四句话，哪句"信息量最大"？把它们从信息量最小到最大排个序，并说说你的理由。

- A. "明天太阳会从东边升起"
- B. "明天北京会下雨"
- C. "明天北京会下雪"（假设现在是七月）
- D. "明天你会中五百万彩票"

**问题 2**：假设我抛一枚**均匀的硬币**告诉你结果，和我抛一枚**作弊的硬币（99% 出正面）**告诉你结果——哪一次我给你的"信息"更多？为什么？
