# 模块 1：什么是"信息"？

### 一、从一个反直觉的事实开始

我们日常说"信息"这个词，含义很模糊：一篇文章有信息、一张照片有信息、一句话有信息。但这些"信息"的多少，怎么量化？

香农在 1948 年给出了一个石破天惊的回答：**信息的多少，与内容是什么无关，只与"它有多让人意外"有关。**

这是什么意思？我们看那四句话：

- A. "明天太阳从东边升起"——你早就知道，**毫无意外**，信息量 = 0
- B. "明天北京下雨"——挺常见的事，**有点意外**，信息量较小
- C. "七月的北京下雪"——几乎不可能发生，**非常意外**，信息量较大
- D. "你中五百万彩票"——概率极低，**极度意外**，信息量最大

**信息量的排序：A < B < C < D**

注意：D 对你的"实际价值"显然最大（你发财了），但 A 这种"太阳升起"的废话也并非毫无价值（万一哪天太阳不升起呢？）。香农直接抛弃了"价值""意义""真假"这些主观维度，只盯住一个客观的东西——**概率**。

> **核心洞察**：一件事发生的概率越小，它真的发生时带来的信息量就越大。

### 二、那枚作弊的硬币

再看第二个问题。一枚均匀硬币，正反各 50%；一枚作弊硬币，99% 出正面。

我抛硬币告诉你结果——哪种情况你获得的信息更多？

直觉上你可能觉得"作弊硬币更不寻常，信息更多"。**错了**。

想想看：作弊硬币我还没抛，你就能猜到 99% 是正面。我告诉你"正面"，你心里说"我早猜到了"——**没什么信息**。偶尔出反面才让你意外，但这种意外只有 1% 的概率发生。

均匀硬币呢？抛之前你完全猜不到，正反各半。我告诉你结果，你**真的从"不知道"变成了"知道"**——这才是实打实的信息。

> **核心洞察**：信息量衡量的是**不确定性的消除**。事先越不确定，告知后获得的信息就越多。

均匀硬币的不确定性最大，所以它的结果信息量最大。这个规律可以推广：**等概率分布的随机事件，平均信息量最大。**

### 三、给"信息量"找一个公式

现在我们知道：信息量是概率的某种函数。设事件 xx 发生的概率是 p(x)p(x)，它的信息量记作 I(x)I(x)。我们希望 I(x)I(x) 满足三个条件：

**条件 1：概率越小，信息量越大。** pp 越小，II 越大。所以 II 是 pp 的**减函数**。

**条件 2：必然事件信息量为零。** 当 p(x)=1p(x)=1 时，I(x)=0I(x)=0。

**条件 3：独立事件信息量可加。** 这条最关键，我们仔细看。

假设我抛两次独立的均匀硬币。

- 第一次告诉你结果：消除了 2 种可能 → 提供了某种"1 单位"的信息
- 第二次告诉你结果：又消除了 2 种可能 → 又提供了"1 单位"的信息

加起来应该是 2 个单位的信息。

但从概率角度看，"两次结果"作为一个整体事件，概率是 12×12=1421​×21​=41​。

也就是说：

I(p1⋅p2)=I(p1)+I(p2)I(p1​⋅p2​)=I(p1​)+I(p2​)

**概率相乘 → 信息量相加**。这是什么函数？**对数函数**！

log⁡(a⋅b)=log⁡a+log⁡blog(a⋅b)=loga+logb

### 四、香农的定义

把上面的要求合在一起，香农给出了信息量的定义：

 I(x)=−log⁡p(x) I(x)=−logp(x)​

为什么前面有个负号？因为 0<p(x)≤10<p(x)≤1，对数 log⁡p(x)≤0logp(x)≤0，加上负号让 I(x)≥0I(x)≥0，符合"信息量非负"的直觉。

我们验证一下三个条件：

- pp 越小 → log⁡plogp 越小（绝对值越大）→ −log⁡p−logp 越大 ✓
- p=1p=1 时 → log⁡1=0log1=0 → I=0I=0 ✓
- I(p1p2)=−log⁡(p1p2)=−log⁡p1−log⁡p2=I(p1)+I(p2)I(p1​p2​)=−log(p1​p2​)=−logp1​−logp2​=I(p1​)+I(p2​) ✓

完美。

### 五、对数的底——单位问题

对数的底数还没定。底取什么，就决定了信息的"单位"：

|底数|单位|用在哪|
|---|---|---|
|2|**比特 (bit)**|通信、计算机、信息论标准单位|
|ee|nat|数学推导中常用|
|10|hartley / dit|历史上用过|

**信息论里默认用 2 为底**，因为计算机就是二进制的，一个比特对应一次"是/否"的选择。本课程后续没有特别说明都是 log⁡2log2​。

### 六、用公式重新算那四句话

假设：

- 太阳从东方升起：p=1p=1（必然）
- 北京下雨：p=0.3p=0.3
- 七月北京下雪：p=0.0001p=0.0001
- 中五百万彩票：p=10−7p=10−7

信息量分别是：

- IA=−log⁡21=0IA​=−log2​1=0 比特
- IB=−log⁡20.3≈1.74IB​=−log2​0.3≈1.74 比特

数学和直觉吻合。

### 七、一个关键例子：均匀硬币

均匀硬币 p(正)=p(反)=12p(正)=p(反)=21​，每个结果的信息量：

I=−log⁡212=1 比特I=−log2​21​=1 比特

**这就是"1 比特"的物理含义**：一次等概率二选一所携带的信息。这也是计算机里"1 比特"这个名字的由来——它不是字节意义上的比特，而是**信息论意义上的比特**。

### 八、本模块小结

记住三件事：

1. **信息量衡量"意外程度"**，与概率成反向关系：I(x)=−log⁡p(x)I(x)=−logp(x)。
2. **公式从三个公理推出**：减函数、必然事件为零、独立事件可加 → 必然是对数形式。
3. **比特是基本单位**，对应一次等概率二选一的信息量。
