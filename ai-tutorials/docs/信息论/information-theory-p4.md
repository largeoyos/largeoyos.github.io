# 模块 3：联合熵、条件熵、互信息

到现在我们处理的都是**单个**随机变量。但通信的本质是"输入 → 信道 → 输出"——发送 XX，收到 YY。XX 和 YY 是成对的。

这一节我们要回答四个问题：

1. **(X,Y)(X,Y) 整体**有多少不确定性？→ 联合熵
2. **已知 YY 之后**，XX 还剩多少不确定性？→ 条件熵
3. **YY 告诉了我们关于 XX 的多少信息**？→ 互信息
4. 这些量之间有什么关系？→ 一张神图（韦恩图）

---

### 一、补一个概念：联合分布与条件分布

你只学过高中概率，这里需要先补两个概念。

#### 联合分布

设 XX 和 YY 是两个随机变量。它们一起取值 (x,y)(x,y) 的概率记作：

p(x,y)=P(X=x,Y=y)p(x,y)=P(X=x,Y=y)

这叫**联合概率分布**。它描述了 XX 和 YY "同时"取某对值的概率。

举例：抛两枚硬币，XX 是第一枚，YY 是第二枚。

||Y=Y=正|Y=Y=反|
|---|---|---|
|X=X=正|0.25|0.25|
|X=X=反|0.25|0.25|

这就是 p(x,y)p(x,y) 的全部信息（一个 2×22×2 表）。

#### 边缘分布

如果我只关心 XX、不关心 YY，可以把 YY "求和掉"：

p(x)=∑yp(x,y)p(x)=y∑​p(x,y)

这叫**边缘分布**（之所以叫"边缘"，是因为在表格里它出现在表格边缘的求和栏）。

#### 条件分布

已知 Y=yY=y 的前提下，X=xX=x 的概率：

p(x∣y)=p(x,y)p(y)p(x∣y)=p(y)p(x,y)​

这就是高中你学过的条件概率，只是写成随机变量的形式。

#### 一个要记住的关系

p(x,y)=p(y)⋅p(x∣y)=p(x)⋅p(y∣x)p(x,y)=p(y)⋅p(x∣y)=p(x)⋅p(y∣x)

这叫**乘法公式**，后面反复要用。

#### 独立的定义

XX 与 YY **独立**，当且仅当对所有 x,yx,y：

p(x,y)=p(x)⋅p(y)p(x,y)=p(x)⋅p(y)

或者等价地 p(x∣y)=p(x)p(x∣y)=p(x)（已知 YY 不改变 XX 的分布）。

好，概率工具齐了。下面进入正题。

---

### 二、联合熵 H(X,Y)H(X,Y)

把 (X,Y)(X,Y) 看成**一个整体随机变量**——它的取值是一对 (x,y)(x,y)，概率是 p(x,y)p(x,y)。直接套上一节熵的定义：

 H(X,Y)=−∑x,yp(x,y)log⁡p(x,y) H(X,Y)=−x,y∑​p(x,y)logp(x,y)​

它衡量的是 **(X,Y)(X,Y) 这一对的总不确定性**。

举例：两枚独立均匀硬币，4 种等概率结果，每种 0.250.25：

H(X,Y)=−4×0.25log⁡20.25=−log⁡20.25=2 比特H(X,Y)=−4×0.25log2​0.25=−log2​0.25=2 比特

正好是 H(X)+H(Y)=1+1=2H(X)+H(Y)=1+1=2。这不是巧合——独立时，联合熵等于两个熵之和（后面会证）。

---

### 三、条件熵 H(X∣Y)H(X∣Y)

这是这一节**最重要也最容易混的概念**，慢慢讲。

#### 先想一个具体场景

设 XX 是"明天是否下雨"，YY 是"今天的气压读数"。YY 不能完全确定 XX，但**已知 YY 之后，XX 的不确定性会减小**。

我们想量化：**已知 YY 之后，XX 还剩多少不确定性？**

#### 第一步：固定 Y=yY=y 时的熵

如果 YY 已经确定取了某个具体值 yy，那 XX 服从条件分布 p(x∣y)p(x∣y)。这个分布的熵是：

H(X∣Y=y)=−∑xp(x∣y)log⁡p(x∣y)H(X∣Y=y)=−x∑​p(x∣y)logp(x∣y)

注意：**这是一个具体数值**，对应 YY 取某一个特定值 yy 时，XX 还剩的不确定性。

#### 第二步：对所有 yy 取期望

但 YY 本身也是随机的，会取不同的值。我们要的是**平均**意义下的"已知 YY 后 XX 的不确定性"——也就是把上面那个数对 YY 取期望：

H(X∣Y)=∑yp(y)⋅H(X∣Y=y)H(X∣Y)=y∑​p(y)⋅H(X∣Y=y)

展开来看：

H(X∣Y)=∑yp(y)[−∑xp(x∣y)log⁡p(x∣y)]H(X∣Y)=y∑​p(y)[−x∑​p(x∣y)logp(x∣y)]

把 p(y)p(y) 乘进去，再用 p(x,y)=p(y)p(x∣y)p(x,y)=p(y)p(x∣y)：

 H(X∣Y)=−∑x,yp(x,y)log⁡p(x∣y) H(X∣Y)=−x,y∑​p(x,y)logp(x∣y)​

这就是**条件熵**的定义式。

#### 务必理清的细节

注意公式里有两个不同的概率，**不要搞混**：

- 求和的"权重"是 p(x,y)p(x,y)（联合概率）
- 取对数的对象是 p(x∣y)p(x∣y)（条件概率）

**口诀**：联合概率加权，条件概率取对数。

#### 直觉理解

- H(X)H(X)：你对 XX 的不确定性。
- H(X∣Y)H(X∣Y)：**已经看到 YY 之后**，你对 XX 还剩的不确定性。
- 显然 H(X∣Y)≤H(X)H(X∣Y)≤H(X)——多看到一个相关变量，不确定性只会减少或不变，**不可能增加**。这条不等式叫"**条件熵不大于熵**"，是信息论中极其重要的事实。

#### 两个极端例子

**例 1：XX 与 YY 独立**

p(x∣y)=p(x)p(x∣y)=p(x)，所以：

H(X∣Y)=−∑x,yp(x,y)log⁡p(x)=−∑xp(x)log⁡p(x)=H(X)H(X∣Y)=−x,y∑​p(x,y)logp(x)=−x∑​p(x)logp(x)=H(X)

**YY 啥都没告诉你关于 XX 的信息**，所以 XX 的不确定性没变。

**例 2：YY 完全决定 XX**（比如 X=YX=Y 或 X=f(Y)X=f(Y)）

已知 YY 后 XX 完全确定，H(X∣Y=y)=0H(X∣Y=y)=0 对所有 yy 成立，所以：

H(X∣Y)=0H(X∣Y)=0

### 四、链式法则——一个漂亮的恒等式

把条件熵的定义和联合熵串起来，能得到一个非常重要的恒等式。

我们看 H(X,Y)H(X,Y)。利用 p(x,y)=p(y)p(x∣y)p(x,y)=p(y)p(x∣y)：

H(X,Y)=−∑x,yp(x,y)log⁡p(x,y)=−∑x,yp(x,y)log⁡[p(y)p(x∣y)]=−∑x,yp(x,y)log⁡p(y)−∑x,yp(x,y)log⁡p(x∣y)=−∑yp(y)log⁡p(y)+H(X∣Y)=H(Y)+H(X∣Y)H(X,Y)​=−x,y∑​p(x,y)logp(x,y)=−x,y∑​p(x,y)log[p(y)p(x∣y)]=−x,y∑​p(x,y)logp(y)−x,y∑​p(x,y)logp(x∣y)=−y∑​p(y)logp(y)+H(X∣Y)=H(Y)+H(X∣Y)​

（中间倒数第二步用了 ∑xp(x,y)=p(y)∑x​p(x,y)=p(y) 把 xx 求和掉。）

得到**链式法则**：

 H(X,Y)=H(Y)+H(X∣Y)=H(X)+H(Y∣X) H(X,Y)=H(Y)+H(X∣Y)=H(X)+H(Y∣X)​

**直观解读**：(X,Y)(X,Y) 总的不确定性 = "先看 YY" 的不确定性 + "已知 YY 后再看 XX" 的不确定性。这非常符合直觉——你可以分两步揭开 (X,Y)(X,Y)。

### 五、互信息 I(X;Y)I(X;Y)

终于到本节的高潮——互信息。

#### 提出问题

我们已经有两个量：

- H(X)H(X)：XX 原本的不确定性
- H(X∣Y)H(X∣Y)：已知 YY 之后 XX 还剩的不确定性

**两者之差就是"YY 告诉了我们关于 XX 的多少信息"**：

 I(X;Y)=H(X)−H(X∣Y) I(X;Y)=H(X)−H(X∣Y)​

这个量叫做 **XX 与 YY 的互信息**（Mutual Information），是信息论中最重要的概念之一，地位甚至高于熵。

#### 互信息的对称性

直觉上"YY 告诉我 XX 多少"应该等于"XX 告诉我 YY 多少"。我们验证一下。

由链式法则：

H(X)+H(Y∣X)=H(Y)+H(X∣Y)H(X)+H(Y∣X)=H(Y)+H(X∣Y)

整理得：

H(X)−H(X∣Y)=H(Y)−H(Y∣X)H(X)−H(X∣Y)=H(Y)−H(Y∣X)

也就是：

I(X;Y)=I(Y;X)I(X;Y)=I(Y;X)

**互信息是对称的**。所以记号写 I(X;Y)I(X;Y)（用分号，强调对称），不写箭头方向。

#### 互信息的另一个公式

利用 H(X∣Y)=H(X,Y)−H(Y)H(X∣Y)=H(X,Y)−H(Y)：

I(X;Y)=H(X)+H(Y)−H(X,Y)I(X;Y)=H(X)+H(Y)−H(X,Y)

直接展开成概率形式：

 I(X;Y)=∑x,yp(x,y)log⁡p(x,y)p(x)p(y) I(X;Y)=x,y∑​p(x,y)logp(x)p(y)p(x,y)​​

这个形式很有内涵：它衡量的是**联合分布 p(x,y)p(x,y) 与"独立分布乘积" p(x)p(y)p(x)p(y) 之间的差异**。

- 如果 X,YX,Y 独立 → p(x,y)=p(x)p(y)p(x,y)=p(x)p(y) → 比值为 1 → log⁡1=0log1=0 → I(X;Y)=0I(X;Y)=0
- 如果 X,YX,Y 越"紧密相关" → 联合分布偏离独立越远 → 互信息越大

下一模块讲 KL 散度时你会发现：**互信息其实就是联合分布与边缘乘积分布的 KL 散度**。先记住这个伏笔。

#### 互信息的性质

**性质 1：非负性** I(X;Y)≥0I(X;Y)≥0

等号成立 ⟺ XX 与 YY 独立。

**直觉**：知道 YY 至多帮你减少对 XX 的不确定性，不可能增加。**多一个观测不会让你"更糊涂"**——这是信息论的基本伦理。

严格证明需要 Jensen 不等式或 Gibbs 不等式，下一模块讲 KL 散度时一并证。

**性质 2：上界**

I(X;Y)≤min⁡{H(X),H(Y)}I(X;Y)≤min{H(X),H(Y)}

因为 I(X;Y)=H(X)−H(X∣Y)≤H(X)I(X;Y)=H(X)−H(X∣Y)≤H(X)（条件熵非负），同理 ≤H(Y)≤H(Y)。

**直觉**：YY 告诉你关于 XX 的信息，不可能超过 XX 本身的总信息量。

**性质 3：自互信息**

I(X;X)=H(X)−H(X∣X)=H(X)−0=H(X)I(X;X)=H(X)−H(X∣X)=H(X)−0=H(X)

**XX 跟自己的互信息就是熵**。所以熵也叫"自信息"。这从另一个角度说明熵的本质。

### 六、那张神图：信息韦恩图

把上面所有量画在一张图里，你会看到惊人的结构对应。

```
          H(X, Y)
   ┌─────────────────────┐
   │                     │
   │   H(X | Y)   I(X;Y)   H(Y | X)
   │   ┌─────┬───────┬─────┐
   │   │     │       │     │
   │   │  ●  │   ●   │  ●  │
   │   │     │       │     │
   │   └─────┴───────┴─────┘
   │   ←── H(X) ──→
   │           ←── H(Y) ──→
   └─────────────────────┘
```

把 H(X)H(X) 看作一个圆，H(Y)H(Y) 看作另一个圆。两个圆**部分重叠**：

- 整个左圆 = H(X)H(X)
- 整个右圆 = H(Y)H(Y)
- **左圆独有的部分** = H(X∣Y)H(X∣Y)（YY 不知道的关于 XX 的信息）
- **右圆独有的部分** = H(Y∣X)H(Y∣X)
- **两圆相交的部分** = I(X;Y)I(X;Y)（共享信息）
- **两圆并集** = H(X,Y)H(X,Y)（总信息）

这张图把模块 3 所有公式可视化：

|公式|韦恩图含义|
|---|---|
|H(X,Y)=H(X)+H(Y∣X)H(X,Y)=H(X)+H(Y∣X)|并集 = 左圆 + 右圆独有|
|H(X,Y)=H(X)+H(Y)−I(X;Y)H(X,Y)=H(X)+H(Y)−I(X;Y)|并集 = 左 + 右 − 相交|
|I(X;Y)=H(X)−H(X∣Y)I(X;Y)=H(X)−H(X∣Y)|相交 = 左圆 − 左圆独有|
|I(X;Y)=H(X)+H(Y)−H(X,Y)I(X;Y)=H(X)+H(Y)−H(X,Y)|相交 = 左 + 右 − 并集|

**强烈建议你把这张图记在脑子里**。后面所有的信息量关系，看图就能秒推。考试也是这张图的天下。

### 七、一个完整算例

设联合分布如下：

||Y=0Y=0|Y=1Y=1|p(x)p(x)|
|---|---|---|---|
|X=0X=0|1/8|3/8|1/2|
|X=1X=1|3/8|1/8|1/2|
|p(y)p(y)|1/2|1/2||

**算 H(X)H(X)**：均匀，H(X)=1H(X)=1。 **算 H(Y)H(Y)**：均匀，H(Y)=1H(Y)=1。

**算 H(X,Y)H(X,Y)**：四个值，两个 1/8 两个 3/8。

H(X,Y)=−2⋅18log⁡218−2⋅38log⁡238H(X,Y)=−2⋅81​log2​81​−2⋅83​log2​83​

log⁡2(1/8)=−3log2​(1/8)=−3，log⁡2(3/8)=log⁡23−3≈−1.415log2​(3/8)=log2​3−3≈−1.415。

H(X,Y)≈14⋅3+34⋅1.415=0.75+1.061=1.811 比特H(X,Y)≈41​⋅3+43​⋅1.415=0.75+1.061=1.811 比特

**算 I(X;Y)I(X;Y)**：用韦恩图公式

I(X;Y)=H(X)+H(Y)−H(X,Y)=1+1−1.811=0.189 比特I(X;Y)=H(X)+H(Y)−H(X,Y)=1+1−1.811=0.189 比特

**算 H(X∣Y)H(X∣Y)**：

H(X∣Y)=H(X,Y)−H(Y)=1.811−1=0.811 比特H(X∣Y)=H(X,Y)−H(Y)=1.811−1=0.811 比特

或者直接验证：H(X)−I(X;Y)=1−0.189=0.811H(X)−I(X;Y)=1−0.189=0.811。✓

**解读**：XX 原本不确定性 1 比特，看到 YY 之后还剩 0.811 比特，YY 帮你消除了 0.189 比特的不确定性。XX 和 YY 不独立但也不强相关。

### 八、为什么互信息这么重要

提前剧透一下后面会看到的应用：

1. **信道容量**（模块 6）：信道容量 C=max⁡p(x)I(X;Y)C=maxp(x)​I(X;Y)。即"在所有可能的输入分布中，能传递的最大互信息"。这是后面整个理论的支柱。
2. **机器学习中的特征选择**：选哪些特征 YY 来预测标签 XX？挑 I(X;Y)I(X;Y) 大的——这些特征"含 XX 的信息"最多。
3. **神经网络的信息瓶颈理论**：深度学习里有一派理论用互信息分析每一层"保留了多少输入信息""学到了多少标签信息"。

互信息几乎是现代信息论应用最广的概念。

### 九、本模块小结

**四个量，一张图**：

|量|含义|公式|
|---|---|---|
|H(X,Y)H(X,Y)|总不确定性|−∑p(x,y)log⁡p(x,y)−∑p(x,y)logp(x,y)|
|H(X∣Y)H(X∣Y)|已知 YY 后 XX 的不确定性|−∑p(x,y)log⁡p(x∣y)−∑p(x,y)logp(x∣y)|
|I(X;Y)I(X;Y)|X,YX,Y 共享的信息|∑p(x,y)log⁡p(x,y)p(x)p(y)∑p(x,y)logp(x)p(y)p(x,y)​|
|H(X)H(X)|XX 的总不确定性|上一节|

**核心关系**：

H(X,Y)=H(X)+H(Y∣X)H(X,Y)=H(X)+H(Y∣X)I(X;Y)=H(X)+H(Y)−H(X,Y)=H(X)−H(X∣Y)I(X;Y)=H(X)+H(Y)−H(X,Y)=H(X)−H(X∣Y)

**核心不等式**：

0≤I(X;Y)≤min⁡{H(X),H(Y)}0≤I(X;Y)≤min{H(X),H(Y)}H(X∣Y)≤H(X)H(X∣Y)≤H(X)

**两个直觉**：

- 多一个观测不会增加不确定性。
- 互信息衡量"两个变量有多依赖"。

---

到现在，我们已经能用熵、条件熵、互信息描述各种信息关系了。但你可能注意到了一个事——互信息的公式 ∑p(x,y)log⁡p(x,y)p(x)p(y)∑p(x,y)logp(x)p(y)p(x,y)​ 看起来是在比较两个分布：实际的 p(x,y)p(x,y) 和"假装独立"的 p(x)p(y)p(x)p(y)。

这种"比较两个分布"的思路非常重要。下一模块我们要把它一般化，得到一个量——**KL 散度**（相对熵）——它是机器学习里几乎无处不在的概念，也是证明前面那些不等式的钥匙。
