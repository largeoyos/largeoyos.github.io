# 第一课:两个值的世界

在开始之前,我想先了解一下你的起点先问你两个问题:

**第一个**:你之前接触过布尔代数吗?哪怕是零碎的也算,比如:

- 编程里写过 `if (a && b || !c)` 这种条件
- 数字电路里见过与门、或门、非门
- 数学里学过集合的交、并、补
- 完全没接触过,纯零基础

**第二个**:你学布尔代数的目的是什么?

- 为了搞懂数字电路 / 计算机硬件
- 为了写更好的程序逻辑
- 为了应付某门课程(比如离散数学)
- 纯粹好奇,想理解它的数学结构

---

我先给你一个**最核心的直觉**,作为我们讨论的起点:

> 普通代数研究的是**数**,运算是加减乘除,变量可以取任意实数。
> 
> 布尔代数研究的是**真假**(或者说 0 和 1),运算是"与、或、非",变量只能取两个值之一。

这听起来很简单,但它的威力在于:**整个数字世界——你的手机、电脑、网络——底层全是布尔代数在跑**。乔治·布尔在 1854 年发明它的时候,只是想用代数的方式描述人类的逻辑推理,他完全没想到一百年后香农会用它来设计电路,从而开启计算机时代。

---


我会从零基础开始,按"基本运算 → 真值表 → 公理与定律 → 化简 → 实际应用"这个顺序展开。

布尔代数的全部基础,是承认这样一个设定:**变量只能取 0 或 1**。

在不同语境里,这两个值有不同名字:

|语境|"1"|"0"|
|---|---|---|
|逻辑|真 (True)|假 (False)|
|电路|高电平|低电平|
|编程|true|false|

但它们是同一个东西。**这很重要**——你看到电路图里的高低电平,和代码里的 `true/false`,和真值表里的 1/0,大脑要能立刻知道这是同一回事。

---

### 三个基本运算

普通代数有加减乘除。布尔代数只有三个基本运算:**与、或、非**。

我先不告诉你它们的定义,我让你自己推。我们从"非"开始,因为它最简单。

**非 (NOT)**:输入一个值,输出它的反面。记作 `¬A` 或 `A'` 或 `!A`(C语言风格)。

如果 A = 1,那么 ¬A = ? 如果 A = 0,那么 ¬A = ?

——这个不用回答了,你肯定知道。重点是这个表格,我们叫它**真值表 (truth table)**:

```
 A | ¬A
---|----
 0 |  1
 1 |  0
```

真值表是布尔代数最重要的工具。**任何布尔表达式,无论多复杂,都可以用真值表完整定义**。记住这句话,后面你会反复用到。

---

现在轮到你思考了。

**与 (AND)**,记作 `A ∧ B` 或 `A · B` 或 `A && B`(代码里),它的语义是"两个都为真,结果才为真"。

**或 (OR)**,记作 `A ∨ B` 或 `A + B` 或 `A || B`,它的语义是"至少一个为真,结果就为真"。

请你自己写出这两个的真值表。两个输入变量 A 和 B,所以一共有 4 行(因为 2² = 4 种组合)。

格式像这样:

```
 A | B | A∧B | A∨B
---|---|-----|-----
 0 | 0 |  ?  |  ?
 0 | 1 |  ?  |  ?
 1 | 0 |  ?  |  ?
 1 | 1 |  ?  |  ?
```
