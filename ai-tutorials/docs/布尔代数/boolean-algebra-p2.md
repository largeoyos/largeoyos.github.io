# 第二课:为什么叫"代数"

普通代数里你熟悉这些等式:

- `a + 0 = a`
- `a · 1 = a`
- `a + b = b + a`(交换律)
- `a · (b + c) = a·b + a·c`(分配律)

布尔代数之所以叫"代数",是因为它也有自己的一套**等式定律**,而且其中很多和普通代数惊人地相似。

但是——**有几条定律,会让你觉得布尔代数是个怪东西**。

我先列出基本定律,你扫一遍,我会标出"和普通代数一样"的和"完全不一样"的:

**和普通代数一样的(看一眼就过)**:

```
A + 0 = A           A · 1 = A           (同一律)
A + B = B + A       A · B = B · A       (交换律)
(A+B)+C = A+(B+C)   (A·B)·C = A·(B·C)   (结合律)
A · (B + C) = A·B + A·C                  (分配律)
```

**布尔代数特有的,要警觉**:

```
A + 1 = 1           A · 0 = 0           (零一律)
A + A = A           A · A = A           (幂等律) ← 注意!
A + A' = 1          A · A' = 0          (互补律)
(A')' = A                                (双重否定)
```

**最反直觉的一条**:

```
A + A·B = A         A · (A + B) = A     (吸收律)
```

---

我们一条条看。先看**幂等律**:`A + A = A`。

普通代数里 `a + a = 2a`,但布尔里没有"2",只有 0 和 1。你用真值表验证一下就明白:A 是 0 时,0+0=0;A 是 1 时,1+1=1(因为"或"运算里"真 或 真 = 真")。所以 `A + A` 永远等于 A 本身。

这条律的硬件意义很大:**两根一样的信号线连到 OR 门,等于浪费一个门**。化简电路时,看到 `A + A` 立刻消成 `A`。

---

再看**互补律**:`A + A' = 1`。

"A 或 非A",必然为真——要么 A 真,要么 A 假,总有一个真。这是**排中律**,亚里士多德两千多年前就讲过的逻辑公理。

`A · A' = 0`,"A 且 非A",永远为假。这是**矛盾律**。

这两条是布尔代数的灵魂。普通代数里 `a + (-a) = 0`,但 `a · (-a) = -a²` 不是常数。布尔代数里却是:**A 和 A' 的"或"恒为 1,"与"恒为 0**。这种对称性,普通代数没有。

---

**吸收律**值得你停下来想一下:`A + A·B = A`。

为什么?我用逻辑语言给你翻译一遍:

> "A 为真,**或者**(A 为真且 B 为真)"
> 
> ——但只要 A 为真,整个表达式就为真了,B 是什么根本不重要。所以这等价于"A 为真"。

你也可以代数推导:

```
A + A·B 
= A·1 + A·B    (同一律,把 A 写成 A·1)
= A·(1 + B)    (分配律,提取 A)
= A·1          (零一律,1 + 任何 = 1)
= A            (同一律)
```

吸收律在化简电路时**极其常用**。看到 `A + AB`,直接划掉 `AB`。

---

到这里,我想让你做一件事,确认你跟上了。

请你用**纯代数推导**(不查真值表)证明下面这个等式:

```
A + A'·B = A + B
```

提示:你需要用到分配律、互补律、零一律、同一律。这是吸收律的"扩展版",在化简里出现频率比吸收律本身还高。

推完之后我们进入第三课:**德摩根定律**——这是布尔代数里最优雅、也最重要的定理,直接对应硬件里"与门和或门可以互换"的关键技巧。
