### 模拟电路的全景地图

模拟电路研究的是**连续变化的电信号**(相对于只有0和1的数字电路)。整门课其实围绕一个核心问题展开:

> **怎样用电子元件,把一个微弱的、带噪声的、形状不好的电信号,变成我们想要的样子?**

"想要的样子"可能是:更大(放大器)、更干净(滤波器)、更稳定(稳压电源)、更有特定形状(振荡器、比较器)……

整门课的知识大致是这样一层层叠起来的:

```
第 4 层:系统应用(运放电路、滤波器、振荡器、电源)
            ↑
第 3 层:核心器件(运算放大器 Op-Amp)
            ↑
第 2 层:有源器件(二极管、BJT三极管、MOS管)
            ↑
第 1 层:无源器件 + 分析方法(电阻、电容、电感 + 交流分析、相量、频率响应)
            ↑
第 0 层:基础(你已有的:KCL/KVL、欧姆定律)
```

我们会一层一层往上走。**每一层都必须牢固,否则上面会塌**——这是模电这门课很多人学崩的主要原因:他们在第2层还没懂的时候,老师已经在讲第4层了。
