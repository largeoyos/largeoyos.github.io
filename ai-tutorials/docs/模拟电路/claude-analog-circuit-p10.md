# 第 9 课:集成运算放大器(Op-Amp)与理想运放模型

### 9.1 运放是什么——"封装好的放大器"

前面我们花了大力气学 BJT、Q 点、差放。现在把它们全部打包成**一块芯片**,留出必要的引脚给用户——这就是**集成运算放大器(Operational Amplifier, Op-Amp)**。

"运算"这个名字来自历史——最早运放用来做模拟计算机里的加减乘除和微积分运算。现在它的应用已经远超这个范畴。

#### 符号

```
          v_+ ●───┐
                  │ \
                  │  \
                  │   >───● v_o
                  │  /
          v_− ●───┘ /
                  │/
                  
   两个输入:
     v_+  同相输入端(Non-inverting Input,标 "+")
     v_−  反相输入端(Inverting Input,标 "−")
   一个输出:v_o
```

**注意**:"+"和"−"不是接电源的意思!它们表示两个输入端的**相位属性**:

- v+v+​ 增加 → vovo​ 增加(同相)
- v−v−​ 增加 → vovo​ 减少(反相)

**电源引脚**(VCCVCC​、VEEVEE​)一般画图时省略,但实际电路里必须接。通常用双电源 ±15V、±12V、±5V 等。

### 9.2 内部结构——前八课的集大成

一个典型运放(如经典的 μA741)内部可以粗分三级:

```
   ┌──────────┐     ┌──────────┐     ┌──────────┐
   │ 差分输入级 │──→│ 共射增益级 │──→│ 共集输出级 │──→ v_o
   │  Diff Pair │     │ Gain Stage │     │ Output Buf │
   └──────────┘     └──────────┘     └──────────┘
        ↑                 ↑                 ↑
    提供高 R_i          提供大 A_v       提供低 R_o
    高 CMRR          (带有源负载)        强驱动
    两个输入端        进一步升压           单端输出
```

- **输入级**:差分放大器(第 8 课)——提供两个输入端、高输入阻抗、高 CMRR、低零漂
- **中间级**:共射放大器(第 6 课)——贡献主要电压增益,常带有源负载使增益极大化
- **输出级**:共集放大器(第 7 课)——低输出阻抗,强驱动能力(有时是"推挽"结构,后面讲功放时再说)

各级用直接耦合串联,整体增益可达 105∼106105∼106 倍。

**重点**:**这一课开始,你不需要再考虑里面的 BJT**。运放是一个"整体元件",你只需要知道它的**端口特性**。

### 9.4 开环特性——运放"裸用"是什么样的?

"开环"的意思是:**输出不反馈回输入,直接让运放按自己的增益工作**。

vo=Aod(v+−v−)vo​=Aod​(v+​−v−​)

假设 Aod=105Aod​=105,电源 ±15V。

- v+−v−=+0.2mVv+​−v−​=+0.2mV:vo=20vo​=20V?不——**输出被电源钳位在 +15V 附近(通常是 +13V 到 +14V,受内部电路限制)**
- v+−v−=−0.2mVv+​−v−​=−0.2mV:vo=−14vo​=−14V 左右
- v+−v−=0v+​−v−​=0:理论上 vo=0vo​=0,实际有失调

**结论**:开环运放对输入差极其敏感,**稍微有点差异输出就"撞墙"**,实际工作区极窄(几十 μV 量级)。

这说明**开环运放不能用作线性放大器**——它只能判断"v+v+​ 和 v−v−​ 哪个大"。这种用法叫**比较器(Comparator)**,是运放少数几种开环应用之一。

想让运放做线性放大、做运算、做滤波……必须加**反馈**。

### 9.5 引入反馈——运放变革的开始

**反馈(Feedback)**:把输出的一部分送回输入,和原输入一起决定运放的行为。

反馈分两种:

- **负反馈**:反馈到**反相输入端**(v−v−​),抑制原输入的变化趋势
- **正反馈**:反馈到**同相输入端**(v+v+​),放大原输入的变化趋势

**这节课只讲负反馈**——这是运放 99% 的应用模式。正反馈用于振荡器、施密特触发器,后面的课再讲。

#### 负反馈的神奇效应

加了负反馈后,运放会自动调整输出,使得:

v+−v−≈0v+​−v−​≈0

**直觉**:如果 v+>v−v+​>v−​,输出会增大;输出通过反馈网络送回 v−v−​,让 v−v−​ 也增大,直到 v+=v−v+​=v−​;反之同理。**负反馈让运放"自我平衡"到 v+=v−v+​=v−​ 的状态**。

为什么这能成立?因为开环增益 AodAod​ 极大,哪怕 v+−v−v+​−v−​ 只差一点点,输出就会大幅变化,迫使反馈拉回平衡。换句话说:**v+−v−v+​−v−​ = vo/Aodvo​/Aod​,Aod→∞Aod​→∞,所以 v+−v−→0v+​−v−​→0**。

### 9.6 理想运放的两条黄金法则

这是模电的"牛顿定律"——**所有运放电路分析都从这两条出发**。

#### 法则 1:虚短(Virtual Short)

v+=v−v+​=v−​​

两个输入端电位**相等**,就像被一根看不见的导线短接一样——但**没有电流在这根"虚导线"上流动**。故称"虚短"。

**使用条件**:运放工作在**线性区**(加了负反馈,没有饱和)。如果运放是开环用作比较器,或者反馈不足导致饱和,虚短就不成立。

#### 法则 2:虚断(Virtual Open)

i+=i−=0i+​=i−​=0​

两个输入端**没有电流流入或流出**。

**为什么?** 因为运放输入阻抗极大(理想为 ∞∞),任何有限的电压差都只能产生可忽略的电流。

**使用条件**:理想运放(实际运放输入偏置电流 IIBIIB​ 很小但不为零,大多数场合可以忽略)。

#### 这两条法则的威力

**有了虚短 + 虚断,你不需要再考虑运放内部**——整个运放电路的分析变成**外部电阻网络 + 两个约束方程**的问题。

你会发现接下来所有经典运放电路的分析,**套路完全一样**:

1. 标出节点电位(用虚短)
2. 写 KCL(用虚断)
3. 解出 vovo​

### 9.7 经典电路 1:反相放大器(Inverting Amplifier)

这是最常用、也最经典的运放应用。

#### 电路结构

```
                    R_f
           ┌───────[R_f]───────┐
           │                    │
           │                    │
   v_i ──[R_1]──●──┐         ┌──●── v_o
                   │\        │
                   │ \       │
              v_−  │  \      │
                   │   >─────┤
              v_+  │  /      
               ────┤ /       
               │  │/         
               │              
              地
```

- vivi​ 通过 R1R1​ 接到**反相输入端** v−v−​
- 反馈电阻 RfRf​ 从输出 vovo​ 连回 v−v−​
- 同相输入端 v+v+​ 接地

#### 用黄金法则分析

**步骤 1:虚短**。v+=0v+​=0(接地),所以 v−=0v−​=0。

**这个 v−v−​ 点虽然没有接地,但电位是 0——叫"虚地(Virtual Ground)"**。这是反相放大器的标志。

**步骤 2:虚断**。i−=0i−​=0,意味着流进 v−v−​ 节点的电流**不会流入运放**,只能全部流向 RfRf​。

**步骤 3:在 v−v−​ 节点写 KCL**(设电流方向:从 vivi​ 流向 v−v−​ 为正):

vi−v−R1⏟流入 v−=v−−voRf⏟流出 v−流入 v−​R1​vi​−v−​​​​=流出 v−​Rf​v−​−vo​​​​

代入 v−=0v−​=0:

viR1=−voRfR1​vi​​=Rf​−vo​​

**步骤 4:解出 vovo​**:

vo=−RfR1vi⇒Av=−RfR1vo​=−R1​Rf​​vi​⇒Av​=−R1​Rf​​​

#### 解读

1. **负号**:反相——输入升则输出降,名副其实
2. **增益由两个电阻比例决定**——不依赖运放内部参数!只要运放"足够好",电阻精度就是增益精度
3. **电阻比例可以大于 1 也可以小于 1**:Rf=10R1Rf​=10R1​ 时增益 −10,Rf=R1/10Rf​=R1​/10 时增益 −0.1(衰减)

#### 输入阻抗

从 vivi​ 往里看:vivi​ 通过 R1R1​ 接到**虚地**(电位恒为 0),所以输入端看到的阻抗就是 R1R1​ 到地:

Ri=R1Ri​=R1​

**缺点**:反相放大器的输入阻抗**只有 R1R1​**——不能做很大(太大会引入噪声和偏置电流误差)。这是反相组态的局限。

#### 输出阻抗

由于运放本身输出阻抗很小,加上负反馈的"稳压"效应,反相放大器的输出阻抗**极小**(理想为 0)。

### 9.8 经典电路 2:同相放大器(Non-inverting Amplifier)

#### 电路结构

```
                    R_f
           ┌───────[R_f]───────┐
           │                    │
           │                    │
                   │\        │
              v_−  │ \       │
              ●────┤  \      │
              │    │   >─────●── v_o
              │    │  /
             [R_1] │ /
              │    │/
              │   ┌┘
             地   │
                  │
           v_i ── ● v_+ (同相输入端)
```

- vivi​ 直接接到**同相输入端** v+v+​
- R1R1​ 接在 v−v−​ 和地之间
- RfRf​ 接在 v−v−​ 和 vovo​ 之间(反馈)

#### 分析

**虚短**:v−=v+=viv−​=v+​=vi​。

**虚断**:i−=0i−​=0,所以流过 R1R1​ 和 RfRf​ 的电流相等,是一条"**串联分压**"的关系。

v−v−​ 节点相当于 R1R1​、RfRf​ 分压网络的中点:

v−=vo⋅R1R1+Rfv−​=vo​⋅R1​+Rf​R1​​

代入 v−=viv−​=vi​:

vi=vo⋅R1R1+Rf⇒vo=(1+RfR1)vivi​=vo​⋅R1​+Rf​R1​​⇒vo​=(1+R1​Rf​​)vi​​Av=1+RfR1Av​=1+R1​Rf​​

#### 解读

1. **没有负号**:同相——输入升,输出也升
2. **增益 ≥ 1**(因为 Rf/R1≥0Rf​/R1​≥0)——**同相放大器无法衰减**
3. **增益由电阻比例决定**,和反相一样

#### 输入阻抗

vivi​ 直接接到同相输入端,输入端是运放自己的输入端,**输入阻抗极高**(运放开环 RidRid​ 乘以一个反馈因子,可达几十 MΩ 甚至 GΩ)。

Ri→∞Ri​→∞

这是同相放大器的**最大优点**——**适合接高阻信号源**(如电容麦克风、光电二极管)。

#### 反相 vs 同相——重要对比

|特性|反相放大器|同相放大器|
|---|---|---|
|增益|−Rf/R1−Rf​/R1​(可小于1)|1+Rf/R11+Rf​/R1​(≥1≥1)|
|相位|反相|同相|
|输入阻抗|有限(=R1=R1​)|极高|
|虚地|有(分析方便)|无|
|常用场合|精密放大、加法器、积分器|缓冲、高阻匹配|

### 9.9 经典电路 3:电压跟随器(Voltage Follower)

**同相放大器的特殊情况**:把 RfRf​ 短路、R1R1​ 断开:

```
           ┌────────────┐
           │            │
   v_i ────●── v_+      │
               │\       │
               │ \      │
               │  >─────●── v_o
               │  /     │
               │ /      │
           ●───│/       │
           │            │
           └────────────┘
   (v_− 直接和 v_o 连在一起)
```

#### 分析

虚短:v−=v+=viv−​=v+​=vi​。 v−v−​ 直接连 vovo​,所以 vo=v−vo​=v−​。

vo=vi⇒Av=1vo​=vi​⇒Av​=1​

#### 意义

看起来没用——增益是 1,输入输出一样。**但它的价值在阻抗变换**:

- 输入阻抗:极高(MΩ 级)
- 输出阻抗:极低(运放输出阻抗 /(1+A)/(1+A),接近 0)

这不就是一个**"完美的射极跟随器"**吗?——对,电压跟随器是用运放实现的**理想缓冲器**。

**用途**:

- 把高阻信号源的信号"搬运"到低阻负载,中间不损失电压
- 隔离前后级,避免相互干扰
- 作为多级电路中的"缓冲"模块

### 9.10 反馈的威力——为什么增益公式这么简洁?

你可能注意到:反相放大器增益 −Rf/R1−Rf​/R1​、同相放大器增益 1+Rf/R11+Rf​/R1​——**里面完全没有运放的参数**!

这是**负反馈的魔力**。其原理(下一课会更深入)简述:

**开环增益 AodAod​ 非常大**,通过反馈网络送回的信号让运放的"净输入" v+−v−v+​−v−​ 变得极小。**这使得运放的具体增益值不重要,关键是反馈网络的分压比**。

反馈用"**牺牲巨大的开环增益**"换取了:

- 增益精度(由电阻决定,可达 0.01% 精度)
- 带宽扩展
- 线性度提升
- 输入/输出阻抗改善

这个思想不仅适用于运放,也适用于所有放大器——**是模电设计的灵魂**。下一课我们会系统讲反馈的四种基本结构。

### 9.11 理想运放 vs 实际运放——你需要知道的误差来源

分析时用"理想"假设,但实际设计时要知道偏离理想的地方在哪:

|误差源|理想假设|实际情况|影响|
|---|---|---|---|
|开环增益有限|Aod=∞Aod​=∞|105∼106105∼106|虚短不完美,高频更差|
|输入阻抗有限|Rid=∞Rid​=∞|MΩ 级|高阻电路中有分压误差|
|输出阻抗非零|Ro=0Ro​=0|几十 Ω|重载下有压降|
|输入失调电压|VIO=0VIO​=0|≈1 mV|输出有直流偏移|
|输入偏置电流|IB=0IB​=0|≈几十 nA|高阻电路中引起电压误差|
|带宽有限|无限|GBW=1 MHz(741)|高频增益下降|
|转换速率有限|无限|0.5 V/μs(741)|大信号快速变化时失真|

**普通电路不用担心这些**——把运放当理想的,结果够准。只有高精度、高频、大信号场合才需要考虑。

### 9.12 这一课你必须带走的东西

1. **运放是前面八课内容的集大成封装**:差放输入级 + 共射增益级 + 共集输出级,直接耦合串联。
2. **运放的三大特性**:开环增益极大(105105+)、输入阻抗极大、输出阻抗极小——这三条直接导出黄金法则。
3. **两条黄金法则**(工作在线性区时成立):
    - **虚短**:v+=v−v+​=v−​(因为开环增益无限大,只有微小输入差就能驱动输出)
    - **虚断**:i+=i−=0i+​=i−​=0(因为输入阻抗无限大)
4. **分析套路**:标电位(虚短)→ 写 KCL(虚断)→ 解 vovo​。
5. **三个最基本电路**:
    - 反相放大器:Av=−Rf/R1Av​=−Rf​/R1​,虚地,输入阻抗 =R1=R1​
    - 同相放大器:Av=1+Rf/R1Av​=1+Rf​/R1​,输入阻抗极高
    - 电压跟随器:Av=1Av​=1,完美缓冲
6. **运放电路增益只和外部电阻比例有关**——这是反馈赋予的最大好处。
7. **开环运放必须加反馈才能做线性放大**,否则就是比较器。
