# 第 10 课:运放的运算电路——从加减乘除到微积分

上一课我们有了两把钥匙:**虚短**和**虚断**。这一课我们拿着这两把钥匙,打开运放"运算"能力的大门。

你会发现一件惊人的事:**把反相放大器里的电阻换成电容、换成二极管、多加几个输入口——运放就能做加法、减法、积分、微分、对数……**

分析套路**完全不变**:标电位(虚短)→ 写 KCL(虚断)→ 解 vovo​。这一课的所有电路都是这个模板的变奏。

---

### 10.1 加法器(Summing Amplifier)——反相求和

#### 电路结构

把反相放大器的输入端"复制几份":

```
   v_1 ──[R_1]──┐
                │
   v_2 ──[R_2]──┼─── v_−  ──┐
                │           │\
   v_3 ──[R_3]──┘           │ \
                             │  >───● v_o
   v_+ ●───── 地             │  /
                             │ /
              ┌─[R_f]────────┘
              │              
              └──────────────  v_o (反馈到 v_−)
```

#### 分析

**虚短**:v+=0⇒v−=0v+​=0⇒v−​=0(虚地)。

**虚断**:i−=0⇒i−​=0⇒ 流入 v−v−​ 节点的电流全部流向 RfRf​。

**KCL**(以 v−v−​ 为节点):

v1−0R1+v2−0R2+v3−0R3=0−voRfR1​v1​−0​+R2​v2​−0​+R3​v3​−0​=Rf​0−vo​​vo=−(RfR1v1+RfR2v2+RfR3v3)vo​=−(R1​Rf​​v1​+R2​Rf​​v2​+R3​Rf​​v3​)​

#### 解读

1. **每个输入独立加权**,权重是 Rf/RkRf​/Rk​
2. **加权后的和再取负号**——反相求和
3. 如果 R1=R2=R3=RfR1​=R2​=R3​=Rf​,得到:

vo=−(v1+v2+v3)vo​=−(v1​+v2​+v3​)

**干干净净的求和**(带一个负号)。

#### 为什么能这样加?——虚地的神奇

关键在**虚地**。由于 v−v−​ 节点电位恒为 0,三个输入信号**互不干扰**——每个 vkvk​ 只通过自己的 RkRk​ "看到"虚地,各自独立贡献电流。

**如果没有虚地**,三个输入会通过 v−v−​ 节点相互影响,形成复杂的分压网络,结果就不是简单的求和了。**虚地让加法器实现了"电流的叠加"**。

#### 应用

- **音频混音台**:每个通道是一路输入,通过加法器合成
- **模拟计算**:求信号的线性组合(这是"运算"放大器名字的由来)
- **D/A 转换**:把数字位用不同电阻权重加起来,得到模拟电压(R-2R 梯形网络就是这原理)

### 10.2 减法器(Subtractor / Difference Amplifier)——精确求差

#### 电路结构

```
                              R_f
                     ┌───────[R_f]───────┐
                     │                    │
   v_1 ──[R_1]──── v_−                   │
                     │                    │
                     │ \                  │
                     │  \                 │
                     │   >────────────────●── v_o
                     │  /
                     │ /
   v_2 ──[R_2]──── v_+ 
                     │
                    [R_3]
                     │
                    地
```

- v1v1​ 通过 R1R1​ 接反相输入
- v2v2​ 通过 R2R2​ 接同相输入
- R3R3​ 从同相输入端到地(起对称和分压的作用)
- RfRf​ 是反馈电阻

#### 分析

**求 v+v+​**:v2v2​ 通过 R2R2​ 和 R3R3​ 分压(注意:由于虚断,没有电流进入运放,R2R2​ 和 R3R3​ 是串联关系):

v+=v2⋅R3R2+R3v+​=v2​⋅R2​+R3​R3​​

**虚短**:v−=v+v−​=v+​。

**虚断 + KCL 在 v−v−​**:

v1−v−R1=v−−voRfR1​v1​−v−​​=Rf​v−​−vo​​vo=v−(1+RfR1)−RfR1v1vo​=v−​(1+R1​Rf​​)−R1​Rf​​v1​

代入 v−=v2⋅R3/(R2+R3)v−​=v2​⋅R3​/(R2​+R3​):

vo=v2⋅R3R2+R3⋅R1+RfR1−RfR1v1vo​=v2​⋅R2​+R3​R3​​⋅R1​R1​+Rf​​−R1​Rf​​v1​

**对称情况**(通常这样设计):R1=R2R1​=R2​,Rf=R3Rf​=R3​。此时:

v+=v2⋅RfR1+Rf,vo=v2⋅RfR1+Rf⋅R1+RfR1−RfR1v1v+​=v2​⋅R1​+Rf​Rf​​,vo​=v2​⋅R1​+Rf​Rf​​⋅R1​R1​+Rf​​−R1​Rf​​v1​vo=RfR1(v2−v1)vo​=R1​Rf​​(v2​−v1​)​

#### 解读

1. **输出正比于两输入之差**——这是一个"一体化差分放大器"
2. **增益由 Rf/R1Rf​/R1​ 控制**,对称设计下简洁漂亮
3. 如果 Rf=R1Rf​=R1​,得到 vo=v2−v1vo​=v2​−v1​,干净的减法

#### 缺点

1. **输入阻抗不高**:v1v1​ 看到的是 R1R1​,v2v2​ 看到的是 R2+R3R2​+R3​——几 kΩ 的量级,**不适合高阻源**
2. **电阻匹配要求苛刻**:R1/RfR1​/Rf​ 要严格等于 R2/R3R2​/R3​,稍有偏差共模抑制比就会急剧下降(CMRR 对电阻误差极敏感)
3. **增益调节困难**:改一个电阻,对称关系就破了

这些缺点在精密测量场合不能接受——所以工程上用改进版本:**仪表放大器**。

### 10.3 仪表放大器(Instrumentation Amplifier)——差分运算的工程王者

#### 电路结构(经典三运放结构)

```
            ┌───[R_f]───┐
            │           │
    v_1 ──○─● A1        │     ┌─── 单增益电阻 R_g ───┐
              >─[R]─┐    │     │                        │
            ─●      │    │     └───────── 可调 ────────┘
            │       │    │
     ●──────┘       ●────┴──●
     │              │       │  (前两级)
     │              │       │
     v_1 →  缓冲    ●── A3 ─── v_o
     v_2 →  缓冲    │    (减法器)
            │       │       │
     ●──────┐       ●───────●
            │       │
            ○─● A2  │
              >─[R]─┘
           ─●
```

我画得比较乱。**核心结构**是:

- **前两级**:两个运放 A1、A2 做**同相放大器**(高输入阻抗!)
- **中间**:一个"增益电阻" RgRg​ 连接两个运放的反相输入端
- **后一级**:运放 A3 做减法器

#### 最终公式(直接给)

vo=(1+2RfRg)(v2−v1)vo​=(1+Rg​2Rf​​)(v2​−v1​)​

#### 仪表放大器的三大优点

1. **输入阻抗极高**——两个输入都直接进同相端,运放固有的高输入阻抗
2. **CMRR 极高**(100 dB 以上)——前级两个运放对称,后级只需处理差模信号
3. **增益由单一电阻 RgRg​ 调节**——不破坏对称性

#### 应用

- **生物医学**:心电图(ECG)、脑电图(EEG)——信号只有几十 μV,还叠加在人体几十 V 的共模噪声上,仪表放大器是唯一选择
- **传感器信号调理**:热电偶、应变片、惠斯通电桥
- **精密测量仪器**

实际市面上有专用芯片,如 AD620、INA128 等,内部就是三运放结构,封装成一个 8 脚小器件,外接一个 RgRg​ 就能用。

### 10.4 积分器(Integrator)——反馈电容的魔力

#### 电路结构

把反相放大器里的**反馈电阻 RfRf​ 换成电容 CC**:

```
                   C (反馈电容)
            ┌─────[ ]├──────┐
            │               │
            │               │
   v_i ──[R]──●── v_−       │
                 │\         │
                 │ \        │
                 │  >───────●── v_o
                 │  /
                 │ /
             ●───│/
             │
            地
```

#### 分析

**虚短**:v+=0⇒v−=0v+​=0⇒v−​=0(虚地)。

**虚断**:i−=0⇒i−​=0⇒ 流入电阻 RR 的电流 = 流入电容 CC 的电流。

**流过 RR 的电流**:

iR=vi−0R=viRiR​=Rvi​−0​=Rvi​​

**流过 CC 的电流**(电容的 ii-vv 关系:i=Cdvdti=Cdtdv​,这里跨电容的电压是 v−−vo=−vov−​−vo​=−vo​):

iC=Cd(v−−vo)dt=−CdvodtiC​=Cdtd(v−​−vo​)​=−Cdtdvo​​

**KCL**:iR=iCiR​=iC​(电流从输入端经电阻进节点,再经电容流向输出)

viR=−CdvodtRvi​​=−Cdtdvo​​

积分:

vo(t)=−1RC∫0tvi(τ) dτ+vo(0)vo​(t)=−RC1​∫0t​vi​(τ)dτ+vo​(0)​

#### 解读

1. **输出 = 输入的积分**(加负号、除以 RCRC)
2. 时间常数 τ=RCτ=RC 决定积分的"速度"
3. vo(0)vo​(0) 是初始电容电压,**是积分常数的物理体现**

#### 波形直觉

- 输入一个**常数** V0V0​:vo(t)=−V0t/(RC)vo​(t)=−V0​t/(RC)——**线性增长**的斜坡
- 输入一个**方波**:输出是**三角波**(正半周积分变线性上升,负半周线性下降)
- 输入一个**正弦** Vmsin⁡ωtVm​sinωt:输出是 VmωRCcos⁡ωtωRCVm​​cosωt——**相位落后 90°,幅度除以 ωω**

这里藏着一个深刻的频域解读:**积分器对低频放大、高频衰减**——它是一个**低通滤波器**。后面讲频率响应时我们再深挖。

#### 实际积分器的问题——必须加泄放电阻

理想积分器有一个致命问题:**任何微小的直流偏移都会被"积累"成无限大**。

实际运放有输入失调电压 VIOVIO​ 和偏置电流,这些小量经过长时间积分后会让输出**"撞墙"——漂到电源电压,运放饱和**。

**解决办法**:在 CC 两端**并联一个大电阻 RfRf​**(远大于 RR)。

```
               R_f (大阻值泄放电阻)
           ┌──[R_f]──┐
           │         │
           │    C    │
           ├──[ ]├──┤
           │         │
   v_i ──[R]──●── v_−
                 │\
                 │ \
                 │  >───── v_o
```

这样做:

- 对**高频信号**:电容阻抗小,RfRf​ 阻抗大,**积分特性基本不变**
- 对**直流**:电容开路,电路退化成反相放大器,Av=−Rf/RAv​=−Rf​/R——**直流增益被限制**,不会无限累积

代价:**积分特性在极低频偏离理想**,但工程上完全可以接受。

### 10.5 微分器(Differentiator)——输入电容的对偶

#### 电路结构

**把反相放大器输入端的电阻换成电容**:

```
                    R_f
              ┌───[R_f]───┐
              │           │
              │           │
              C           │
   v_i ──[ ]├──●── v_−    │
                 │\       │
                 │ \      │
                 │  >─────●── v_o
                 │  /
                 │ /
             ●───│/
             │
            地
```

#### 分析

**虚短**:v−=0v−​=0。

**流过电容的电流**(跨电容电压 = vi−0=vivi​−0=vi​):

iC=Cd(vi−0)dt=CdvidtiC​=Cdtd(vi​−0)​=Cdtdvi​​

**虚断 + KCL**:电容电流 = 流出 RfRf​ 的电流

Cdvidt=0−voRfCdtdvi​​=Rf​0−vo​​vo=−RfCdvidtvo​=−Rf​Cdtdvi​​​

#### 解读

1. **输出 = 输入的微分**(加负号、乘 RfCRf​C)
2. 输入**变化越快、输出越大**
3. 输入**恒定时,输出为 0**(直流被完全阻断)

#### 波形直觉

- 输入**方波**:上升沿/下降沿微分后是**冲激**(实际是很尖的窄脉冲)
- 输入**三角波**:微分后是**方波**
- 输入**正弦**:微分得余弦,**幅度 × ωω**——**高频放大,低频衰减**,是**高通滤波器**

#### 实际微分器的问题——噪声放大器

微分器从频域看是**高频增益大**,这意味着:

**它会把高频噪声放大得很厉害**——哪怕信号很干净,一点点高频噪声就能毁掉输出。

工程上**纯微分器很少用**,必要时加限制:**在输入端串一个小电阻 R1R1​**,组成"带限微分器":

```
   v_i ──[R_1]──[C]──●── v_−
                         ↓ (后续同上)
```

这样在高频(超过 1/(R1C)1/(R1​C))处,增益不再继续上升,**限制了噪声放大**。

### 10.6 一个重要的对称美:微分和积分是"对偶"

把反相放大器模板记在心里:

Av=−ZfZinAv​=−Zin​Zf​​

其中 ZfZf​ 是反馈阻抗,ZinZin​ 是输入阻抗。这个公式**对复阻抗也成立**(相量分析的视角)。

|电路|ZinZin​|ZfZf​|增益形式|
|---|---|---|---|
|反相放大器|RR|RR|−Rf/R−Rf​/R|
|积分器|RR|1/(jωC)1/(jωC)|−1/(jωRC)−1/(jωRC)(低频大)|
|微分器|1/(jωC)1/(jωC)|RR|−jωRC−jωRC(高频大)|

**积分器和微分器是把 RR 和 CC 换位置的结果**——这种对偶性贯穿整个模电,值得内化。

### 10.7 对数与指数运算电路

#### 对数放大器(Logarithmic Amplifier)

**利用二极管或 BJT 的指数伏安特性**:在反馈回路里用 PN 结代替电阻。

```
               D (二极管,阴极接 v_o,阳极接 v_−)
            ┌──┤◀├────┐
            │         │
   v_i ──[R]──●── v_−
                 │\
                 │ \
                 │  >───● v_o
                 │  /
                 │ /
             ●───│/
             │
            地
```

#### 分析

流过二极管的电流 = 流过 RR 的电流 = vi/Rvi​/R。

二极管的肖克利方程:

iD=IS(evD/VT−1)≈ISevD/VTiD​=IS​(evD​/VT​−1)≈IS​evD​/VT​

跨二极管的电压 vD=v−−vo=−vovD​=v−​−vo​=−vo​,所以:

viR=ISe−vo/VTRvi​​=IS​e−vo​/VT​

解出 vovo​:

vo=−VTln⁡(viRIS)vo​=−VT​ln(RIS​vi​​)​

**输出正比于输入的自然对数**。

#### 指数放大器(Antilog Amplifier)

**把输入端和反馈端换位**:

```
                R_f (反馈电阻)
           ┌──[R_f]──┐
           │         │
   v_i ──┤◀├──●── v_−
                │\
                │ \
                │  >──── v_o
                │ ...
```

推导类似,得到:

vo∝−evi/VTvo​∝−evi​/VT​

**输出正比于输入的指数**。

#### 应用

- **模拟乘法**:ln⁡(a)+ln⁡(b)=ln⁡(ab)ln(a)+ln(b)=ln(ab),所以"对数 → 加 → 反对数" = "乘法"。两个对数放大器 + 加法器 + 指数放大器 = 模拟乘法器。
- **信号压缩**:音频、通信中常用对数变换压缩动态范围(dB 本身就是对数尺度)
- **精密测量**:把指数关系的传感器信号(如温度、光强)线性化

**温度补偿要求高**:因为 VT=kT/qVT​=kT/q 直接依赖温度,对数放大器必须有温度补偿,否则精度很差。

### 10.8 一个大视野:运放电路的统一模板

你可能已经注意到,这一课所有电路分析步骤完全一样。让我把这个"**万能模板**"总结出来——这是你分析任何运放电路的通用方法:

**步骤 1:判定是否线性工作**(有没有负反馈?反馈够不够?)

**步骤 2:用虚短写 v−v−​ 和 v+v+​ 的关系**

**步骤 3:用虚断,在 v−v−​ 节点(或其他关键节点)写 KCL**

**步骤 4:代入元件的 ii-vv 关系**

- 电阻:i=v/Ri=v/R
- 电容:i=C dv/dti=Cdv/dt
- 电感:v=L di/dtv=Ldi/dt
- 二极管:i=IS(ev/VT−1)i=IS​(ev/VT​−1)

**步骤 5:解方程**

**整门模电的运放分析都是这个模板**。一旦熟练,看到电路几秒钟就知道它在干什么。

### 10.9 这一课你必须带走的东西

1. **加法器**:反相端多个输入独立加权求和,利用虚地让各路互不干扰。
2. **减法器**:利用对称电阻网络,输出两输入之差。严格依赖电阻匹配,实际中用**仪表放大器**(三运放结构)替代。
3. **积分器**:反馈 RR 换成 CC——vo=−1RC∫vi dtvo​=−RC1​∫vi​dt。实际用时必须加**泄放电阻**防止直流漂移。
4. **微分器**:输入 RR 换成 CC——vo=−RC dvi/dtvo​=−RCdvi​/dt。实际用时**加限流电阻**防止高频噪声放大。
5. **积分器 / 微分器是对偶的**——RR 和 CC 的位置互换。
6. **对数 / 指数放大器**:利用 PN 结的指数特性做对数/指数运算,组合起来实现**模拟乘除法**。
7. **运放分析的万能模板**:虚短 → 虚断 → KCL → 代 ii-vv 关系 → 解方程。
8. **反相放大器模板公式** Av=−Zf/ZinAv​=−Zf​/Zin​ **对复阻抗也成立**——这是理解积分器和微分器频域行为的入口。
