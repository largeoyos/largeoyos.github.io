# 第 11 课:反馈——模电的核心思想

### 11.1 反馈到底是什么

**反馈(Feedback)**:把输出的一部分"取回来",和原输入叠加,共同决定系统的行为。

这不是电子学独有的概念——它是**一切自调节系统的灵魂**:

- 人端水杯:眼睛看到手的位置(输出),大脑调整肌肉(输入)——视觉—运动反馈
- 空调:温度传感器测室温(输出),控制压缩机(输入)——恒温反馈
- 央行调利率:看通胀数据(输出),调控货币政策(输入)——经济反馈

**电子电路的反馈是同样的思想**。

---

### 11.2 反馈系统的一般框图

所有反馈系统都可以抽象为这个结构:

```
               x_i         x_id        x_o
    x_s ──●──────→ [ A  基本放大器 ] ──────●────→ 输出
          ▲                                │
          │                                │
          │      x_f                       │
          └──────── [ F  反馈网络 ] ←───────┘
      (比较/叠加点)
```

**四个核心量**:

- xsxs​:输入源信号
- xfxf​:反馈信号(输出经 F 网络后送回的量)
- xid=xs−xfxid​=xs​−xf​:**净输入**(进入放大器的实际信号,叫"误差信号")
- xoxo​:输出信号

**两个核心模块**:

- **基本放大器 A**:开环增益 A=xo/xidA=xo​/xid​
- **反馈网络 F**:反馈系数 F=xf/xoF=xf​/xo​

**负反馈的标志**:比较点是**减号**(xid=xs−xfxid​=xs​−xf​),反馈信号"抵消"输入趋势。

### 11.3 核心公式推导——反馈方程

从框图直接推导:

xo=A⋅xid=A(xs−xf)=A(xs−Fxo)xo​=A⋅xid​=A(xs​−xf​)=A(xs​−Fxo​)

整理:

xo(1+AF)=Axsxo​(1+AF)=Axs​Af=xoxs=A1+AFAf​=xs​xo​​=1+AFA​​

**这是反馈的万能公式,务必刻进脑子**。

几个关键量的名字:

- AA:**开环增益**
- FF:**反馈系数**
- AFAF:**环路增益**(信号绕环路一圈被放大的倍数)
- 1+AF1+AF:**反馈深度**
- AfAf​:**闭环增益**

---

### 11.4 深度负反馈的"极限美"

当 AF≫1AF≫1(**深度负反馈**),反馈公式简化为:

Af≈AAF=1FAf​≈AFA​=F1​

**闭环增益只由反馈网络 F 决定,与开环增益 A 几乎无关!**

回头看第 9 课的运放公式:

**同相放大器**:Av=1+Rf/R1Av​=1+Rf​/R1​。推导中出现过 v−=vo⋅R1/(R1+Rf)v−​=vo​⋅R1​/(R1​+Rf​)——这不就是 F=R1/(R1+Rf)F=R1​/(R1​+Rf​) 吗?所以 Af=1/F=1+Rf/R1Af​=1/F=1+Rf​/R1​。完全吻合!

**反相放大器**:Av=−Rf/R1Av​=−Rf​/R1​,虽然形式稍有不同(电路结构决定),但核心还是"增益只由电阻比决定"。

**这就是为什么运放电路增益公式这么简洁**——本质是深度负反馈把开环增益的不稳定性完全压下去,换来了反馈网络的精确控制。

---

### 11.5 负反馈的五大好处(代价都是"损失增益")

这五条是模电的精华。每一条都要用 Af=A/(1+AF)Af​=A/(1+AF) 的变体来理解。

#### 好处 1:稳定增益

开环 AA 变化 ΔAΔA(比如因温度、器件老化),闭环 AfAf​ 变化多少?

求导(或者做微扰分析):

dAfAf=11+AF⋅dAAAf​dAf​​=1+AF1​⋅AdA​

**增益的相对变化被缩小了 (1+AF)(1+AF) 倍**!

**例子**:开环增益 AA 从 105105 变到 0.5×1050.5×105(变化 50%!),若 F=0.01F=0.01(AF=1000AF=1000):

闭环变化 ≈50%/1001≈0.05%≈50%/1001≈0.05%。

**用 BJT 的 ββ 不稳定,用反馈 + 精密电阻换来稳定增益**——这是整个运放电路设计哲学的缩影。

#### 好处 2:扩展带宽

放大器的增益在高频下会下降(我们下下节课专讲频率响应)。设放大器的增益随频率变化:

A(ω)=A01+jω/ωHA(ω)=1+jω/ωH​A0​​

ωHωH​ 是上截止频率(开环带宽)。

加反馈后,闭环增益:

Af(ω)=A(ω)1+A(ω)FAf​(ω)=1+A(ω)FA(ω)​

推导后得到闭环上截止频率:

ωHf=ωH(1+A0F)ωHf​=ωH​(1+A0​F)

**闭环带宽扩展了 (1+A0F)(1+A0​F) 倍**,但**增益缩小了同样的倍数**。乘起来:

Af⋅ωHf=A01+A0F⋅ωH(1+A0F)=A0ωHAf​⋅ωHf​=1+A0​FA0​​⋅ωH​(1+A0​F)=A0​ωH​

**增益带宽积(GBW)是常数**!

**这是反馈的守恒律**——想要更大带宽?就得降低增益。想要更大增益?带宽就缩。

一个 1 MHz GBW 的运放:设为增益 10 倍,带宽 100 kHz;设为增益 100 倍,带宽只剩 10 kHz。这是实际选型时常考虑的权衡。

#### 好处 3:减小非线性失真

基本放大器对小信号近似线性,但大信号下有非线性(BJT 的指数特性、饱和/截止失真等)。非线性可以看作**输出里出现"不该有"的谐波分量**——记为 xdxd​。

带反馈后,这些谐波也被"反馈回输入",然后被放大器反向放大,**部分抵消**原谐波。结果:

xd,f=xd1+AFxd,f​=1+AFxd​​

**失真也被压缩 (1+AF)(1+AF) 倍**。这是为什么高保真音频放大器几乎都用深度反馈。

#### 好处 4:改变输入/输出阻抗(按反馈类型)

这里有个重要的规律:**反馈"取什么样"和"送回什么样",决定了对阻抗的影响**。下面第 11.7 节展开。

#### 好处 5:抑制噪声和干扰(但有限制!)

反馈能抑制**放大器内部**产生的噪声/干扰(比如电源纹波通过电源线注入放大器内部),衰减 (1+AF)(1+AF) 倍。

**但是**:反馈**不能**抑制输入端混入的噪声——因为那些噪声和真信号完全混在一起,对放大器来说看起来一模一样。反馈只能区分"环路内产生的"和"输入进来的"。

---

### 11.6 上面这些"代价"总结

每一条好处都是用 **损失 (1+AF)(1+AF) 倍的增益** 换来的:

|好处|改善倍数|
|---|---|
|增益稳定度|1+AF1+AF|
|带宽|1+AF1+AF|
|失真|1+AF1+AF|
|内部噪声|1+AF1+AF|

而闭环增益变为 A/(1+AF)A/(1+AF)——也是除以 1+AF1+AF。

**反馈不创造增益,它"分配"增益——把原始增益的一部分拿出来交换其他性能**。这是模电里最深的设计哲学之一。

### 11.7 四种基本反馈结构

这是本课最"需要记"的部分,但逻辑非常清楚。

反馈有两个独立的维度:

**维度 1:从输出"取样"什么?**

- 取样**电压**:在输出端**并联**采样(电压越高采样越大)→ "电压反馈"
- 取样**电流**:在输出回路**串联**采样(电流越大采样越大)→ "电流反馈"

**维度 2:送回输入端怎么"比较"?**

- 反馈信号**串联**进输入回路(和输入电压比较)→ "串联反馈"
- 反馈信号**并联**接到输入节点(和输入电流比较)→ "并联反馈"

2 × 2 = 4 种组合:

|类型|取样|比较|输入端|输出端|
|---|---|---|---|---|
|**电压串联**|电压|串联|输入阻抗 ↑|输出阻抗 ↓|
|**电流串联**|电流|串联|输入阻抗 ↑|输出阻抗 ↑|
|**电压并联**|电压|并联|输入阻抗 ↓|输出阻抗 ↓|
|**电流并联**|电流|并联|输入阻抗 ↓|输出阻抗 ↑|

#### 记忆法(核心规律)

- **"取样什么,稳定什么"**:取电压 → 稳定输出电压 → 输出阻抗↓;取电流 → 稳定输出电流 → 输出阻抗↑
- **"比较什么,输入像什么"**:串联比较 → 像电压源 → 输入阻抗↑;并联比较 → 像电流源 → 输入阻抗↓

记住这两条规律,四种结构就不用背。

---

### 11.8 四种反馈的识别方法——实用技巧

#### 识别维度 1:电压反馈 vs 电流反馈

**看反馈网络"采样点"在哪里**:

- 反馈网络从**输出节点**引出(和负载**并联**)→ **电压反馈**(因为"采样"的是输出电压)
- 反馈网络从**负载回路**里串联位置引出(通常是负载和地之间的某个电阻)→ **电流反馈**(采样的是流过负载的电流)

**另一个判断方法**:把输出端短路——

- 如果反馈信号消失 → **电压反馈**(输出电压被短路拉到 0,反馈没了)
- 如果反馈信号还在 → **电流反馈**

#### 识别维度 2:串联反馈 vs 并联反馈

**看反馈信号接到输入端的哪里**:

- 反馈电压和输入信号**串联**(加在 BJT 的 BE 结两端之一、或运放两个输入端之一)→ **串联反馈**
- 反馈电流和输入信号在**同一节点相加**(接到同一个点)→ **并联反馈**

**运放电路的简化判断**:

- 输入信号进 v+v+​,反馈到 v−v−​ → **串联反馈**(同相放大器模式)
- 输入信号和反馈都进 v−v−​ → **并联反馈**(反相放大器模式)

### 11.9 经典例子:四种反馈各自代表什么电路

#### 电压串联——同相放大器(第 9 课)

```
  v_i ── v_+
               │\
               │ \
               │  >────●── v_o
               │  /    │
           ●───│/      │
           │           │
          [R_1]       [R_f]
           │           │
           地          │
           │           │
           └───────────┘
```

- 输出取样:反馈网络(R1R1​、RfRf​ 分压)**和输出并联**——电压反馈
- 输入比较:反馈电压接到 v−v−​,输入信号接到 v+v+​,两者**串联**形成净输入 v+−v−v+​−v−​——串联反馈
- **结果**:输入阻抗**极高**(MΩ 级),输出阻抗**极低**——完美的"电压放大器"

**电压放大器的理想形态:高 RiRi​,低 RoRo​**。电压串联反馈让任何放大器向这个理想靠近。

#### 电压并联——反相放大器(第 9 课)

```
       R_f
  ┌───[R_f]───┐
  │           │
 v_i──[R_1]──●── v_−
                │\
                │ \
                │  >───● v_o
                │  /
            ●───│/
            │
           地
```

- 输出取样:RfRf​ 一端接输出(并联)——电压反馈
- 输入比较:反馈电流和输入电流都流进 v−v−​ 节点(并联相加)——并联反馈
- **结果**:输入阻抗 =R1=R1​(不高),输出阻抗低——适合做"电流输入、电压输出"的**跨阻放大器**(光电二极管放大就常用这个)

#### 电流串联——共射放大器(发射极带 RERE​,不接旁路电容)

第 6 课讲过:不加 CECE​ 时,Av≈−RC/REAv​≈−RC​/RE​,**增益由外部电阻决定**。这就是反馈的威力——电流串联反馈。

```
  v_i ──[R_B]── B       C ── v_o
                 │       │
                 BJT    [R_C]
                 │       │
                 E       V_CC
                 │
                [R_E]   ← 这就是反馈网络
                 │
                地
```

- 输出取样:RERE​ 串联在发射极,流过它的电流 ≈IC≈IC​——电流反馈
- 输入比较:反馈电压(落在 RERE​ 上的 vREvRE​​)与 vivi​ 串联形成 vBEvBE​——串联反馈
- **结果**:输入/输出阻抗都提高,适合做"恒流输出"的场合

#### 电流并联——较少见,但存在

比如反相放大器的变种,输出串一个电阻做电流采样,反馈回 v−v−​。较少直接以这种结构出现,但在某些驱动电路(如 LED 恒流驱动、激光二极管驱动)里有应用。

### 11.10 一个深刻的总结:"负反馈塑造理想化"

把四种反馈的效果列出来,你会发现它们在做一件事——**把放大器改造成四种"受控源"的理想形态**:

|反馈类型|改造目标|理想特性|
|---|---|---|
|电压串联|理想电压放大器(VCVS)|Ri=∞Ri​=∞, Ro=0Ro​=0|
|电流串联|理想跨导放大器(VCCS)|Ri=∞Ri​=∞, Ro=∞Ro​=∞|
|电流并联|理想电流放大器(CCCS)|Ri=0Ri​=0, Ro=∞Ro​=∞|
|电压并联|理想跨阻放大器(CCVS)|Ri=0Ri​=0, Ro=0Ro​=0|

**根据你需要的"理想化方向",选择对应的反馈类型**。这是放大器设计的顶层思考。

### 11.11 正反馈与振荡条件——反馈的"另一面"

如果反馈信号不是减掉,而是**加上**原输入:

Af=A1−AFAf​=1−AFA​

(注意符号变了)

**当 AF=1AF=1 时**,分母为零,Af→∞Af​→∞——**输入哪怕是零,输出也能持续存在**。

这就是**振荡器**的原理。

**巴克豪森判据(Barkhausen Criterion)**——自激振荡的条件:

1. **环路增益幅值 ∣AF∣=1∣AF∣=1**(能量刚好补偿损耗)
2. **环路相移为 2π2π 的整数倍**(反馈信号与输入同相,形成正反馈)

**这是所有正弦振荡器的设计基础**。下两节课我们讲完频率响应后,就会用这个判据设计 RC 振荡器、LC 振荡器、晶体振荡器。

### 11.12 负反馈的稳定性问题——自激振荡

"负反馈"本来是减去,但放大器本身在**不同频率下有相移**。如果某个频率下**反馈信号被额外相移了 180°**——

**"减"变成了"加"——负反馈在这个频率下变成了正反馈!**

如果此频率下恰好 ∣AF∣≥1∣AF∣≥1,电路就**自激振荡**——变成了一个振荡器,失去了放大功能。

这就是**反馈系统稳定性问题**。

#### 相位裕量(Phase Margin)

定义:在 ∣AF∣=1∣AF∣=1 的频率点,环路相移偏离 180° 的余量。

- 相位裕量 > 60°:稳定,响应良好
- 相位裕量 = 30°~60°:勉强稳定,可能有振铃
- 相位裕量 < 30°:非常接近振荡
- 相位裕量 ≤ 0°:自激振荡

#### 工程上的"相位补偿"

运放内部有时有一个**小电容**,故意让高频响应提前下降——**在 ∣AF∣→1∣AF∣→1 之前,把增益拉下来**,从根本上避免自激。这叫**内部相位补偿**(如 μA741 内部就有一个 30 pF 补偿电容)。

代价:**高频增益牺牲了**——这就是为什么 741 的 GBW 只有 1 MHz,不是因为设计不好,而是为稳定性故意"压住"。

### 11.13 这一课你必须带走的东西

1. **反馈方程** Af=A/(1+AF)Af​=A/(1+AF)——所有反馈分析的起点。
2. **深度负反馈** (AF≫1AF≫1) 下,Af≈1/FAf​≈1/F——闭环增益只由反馈网络决定,这就是"运放电路只看外部电阻"的根本原因。
3. **负反馈五大好处**(都是 (1+AF)(1+AF) 倍的改善,以损失同样倍数的增益为代价):
    - 稳定增益
    - 扩展带宽(增益带宽积守恒)
    - 减小失真
    - 改善输入/输出阻抗
    - 抑制内部噪声
4. **四种反馈结构**,由"取样类型 × 比较方式"构成:
    - "取样什么,稳定什么"——输出阻抗
    - "比较什么,输入像什么"——输入阻抗
5. **反馈的本质是"塑造理想化"**——把真实放大器推向四种理想受控源的极限。
6. **正反馈 + ∣AF∣=1∣AF∣=1 = 振荡器**(巴克豪森判据)——反馈的"暗面"就是下下节课的振荡器。
7. **反馈稳定性**靠相位裕量衡量,相位补偿是工程手段——这是为什么运放都有"主极点补偿"。

### 下一课预告:频率响应——让时间说话

到现在为止,我们的分析默认信号是"某个固定频率",或者干脆忽略频率。**但所有实际放大器都有带宽限制**:

- 低频下,耦合电容阻碍信号通过,**增益下降**(下限截止频率 fLfL​)
- 高频下,BJT 的寄生电容短路信号,**增益下降**(上限截止频率 fHfH​)
- 中频段——增益平稳,才是我们前面分析过的"增益"

下一课讲:

- **幅频响应、相频响应、伯德图**——工程师眼中的"频率地图"
- **耦合电容怎么决定低频截止 fLfL​**
- **BJT 内部寄生电容**(CπCπ​、CμCμ​)怎么决定高频截止 fHfH​
- **密勒效应**(Miller Effect)——共射为什么高频差,共基为什么高频好,这里揭晓
- **增益带宽积**——运放选型的关键指标
- **多级放大器的带宽为什么比单级窄**

频率响应是模电"数学味"最浓的一章,用到复数、极点零点、对数坐标——但一旦掌握,你看电路就从"时域"升级到"频域",设计能力会发生质变。
