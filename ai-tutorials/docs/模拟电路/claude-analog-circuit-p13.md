# 第 12 课:频率响应——从时域到频域

### 12.1 为什么前面的分析"不完整"

前面十一课,我们算增益都得到一个**数字**:Av=−100Av​=−100、Av=1+Rf/R1Av​=1+Rf​/R1​ 等等。

但这些数字背后,有一个隐含假设:**信号频率不变,或者频率"无所谓"**。

**现实却是**:

- 你把 10 Hz 的信号送进一个音频放大器——没反应(低频被耦合电容挡住)
- 你把 10 MHz 的信号送进 μA741——严重衰减(高频超过带宽)
- 中频段(比如 1 kHz)——增益和我们算的一致

**放大器的增益是频率的函数**,记为 A(ω)A(ω) 或 A(f)A(f)。

这一课就是要把"一个数字"扩展成"一条曲线",让我们看到放大器在整个频率范围内的完整行为。

### 12.3 伯德图(Bode Plot)——工程师的频率地图

把幅频和相频画出来,就得到**伯德图**。工程上用**对数坐标**:

- 横轴:频率 ff(或 ωω),对数刻度(每格是 10 倍)
- 纵轴(幅频):**分贝** 20log⁡10∣A∣20log10​∣A∣ 的线性刻度
- 纵轴(相频):相位,线性刻度(单位:度 或 弧度)

**为什么用对数?**

1. **频率范围动辄横跨 6~7 个数量级**(几 Hz 到几 MHz),线性坐标画不下
2. **分贝让"增益级联"变成"加法"**:20log⁡(A1A2)=20log⁡A1+20log⁡A220log(A1​A2​)=20logA1​+20logA2​——多级放大器的幅频图直接叠加
3. **一阶系统的幅频在对数坐标下是两条直线**,方便手绘近似

### 12.5 放大器的完整频率响应

真实放大器的幅频图长这样:

```
   20 log|A|
         ↑
          │
       A_m ┤     ┌───────────────────┐
          │    /                     \
          │   /                       \
          │  /                         \
          │ /                           \
        ──┴──────┼────────────┼──────────→ log f
                f_L          f_H
         ↑      ↑              ↑          ↑
       低频段 下截止          上截止      高频段
              频率            频率
              
       ◄──── 中频段(平坦)───────►
            "通频带" BW = f_H − f_L
```

**三段**:

- **低频段**(f<fLf<fL​):增益随频率升高而上升(高通行为)
- **中频段**(fL<f<fHfL​<f<fH​):增益平坦,值为 AmAm​
- **高频段**(f>fHf>fH​):增益随频率升高而下降(低通行为)

**两个关键频率**:

- **下限截止频率 fLfL​**:低频端增益下降 3 dB 的频率
- **上限截止频率 fHfH​**:高频端增益下降 3 dB 的频率
- **通频带** BW=fH−fLBW=fH​−fL​

**关键**:

- **fLfL​ 由耦合电容、旁路电容决定**(低频下它们的阻抗不够小,开始阻碍信号)
- **fHfH​ 由 BJT/MOS 内部寄生电容决定**(高频下它们的阻抗变小,开始短路信号)

两段影响**完全独立,分别分析**。

### 12.6 低频响应——耦合电容的"退化"

#### 耦合电容的作用回顾

耦合电容 C1C1​ 把输入信号从前级"送"进下一级,同时阻隔直流。但这个"送"不是万能的:

**电容的阻抗** ZC=1/(jωC)ZC​=1/(jωC),频率越低阻抗越大。

- 高频:ZCZC​ 小,电容近似短路,信号顺利通过
- 低频:ZCZC​ 大,电容对信号有阻碍,形成高通滤波器

#### 分析模型

简化一下:把共射放大器输入端(耦合电容 C1C1​ 和输入阻抗 RiRi​)孤立出来看:

```
     v_s ──[C_1]──┬── v_i'(进入 BJT)
                  │
                 [R_i] (放大器输入阻抗)
                  │
                 地
```

这是一个典型的**RC 高通滤波器**:

vi′vs=RiRi+1/(jωC1)=jωRiC11+jωRiC1vs​vi′​​=Ri​+1/(jωC1​)Ri​​=1+jωRi​C1​jωRi​C1​​

**下限截止频率**:

fL1=12πRiC1fL1​=2πRi​C1​1​​

RiRi​ 越大、C1C1​ 越大,fL1fL1​ 越低——信号能"通过"到更低的频率。

#### 三个电容,三个极点

一个典型的共射放大器有三个电容:

- 输入耦合 C1C1​ → 产生极点 fL1fL1​
- 输出耦合 C2C2​ → 产生极点 fL2fL2​
- 发射极旁路 CECE​ → 产生极点 fLEfLE​

每个都是一个"高通极点"。

**下限截止频率**由**最高的那个极点**决定(近似):

fL≈max⁡(fL1,fL2,fLE)fL​≈max(fL1​,fL2​,fLE​)

更精确地,当三个极点相近时,fLfL​ 比最大那个略高。

#### 工程含义

**想让放大器响应到低频**(比如音频放大器要响应到 20 Hz):电容取大。

**高保真音频放大器的耦合电容常常是几十 μF 甚至几百 μF**——有时不得不用电解电容(大电容),但电解电容极性敏感、漏电、老化,所以音频发烧友又会挑高质量的薄膜电容……

**直接耦合放大器(无电容)**:从 f=0f=0(直流)开始就能工作,**没有下限截止频率问题**。这就是运放为什么能用作直流放大器的根本原因。

### 12.7 高频响应——BJT 寄生电容的"暴露"

这是这一课最精彩的部分——**密勒效应登场**。

#### 混合 π 模型加上电容

前面第 6 课的 BJT 小信号模型是"干净"的——没有电容。那是**中频近似**,忽略了 BJT 内部两个重要的电容:

```
            C_μ
     ┌────[C_μ]────┐
     │             │
     B ●──┬────────┼─────● C
         │         │
        [r_be]   [C_π]  ↓ g_m v_be
         │         │
         E ●───────●─────────
```

- **CπCπ​**(BE 结电容,几 pF 到几十 pF):发射结正偏时的扩散电容,较大
- **CμCμ​**(BC 结电容,0.5 pF 到几 pF):集电结反偏时的势垒电容,较小

**低/中频**:ZCπZCπ​​、ZCμZCμ​​ 都极大,近似开路,可以忽略。 **高频**:它们阻抗变小,开始起作用——**短路信号,使增益下降**。

#### 密勒效应(Miller Effect)——共射的"阿喀琉斯之踵"

这是一个非常重要、一开始不容易理解的概念。

**CμCμ​ 跨接在 B 和 C 之间**。B 是输入端,C 是输出端。**输出相对输入反相且幅度放大了 ∣Av∣∣Av​∣ 倍**。

从输入端看,CμCμ​ 两端的电压不是 vbevbe​,而是:

vμ=vB−vC=vbe−(−∣Av∣vbe)=(1+∣Av∣)vbevμ​=vB​−vC​=vbe​−(−∣Av​∣vbe​)=(1+∣Av​∣)vbe​

所以,流过 CμCμ​ 的电流是:

iCμ=jωCμ⋅(1+∣Av∣)vbeiCμ​​=jωCμ​⋅(1+∣Av​∣)vbe​

**从输入端往里看,CμCμ​ "看起来"像一个更大的电容**:

CM=(1+∣Av∣)CμCM​=(1+∣Av​∣)Cμ​​

这就是**密勒效应**——跨接在"反相放大器两端"的电容,从输入端看被**放大 (1+∣Av∣)(1+∣Av​∣) 倍**。

**数值冲击**:若 ∣Av∣=100∣Av​∣=100,Cμ=2Cμ​=2 pF,则从输入端看 CM=202CM​=202 pF!**原本很小的 CμCμ​ 被放大了 100 倍**,成为限制高频响应的**主要因素**。

#### 高频等效模型(简化)

用密勒效应把 CμCμ​ "等效"到输入端,加上原有的 CπCπ​:

```
   v_s ──[R_s]──● B ────[r_be]────● ... 
                │
                │           
             [C_π + C_M]     ← 总输入电容
                │
                地
```

输入端形成一个**RC 低通滤波器**,截止频率:

fH=12π(Rs∥rbe)(Cπ+CM)fH​=2π(Rs​∥rbe​)(Cπ​+CM​)1​​

**CM=(1+∣Av∣)CμCM​=(1+∣Av​∣)Cμ​ 主导** → **共射放大器高频响应受密勒效应强烈限制**。

#### 为什么共基、共集高频好?(回头看第 7 课)

- **共基**:输入端是 E,输出端是 C。**CμCμ​ 不跨接在输入输出之间**(它跨在 B 和 C 之间,B 是交流地)——**没有密勒效应**!所以共基的高频截止频率比共射高得多。
- **共集(射随)**:电压增益 ≈1≈1,**(1+∣Av∣)≈2(1+∣Av​∣)≈2,密勒倍增几乎不起作用**,所以高频也很好。

**密勒效应是区分三种组态高频性能的根本**。工程上用的 **Cascode**(共射 + 共基)结构:前级共射提供电压增益,但输出端"压在"共基的低阻输入上(共基输入阻抗 ≈re≈re​ 只有几十 Ω),使共射的实际电压增益被压到 1 左右——**密勒效应被抑制**。然后共基把信号"转播"到真正的负载,提供高频所需的高增益。**Cascode 是高频放大的标配结构**。

#### 特征频率 fTfT​

BJT 的一个重要高频指标:**电流增益降到 1 时的频率**:

β(jω)=β01+jω/ωββ(jω)=1+jω/ωβ​β0​​ωT≈β0ωβ=gmCπ+CμωT​≈β0​ωβ​=Cπ​+Cμ​gm​​

fTfT​(特征频率)是 BJT 高频性能的"上限"——信号频率超过 fTfT​ 时,BJT 已经基本不能放大电流。典型值:低频管几十 MHz,高频管几 GHz,微波管几十 GHz。

### 12.8 增益带宽积(GBW)——反馈的守恒律

第 11 课讲过这个,现在有了频率响应的框架,可以正式推导。

运放开环增益典型:

A(jω)=A01+jω/ωpA(jω)=1+jω/ωp​A0​​

加负反馈,闭环:

Af(jω)=A(jω)1+A(jω)FAf​(jω)=1+A(jω)FA(jω)​

代入推导,闭环上限频率:

ωHf=ωp(1+A0F)ωHf​=ωp​(1+A0​F)

中频闭环增益:

Amf=A01+A0FAmf​=1+A0​FA0​​

**乘积**:

Amf⋅ωHf=A0⋅ωp=GBWAmf​⋅ωHf​=A0​⋅ωp​=GBW

**增益带宽积是常数,与反馈量无关**。

**工程用法**:μA741 的 GBW = 1 MHz。

- 设计为增益 10(配 $R_f:R_1 = 9:1$):带宽 100 kHz
- 设计为增益 100:带宽 10 kHz
- 设计为增益 1000:带宽 1 kHz——**实际还是音频范围,不适合视频信号**

要做带宽 10 MHz 的放大器?**换一个 GBW 更高的运放**(如 LM318 的 15 MHz、LT1055 等)。

### 12.10 频率响应与"快慢"——时域的对应

**带宽越大,时域响应越快**。

对阶跃输入(方波),一阶低通系统的输出是指数上升:

vo(t)=V∞(1−e−t/τ),τ=12πfHvo​(t)=V∞​(1−e−t/τ),τ=2πfH​1​

**上升时间**(10% 到 90%):

tr≈0.35fHtr​≈fH​0.35​

如果一个放大器带宽 1 MHz,阶跃响应上升时间约 0.35 μs。**带宽和速度是一体两面**。

**应用**:示波器探头、高速数字电路、脉冲放大器……都要关注上升时间指标。

### 12.11 这一课你必须带走的东西

1. **增益是频率的函数 A(jω)A(jω)**,复数,有幅有相。
2. **伯德图**用对数—分贝坐标,因为频率范围大、对数让级联变加法、一阶系统在对数坐标下是折线近似。
3. **放大器三段**:
    - 低频段被**耦合电容**的高通特性限制,fLfL​ 由 max⁡max 的 RC 极点决定
    - 中频段**平坦**,就是前面算的那个"增益"
    - 高频段被**BJT 寄生电容**的低通特性限制
4. **关键频率**:fLfL​、fHfH​、带宽 BW=fH−fLBW=fH​−fL​,−3−3 dB 是工程定义的截止点。
5. **密勒效应**:跨接在反相放大器两端的小电容,从输入端看被放大 (1+∣Av∣)(1+∣Av​∣) 倍——**共射高频差的根本原因**。
6. **组态的高频对比**:共射差(密勒)< 共集好(增益≈1,密勒弱)< 共基最好(无密勒)。**Cascode 结构**是高频设计的标配。
7. **增益带宽积 GBW = 常数**——反馈可以增加带宽,但必须以同比例损失增益为代价。
8. **多级放大器总带宽比单级窄**,fH,total=fH21/n−1fH,total​=fH​21/n−1​。
9. **频域与时域对偶**:带宽大 ⇔ 响应快。tr≈0.35/fHtr​≈0.35/fH​。

### 下一课预告:功率放大器与效率

到现在为止,我们讲的都是**小信号**放大——BJT 工作在放大区中心,信号摆幅小,输出功率可能只有 mW 级。

但是你的音箱要输出 10W,你的发射机要输出 100W——**需要功率放大器(Power Amplifier)**。

功放的核心问题完全不同:**怎么把电源的直流功率高效转换成信号的交流功率?**

- **效率 ηη**:输出信号功率 / 电源提供的总功率
- **热功耗**:消耗在 BJT 上的功率会发热,必须散热
- **失真**:大信号工作,非线性严重,怎么压制?

下一课讲:

- **A 类**(甲类)放大器:晶体管全周导通,线性最好,效率最低(< 25%)
- **B 类**(乙类)推挽放大器:两个管子轮流工作,效率高(< 78.5%),但有"交越失真"
- **AB 类**:实际用得最多的折中——几乎无交越失真,效率可接受
- **D 类**(开关功放):晶体管当开关用,效率可达 90%+——音箱、手机里的功放多数是 D 类
- **散热、热阻、安全工作区**——功放的工程细节

这一课是模电"从理论走向实际应用"的关键。准备好说"继续"。
