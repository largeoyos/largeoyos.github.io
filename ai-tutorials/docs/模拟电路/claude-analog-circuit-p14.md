# 第 13 课:功率放大器——把电源功率送到负载

### 13.1 功放是一个"不同的问题"

前面所有课讲的放大器,目标都是**把小信号放大成更大的电压信号**——输入几 mV,输出几 V 就够了,负载是后一级的输入阻抗(通常几 kΩ),**消耗的功率极小**。

现在情况变了。假设你要驱动:

- **音箱扬声器**:4 Ω 或 8 Ω 负载,输出 10 W
- **电机驱动**:几 A 的电流
- **射频发射**:50 Ω 天线,输出几十 W

这些场合**关心的不再是电压增益,而是功率**。核心问题变成:

1. **怎么输出足够大的功率?** 电压摆幅要接近电源电压,电流要足够大
2. **怎么让电源的能量高效转化为信号能量?** —— 这叫**效率**
3. **怎么控制晶体管发热?** 大功率工作下,晶体管消耗的能量全变成热,烧管子就坏了
4. **怎么避免大信号下的非线性失真?**

**效率**和**发热**是功放独有的核心问题,这节课围绕它们展开。

---

### 13.2 几个必须先搞清楚的量

对一个功率放大器,我们关心三种功率:

- **PLPL​**:**负载获得的功率**(有用功率,信号功率)
- **PCCPCC​**:**电源供给的总功率**(电源"花的钱")
- **PCPC​**:**晶体管消耗的功率**(变成热量,必须散掉)

**能量守恒**:

PCC=PL+PC+P其他损耗PCC​=PL​+PC​+P其他损耗​

**效率**定义:

η=PLPCCη=PCC​PL​​​

这是功放的"圣杯指标"——越高越好。

**典型值**(一会儿逐类推导):

- A 类:最高 25% (变压器耦合 50%)
- B 类:最高 78.5%
- AB 类:70% 左右(实用)
- D 类:90%+(现代主流)

---

### 13.3 A 类(甲类)放大器——前面讲的都是

#### 定义

**A 类**:晶体管在整个信号周期内都处于**导通状态**(Q 点在放大区正中央,信号正负半周都不截止)。

你前面学的共射、共集、共基——都是 A 类。

#### 为什么效率低?

先看一个核心事实:**静态时(没有信号),晶体管仍在消耗电流**——Q 点 ICQICQ​ 始终流过,这本身就消耗电源功率。

设电源 VCCVCC​,Q 点电流 ICQICQ​,静态功耗:

PCC,静态=VCC⋅ICQPCC,静态​=VCC​⋅ICQ​

即使不发信号,这部分功率也始终被消耗掉——**大部分变成晶体管的热**,只有少量(通过 RCRC​)被浪费。

#### 电阻负载的 A 类效率上限

对简单的电阻耦合 A 类,最大不失真输出功率对应 VCC/2VCC​/2 的摆幅:

- 输出峰值电压 Vom≤VCC/2Vom​≤VCC​/2(Q 点居中时)
- 负载功率:PL=Vom2/(2RL)=VCC2/(8RL)PL​=Vom2​/(2RL​)=VCC2​/(8RL​)(最大时)
- 电源功率(静态):PCC=VCC⋅ICQ=VCC2/(2RL)PCC​=VCC​⋅ICQ​=VCC2​/(2RL​)(典型设计)

ηmax⁡=VCC2/(8RL)VCC2/(2RL)=25%ηmax​=VCC2​/(2RL​)VCC2​/(8RL​)​=25%

**只有 25%**——**四分之三的能量变成晶体管热量**。

**如果改用变压器耦合**(让负载直流电阻接近 0),可以提升到 50%,但变压器笨重昂贵,现在很少这样用。

#### A 类的优缺点

**优点**:

- **线性最好**——晶体管始终在放大区,远离截止和饱和,失真极小
- **电路最简单**
- **无交越失真**(B 类才有的问题)

**缺点**:

- **效率低**(25% ~ 50%)
- **静态功耗大**——即使没信号也发热,浪费电

**适用场合**:高保真音频前级、小信号放大、要求极低失真的场合(如甲类胆机发烧音响)。**大功率场合几乎不用 A 类**。

---

### 13.4 B 类(乙类)推挽放大器——效率大幅跃升

#### 核心思想

A 类的浪费来自"**静态电流**"——没信号也耗电。

**B 类的想法**:让晶体管**只在信号"需要"它时才导通**。具体做法:**两个管子,一个负责信号正半周,一个负责负半周,轮流工作**。

#### 互补对称推挽电路

```
                    V_CC
                     │
                     │
               NPN  ─┤ T1(正半周导通)
            ┌───────│
            │        E
            │        │
    v_i ────●        ●──── v_o ──── R_L ──── 地
            │        │
            │       ─┤ B
            │        │
               PNP  ─┤ T2(负半周导通)
                     │
                   V_EE (= -V_CC)
```

- T1 是 NPN,**输入大于 0 时导通**,电流从 VCCVCC​ 经 T1 流向负载(正半周)
- T2 是 PNP,**输入小于 0 时导通**,电流从负载经 T2 流向 VEEVEE​(负半周)
- 两个管子**互补对称**,共同完成一个完整周期

**两个管子都接成射极跟随器**(共集),所以电压增益 ≈1≈1,阻抗变换——这里的"放大"主要是**电流放大/功率放大**,不是电压放大。前面需要有电压放大级,后面接功放。

#### B 类的效率推导

**关键**:每个管子只在半个周期导通,另半个周期截止——**截止时电流为 0**。

正弦输入峰值 VmVm​,输出 vo(t)=Vmsin⁡(ωt)vo​(t)=Vm​sin(ωt):

**负载功率**:

PL=Vm22RLPL​=2RL​Vm2​​

**电源电流**:每个电源只在各自半周有电流,且是半波正弦。平均值:

ICC,avg=VmπRLICC,avg​=πRL​Vm​​

**电源功率**(双电源):

PCC=2VCC⋅VmπRL=2VCCVmπRLPCC​=2VCC​⋅πRL​Vm​​=πRL​2VCC​Vm​​

**效率**:

η=PLPCC=Vm2/(2RL)2VCCVm/(πRL)=πVm4VCCη=PCC​PL​​=2VCC​Vm​/(πRL​)Vm2​/(2RL​)​=4VCC​πVm​​

**理想上限**(当 Vm=VCCVm​=VCC​,最大摆幅):

ηB,max⁡=π4≈78.5%ηB,max​=4π​≈78.5%​

**比 A 类高得多**。实际做不到 78.5%(有各种损耗),但 70% 左右完全可行。

#### 交越失真(Crossover Distortion)——B 类的致命问题

**理想**:T1 在 vi>0vi​>0 时导通,T2 在 vi<0vi​<0 时导通,信号过零点平滑切换。

**现实**:T1 要等 vi>0.7Vvi​>0.7V 才导通(PN 结导通门槛),T2 要等 vi<−0.7Vvi​<−0.7V 才导通。

**在 −0.7V<vi<0.7V−0.7V<vi​<0.7V 之间,两个管子都截止,输出为 0**。

**结果**:输出波形在过零点附近"**平坦一段**":

```
      理想输出(sine)               B 类实际输出
       ___                          ___
      /   \                        /   \
     /     \                      /     \
    /       \                    /       \
   /         \    →           __/         \__
  /           \              /              \
  过零平滑          过零"平台"——交越失真
```

在音频上这**极其难听**——高频谐波丰富,听起来刺耳。

**这是 B 类在音频中几乎不用的原因**。

---

### 13.5 AB 类——实用的王者

#### 核心思想

**给 T1 和 T2 一点点"偏置",让它们在静态时就"微微导通"**。这样信号过零时,两个管子都已经在导通状态,平滑切换。

#### 电路实现

在 T1 和 T2 的基极之间加一个小的偏压(约 1.4V,两个 PN 结的导通压降),让它们**静态时都有一个小的静态电流** ICQICQ​(几 mA 到几十 mA)。

典型做法:用**两个二极管串联**(或两个 BJT 接成"偏置镜"结构)产生这个 1.4V 偏压。二极管的温度特性还能**自动补偿**晶体管的温度漂移——这是一个聪明的设计。

```
         V_CC
          │
     ──┤ T1 (NPN)
    ────┤
    ┌───●─── ...
    │   │
   ─▼──  ← 二极管 D1
    │
    │   ← 偏置中点(这里注入信号)
   ─▼──  ← 二极管 D2
    │
    ┌───●─── v_o
    │   │
     ──┤ T2 (PNP)
    ────┤
          │
         V_EE
```

#### AB 类的特性

- **静态电流小**(远小于 A 类),但不是 0(不像 B 类)
- **效率**略低于 B 类(因为有静态功耗),典型 60~75%
- **交越失真几乎消除**——因为两个管子始终都导通一点
- **线性度大幅改善**

**AB 类是实际音频功放的绝对主流**——几乎所有传统音响、耳机放大器、家用功放都是 AB 类。在"效率"和"音质"之间取得了最好的折中。

---

### 13.6 D 类(开关功放)——现代主流

#### 革命性思想

前面 A、AB、B 类都是"**线性功放**"——晶体管工作在放大区,VCEVCE​ 和 ICIC​ 都不为 0,功耗 VCE⋅IC>0VCE​⋅IC​>0 始终存在。

**D 类的想法**:让晶体管**只工作在两种状态**——**完全导通(饱和)或完全截止**。

- **导通时**:VCE≈0VCE​≈0,功耗 =VCE⋅IC≈0=VCE​⋅IC​≈0
- **截止时**:IC=0IC​=0,功耗 =0=0

**两种状态下晶体管功耗都接近 0**——效率理论上 100%,实际 85~95%。

#### 怎么用开关做放大?

**PWM 脉宽调制**:

1. 把输入模拟信号和一个高频三角波比较,得到**脉宽随信号变化的方波**(PWM)
2. 这个方波驱动晶体管**全速开关**(导通/截止,典型开关频率 100 kHz ~ 几 MHz)
3. 输出经过**LC 低通滤波器**,把高频开关成分滤掉,留下原始音频信号

```
   模拟输入 v_i
       │
       ▼
    [PWM 调制器] ──→ 高频方波(脉宽编码信号)
                        │
                        ▼
                [开关功放输出级] ── 接近方波
                        │
                        ▼
                   [LC 低通滤波器]
                        │
                        ▼
                  还原的音频信号 → 扬声器
```

#### D 类的优缺点

**优点**:

- **效率极高**(85~95%),发热小
- **电池续航长**(手机、便携音箱的优势)
- **小型化**——散热片小,集成度高

**缺点**:

- **电路复杂**——需要 PWM 调制、快速开关管、输出滤波器
- **EMI(电磁干扰)大**——高频开关产生辐射,需要仔细的布局和屏蔽
- **早期 D 类音质不如 AB 类**,但现代 D 类(高开关频率、先进调制方式)已经非常接近甚至超越 AB 类

**应用**:

- **手机扬声器功放**:电池续航是关键
- **蓝牙音箱、无线音箱**:小型化需求
- **汽车音响**:多通道、发热控制
- **专业音箱**(如 Class-D 舞台音箱):高功率输出,低发热

**D 类是现代功放的绝对主流**,传统 AB 类只在高端 HiFi 音响里还有一席之地。

### 13.7 散热与热阻——功放的工程现实

#### 为什么散热这么重要

晶体管消耗的功率变成热,导致**结温 TjTj​ 上升**。结温超过器件上限(硅器件通常 125°C ~ 150°C),**BJT 会"热击穿"——永久损坏**。

#### 热阻模型(电阻的类比)

把热流类比成电流,温差类比成电压:

ΔT=P⋅RθΔT=P⋅Rθ​

- PP:热功率(W)
- RθRθ​:热阻(°C/W)
- ΔTΔT:两点温差(°C)

**典型的热阻链**:

Tj=Ta+P⋅(Rθ,jc+Rθ,cs+Rθ,sa)Tj​=Ta​+P⋅(Rθ,jc​+Rθ,cs​+Rθ,sa​)

- TjTj​:结温
- TaTa​:环境温度
- Rθ,jcRθ,jc​:结到外壳热阻(晶体管内部,datasheet 给)
- Rθ,csRθ,cs​:外壳到散热片热阻(由导热硅脂、云母片决定)
- Rθ,saRθ,sa​:散热片到环境热阻(由散热片大小、风扇决定)

#### 设计例子

用一个 TO-220 封装的晶体管做功放,Rθ,jc=2Rθ,jc​=2 °C/W,Rθ,cs=0.5Rθ,cs​=0.5 °C/W。散热片 Rθ,sa=3Rθ,sa​=3 °C/W。环境 25°C。

功率 10 W:

Tj=25+10×(2+0.5+3)=25+55=80°C✓Tj​=25+10×(2+0.5+3)=25+55=80°C✓

远低于 150°C 上限,安全。

**如果不装散热片**(TO-220 裸露到空气的 RθRθ​ 典型 60 °C/W):

Tj=25+10×(2+60)=645°C完蛋Tj​=25+10×(2+60)=645°C完蛋

**结论**:功放必须算散热,散热片大小不是装饰。

### 13.8 安全工作区(SOA, Safe Operating Area)

除了不能"烧",BJT 还有**三个限制同时存在**:

1. **最大电流** IC,max⁡IC,max​
2. **最大电压** VCEO,max⁡VCEO,max​(击穿电压)
3. **最大功耗** PC,max⁡=VCE⋅ICPC,max​=VCE​⋅IC​——功耗等值线(双曲线)

三者围成一个区域,工作点必须始终在这个区域内——**安全工作区**:

```
    I_C
     │╲
     │  ╲          ← 恒定功耗双曲线 P_Cmax
     │    ╲
     │      ╲     
     │        ╲────
     ├─────────────┤   ← 最大电流 I_Cmax
     │              │
     │              │
     │              │
     │              │
    ─┴──────────────┴─── V_CE
                     V_CEOmax
```

**动态工作时**,信号摆幅决定工作轨迹。轨迹必须在 SOA 内——这约束了最大输出功率。

功放设计时常常要**画出负载线和 SOA**,验证信号最大时不会越界。这是一个经常被初学者忽略、实际烧管子的地方。

### 13.9 推挽功放的一个实际电路(示意)

把这一课所学组合起来,一个典型 AB 类音频功放的结构:

```
   输入 ──→ [输入级:差分放大] 
                     │
                     ▼
           [电压放大级:共射,高增益]
                     │
                     ▼
           [AB 类互补推挽输出级]
                     │
                     ▼
                  扬声器(8 Ω)
   
   + 全局负反馈网络(从输出采样回输入)
```

**各级分工**:

- 差分输入:高输入阻抗、双端接受信号
- 共射电压放大:提供主要电压增益
- AB 推挽输出:**电流放大,低阻驱动**
- **全局负反馈**:降低失真、稳定增益、扩展带宽——上节课讲的反馈思想再次发挥威力

这个结构和运放的内部结构高度相似——差不多就是"能驱动几 W 负载的大功率运放"。事实上许多功放芯片(如 LM3886、TDA7293)就是这样设计的,外面接几个电阻就能搭一个完整功放。

### 13.10 这一课你必须带走的东西

1. **功放不同于小信号放大**:核心关注"**效率**"和"**散热**",而不是电压增益。
2. **四类功放的特性对比**:

|类型|效率上限|失真|静态功耗|典型应用|
|---|---|---|---|---|
|A 类|25%(50%变压器)|极小|大|发烧音频前级|
|B 类|78.5%|交越失真严重|0|理论价值,实际少用|
|AB 类|~75%|很小|小|传统音频主流|
|D 类|90%+|取决于调制和滤波|极小|现代主流(手机、蓝牙音箱、汽车音响)|

3. **效率和线性矛盾**:A 类最线性但最费电,D 类最省电但电路复杂。**AB 类是在效率和线性之间的最好折中**,所以成为了传统主流。
4. **交越失真**是 B 类的核心问题,**AB 类通过加小偏置解决**——静态电流非零但很小。
5. **D 类的革命**:晶体管只做开关,摆脱线性工作区的功耗损失,配合 PWM + LC 滤波还原信号。
6. **散热是功放的工程现实**:用热阻链模型算结温,必须保证 Tj<Tj,max⁡Tj​<Tj,max​。
7. **安全工作区 SOA** 由电流、电压、功耗三个限制围成,大信号工作不能越界。
8. **实际功放 = 多级组合 + 全局负反馈**:和运放内部结构高度类似,就是"可以输出大功率的运放"。

### 下一课预告:振荡器与波形发生器

到现在为止我们都在放大"外来的"信号。但有些场合我们需要**让电路自己产生信号**——这就是**振荡器(Oscillator)** 的工作。

振荡器无处不在:

- 手机、WiFi、蓝牙的**本振频率**
- 微处理器的**时钟信号**
- 电子琴的**音调发生**
- 信号发生器的**测试信号**
- 开关电源的**开关频率**

下一课讲:

- **正弦振荡器** 的工作原理——正反馈 + 巴克豪森判据的工程应用
- **RC 振荡器**(文氏电桥、移相振荡器)——低频到中频,音频信号发生
- **LC 振荡器**(三点式电容反馈、电感反馈)——中频到高频,RF 应用
- **晶体振荡器**——高精度、高稳定度的时钟源,手机/计算机里的核心
- **非正弦波发生器**:方波、三角波、锯齿波——比较器 + 积分器的组合
- **555 定时器**——传奇芯片,搭一堆经典电路

这一课会让你看到**正反馈的"正面价值"**——不是设计缺陷,而是主动利用的工具。
