# 第 14 课:振荡器与波形发生器

### 14.1 振荡器是什么——反馈的"反面应用"

前 13 课讲的所有电路都在**放大外来的信号**。但有些场合,我们需要电路**自己产生**信号——没有输入,输出却能自己产生一个稳定的周期波形。

这就是**振荡器(Oscillator)**。

振荡器无处不在:

- **时钟信号**:CPU、单片机、手机处理器内部都有振荡器提供时钟
- **载波信号**:手机、WiFi、蓝牙、广播电视的无线发射需要高频载波
- **音调发生**:电子琴、合成器、测试仪器
- **开关频率**:开关电源、D 类功放内部需要高频方波
- **扫描信号**:示波器的时基、老式电视的行扫描

#### 振荡器的思想来源——正反馈

第 11 课讲反馈时,我们推导过闭环增益:

Af=A1−AF(正反馈)Af​=1−AFA​(正反馈)

**当 AF=1AF=1 时,分母为 0,Af→∞Af​→∞**——即使输入为零,输出也能有限。这时电路**自我维持振荡**。

**巴克豪森判据**——自激振荡的两个条件:

1. **幅值条件**:∣AF∣=1∣AF∣=1(环路增益 = 1,信号绕一圈刚好补偿损耗)
2. **相位条件**:环路相移 = 2πn2πn(nn 为整数)(反馈信号与原信号**同相**,形成正反馈)

> **放大器的缺陷(自激振荡)= 振荡器的功能**
> 
> 设计时要避免的事,变成我们主动追求的事。

#### 起振与稳幅的工程细节

严格满足 ∣AF∣=1∣AF∣=1 时电路处在"临界"状态——理论上能振荡,实际中噪声、温漂会让振荡消失。

**工程做法**:

1. **起振阶段**:设计 ∣AF∣>1∣AF∣>1(典型 2~3 倍),电路噪声(电阻热噪声、电源扰动等)被放大成振荡
2. **稳态阶段**:需要一个**自动稳幅机制**,让振幅增大时把 ∣AF∣∣AF∣ 拉回到 1,达到稳定振幅

**稳幅方法**:

- **非线性元件**:二极管、JFET、热敏电阻在幅值变大时阻抗变化,降低环路增益
- **自动增益控制(AGC)**:用电子电路主动调节放大器增益

### 14.2 正弦振荡器 vs 非正弦振荡器

两大类:

**正弦振荡器**:输出稳定的正弦波。核心是"选频网络"——让某一个频率满足巴克豪森判据,其他频率不满足。

- RC 振荡器:低频到中频
- LC 振荡器:中频到高频
- 晶体振荡器:高精度高稳定度

**非正弦振荡器**:输出方波、三角波、锯齿波。核心是"**弛豫**"——电容充放电 + 开关切换。

- 比较器 + 积分器结构
- 555 定时器
- 施密特触发器 + RC 电路

这两类工作原理完全不同,分开讲。

### 14.3 RC 正弦振荡器——文氏电桥

#### 为什么叫"文氏"(Wien Bridge)

1891 年德国物理学家 Max Wien 提出一种频率选择电桥,后来被用于振荡器。这是**音频范围最常用的正弦振荡器**,信号发生器的经典电路。

#### 电路结构

核心是一个**同相放大器 + 文氏网络正反馈**:

```
                            R_f
                   ┌───────[R_f]───────┐
                   │                    │
                   │                    │
          v_-  ●───┤                    │
                   │\                   │
              ●────┤ \                  │
              │    │  >─────────────────●──── v_o
             [R_1] │ /                  │
              │    │/                   │
              地   │                    │
                   ●──────────────●     │
                  v_+         ┌───┴───┐ │
                              │       │ │
                             [R]   [C]─┤ 反馈网络
                              │       │ │
                              └───●───┘ │
                                  │     │
                              ┌───●───┐ │
                              │       │ │
                             [R]    [C]─┤ (并联)
                              │       │ │
                              └───●───┘ │
                                  │     │
                                 地
```

**两部分**:

- **上方**:运放 + R1R1​ + RfRf​ 构成同相放大器,增益 A=1+Rf/R1A=1+Rf​/R1​
- **下方**:由 RR、CC 构成的**文氏反馈网络**,把输出送回同相输入端

#### 文氏网络的频率特性

文氏网络(串联 RCRC + 并联 RCRC):

```
  v_o ──[R串]──┬── R并 ┬── v_f(到 v_+)
               │        │
              [C串]    [C并]
               │        │
              地       地
```

反馈系数(频域):

F(jω)=vfvo=13+j(ωRC−1/(ωRC))F(jω)=vo​vf​​=3+j(ωRC−1/(ωRC))1​

**关键观察**:当 ω=1/RCω=1/RC 时,虚部为 0,F=1/3F=1/3——**实数,即相位为 0°**。

对其他频率:相位非零,即有相移。

**这意味着**:只有在 ω0=1/RCω0​=1/RC 这一个频率上,反馈信号与原信号同相。

#### 振荡条件

巴克豪森判据要求 AF=1AF=1(实数,正,即相位 0°):

- 频率:ω0=1/RCω0​=1/RC(文氏网络本身选出)
- 幅值:A⋅F=A⋅1/3=1⇒A=3A⋅F=A⋅1/3=1⇒A=3

也就是 1+Rf/R1=31+Rf​/R1​=3,**即 Rf=2R1Rf​=2R1​**。

#### 振荡频率

f0=12πRCf0​=2πRC1​​

**调频方法**:同步改变两个 RR 或两个 CC。传统做法是用双联电位器或双联可变电容。

#### 稳幅——钨丝灯泡的古典绝技

**问题**:Rf/R1=2Rf​/R1​=2 是临界条件,精度差一点就不振荡或振幅无界增大。

**经典方案(HP 公司开创)**:用**小钨丝灯泡**代替 R1R1​。

- 振幅小 → 灯泡电流小 → 灯丝温度低 → 电阻小 → Rf/R1Rf​/R1​ 大 → 增益大 → 振幅增大
- 振幅大 → 灯泡电流大 → 灯丝热 → 电阻大 → Rf/R1Rf​/R1​ 小 → 增益小 → 振幅减小

**自动平衡**在 Rf/R1=2Rf​/R1​=2 的临界点附近。

这是 HP 公司 1939 年第一款产品 HP 200A 的核心技术,奠定了整个硅谷。现代方案用 JFET、光耦、二极管整流等替代灯泡,但思想一样。

#### 适用范围

文氏电桥适用于 **几 Hz 到几百 kHz**,覆盖整个音频范围和中频低段。失真度可以做到 0.01%——非常干净的正弦波。

### 14.4 LC 振荡器——高频场合

#### 为什么高频要用 LC

RC 振荡器在高频遇到问题:

- RR 和 CC 取很小才能做到高频,但 RR 太小使信号源被"短路",CC 太小与寄生电容相比可以忽略
- 运放带宽不够(几 MHz 以上大多数运放不工作)

**高频用 LC**:电感 + 电容构成并联谐振回路,有天然的频率选择能力。

#### LC 并联谐振

LC 并联电路在谐振频率处阻抗极大:

ω0=1LCω0​=LC​1​

**Q 值**(品质因数):衡量谐振的"尖锐度",Q 越高选频越尖锐。典型 LC 谐振回路 Q = 几十到几百。

#### 电容反馈振荡器(Colpitts)

```
                    V_CC
                     │
                    [R_C]   ← 集电极偏置
                     │
                     ├──────────●──── v_o
                     │         │
                    ─┤         │
                  ──┤          │
                     │ (BJT)    │
                     │         │
                     ●─────────┤
                     │         │
                    [L]       [C1]
                     │         │
                     ●────────●
                     │         │
                    ─┴─       [C2]
                    地         │
                               │
                              地
```

反馈从 C2 两端取,送回 BJT 基极。C1C1​、C2C2​ 串联再与 LL 并联,振荡频率:

f0=12πL⋅C1C2/(C1+C2)f0​=2πL⋅C1​C2​/(C1​+C2​)​1​

反馈系数(电容分压):F=C1/C2F=C1​/C2​。

#### 电感反馈振荡器(Hartley)

结构类似,用**抽头电感**代替电容分压。

f0=12π(L1+L2)Cf0​=2π(L1​+L2​)C​1​

#### Colpitts vs Hartley

历史上都很常见,现代更多用 Colpitts,因为电容比抽头电感更容易精确制造。

#### 适用范围

LC 振荡器覆盖 **几百 kHz 到几 GHz**——广播发射、电视调谐、RF 通信、微波。手机射频前端里有大量 LC 振荡器(压控振荡器 VCO 是关键模块)。

### 14.5 晶体振荡器——精度之王

#### 为什么需要晶体

LC 振荡器的频率稳定度约 10−410−4——随温度、老化会漂移几十 ppm。这对**计时、通信的载波同步**是不够的。

**石英晶体(Quartz Crystal)利用压电效应**:外加电场使晶体机械振动,机械振动反过来产生电场。电学上,晶体等效为一个**高 Q 值**(几万到几十万)的 LC 谐振电路。

**典型石英晶体的稳定度**:

- 普通晶体:10−510−5 ~ 10−610−6(几 ppm)
- 恒温晶振(OCXO):10−910−9
- 原子频率标准:10−1210−12 以上

#### 晶体的电学等效

```
           L_s   C_s   R_s
   ●──────[L]───[C]───[R]─────●
   │                            │
   │          ──[C_p]──         │  ← 并联寄生电容
   │                            │
   ●────────────────────────────●
```

两个谐振频率:

- **串联谐振** fs=1/(2πLsCs)fs​=1/(2πLs​Cs​​)
- **并联谐振** fpfp​ 略高于 fsfs​

晶振电路利用这两个频率中的一个。

#### 应用

- **CPU 时钟**:笔记本的时钟是从几 MHz 晶振倍频出来的
- **手机基带时钟**:26 MHz 或 19.2 MHz 晶振是所有通信频率的基准
- **实时钟 RTC**:32.768 kHz 表用晶体(215215 Hz,方便二进制分频)
- **GPS 接收机、精密测量仪器**

**几乎所有需要精确时间或频率的电路都用晶振**。

### 14.6 非正弦波发生器——比较器 + 积分器

#### 方波发生器(多谐振荡器)

用**一个运放做比较器 + RC 充放电**:

```
                              R_f
                     ┌───────[R_f]───────┐
                     │                    │
                     │                    │
     v_i  ●──────────┤                    │
                     │\                   │
                     │ \                  │
         ┌───[R]─────┤  \                 │
         │           │   >────────────────●──── v_o (方波输出)
         │           │  /                 
         │           │ /                  
         ●───────────│/                   
         │          v_+                   
        [C]                                
         │           
         │           [R_2]   
         │            │      (正反馈上拉电阻到 v_+ 和 v_o)
         │            │      
        地           ...     
                     
   (这是施密特触发器 + RC 积分的反馈结构)
```

精确图略复杂,核心工作原理:

1. vovo​ 为高时,RR 对 CC 充电,v−v−​ 上升
2. 当 v−>v+v−​>v+​(正反馈门槛),比较器翻转,vovo​ 变低
3. CC 反向充电,v−v−​ 下降
4. 当 v−<v+v−​<v+​ 新门槛,再次翻转,vovo​ 变高
5. 循环——输出方波,频率由 RCRC 决定

#### 三角波发生器

**方波发生器 + 积分器**:把方波送入积分器,输出就是三角波。

```
   方波发生器 ──────→ 积分器 ──────→ 三角波
     (方波)                           (三角波)
```

更精妙的是,**把三角波送回方波发生器作为比较输入**,可以构成**自持续的方波+三角波组合输出电路**——信号发生器的核心原型。

#### 锯齿波

三角波的特例:上升慢、下降快(或反之)。

**应用**:

- 示波器扫描线性
- 电视行扫描(CRT 时代)
- PWM 调制中的参考波

### 14.7 555 定时器——传奇芯片

#### 为什么单独讲 555

555 是 1972 年 Signetics 公司设计的定时器芯片。**50 多年过去,累计出货量超过 10 亿片**,至今仍在生产。它简单、便宜、功能多——几乎是电子爱好者的"入门图腾"。

#### 内部结构

```
       V_CC
        │
       ─┴─
        │
       [5kΩ]
        │     
        ●───── 阈值比较器 (V_CC × 2/3)
        │      │
       [5kΩ]   │
        │      │
        ●───── 触发比较器 (V_CC × 1/3)
        │      │
       [5kΩ]   │
        │      │
       地      ─── RS 触发器 ── 输出驱动 ── 输出
                      │
                   放电三极管
```

**三个 5 kΩ 电阻分压**(这就是"555"名字来源)产生两个阈值:VCC/3VCC​/3 和 2VCC/32VCC​/3。

**两个比较器 + 一个 RS 触发器**组成核心逻辑。

**8 个引脚**:电源、地、触发、阈值、输出、复位、控制电压、放电。

#### 两种经典模式

##### 模式 1:单稳态(单触发脉冲发生器)

外接一个电阻 RR 和电容 CC。触发输入一个下降沿,输出一个宽度确定的脉冲:

T=1.1RCT=1.1RC

**应用**:按键去抖、精确延时、脉冲整形。

##### 模式 2:无稳态(方波振荡器)

外接两个电阻 R1R1​、R2R2​ 和电容 CC。自动连续振荡,输出方波:

- 高电平时间:TH=0.693(R1+R2)CTH​=0.693(R1​+R2​)C
- 低电平时间:TL=0.693R2CTL​=0.693R2​C
- 频率:f=1.44/((R1+2R2)C)f=1.44/((R1​+2R2​)C)
- 占空比:D=(R1+R2)/(R1+2R2)D=(R1​+R2​)/(R1​+2R2​)(永远 > 50%)

**应用**:简单时钟、LED 闪烁、PWM 发生、音频提示音。

#### 555 的价值

1. **教学价值**:集成了比较器、触发器、驱动器,是学习时序电路的绝佳载体
2. **工程价值**:低成本、高可靠、易设计——很多小应用根本不需要 MCU,一片 555 + 几个电阻搞定
3. **爱好者价值**:无数经典电路案例——报警器、玩具、灯光控制、乐器……

### 14.8 压控振荡器(VCO)——现代通信的基石

#### 定义

**压控振荡器(Voltage-Controlled Oscillator, VCO)**:频率由**控制电压 vctrlvctrl​** 决定:

fout=f0+KVCO⋅vctrlfout​=f0​+KVCO​⋅vctrl​

KVCOKVCO​ 是"调频灵敏度"(Hz/V)。

#### 实现方法

把 LC 振荡器中的电容替换为**变容二极管**(反偏 PN 结的结电容随反偏电压变化)。控制电压改变变容二极管的电容,从而改变振荡频率。

#### 应用——锁相环 PLL

VCO 的杀手级应用是**锁相环(Phase-Locked Loop, PLL)**:

```
  参考频率 ─→ [相位比较器] ─→ [环路滤波] ─→ [VCO] ─→ 输出频率
                  ↑                              │
                  └──────────[分频器]←───────────┘
```

PLL 用一个低频、稳定的**参考晶振**(比如 10 MHz),通过反馈锁定让 VCO 输出一个**可编程的高频信号**(比如 2.4 GHz)。

**所有现代通信设备都依赖 PLL**——手机、WiFi、蓝牙、5G 基站,都靠 PLL 合成各个信道的载波频率。

### 14.9 振荡器的选择——工程速查

|频率范围|首选|
|---|---|
|0.01 Hz ~ 1 MHz|RC(文氏电桥、555 定时器、运放弛豫)|
|1 MHz ~ 1 GHz|LC 振荡器(Colpitts、Hartley)|
|需要高精度(任何频率)|晶体振荡器|
|1 GHz 以上(射频/微波)|LC VCO + PLL|
|开关电源 / PWM|锯齿波/三角波发生器,通常集成在芯片内|

### 14.10 这一课你必须带走的东西

1. **振荡器 = 正反馈 + 巴克豪森判据**(幅值 1、相移 2πn2πn)。**放大器的缺陷就是振荡器的功能**。
2. **工程起振条件**:初始 ∣AF∣>1∣AF∣>1,靠非线性稳幅收敛到 ∣AF∣=1∣AF∣=1。
3. **文氏电桥振荡器**:中心频率 f0=1/(2πRC)f0​=1/(2πRC),增益条件 Rf=2R1Rf​=2R1​。音频范围标准振荡器。
4. **LC 振荡器**:Colpitts(电容分压)、Hartley(电感抽头),中高频应用。
5. **晶体振荡器**:用石英晶体的机械谐振(Q 值极高)做频率基准,稳定度 10−610−6 或更高。现代电子设备的时基几乎都依赖晶振。
6. **非正弦振荡**:比较器 + RC 充放电 = 方波。方波 + 积分器 = 三角波。
7. **555 定时器**是集成化的"迷你振荡器/定时器系统",两种典型模式(单稳态、无稳态)覆盖大量应用。
8. **压控振荡器 VCO + PLL**:现代通信的频率合成基石。
9. **正反馈不再是"敌人"**,而是**主动利用的工具**。关键是控制——让电路在我们想要的频率振荡,不在其他频率自激。

### 下一课预告:有源滤波器

13 课的功放、14 课的振荡器分别对应"**能量**"和"**时间/频率**"两个维度的应用。下一课把这一切整合到一个最普遍的任务——**滤波**。

信号处理最基本的操作之一是**"选择性地让某些频率通过,压制其他频率"**。这就是滤波器。

我们以前用过 RC 低通、RC 高通——**无源滤波**。但是无源滤波有两个大问题:

1. 增益永远 ≤ 1,没有放大能力
2. 带负载时滤波特性被破坏

**有源滤波器**用运放 + RC,解决这两个问题,还能做出 RC 做不到的复杂特性。
