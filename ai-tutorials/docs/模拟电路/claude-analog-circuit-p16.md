# 第 15 课:有源滤波器——信号处理的基石

### 15.1 滤波器是在干什么

信号处理最基本的操作之一:**让我们想要的频率通过,压制我们不想要的频率**。

例子:

- 收音机:频谱里有无数电台,我们只要其中一个——**带通滤波**
- 音响:音源里有 60 Hz 市电嗡嗡声(哼声)——**带阻(陷波)滤波**
- 音频采样前:超过采样率一半的高频要先去掉,否则会混叠——**低通滤波**(抗混叠滤波)
- 心电图采集:微弱信号被肌电、工频干扰淹没——**组合滤波**

**滤波器的核心任务就是"频域选择性"**。

---

### 15.2 四大理想滤波器

按"通过"和"阻止"的频率范围,滤波器分四类:

```
   低通 (LPF)               高通 (HPF)
   |A|                     |A|
   ─────┐                    ┌─────
        │                    │
        └────────         ───┘
   通过 ↑ 阻止 ↑       阻止 ↑ 通过 ↑
         f_c                 f_c

   带通 (BPF)              带阻/陷波 (BSF/Notch)
   |A|                     |A|
       ┌─┐                 ───┐   ┌───
       │ │                    │   │
   ────┘ └────                └───┘
       f_c                     f_c
```

**理想**是矩形切换。**现实**中不可能——矩形幅频响应对应无限长的时域冲激响应,物理不可实现。

所有真实滤波器都是理想形状的**近似**,近似的方向和手段决定了滤波器的"种类"。

---

### 15.3 关键指标

#### 截止频率 fcfc​

增益下降 3 dB 的频率,习惯性定义——这是"通带"和"阻带"的分界。

#### 阶数 nn

决定**过渡带的陡度**。一阶滤波器过渡带斜率 20 dB/decade(每十倍频程衰减 20 dB);二阶 40 dB/decade;nn 阶 20n20n dB/decade。

**阶数越高,越接近理想矩形,但电路越复杂**。

#### 通带平坦度

通带内增益是不是平坦?有无纹波?

#### 阻带衰减

要求阻带至少衰减多少(如 60 dB、80 dB)?

#### 相位特性

线性相位滤波器保持波形形状,非线性相位会让波形失真——**音频、视频需要关注相位**。

### 15.4 无源滤波器的局限

之前用过的 RC 低通:

```
   v_i ──[R]──┬── v_o
              │
             [C]
              │
             地
```

传递函数:

H(jω)=11+jωRCH(jω)=1+jωRC1​

截止频率 fc=1/(2πRC)fc​=1/(2πRC),**一阶低通**。

**三大局限**:

1. **增益 ≤ 1**——无法放大,甚至通带都有损耗
2. **带负载影响特性**——负载阻抗和 RR、CC 相互作用,破坏滤波形状
3. **多阶级联时阻抗相互耦合**——两个 RC 低通串起来不是真正的二阶滤波器,特性退化

**LC 滤波器**可以避免负载问题,也能做二阶以上,但:

- 低频下电感巨大笨重
- 电感有损耗、饱和、磁耦合问题
- 集成电路里做不出来像样的电感

---

### 15.5 有源滤波器——运放登场

**有源滤波器 = 运放 + RC 网络**。

优势:

1. **运放输出阻抗极低,隔离负载**——后面接什么不影响滤波特性
2. **有增益**——通带可以放大而不是衰减
3. **用运放反馈"制造"出电感效果**——低频也能做高阶滤波
4. **精确、可调、易集成**

代价:

- 需要电源
- 带宽受运放限制,高频(MHz 以上)不适合
- 有噪声、失调

**音频、仪表、生物医学信号处理——几乎都用有源滤波**。

### 15.7 Sallen-Key 二阶低通——经典中的经典

#### 电路结构

最著名的有源滤波拓扑,1955 年由 MIT 林肯实验室的 R.P. Sallen 和 E.L. Key 提出。

```
             C_2
        ┌───[C_2]───┐
        │           │
        │           │
   v_i─[R_1]─●─[R_2]─●─── v_+
                         │\
                         │ \
                     ●───┤  \
                     │   │   >────●── v_o
                     │   │  /     │
                    [C_1]│ /      │
                     │  ●│/       │
                    地   │        │
                         │        │
                     (如果是单位增益跟随器,v_− 直接接输出)
                         │        │
                         └────────┘
```

**工作原理**(直观理解):

- 低频:C1C1​、C2C2​ 开路,信号通过 R1R1​、R2R2​ 到运放 → 输出 = 输入
- 高频:C1C1​、C2C2​ 短路,信号被导地,输出趋于 0
- **中间**:C2C2​ 作为"**正反馈元件**",在截止频率附近产生一个**谐振峰**——这是它"二阶"的关键

#### 单位增益版本的传递函数

K=1K=1(同相接成电压跟随器),设 R1=R2=RR1​=R2​=R,C1=C2=CC1​=C2​=C:

H(jω)=1(jωRC)2+3(jωRC)+1⋅(某种形式)H(jω)=(jωRC)2+3(jωRC)+11​⋅(某种形式)

一般式:

H(s)=ω02s2+(ω0/Q)s+ω02H(s)=s2+(ω0​/Q)s+ω02​ω02​​

其中 s=jωs=jω 为拉普拉斯变量(可以暂且当成复频率)。

**两个关键参数**:

- **谐振频率** ω0=1/R1R2C1C2ω0​=1/R1​R2​C1​C2​​
- **品质因数** QQ——由电阻电容比值和增益决定

#### Q 值决定通带形状

这是二阶滤波的核心:**同样的 ω0ω0​,不同的 QQ,通带附近形状完全不同**。

```
    |H|
     │
     │      ╱╲       Q = 2(有峰)
     │     ╱  ╲
     │    ╱    ╲
  1  ────      ╲__     Q = 0.707(巴特沃斯,最大平坦)
     │            ╲
     │             ╲   Q = 0.5(过阻尼,下沉早)
     │              ╲
   ──┴───────────────╲──→ f
                     f_c
```

- Q=0.5Q=0.5:**临界阻尼**(贝塞尔滤波器接近这个)
- Q=0.707Q=0.707:**巴特沃斯**(最大平坦通带)
- Q>0.707Q>0.707:通带出现峰——切比雪夫族用这个
- Q→∞Q→∞:变成振荡器!(阻尼消失)

**低通滤波器的设计本质:选 ω0ω0​(截止频率)和 QQ(通带形状)**。

#### Sallen-Key 高通

**把 Sallen-Key 低通里的 R 和 C 全部互换**,就是 Sallen-Key 高通。频率特性镜像。

### 15.8 三大经典滤波器家族

对给定阶数,根据**通带形状、阻带陡度、相位特性的权衡**,有三大经典家族:

#### 巴特沃斯(Butterworth)——最大平坦

**特点**:通带极平坦(无纹波),阻带单调下降,过渡带中等陡。

**二阶 Q=0.707Q=0.707**。

**适用场合**:**通用首选**,特别是要求通带内幅度一致的场合(音频、测量仪器)。

#### 切比雪夫(Chebyshev)——陡峭过渡

**特点**:通带有**等纹波**(ripple,设计时选 0.5 dB、1 dB 等),换来**更陡的过渡带**。阶数相同的情况下,比巴特沃斯更接近理想矩形。

**代价**:

- 通带有波动
- 相位特性更差(波形失真大)

**适用场合**:对过渡带陡度要求高、能容忍通带波动的场合。

**反切比雪夫(Inverse Chebyshev)**:通带平坦,**阻带有等纹波**,过渡带同样陡。换了一种分配波动的方式。

**椭圆滤波器(Cauer)**:通带和阻带都有等纹波,**过渡带最陡**——给定阶数下性能最强,但相位特性最差。

#### 贝塞尔(Bessel)——线性相位

**特点**:**通带内群延迟几乎是常数(相位线性)**,不同频率分量延时一致,**波形不失真**。过渡带比巴特沃斯更缓。

**适用场合**:**音频**(保持音色)、**示波器前端**(保持阶跃响应波形)、**数字通信**(保持脉冲形状)。

#### 对比

```
               过渡带陡度 ←─────────────────→   陡
                    贝塞尔  巴特沃斯  切比雪夫  椭圆
                    
    通带平坦度        好     最好     差(纹波)差
    阻带衰减          缓     中       陡         最陡
    相位线性         最好    中       差         最差
    时域波形保真度    最好    中       差         最差
```

**没有"最好"的家族,只有最适合某个任务的家族**。

### 15.9 高阶滤波器——级联二阶

要做六阶低通?用三个二阶 Sallen-Key 级联,每个二阶配合适的 ω0ω0​ 和 QQ。

**注意**:六阶巴特沃斯不是"三个相同的二阶巴特沃斯",而是**三个 ω0ω0​ 相同、QQ 值不同的二阶**,按特定数学规律组合(查表或用设计软件)。

**奇数阶**:需要一个一阶级 + 若干二阶级。

**级联时需要缓冲**:运放输出阻抗低,各级天然隔离——这是有源滤波相对无源最大的优势之一。

### 15.10 带通滤波器

两种实现方法:

#### 方法 1:低通 + 高通级联

低通的截止频率 fHfH​,高通的截止频率 fLfL​(fL<fHfL​<fH​)。

- f<fLf<fL​:高通阻止
- f>fHf>fH​:低通阻止
- fL<f<fHfL​<f<fH​:都通过

适合**宽带带通**(fH/fL>2fH​/fL​>2)。

#### 方法 2:多反馈带通(MFB)

专门的二阶带通电路,一个运放实现:

```
                C_1
           ┌───[C_1]───┐
           │           │
   v_i──[R_1]─●        │
              │        │
              │  C_2   │
              ├──[C_2]─●── v_−
              │           │\
              │           │ \
             [R_2]        │  >────● v_o
              │           │  /
              │       ●───│/
              │       │
              地      地
```

**三个参数**:中心频率 ω0ω0​、品质因数 QQ、中心频率处增益 A0A0​。

QQ 决定带宽:BW=f0/QBW=f0​/Q。QQ 越高,带宽越窄(越"尖")。

适合**窄带带通**——比如选出某个特定频率的信号。

### 15.11 带阻/陷波滤波器

#### 应用——去除工频干扰

50 Hz 或 60 Hz 市电通过各种耦合混入信号。生物医学信号(ECG、EEG)、精密测量常要**精确去掉 50 Hz**——这就是**陷波滤波器**。

#### 双 T 陷波网络

```
           R       R
   v_i──[R]──●──[R]───●── v_+
              │        │\
             [C]       │ \
              │        │  >──── v_o
             地        │ /
                       │/
           C       C
   v_i──[C]──●──[C]───┘
              │
             [R]
              │
             地
```

上面一路 R-R-C 到地,下面一路 C-C-R 到地。在 f0=1/(2πRC)f0​=1/(2πRC) 处,两路的信号**反相抵消**——形成深深的陷波。

实际电路把双 T 网络嵌入运放正反馈结构(Q 增强),得到尖锐的 50 Hz 陷波,**陷波深度可达 60 dB 以上**。

### 15.12 设计流程(实用提示)

工程上设计有源滤波器的标准流程:

1. **明确需求**:截止频率 fcfc​、通带增益、阻带衰减、相位要求、成本等
2. **选家族**:巴特沃斯(通用)、切比雪夫(陡)、贝塞尔(相位)、椭圆(最陡)
3. **定阶数**:根据阻带衰减要求。每阶 20 dB/dec,所以要 60 dB 衰减至少需要 3 阶
4. **查表**:滤波器手册或 TI/ADI 等芯片厂商的设计工具(FilterPro、Filter Wizard 等),直接给出 ω0ω0​、QQ 参数
5. **选拓扑**:Sallen-Key(简单,参数灵敏度中)、多反馈 MFB(适合带通、高 Q)、状态变量(同时输出 LPF/HPF/BPF)
6. **算元件值**:根据 ω0ω0​、QQ、增益算 R、C
7. **取标准值 + 仿真**:实际电阻电容有标称值序列(E12、E24、E96),选取最接近的,然后 SPICE 仿真验证

**现代实践**:几乎不用手算,直接用厂商提供的 Web 设计工具,输入指标,工具给完整电路图 + 元件值 + 仿真结果。

### 15.13 超越:开关电容滤波器 & 数字滤波器

#### 开关电容滤波器

用 **MOS 开关 + 电容**模拟电阻,然后和运放组成滤波器。**截止频率由开关时钟频率决定**,可以用控制信号精确调节 → 容易做可编程滤波器。集成电路里常见,如 MAX7400、MF10 等。

#### 数字滤波器

**把模拟信号采样成数字,用处理器做数学运算滤波,再还原成模拟**。

优势:

- 精度由计算决定,可以做无限高阶
- 参数可以**动态改变**,甚至用算法自适应
- 线性相位(FIR 滤波器)容易实现
- 不受元件公差、温漂影响

代价:

- 需要 ADC + 处理器 + DAC
- 有采样延时
- 高频受限于采样率

**现代主流信号处理是数字滤波**。模拟滤波器主要用在:**采样前的抗混叠滤波**、**采样后的平滑滤波**、**高频场合(GHz)不适合数字的场合**。

这是模电到数电/DSP 的桥梁——不同工具的分工。

### 15.14 这一课你必须带走的东西

1. **四大滤波族**:低通、高通、带通、带阻。所有实际滤波器都是理想形状的**近似**。
2. **关键指标**:截止频率、阶数(决定过渡带陡度)、通带平坦度、阻带衰减、相位特性。
3. **有源滤波器相对无源的优势**:
    - 有增益
    - 输出阻抗低,级间不相互影响
    - 低频不需要电感
4. **Sallen-Key** 是经典二阶拓扑,ω0ω0​ 决定截止频率,**QQ 决定通带形状**:Q=0.707Q=0.707 巴特沃斯,Q>0.707Q>0.707 有峰,Q→∞Q→∞ 振荡。
5. **三大经典家族的权衡**:
    - **巴特沃斯**:通带最平,通用首选
    - **切比雪夫**:过渡带陡,通带有纹波
    - **贝塞尔**:相位线性,波形保真,过渡最缓
    - **椭圆**:过渡最陡,但相位差
6. **高阶 = 二阶级联**,但不同级的 ω0ω0​、QQ 按家族规则各不相同,需查表或用设计工具。
7. **陷波滤波器** 去除特定频率干扰(如 50 Hz 工频)。
8. **现代滤波的边界**:高频/超高频 → 模拟有源,中频/低频/精确参数 → 数字滤波。模数并存,各有所长。

### 下一课预告:MOSFET 与 CMOS

到现在为止,所有的放大电路我们都用 BJT。但现实中:

- **手机、电脑、所有数字 IC**:用的都不是 BJT,是 MOSFET
- **现代模拟 IC(运放、电源管理、RF)**:主流也是 MOSFET 或 BiCMOS
- **Intel、TSMC 的 7nm、5nm、3nm 工艺**:指的是 MOS 管的特征尺寸

为什么 MOS 管赢了?它和 BJT 有什么本质区别?怎么分析 MOS 电路?

下一课讲:

- **MOSFET 的物理结构**:NMOS、PMOS,为什么是"**场效应**"管
- **MOS 的三个工作区**:截止、可变电阻区、饱和(放大)区
- MOS 放大器——和 BJT 有什么异同
- **CMOS(互补 MOS)反相器**——数字逻辑的基石,但它也是模拟电路的基础单元
- **电流镜**——集成模拟电路的"万能工具",运放内部的核心模块
- 为什么 MOS 做数字电路几乎没有静态功耗(这是电池设备能存在的关键)

这是**从分立元件模电到集成电路模电**的转折。掌握这一课,你才真正摸到现代模拟 IC 的门。准备好说"继续"。
