# 第 16 课:MOSFET 与 CMOS——现代电子的主角

### 16.1 BJT 为什么"输了"

到现在我们讲的所有放大器都用 BJT。但现实是:

- **所有数字 IC**(CPU、内存、手机芯片):**99.99% 是 MOSFET**
- **现代模拟 IC**(运放、电源管理、ADC/DAC、RF):**主流是 CMOS 或 BiCMOS**
- **3nm、5nm、7nm 工艺**:指的是 MOSFET 的最小尺寸

BJT 几乎只在分立元件、特殊高频/高功率场合存活。为什么?

**BJT 的根本问题**:

1. **基极有电流**(IB=IC/βIB​=IC​/β)——大规模集成时几亿管子的基极电流加起来巨大,功耗不可接受
2. **集成密度受限**——BJT 的物理结构(发射区/基区/集电区垂直堆叠)难以小型化
3. **数字逻辑功耗高**——BJT 的导通需要持续的基极电流,不像 MOS 几乎"零静态功耗"

**MOSFET 的核心优势**:

1. **栅极几乎不消耗电流**——纯电压控制,功耗极低
2. **结构简单**,适合大规模集成,容易微缩
3. **CMOS 数字逻辑静态功耗几乎为零**——这是手机能用电池工作的关键

这一课带你从 BJT 的世界跨入 MOS 的世界。

### 16.2 MOSFET 的物理结构

**MOSFET = Metal-Oxide-Semiconductor Field-Effect Transistor**——**金属-氧化物-半导体场效应管**。

#### NMOS 结构

```
           栅 (G, Gate)
           │
        ───┴───  ← 多晶硅栅(过去是金属)
       │       │
    ───┤◤◢◤◢◤◢├───  ← 二氧化硅 SiO₂ 绝缘层(几纳米厚)
       │       │
    ───●───────●───
    │  N+  P衬底  N+ │   
    │ 源 ─沟道─ 漏 │     ← 衬底是 P 型,源漏是 N+
    │      │       │
    └──────┼───────┘
           │
           衬底 B(Body)
   
   引脚:G(栅)、D(漏)、S(源)、B(衬底,通常接最低电位)
```

**关键结构特征**:

- **栅极 G**:一片导电材料(早期是金属,现代是重掺杂多晶硅),**与下面的硅完全绝缘**——**栅极电流 = 0**(理想)
- **源 S 和漏 D**:两个 N+ 高掺杂区,埋在 P 型衬底里
- **源漏之间是 P 型衬底**——正常情况下两个 N+ 区被 P 衬底隔开,**不导通**
- **栅极下方是关键区域**——施加正电压时,这里会形成"沟道"

#### 与 BJT 的核心区别

||BJT|MOSFET|
|---|---|---|
|控制方式|**电流**控制(IBIB​)|**电压**控制(VGSVGS​)|
|输入电流|有(IBIB​,小但非零)|**几乎为 0**(只有漏电流)|
|控制原理|PN 结注入|**栅极电场感应沟道**|
|载流子|电子+空穴(双极)|单一(NMOS 用电子,PMOS 用空穴)|

**MOS 是"场效应"——栅极不接触沟道,只通过电场远程控制**。这是它名字"Field-Effect"的由来。

### 16.3 MOSFET 的工作原理

#### 三个工作区(NMOS)

**变量约定**:

- VGSVGS​:栅源电压
- VDSVDS​:漏源电压
- VTHVTH​(或 VTVT​,注意不要和热电压混):**阈值电压**(典型 0.5~1V)

##### 区域 1:截止区(VGS<VTHVGS​<VTH​)

栅极电压不够,**栅极下方还没形成沟道**,源漏之间被 P 衬底隔开,**不导通**:

ID=0ID​=0

相当于一个开关"断开"。

##### 区域 2:可变电阻区/三极管区(线性区)

条件:VGS>VTHVGS​>VTH​ 且 VDS<VGS−VTHVDS​<VGS​−VTH​

栅极电压足够高,把 P 型衬底里的少数载流子(电子)**吸引到表面**,在源漏之间形成一条**N 型反型层(沟道)**——源漏导通。

此时 VDSVDS​ 较小,沟道是均匀的,**MOS 像一个"电压控制的可变电阻"**:

ID=μnCoxWL[(VGS−VTH)VDS−VDS22]ID​=μn​Cox​LW​[(VGS​−VTH​)VDS​−2VDS2​​]

参数说明:

- μnμn​:电子迁移率
- CoxCox​:单位面积栅氧化层电容
- W/LW/L:沟道宽长比(版图设计的关键参数)

**VDSVDS​ 很小时简化**:

ID≈μnCoxWL(VGS−VTH)VDSID​≈μn​Cox​LW​(VGS​−VTH​)VDS​

——**IDID​ 正比于 VDSVDS​,沟道导通电阻 ∝1/(VGS−VTH)∝1/(VGS​−VTH​)**,可以通过调节 VGSVGS​ 改变电阻。

**应用**:**MOS 作开关**(数字电路)、模拟开关、电压控制电阻。

##### 区域 3:饱和区(放大区)

条件:VGS>VTHVGS​>VTH​ 且 VDS>VGS−VTHVDS​>VGS​−VTH​

VDSVDS​ 增大到一定程度,**沟道在漏端被"夹断"**——VGD=VGS−VDS<VTHVGD​=VGS​−VDS​<VTH​,漏端不再有沟道。

但电子仍在沟道里加速,被夹断点处的强电场扫到漏极,IDID​ 几乎不再随 VDSVDS​ 增加:

ID=12μnCoxWL(VGS−VTH)2ID​=21​μn​Cox​LW​(VGS​−VTH​)2​

这就是**MOSFET 的"放大方程"**,类比 BJT 的 IC=βIBIC​=βIB​。

**注意**:

- IDID​ 与 VGS−VTHVGS​−VTH​ 是**平方关系**(BJT 是指数关系)
- IDID​ 几乎不依赖 VDSVDS​——和 BJT 放大区的水平特性曲线类似
- VGS−VTHVGS​−VTH​ 叫**过驱动电压(overdrive)**,记为 VOVVOV​,决定 MOS 的"工作强度"

#### 输出特性曲线

```
   I_D ↑
       │           饱和区(放大区)
       │       ╱──────────── V_GS = V_TH+1.5V
       │      ╱
       │     ╱──────────── V_GS = V_TH+1.0V
       │    ╱
       │   ╱──────────── V_GS = V_TH+0.5V
       │  ╱│
       │ ╱ │ 可变电阻区   
       │╱ │              
      ─┴───┴───────────────→ V_DS
      0  V_GS-V_TH (夹断点)
```

左半边:可变电阻区(类似 BJT 的饱和区) 右半边:饱和区(类似 BJT 的放大区) **注意命名陷阱**:**MOS 的"饱和区" = BJT 的"放大区"**。模电学到这里要在心里调整这个翻译。

### 16.4 NMOS 与 PMOS——互补对偶

#### NMOS

刚才讲的就是 NMOS:

- 衬底 P 型,源漏 N+
- VGS>VTH>0VGS​>VTH​>0 时导通
- 沟道里**电子**导电
- VDSVDS​ 为正(DD 接高电位)

#### PMOS

**完全镜像**:

- 衬底 N 型,源漏 P+
- VGS<VTH<0VGS​<VTH​<0 时导通(注意 VTHVTH​ 对 PMOS 来说是负值)
- 沟道里**空穴**导电
- VDSVDS​ 为负(DD 接低电位)

电路符号上,PMOS 在栅极加一个圆圈表示"反相控制"。

**NMOS 和 PMOS 的互补性是 CMOS 的基础**——两者搭配,相互成就。

### 16.5 MOSFET 的小信号模型

类比 BJT 第 6 课。把 MOS 在 Q 点处线性化:

#### 跨导 gmgm​

对饱和区方程求导:

gm=∂ID∂VGS∣Q=μnCoxWL(VGS−VTH)=2μnCoxWLIDgm​=∂VGS​∂ID​​​Q​=μn​Cox​LW​(VGS​−VTH​)=2μn​Cox​LW​ID​​

也可写为:

gm=2IDVGS−VTH=2IDVOVgm​=VGS​−VTH​2ID​​=VOV​2ID​​

**关键对比**:

- BJT:gm=IC/VTgm​=IC​/VT​,VT≈26VT​≈26 mV(常数)
- MOS:gm=2ID/VOVgm​=2ID​/VOV​,VOVVOV​ 几百 mV(设计可调)

**同样的电流下,BJT 的 gmgm​ 比 MOS 大得多**(因为 VT≪VOVVT​≪VOV​)——这是 MOS 的一个**先天缺点**:跨导效率低。

弥补方法:**做大 W/L,即用大尺寸管子**——这也是为什么模拟 CMOS 电路里管子常常很大,而数字电路追求小管子。

#### 输入阻抗 rgsrgs​

栅极绝缘,理想为 ∞∞。实际有微小漏电流,但通常忽略——**MOS 输入阻抗几乎无穷大**。

这是 MOS 相对 BJT 的一个**重大优势**:接什么阻抗源都不"加载"。

#### 输出阻抗 rdsrds​

实际中 IDID​ 还是会随 VDSVDS​ 缓慢增加(**沟道长度调制效应**),用一个参数 λλ 描述:

ID=12μnCoxWLVOV2(1+λVDS)ID​=21​μn​Cox​LW​VOV2​(1+λVDS​)

输出电阻:

rds=1λIDrds​=λID​1​

典型 rdsrds​ 几十 kΩ 到几 MΩ。

#### 小信号等效电路

```
      G ●───────────────────● D
                              │
                              │
                            [g_m·v_gs]   ↓
                              │
                            [r_ds]
                              │
                              │
      S ●───────────────────●
```

**和 BJT 模型几乎一样**,差别就是没有 rberbe​(栅极绝缘,断开)。

### 16.6 MOSFET 放大器——共源、共漏、共栅

完全对应 BJT 的共射、共集、共基:

|MOS|BJT|
|---|---|
|共源(CS)|共射(CE)|
|共漏(CD,**源极跟随器**)|共集(CC,**射极跟随器**)|
|共栅(CG)|共基(CB)|

#### 共源放大器

```
                V_DD
                 │
                [R_D]
                 │
                 ●──── v_o
                 │
              ──┤ D
        v_i ──┤G       NMOS
              ──┤ S
                 │
                ─┴─ 地  (有时串 R_S 然后旁路 C_S)
```

电压增益:

Av=−gmRDAv​=−gm​RD​

**和共射几乎一样**——负号表示反相,增益值由 gmgm​ 和负载电阻决定。

**主要差异**:

- 输入阻抗**极高**(MOS 栅绝缘),不像共射的几 kΩ
- 同样电流下,gmgm​ 比 BJT 小,所以增益**普遍比 BJT 共射略低**

#### 源极跟随器(共漏)

类比射极跟随器:**电压增益 ≈ 1,输入阻抗极高,输出阻抗低**——更完美的缓冲器。

由于 MOS 输入阻抗本身就极高,源极跟随器作为缓冲特别理想——不需要前级电流,完全的"电压采样"。

#### 共栅

类比共基,高频特性好,与共源组合成 **MOS Cascode**——用于精密模拟设计。

### 16.7 CMOS——数字逻辑的基石

#### CMOS 反相器(Inverter)

**最简单的 CMOS 电路:一个 PMOS + 一个 NMOS**:

```
           V_DD
            │
        ────┤P  ← PMOS
            │
            ●──── 输出 v_o
            │
        ────┤N  ← NMOS
            │
           地
   
   两个栅极连在一起 ── 输入 v_i
```

#### 工作原理

**输入 = 高(V_DD)**:

- NMOS:VGS,N=VDD>VTHVGS,N​=VDD​>VTH​ → **导通**
- PMOS:VGS,P=0−VDD=−VDDVGS,P​=0−VDD​=−VDD​ < VTH,PVTH,P​(负值) → **导通?** 等等

让我重画。PMOS 的源接 VDDVDD​,栅是输入:

- 输入高:PMOS 的 VGS,P=Vin−VDD=0VGS,P​=Vin​−VDD​=0 → **截止**
- 同时 NMOS 导通 → 输出被 NMOS 拉到地 → vo=0vo​=0(低)

**输入 = 低**:

- NMOS:VGS,N=0<VTHVGS,N​=0<VTH​ → **截止**
- PMOS:VGS,P=−VDDVGS,P​=−VDD​,绝对值大于 ∣VTH,P∣∣VTH,P​∣ → **导通**
- 输出被 PMOS 拉到 VDDVDD​ → vo=VDDvo​=VDD​(高)

**总结**:输入高 → 输出低,输入低 → 输出高——**反相器**(NOT 门)。

#### CMOS 的精妙——零静态功耗

**关键观察**:无论输入是高还是低,**总有一个管子截止**——电流路径被切断,**静态电流 = 0**。

只有在输入**翻转的瞬间**,两个管子都短暂导通,有一小段电流——这是**动态功耗**,正比于翻转频率。

**结论**:CMOS 静态功耗 ≈ 0,动态功耗 ∝f⋅C⋅VDD2∝f⋅C⋅VDD2​。

**这就是手机能用电池工作的根本原因**——大量逻辑门,只有正在翻转的少数管子在耗电。如果用 BJT 做同样的逻辑(早期的 TTL 工艺),功耗会高几个数量级,根本不可能集成上亿管子。

#### 从反相器到一切

把反相器扩展:

- **NAND**:两个 NMOS 串联到地,两个 PMOS 并联到 VDDVDD​
- **NOR**:两个 NMOS 并联到地,两个 PMOS 串联到 VDDVDD​
- **复杂逻辑**:任何布尔函数都能用 PMOS 上拉 + NMOS 下拉网络实现

**CPU、GPU、内存的全部数字逻辑——本质是几亿个 CMOS 反相器及其变体的网络**。

### 16.8 电流镜——模拟集成电路的"万能工具"

到这里我们准备进入第二个核心概念——**电流镜**。这是所有现代模拟 IC 的核心积木。

#### 问题:集成电路里怎么提供恒定电流?

分立电路里恒流源可以用电池+电阻或专门的恒流二极管。集成电路里:

- **大电阻又大又费版图**——10 kΩ 在硅片上占巨大面积
- **要给 100 个放大器各提供 100 μA 偏置**——不能放 100 个独立恒流源

**解决方案:电流镜——一个参考电流"复制"到多个地方**。

#### 基本电流镜结构(BJT 版本)

```
              V_CC
               │
              [R]   ← 设定参考电流
               │
               ●────────●
               │        │
              ─┤ T1    ─┤ T2  ← T1、T2 完全相同
            ┌─┤ B    ┌─┤ B
            │  │     │  │
            └──┴─────┘  │
               │        │
               E        E
               │        │
              ─┴─      ─┴─
              地        地
   
   T1 的 B 和 C 短接 ── 起"模板"作用
   T2 与 T1 共用同样的 V_BE ── 输出电流 = 参考电流
```

#### 工作原理

- T1 接成"二极管接法"(BB 和 CC 短接),IREFIREF​ 流过 T1,**确定它的 VBEVBE​**
- T2 与 T1 **完全匹配**(同一硅片,相邻位置,工艺一致),**共用同一个 VBEVBE​**
- 由 BJT 的 IC=ISeVBE/VTIC​=IS​eVBE​/VT​ 关系,**IC,T2=IC,T1=IREFIC,T2​=IC,T1​=IREF​**

**输出电流 = 参考电流**——T2 像一个"镜像",复制了 T1 里的电流。

#### 电流镜的关键性质

1. **输出阻抗高**:T2 工作在放大区,roro​ 几十 kΩ,接近理想恒流源
2. **可"镜像放大"**:让 T2 的发射区面积是 T1 的 N 倍,Iout=N⋅IREFIout​=N⋅IREF​
3. **多输出**:并联多个 T2',T2''……同时给多个电路供电
4. **节省版图**:一个 RR(可以做大)就能给整个芯片提供基准

#### MOS 电流镜

完全类似,把 BJT 换成 NMOS:

```
              V_DD
               │
              [R]
               │
               ●────────●
               │        │
             ──┤        ├──   两个 NMOS 共用 V_GS
        ──┐ ──┤M1     M2├── ──┐
        │  │  │        │  │  │
        └──┴──┘        └──┴──┘
               │        │
              地       (输出端)
   
   M1 的 G 和 D 短接
   M2 复制 M1 的电流
```

通过调节 W/LW/L 比例,可以镜像放大或缩小:

Iout=(W/L)M2(W/L)M1⋅IREFIout​=(W/L)M1​(W/L)M2​​⋅IREF​

#### 电流镜在运放里

第 8 课讲过差分放大器需要"尾电流源"——**就是用电流镜实现的**。

差放的有源负载(把第 8 课最后提到的"用电流源代替集电极电阻")**也是电流镜**。

可以说:**没有电流镜就没有现代集成运放**。运放内部从输入级偏置、有源负载、到输出级,全是各种电流镜的变种。

### 16.9 MOS 与 BJT 的对比——什么时候用什么?

||BJT|MOSFET|
|---|---|---|
|跨导效率|**高**(gm=IC/VTgm​=IC​/VT​)|低(gm=2ID/VOVgm​=2ID​/VOV​)|
|输入阻抗|中(rberbe​)|**极高**(栅绝缘)|
|噪声特性|**低频噪声小**|低频 1/f 噪声大|
|速度|**高频快**(BJT 高频管 fTfT​ 几十 GHz)|取决于工艺(7nm 工艺 fTfT​ 数百 GHz)|
|静态功耗(数字)|高|**极低**|
|集成密度|低|**极高**|
|工艺成熟度|成熟,但工艺停滞|**持续微缩**(摩尔定律)|
|大功率|**大功率传统强势**|现代 MOS 也能做几百 W|

**现代选择**:

- **数字 IC**:几乎全 CMOS
- **大规模模拟 IC**(运放、ADC、DAC、电源、RF):主流 CMOS,高性能用 BiCMOS(BJT + CMOS 混合)
- **离散功放、高频 RF**:BJT 仍有优势,特别是低噪声放大器(LNA)
- **超高频微波**:GaAs、GaN 等化合物半导体,但物理原理类似

### 16.10 这一课你必须带走的东西

1. **MOS 是场效应管**:栅极绝缘,**电压控制**,栅极电流 ≈ 0——这是 MOS 相对 BJT 的根本优势。
2. **MOS 三个工作区**:
    
    - 截止(VGS<VTHVGS​<VTH​):ID=0ID​=0
    - 可变电阻区(VDS<VOVVDS​<VOV​):像电压控制电阻
    - 饱和区(VDS>VOVVDS​>VOV​):**这里才能放大**,ID=12μCox(W/L)VOV2ID​=21​μCox​(W/L)VOV2​
    
    **命名陷阱**:MOS 的"饱和"= BJT 的"放大"。
3. **MOS 小信号参数**:gm=2ID/VOVgm​=2ID​/VOV​,**比 BJT 同电流下小**——MOS 跨导效率较低,要靠大 W/LW/L 弥补。
4. **MOS 放大器三种组态(共源/共漏/共栅)** 完全对应 BJT 的(共射/共集/共基),**分析方法和公式形式几乎一样**。
5. **CMOS 反相器**:NMOS + PMOS 互补,**静态功耗 ≈ 0**,这是手机/便携设备能存在的根本原因。**所有数字逻辑都建立在 CMOS 反相器之上**。
6. **电流镜**:集成电路的"万能工具",一个参考电流复制到多处。运放内部全是电流镜。
7. **现代选择**:数字 = CMOS,大规模模拟 = CMOS 或 BiCMOS,高频/低噪声 = BJT 或化合物半导体。
8. **学到这里,你已经站在分立元件电路和集成电路设计的分水岭上**。前面 15 课几乎全用 BJT,你看到的是经典模电;这一课打开了现代 IC 设计的入口。
