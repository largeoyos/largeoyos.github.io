# 第 17 课:模拟集成电路设计入门

到这里,你已经掌握了模电的全部核心工具:BJT、MOS、差放、反馈、频率响应、运放、滤波器、电流镜……

这一课不是新概念的堆砌,而是**把这些工具组合起来,看真实的集成电路是怎么设计的**。当你看完这一课,再去看一颗运放或电源芯片的 datasheet 内部框图,你应该能认出每一块在干什么。

### 17.1 集成电路设计的"思维转变"

分立元件电路和集成电路设计在思维方式上有几个关键差异,先讲清楚:

#### 1. 电阻贵,管子便宜

分立元件:**电阻最便宜**($1 美分),晶体管贵($10 美分)。 集成电路:**完全相反**——一个 10 kΩ 电阻占的硅片面积远大于一个晶体管,而硅片面积 = 钱。

**结果**:IC 设计师**用晶体管代替电阻**——能用电流镜的地方绝不用电阻,能用 MOS 当电阻的地方绝不用真电阻。这是为什么集成运放电路图里**全是管子**。

#### 2. 绝对值不准,相对值很准

集成电路工艺:

- 单个电阻的绝对值误差 ±20% 是常态
- 但**两个紧挨着的电阻之间的比值,误差可以小于 0.1%**

**结果**:IC 设计**避免依赖绝对值**,**充分利用比值匹配**。电流镜就是典型——输出电流由 W/LW/L 比值决定,绝对值不重要。

#### 3. 温度对所有元件相同

整片硅片温度一致——这是差分对、电流镜能工作的根本前提。

#### 4. 不能用大电感、大电容

电感几乎做不出来(低频下需要的尺寸太大),电容也只能做小(几 pF 到几十 pF)。

**结果**:IC 里的滤波几乎都是 RC 或开关电容,极少 LC。需要大电容时只能"外挂"——这就是芯片为什么有"补偿电容"的外接引脚。

#### 第一级:差分对 + 镜像负载

这是第 8 课的差放 + 第 16 课的电流镜的组合。

- **M1、M2 差分对**:接受输入信号 v+v+​、v−v−​
- **M9 尾电流源**:用电流镜从基准电流复制,提供恒定 ItailItail​
- **M3、M4 镜像负载**:**关键创新**——不再用电阻 RDRD​ 作负载,而是用 PMOS 电流镜

**为什么用电流镜做负载?** 第 8 课提过——"有源负载"。

普通电阻 RDRD​ 几 kΩ,差放增益 gmRDgm​RD​ 顶多几十倍。 **电流镜的输出电阻 roro​ 几百 kΩ 到 MΩ**——增益 gmrogm​ro​ 可达几千甚至上万倍。

**第一级增益**:

Av1=−gm1(ro2∥ro4)Av1​=−gm1​(ro2​∥ro4​)

典型值 100~1000 倍。

#### 第二级:共源放大 + 米勒补偿

- **M6**:N 沟道共源,放大第一级输出
- **M7**:P 沟道电流镜负载

**第二级增益**:

Av2=−gm6(ro6∥ro7)Av2​=−gm6​(ro6​∥ro7​)

典型 100~500 倍。

**总增益**:Av1×Av2≈104∼106Av1​×Av2​≈104∼106——经典运放开环增益。

#### 米勒补偿电容 CcCc​

**反馈稳定性**(第 11 课)的工程实现。

两级运放有两个极点,在用作负反馈放大器时容易自激。CcCc​ 接在第二级共源管的栅漏之间——**密勒效应**(第 12 课):从第二级输入端看,CcCc​ 等效成 (1+∣Av2∣)Cc(1+∣Av2​∣)Cc​,产生一个**很低频的主极点**。

效果:在 ∣AF∣=1∣AF∣=1 处之前,环路相位还远未达 180°,**相位裕量充足,系统稳定**。

**这是密勒效应的"反派变英雄"**——共射/共源放大器的高频性能杀手,在两级运放里被故意利用,变成稳定性的救星。

#### 实际芯片

经典 μA741(BJT 实现)、CA3140(JFET 输入 + BJT)、TL082(JFET 输入 + 双极)、LM358(双极)、MCP6V81(CMOS 自动校零)——内部结构都是上面这个套路的变种。

### 17.3 基准电压源——一个被忽视的英雄

#### 为什么需要"基准"

ADC 把模拟电压转成数字,需要一个"标尺"——0V 对应 0,VREFVREF​ 对应满刻度。**VREFVREF​ 不准,所有测量都不准**。

类似地:

- DAC:把数字数转成模拟电压,需要 VREFVREF​
- 比较器:阈值要稳定
- 电源芯片:输出电压由内部基准决定

**要求**:

- 不随**电源电压变化**(电源 5V 变 4.8V,基准纹丝不动)
- 不随**温度变化**(芯片从室温到 70°C,基准变动 < 0.01%)
- 长期**老化稳定**

#### 难点

**单一物理量很难同时不依赖电源和温度**:

- 电阻分压:依赖电源
- 齐纳二极管:温度系数大
- VBEVBE​:温度系数 −2 mV/°C(负)
- 热电压 VT=kT/qVT​=kT/q:温度系数 +0.085 mV/°C(正)

但是**最后两条加在一起,可能正好抵消**——这就是带隙基准的核心想法。

#### 带隙基准(Bandgap Reference)

**1971 年 Robert Widlar 发明**——模拟 IC 设计史上的里程碑。

**原理**:把两个温度系数相反的电压**按权重相加**,使总温度系数为 0:

VREF=VBE+K⋅VTVREF​=VBE​+K⋅VT​

- VBEVBE​:负温度系数 ≈ −2 mV/°C
- VTVT​:正温度系数 ≈ +0.085 mV/°C
- 选 K≈23K≈23,使两者相加的温度系数为 0

**最终 VREFVREF​ 约 1.25 V**——非常接近硅的禁带宽度(1.12 eV),所以叫"**带隙**"基准。

#### 怎么得到 VTVT​

利用**两个工作在不同电流密度的 BJT 之间的 VBEVBE​ 差值**。

由 VBE=VTln⁡(IC/IS)VBE​=VT​ln(IC​/IS​):

ΔVBE=VBE1−VBE2=VTln⁡(IC1IC2)ΔVBE​=VBE1​−VBE2​=VT​ln(IC2​IC1​​)

**这个差值正比于 VTVT​**,即正比于绝对温度——叫 **PTAT(Proportional To Absolute Temperature)** 电压。

通过电阻把 ΔVBEΔVBE​ 放大 KK 倍,加到一个 VBEVBE​ 上,就得到带隙基准。

#### 经典电路(简化示意)

```
              V_CC
               │
          ┌────●────┐
          │         │
         [I1]      [I2 = N·I1]   (用电流镜得到不同电流)
          │         │
          ●         ●
       Q1│        Q2│   面积比 1 : N
          │         │
       V_BE1     V_BE2
          │         │
         地       [R_1]
                    │
                  [R_2]
                    │
                   地

    输出 V_REF = V_BE1 + (R_1+R_2)/R_2 × ΔV_BE
              ≈ V_BE + 23 V_T ≈ 1.25V
```

实际电路用反馈把 VBE1VBE1​ 和 VBE2VBE2​ 之差精确地建立在 R1R1​ 上,具体推导省略——核心思想就是上面那个抵消。

#### 性能

好的带隙基准:

- 温度系数:几 ppm/°C(0.0001%/°C)
- 电源抑制比:80 dB+
- 长期漂移:几十 ppm/年

经典芯片:LM385(2.5V)、LT1009、MAX6126……

带隙基准嵌在几乎每一颗 ADC、DAC、电源 IC 里。**是最普及但最不被注意的电路**。

### 17.4 比较器与模数转换

#### 比较器 = 没有反馈的运放

比较器的功能:输出告诉你 v+v+​ 和 v−v−​ 哪个大。

vo={VHv+>v−VLv+<v−vo​={VH​VL​​v+​>v−​v+​<v−​​

第 9 课讲过,运放开环增益极大,稍有差异就饱和——**这就是比较器**。

但**专用比较器芯片**(如 LM311、LM393、LMV7235)和运放有重要区别:

- **响应快**:不带补偿电容,几十 ns 响应(普通运放 μs 级)
- **输出可推灌**(open collector / open drain),便于驱动数字逻辑
- **不用作线性放大**,所以稳定性问题不存在

#### 施密特触发器(Schmitt Trigger)——抗噪声比较

普通比较器问题:**输入信号在阈值附近抖动时,输出反复跳变**——产生大量虚假翻转。

**解决**:**正反馈引入双阈值**(回差)。

```
                        R_f
              ┌────────[R_f]────────┐
              │                      │
              │                      │
    v_i ●──── v_−                    │
                 │\                  │
                 │ \                 │
                 │  >────────────────●── v_o
                 │  /
                 │ /
              ●──┘
              │
             [R_1]
              │
             地     (v_+ 接到 R_1 和 R_f 中点,
                     被输出反馈"自我加强")
```

工作原理:

- vovo​ 高时,v+=VH⋅R1/(R1+Rf)>0v+​=VH​⋅R1​/(R1​+Rf​)>0 → 阈值偏高,**抗输入下行干扰**
- vovo​ 低时,v+=VL⋅R1/(R1+Rf)<0v+​=VL​⋅R1​/(R1​+Rf​)<0 → 阈值偏低,**抗输入上行干扰**

输入电压必须**穿过更宽的范围**才能让输出翻转,中间的小抖动被忽略——**回差(hysteresis)**。

应用:

- 数字电路输入端:噪声抑制(74HC14 等芯片)
- 比较器去抖:按键去抖
- 弛豫振荡器:第 14 课的方波发生器内部就是施密特结构

#### 模数转换器(ADC)概览

把模拟电压变成数字数。三种主流架构:

##### Flash ADC(并行 ADC)

**最快,最贵**。

n 位 Flash ADC = 2n−12n−1 个比较器并联,每个的阈值由分压电阻产生。一次比较出全部位。

```
   v_i ──┬── 比较器1 (阈值 = V_REF·1/8)
         ├── 比较器2 (阈值 = V_REF·2/8)
         ├── 比较器3 (阈值 = V_REF·3/8)
         ...                                → 编码器 → 数字输出
         └── 比较器7 (阈值 = V_REF·7/8)
```

8 位需要 255 个比较器,功耗大、面积大。但速度快(单比较周期),**用于 GHz 采样的高速 ADC**(示波器、雷达)。

##### 逐次逼近 ADC(SAR)

**主流通用**,速度中等(MSPS 级),精度高(12~18 位)。

工作流程(二分搜索):

1. 第一步:猜中间值 VREF/2VREF​/2,与 vivi​ 比较 → 决定最高位 MSB
2. 第二步:根据 MSB,猜上半区或下半区的中间 → 第二位
3. 重复 n 次 → 得到 n 位结果

类似猜数字游戏,nn 个时钟周期完成 nn 位转换。

**用一个比较器 + 一个 DAC + 控制逻辑**,效率极高。

应用:工业测量、电池监控、传感器接口——**最常见的 ADC 类型**。

##### Δ-Σ(Delta-Sigma)ADC

**最高精度**(24 位),速度慢(kHz 级)。

工作原理(简述):

- 用极高的采样率(几 MHz)对信号过采样
- 利用反馈把量化噪声推到高频
- 用数字滤波器去除高频噪声,提取低频信号

适合**音频(24 位)、精密测量、称重传感器**。

#### DAC——反向操作

DAC 把数字数变模拟电压。常见结构:

- **电阻分压**:每位对应一个电阻分支
- **R-2R 梯形网络**:用两种电阻值即可实现任意位数,集成性好
- **电流舵 DAC**:高速场合(视频 DAC、RF DAC)
- **Δ-Σ DAC**:高精度音频(数字音频播放器)

### 17.5 电源管理 IC

电池给的电压不稳定(锂电池 3.0~4.2V),芯片要稳定的 1.8V、3.3V、5V——需要电源管理。

#### 线性稳压器(LDO)

**Low Dropout Regulator**——最简单的稳压器。

```
          V_in
           │
        ──┤ M1 (功率管,P 沟道 MOS 或 PNP BJT)
        ──┤
           │
           ●─── V_out ── 负载
           │
       ┌───┤
       │   │
      [R_1]│
       │   │   反馈到运放,与 V_REF 比较
       │   │   形成负反馈,稳定 V_out
      [R_2]│
       │   │
       └───┘
       
    [V_REF]──● 运放 ●──→ 控制 M1 的栅
             │         │
             └─ 反馈 ──┘
```

**原理**:运放比较反馈电压和 VREFVREF​,调节功率管的导通程度,使 VoutVout​ 等于 VREF⋅(R1+R2)/R2VREF​⋅(R1​+R2​)/R2​。

**优点**:

- 输出干净(无开关纹波)
- 设计简单
- 响应快

**缺点**:

- **效率 = Vout/VinVout​/Vin​**——电压差全部变成功率管发热
- 5V 转 1.8V:效率只有 36%,大部分功率被浪费成热

**应用**:小电流(< 1A)、对噪声敏感(模拟电路供电、射频供电)、电压差不大的场合。

经典芯片:LM7805、AMS1117、TPS7A47……

#### 开关电源(DC-DC Converter)

利用**电感储能**和**晶体管开关**,实现**高效率电压变换**。

##### Buck(降压)电路

```
                   L
   V_in ──┤────●──[L]──●─── V_out ── 负载
          │   │         │
          │   │        [C]
          │   │         │
       开关  二极管     地
       (MOS) (或同步MOS)
          │   │
         地  地
```

**工作原理**:

- 开关闭合:VinVin​ 通过电感给负载和 CC 供电,电感储能
- 开关断开:电感释放能量,通过二极管(或同步整流 MOS)继续给负载供电

通过控制开关的**占空比 D**,平均输出电压:

Vout=D⋅VinVout​=D⋅Vin​

**理论效率 100%**(开关无损耗),实际 90%+。

**降压、升压、升降压、隔离(反激/正激)**——同样原理的不同拓扑。

##### 关键控制

开关电源芯片内部:振荡器(几百 kHz 到几 MHz)+ PWM 控制器 + 误差放大器 + 反馈环路 + 过流过压保护。

经典芯片:LM2596(降压)、MP1584(降压)、MT3608(升压)、TPS61040(升压)、LM5117(同步)……

#### LDO vs 开关电源选择

||LDO|开关电源|
|---|---|---|
|效率|低(Vout/VinVout​/Vin​)|高(85-95%)|
|噪声|极低|有开关纹波|
|复杂度|低|高(电感 + 控制)|
|成本|低|高|
|体积|小|较大|
|响应速度|快|慢|

**典型实践**:电池 → 高效降压(开关电源)→ 中间电压 → LDO → 干净电压给敏感电路。两者结合用。

### 17.6 这一课你必须带走的东西

1. **IC 设计的思维**:用管子代替电阻,**靠匹配和比值,不靠绝对值**,温度对所有元件相同。
2. **两级 CMOS 运放架构**:差分对 + 镜像负载(第一级)→ 共源 + 米勒补偿(第二级)。前面所有概念在这里都用到——差放、电流镜、有源负载、密勒效应、负反馈稳定性。
3. **米勒效应的两面**:共源高频性能的杀手 vs 两级运放稳定性的救星。同一物理现象,在不同工程目标下意义相反。
4. **带隙基准**是 IC 里最普遍的"无名英雄"。原理:**VBEVBE​(负温度系数) + K⋅VTK⋅VT​(正温度系数)** 抵消,得到 ≈ 1.25V 的不随温度变化的基准。所有 ADC、DAC、电源芯片内部都有带隙基准。
5. **比较器**:开环运放,但优化了速度。**施密特触发器** 用正反馈引入回差,抗噪声。
6. **三种 ADC 架构**:
    - Flash:最快最贵,GHz 级采样
    - SAR:速度精度平衡,最常用
    - Δ-Σ:最高精度,慢,适合音频/称重
7. **两类电源管理**:
    - LDO:线性,效率 = Vout/VinVout​/Vin​,适合小电流和敏感电路
    - 开关电源:利用电感,效率 90%+,适合大电流和电池供电
8. **从这一课开始,你不再是模电学习者,而是 IC 设计的入门者**——你能看懂芯片厂商的内部框图,理解他们在用什么手段实现什么指标。
