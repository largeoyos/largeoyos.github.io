# 第 1 课:电压、电流、功率——比你想象的更深

### 1.1 电荷是起点

一切电学现象的源头是**电荷**。电子带负电,质子带正电,电荷之间通过**电场**相互作用。

一个电荷 qq 处在电场中,就具有**电势能**——就像一个物体处在重力场中具有重力势能一样。这个类比非常重要,我们后面会反复用。

### 1.2 电压 = 单位电荷的势能差

**电压(电位差)的本质定义**:

UAB=WABqUAB​=qWAB​​

意思是:把一个电荷 qq 从 A 点移动到 B 点,电场力做的功是 WABWAB​ ,那么 A、B 两点之间的电压就是"每单位电荷的能量差"。

单位:1 伏特 = 1 焦耳 / 1 库仑。

**关键理解**:

- 电压是"**两点之间**"的量,说"A 点的电压"其实是默认了一个参考点(通常是地)
- 电压描述的是**能量差**,不是能量本身
- 电压的存在**不依赖于电流**——一节电池放在桌上没接任何东西,它两端照样有 1.5V 电压

用水的类比:电压 ≈ **两点之间的水位差**(高度差)。水位差存在,水**不一定**在流;但水要流,必须有水位差。

### 1.3 电流 = 电荷的流动速率

I=dqdtI=dtdq​

单位:1 安培 = 1 库仑 / 秒。

**关键理解**:

- 电流是**一个时刻、一个截面上**流过的电荷速率
- 电流方向规定为**正电荷流动的方向**(这是历史遗留,实际上金属中流动的是电子,方向相反——你就记住这个约定即可)
- 电流是"穿过"一个点/截面的量,电压是"两点之间"的量——**这个区别很重要**

用水的类比:电流 ≈ **水管某个截面每秒流过的水量**。

### 1.4 电压与电流的关键区别(很多人模糊的地方)

||电压 UU|电流 II|
|---|---|---|
|描述|两点间的"势差"|一点处电荷的流速|
|数学性质|跨接(across)量|穿过(through)量|
|测量方式|电压表**并联**接在两端|电流表**串联**接在回路中|
|独立存在?|可以独立存在(开路电池)|没有闭合回路就没有稳态电流|

**为什么电压表并联、电流表串联?** 因为电压是两点之间的量,必须同时接触两个点;电流是穿过某处的量,必须让它"穿过"电流表。这不是规定,是由它们的物理定义决定的。

### 1.5 功率 = 电压 × 电流

P=UIP=UI

为什么?推一下就清楚了:

P=dWdt=dWdq⋅dqdt=U⋅IP=dtdW​=dqdW​⋅dtdq​=U⋅I

第一步:功率是能量的时间变化率。 第二步:链式法则。 第三步:dW/dqdW/dq 就是电压(每单位电荷的能量),dq/dtdq/dt 就是电流。

单位:1 瓦特 = 1 伏 × 1 安 = 1 焦耳/秒。

**这个公式之所以重要**,是因为它告诉我们:在电路中,能量的传递速率 = 电压 × 电流。一个元件"消耗"多少能量,就看它两端的电压和流过它的电流的乘积。

### 1.6 关联参考方向(一个坑)

这是中国模电教材特别强调、但学生经常混的地方。

当我们写 P=UIP=UI 时,结果的正负有物理含义:

- **关联参考方向**:电流的参考方向从电压的"+"端流入元件 → P=UIP=UI ,P>0P>0 表示元件**吸收**功率(耗能),P<0P<0 表示**发出**功率(供能)。
- **非关联参考方向**:电流从"−"端流入 → P=−UIP=−UI ,符号反过来。

```
关联参考方向:                非关联参考方向:
                                           
    + ─────┐                     + ─────┐
    │      │   I →               │      │   ← I
    U   [元件]                   U   [元件]
    │      │                     │      │
    − ─────┘                     − ─────┘
    
    P = UI                       P = −UI
```

**为什么需要这个约定?** 因为在复杂电路里,你一开始不知道电流实际往哪流、哪端电位高。你**先假设**一个参考方向,算完之后看符号:正号说明假设对了,负号说明实际方向相反。这是一套让你"不用猜"的工程语言。

记住一句话:**参考方向是人为假设的,真实方向由计算结果的正负决定**。

---

### 1.7 小结(这一课你必须带走的东西)

1. **电压是两点间单位电荷的能量差**,它描述"推动电荷的势"。
2. **电流是电荷流动的速率**,它描述"实际在动多少电荷"。
3. **电压可以独立存在,稳态电流必须有闭合回路**。
4. **功率 P=UIP=UI 来自链式法则**,不是凭空规定。
5. **关联参考方向下 P>0P>0 表示吸收功率**——这是判断一个元件是"电源"还是"负载"的方式。
