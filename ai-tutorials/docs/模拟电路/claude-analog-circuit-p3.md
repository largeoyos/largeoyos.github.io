# 第 2 课:半导体与 PN 结——模拟电路的物理起点

模拟电路所有的"神奇"都来自一类器件:**半导体器件**。二极管、三极管、MOS 管——它们全都建立在**PN 结**这一个物理结构上。不理解 PN 结,后面全是死记硬背。

### 2.1 为什么叫"半导体"

导体(如铜)自由电子多,随便导电;绝缘体(如玻璃)几乎没有自由载流子,不导电。**半导体**(硅 Si、锗 Ge)介于两者之间——它的"导电能力"可以通过**掺杂**和**外加电场**来精确控制。

**这个"可控性"就是整个电子学的根基**。如果没有半导体,你就不可能用一个小信号去控制一个大信号,也就没有放大、没有运算、没有数字逻辑。

### 2.2 本征半导体与载流子

纯净的硅是**本征半导体**。硅原子最外层有 4 个价电子,彼此形成共价键构成晶格。

在绝对零度,所有电子都被束缚,不导电。温度升高,一部分电子获得能量挣脱共价键,成为**自由电子**(带负电),同时在原位置留下一个**空穴**(可以看作带正电的"缺位")。

**关键**:半导体中有**两种载流子**——电子和空穴。这和金属只有电子导电不一样,也是半导体器件能做出各种花样的根本原因。

本征半导体里,电子浓度 nn 和空穴浓度 pp 相等,都等于本征浓度 nini​。室温下硅的 ni≈1.5×1010 cm−3ni​≈1.5×1010 cm−3 ——这个数字很小,所以本征硅导电能力很弱。

### 2.3 掺杂:制造 N 型和 P 型

**N 型半导体**:在硅里掺入 5 价元素(如磷 P)。磷的 4 个价电子和硅成键,**多出 1 个电子几乎是自由的**。这样 N 型材料里电子浓度 ≫≫ 空穴浓度,电子是**多数载流子**。

**P 型半导体**:掺入 3 价元素(如硼 B)。硼少 1 个电子,留下一个空穴。P 型里空穴是多数载流子。

**重要关系**:在热平衡下,n⋅p=ni2n⋅p=ni2​ 始终成立(**质量作用定律**)。掺杂只改变 nn 和 pp 的比例,不改变它们的乘积。

```
N 型硅                     P 型硅
                           
 Si─Si─Si─Si              Si─Si─Si─Si
 │  │  │  │               │  │  │  │
 Si─P─Si─Si   ← 多出电子    Si─B─Si─Si   ← 多出空穴
 │  │• │  │               │  ○│ │  │
 Si─Si─Si─Si              Si─Si─Si─Si
```

### 2.4 PN 结的形成——这里是关键

把 N 型和 P 型半导体**紧密接触**(实际上是在同一块硅上通过不同掺杂工艺形成),接触面就是**PN 结**。

接触瞬间会发生什么?

**第 1 步:扩散**。N 区电子浓度高,向 P 区扩散;P 区空穴浓度高,向 N 区扩散。这是**浓度梯度驱动的扩散**,就像墨水滴入清水。

**第 2 步:形成耗尽区**。电子跑到 P 区后和空穴复合,留下 N 区边上**不可移动的正离子**(磷失去电子变成正离子);空穴跑到 N 区同理留下**负离子**。接触面附近形成一个没有自由载流子、只剩固定离子的区域——**耗尽区(空间电荷区)**。

**第 3 步:产生内建电场**。N 侧正离子、P 侧负离子,形成从 N 指向 P 的**内建电场 E0E0​ **。这个电场**阻止扩散继续**——它把电子往 N 推,把空穴往 P 推。

**第 4 步:达到平衡**。扩散电流(浓度梯度驱动)和漂移电流(内建电场驱动)大小相等、方向相反,净电流为零。此时 PN 结两侧存在一个**内建电势差 VbiVbi​ ** ≈ 0.7V(硅)。

```
         耗尽区
    ┌──────┼──────┐
    │  P   │   N  │
    │ ○ ○ ○│⊕ ⊕ ⊕ │
    │ ○ ○ ⊖│⊕ • • │
    │ ○ ○ ⊖│⊕ • • │   ⊖⊕:不动的离子
    │      │      │    ○:空穴, •:电子
    └──────┼──────┘
           │
      E₀ ←─┤ (内建电场, N→P 方向)
           │
    └──V_bi ≈ 0.7V──┘
```

**⚠️ 容易搞混的点**:虽然 N 区电子多、P 区空穴多,但 N 区整体是**电中性**的——电子和磷离子电荷平衡。只有在**耗尽区内**,因为载流子被扫走了,固定离子才表现出净电荷。

### 2.5 PN 结的单向导电性

现在给 PN 结加外部电压,这才是真正有用的部分。

#### 正向偏置(P 接正极,N 接负极)

外电场方向与内建电场**相反**,削弱内建电场。扩散电流占上风——电子从 N 源源不断注入 P 区,空穴从 P 注入 N 区——**大量电流流过**,且外加电压稍微增加,电流指数式增大。

#### 反向偏置(P 接负极,N 接正极)

外电场与内建电场**同向**,耗尽区加宽,内建电场增强。扩散几乎停止,只剩下极微弱的**反向饱和电流 ISIS​ **(来自少数载流子漂移),数量级 nA~pA,可以近似当成零。

**这就是 PN 结的核心性质:单向导电性**——正向导通,反向截止。

### 2.7 "开启电压 0.7V"从哪来

从肖克利方程你就能看出来:iDiD​ 对 vDvD​ 是**指数**关系。电压从 0.6V 升到 0.7V,电流增大约 e(0.7−0.6)/0.026≈47e(0.7−0.6)/0.026≈47 倍。

所以在实际电路里,**一旦 vDvD​ 超过约 0.6~0.7V,电流急剧增大**,几乎就"钉死"在 0.7V 附近——这不是因为存在一个真正的阈值,而是因为**指数函数实在太陡**。工程上近似为"硅二极管导通压降约 0.7V"。

### 2.8 反向击穿

反向电压过大时,耗尽区的电场极强,电子被加速撞击原子,产生链式电离——**雪崩击穿**。或者非常薄的 PN 结里,强电场直接把共价键电子"拽"出来——**齐纳击穿**。

击穿本身不一定烧毁器件,只要功率 P=vDiDP=vD​iD​ 不超标即可。**稳压二极管(齐纳管)就是专门工作在反向击穿区的**——因为击穿时电压几乎恒定,可以用来稳压。

### 这一课你必须带走的东西

1. **半导体有两种载流子:电子(N型多)和空穴(P型多)**。掺杂控制多数载流子类型。
2. **PN 结的本质**:扩散形成耗尽区,耗尽区的固定离子产生内建电场,达到扩散—漂移动态平衡。
3. **单向导电性**:正向偏置削弱内建电场 → 导通;反向偏置加强内建电场 → 截止。
4. **肖克利方程** iD=IS(evD/VT−1)iD​=IS​(evD​/VT​−1) 是 PN 结的"万物之源",VT≈26VT​≈26 mV 要记住。
5. **0.7V 导通电压是指数函数的"工程近似"**,不是一个硬阈值。
