# 第 3 课:二极管——从物理到电路的第一步

上一课我们搞清楚了 PN 结的物理本质。把 PN 结封装起来,加上两个引脚,就是**二极管(Diode)**。

但是——**知道肖克利方程不等于会分析电路**。你在做题时如果每次都去解指数方程,会崩溃。工程上的做法是:**根据精度需要,用不同层次的"模型"去近似真实二极管**。这是模电的核心思想之一:**建模与近似**。

### 3.1 二极管的符号与命名

```
        阳极 (Anode, A)         阴极 (Cathode, K)
         ───┤▶├───
              P  N
              
     i_D 方向 →(正向电流由 A 流向 K)
     v_D 极性: +(A) ─ −(K)
```

三角形箭头指向的方向就是**正向电流**方向,也就是 P→N。阴极端通常有一条横杠(对应符号里的竖线),实物上常用色环或凹槽标记。

---

### 3.2 三种分析模型——核心内容

这是这节课的灵魂。**同一个二极管,根据场景不同,用不同的模型**。

#### 模型 A:理想二极管模型(最粗糙)

**把二极管想象成一个"单向开关"**:

- 正向偏置(vD>0vD​>0 的趋势):**短路**(相当于导线,vD=0vD​=0)
- 反向偏置(vD<0vD​<0 的趋势):**开路**(相当于断路,iD=0iD​=0)

```
    i_D ↑
        │
        │  │  ← 导通后任何电流都行,压降为 0
        │  │
        │  │
    ────┼──┼────→ v_D
        │
   截止时 i_D = 0
```

**什么时候用?** 电路里的电压远大于 0.7V(比如几十 V 的电源),或者你只想定性判断"导不导通",0.7V 的误差可以忽略。

#### 模型 B:恒压降模型(最常用)

**认为二极管导通时两端电压恒为 0.7V**(硅管;锗管是 0.3V,现在几乎不用):

- 导通:vD=0.7VvD​=0.7V,电流由外电路决定
- 截止:iD=0iD​=0,电压由外电路决定

```
    i_D ↑
        │
        │      │  ← 导通后竖直上升
        │      │
        │      │
    ────┼──────┼────→ v_D
        │    0.7V
```

等效电路:导通时相当于**一个 0.7V 的电池**,截止时相当于**断路**。

**什么时候用?** 这是模电课本上和工程中最常用的模型。绝大多数"分析这个电路里二极管导不导通,电流多大"的题目,都用这个模型。

#### 模型 C:折线模型(带正向电阻 rDrD​)

恒压降模型把 I-V 曲线的导通段画成竖线,但实际上是有一定斜率的——电流越大,压降会稍微增加一点。引入一个**正向导通电阻 rDrD​ **:

- 导通:vD=Von+iD⋅rDvD​=Von​+iD​⋅rD​,其中 Von≈0.6VVon​≈0.6V,rDrD​ 几欧到几十欧
- 截止:iD=0iD​=0

等效电路:导通时 = 一个 0.6V 电池 + 一个小电阻 rDrD​ 串联。

**什么时候用?** 大电流场合或需要稍高精度时。课程里出现不多,但要知道有这个模型。

#### 模型 D:小信号模型(非常重要,后面还会深挖)

当二极管**工作在某个直流偏置点 (VD,ID)(VD​,ID​) 上,再叠加一个微小的交流信号**时,我们不关心整个 I-V 曲线,只关心**工作点附近的局部斜率**。

把肖克利方程在 QQ 点处泰勒展开(你数分学过):

iD=ISevD/VT⇒diDdvD∣Q=ISeVD/VTVT=IDVTiD​=IS​evD​/VT​⇒dvD​diD​​​Q​=VT​IS​eVD​/VT​​=VT​ID​​

这个斜率的倒数就是**动态电阻(小信号电阻)** rdrd​:

rd=VTID≈26 mVIDrd​=ID​VT​​≈ID​26mV​​

比如二极管偏置电流 ID=1ID​=1 mA 时,rd≈26 Ωrd​≈26Ω。

**这个公式极其重要**,后面讲三极管放大器时还要用到完全一样的思想——我们把它叫做**"大信号分析 + 小信号分析"的二元方法**,是整个模电的基本功。

### 3.3 直流分析的标准套路

给你一个含二极管的电路,求各点电压电流,怎么做?

**步骤**:

1. **假设**每个二极管的状态(导通 or 截止)
2. 按假设**替换成等效电路**(导通→0.7V 电池,截止→开路)
3. 用 KCL/KVL 求解
4. **验证假设**:
    - 如果假设导通,验证 iD>0iD​>0
    - 如果假设截止,验证 vD<0.7VvD​<0.7V
5. 如果验证失败,换假设重做

**例题 1:简单判断**

```
    +10V ──[R=1kΩ]──┬──┤▶├── 0V (地)
                    │
                   V?
```

10V 通过 1kΩ 电阻和二极管连到地。二极管阳极接 R,阴极接地——如果导通,电流从 +10V 经 R 经二极管到地,是正向。

**假设导通**:二极管两端 0.7V,R 两端 10−0.7=9.310−0.7=9.3V,I=9.3V/1kΩ=9.3I=9.3V/1kΩ=9.3 mA > 0 ✓

假设成立。V 点电压 = 0.7V,电流 = 9.3 mA。

**例题 2:反向的情况**

```
    −10V ──[R=1kΩ]──┬──┤▶├── 0V
                    │
                   V?
```

这次电源是 −10V。阳极试图被拉到低电位,阴极接地——这会让二极管反偏。

**假设截止**:iD=0iD​=0,R 上无压降,V = −10V。二极管两端电压 vD=VA−VK=−10−0=−10vD​=VA​−VK​=−10−0=−10V < 0.7V ✓

假设成立。二极管截止,电路无电流。

### 3.4 整流电路——二极管的经典应用

**整流(Rectification)**:把交流变成直流(或至少是单向脉动的电流)。这是所有电源电路的第一步。

#### 半波整流

```
     ~       ┤▶├          ┌────────┐
   AC 源 ───┬──┬─── R_L ───┤负载     │
             正  负           └────────┘
```

变压器次级输出正弦电压 vs=Vmsin⁡(ωt)vs​=Vm​sin(ωt)。

- **正半周**(vs>0vs​>0):二极管正偏导通,电流流过 RLRL​,输出 vo≈vs−0.7Vvo​≈vs​−0.7V
- **负半周**(vs<0vs​<0):二极管反偏截止,vo=0vo​=0

波形:

```
输入 v_s:   ∿∿∿∿∿  (完整正弦)

输出 v_o:   ∩ ∩ ∩ ∩  (只剩正半周)
          ─ ─ ─ ─ ─
```

缺点:**一半时间没输出,效率低,脉动大**。

#### 全波桥式整流(最常用!)

用 4 个二极管组成一个"桥"。

```
                D1 ▶│       ▶│ D2
     A ────────┤├───┬────────┤├─────── B
                    │
     交流        R_L│负载
     输入             │
                    │
     A ────────┤├───┴────────┤├─────── B
                D4 │◀        │◀ D3
                   
     (图有点难画,核心是:无论 A、B 哪端为正,
      总能找到一条路径让电流从"正端"→经过两个导通的二极管
      →流过 R_L →回到"负端"。)
```

**关键理解**:

- A 端为正(正半周):D1、D3 导通,D2、D4 截止
- B 端为正(负半周):D2、D4 导通,D1、D3 截止
- **无论哪半周**,流过 RLRL​ 的电流方向都一样!

输出波形:

```
输入:    ∿∿∿∿∿

输出:    ∩∩∩∩∩∩  (全部翻到上面,双倍频率)
        ─ ─ ─ ─
```

**代价**:电流经过两个二极管,压降损失 2×0.7=1.4V2×0.7=1.4V。

#### 加滤波电容(整流+滤波)

整流后的波形还是脉动的,不是"真直流"。**并联一个大电容**就能让它平滑得多:

```
      整流输出 ───┬──── 到负载 R_L
                 │
                 C (大电容)
                 │
      ──────────┴────
```

**工作原理**:

- 电压上升时:电容充电,电容电压跟随输入
- 电压下降时:二极管截止,电容通过 RLRL​ **缓慢放电**(时间常数 τ=RLCτ=RL​C)
- 下一个脉冲到来时再充电

波形变成:

```
     ∩∩∩∩∩∩       整流后
     ─ ─ ─ ─ 

     ▔╲▁▔╲▁▔╲▁    加电容后(近似直流,带小纹波)
```

剩下的小波动叫**纹波(ripple)**,CC 越大、RLRL​ 越大,纹波越小。

### 3.5 限幅与钳位——改变波形形状

#### 限幅电路(Clipper)

**把波形超过某个电压的部分"削掉"**。

```
     v_i ──[R]──┬── v_o
                │
              ──┴──
              ▼ D      ← 二极管阳极向上(反过来画的示意)
              ─┬─
               │
              [V_ref]  ← 参考电压
               │
              ─┴─ 地
```

当 vi>Vref+0.7Vvi​>Vref​+0.7V 时,二极管导通,vovo​ 被钉在 Vref+0.7VVref​+0.7V;否则二极管截止,vo=vivo​=vi​。

效果:输入正弦波,输出"顶端被切平"的波形。

应用:**保护电路**(防止电压过高烧坏后级)、**限压**。

#### 钳位电路(Clamper)

**把整个波形平移到某个直流电平上**,形状不变。

```
     v_i ──[C]──┬── v_o
                │
                ▼ D
                │
               ─┴─ 地
```

输入正弦(比如 ±5V 摆动),输出会被钳位到**最高点在某个固定值**(比如 0.7V),整个波形向下平移。关键在电容的**充电记忆**:一旦充了电,就保持那个直流偏移。

### 3.6 稳压二极管(齐纳管)

上一课提过:工作在**反向击穿区**,击穿电压 VZVZ​ 几乎不随电流变化。

符号:普通二极管符号,阴极那条线加个折角(像个 Z)。

**稳压电路**:

```
    V_in ──[R]──┬── V_out (稳定在 V_Z)
                │
              ──┴── 
              ▲ Dz   ← 反向接!阴极接高电位
              ──┬──
                │
    地 ─────────┴──
```

注意接法:**齐纳管反向工作,阴极接高电位**。

工作原理:无论 VinVin​ 怎么波动(在一定范围内),只要流过 DzDz​ 的反向电流足够大,DzDz​ 两端电压就钉在 VZVZ​。R 承担输入电压的波动,Vout=VZVout​=VZ​ 稳定。

### 3.7 这一课你必须带走的东西

1. **三种模型的适用场景**:理想模型用于定性,恒压降模型(0.7V)用于一般计算,小信号模型 rd=VT/IDrd​=VT​/ID​ 用于交流小信号分析。
2. **直流分析的"假设—验证"方法**是标准套路,做题必用。
3. **整流的核心**:半波简单但浪费,桥式全波最常用,加电容滤波后才接近真正的直流。
4. **限幅/钳位**:利用二极管的"开关"特性改造波形形状。
5. **稳压二极管**工作在反向击穿区,是最简单的稳压电路方案。
6. 贯穿整门模电最重要的思想已经出现了:**大信号定工作点,小信号分析增量行为**。这在三极管和运放分析里会大放异彩。

下一课我们进入**双极型晶体管(BJT)**,这是历史上第一种放大器件(1947 年贝尔实验室发明,直接开启了整个电子时代)。我们会讲:

- **放大区的核心关系**:iC=βiBiC​=βiB​,为什么是线性的?
- BJT 的输入输出特性曲线怎么读
- 三种组态:**共射、共基、共集**,它们各自擅长什么
