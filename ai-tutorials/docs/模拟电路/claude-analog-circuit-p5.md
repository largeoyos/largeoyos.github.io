# 第 4 课:双极型晶体管(BJT)——放大的起源

### 4.1 为什么需要"放大"

先想一个问题:麦克风把声音变成电信号,这个电信号只有几毫伏;但音箱驱动扬声器需要几伏、甚至几十伏。中间需要一个东西,**能用小信号"指挥"大信号**——这就是**放大器**。

但要注意:**能量不是凭空产生的**。放大器的能量来自电源(比如 +12V 直流电源),它的作用是"让小信号去**调制**电源向负载输出的能量"。

> **小信号是"指挥棒",电源是"能量池",放大器是"阀门"**。

BJT 就是这样一个"电控阀门"。

---

### 4.2 BJT 的结构——两个 PN 结"背靠背"

BJT = Bipolar Junction Transistor,**双极型晶体管**。"双极"指**电子和空穴两种载流子都参与导电**(这是和后面要讲的 MOS 管的关键区别)。

#### NPN 型结构

```
         发射极 E
             │
         ┌───┴───┐
         │   N   │  ← 发射区(Emitter,高掺杂)
         ├───────┤
         │   P   │  ← 基区 (Base,薄且低掺杂) ──── B 基极
         ├───────┤
         │   N   │  ← 集电区(Collector,中等掺杂)
         └───┬───┘
             │
         集电极 C
```

**关键结构特点**(每一条都有物理意义,不能忽略):

1. **基区极薄**(微米量级)——这是 BJT 能放大的核心秘密,等下就会看到为什么
2. **发射区重掺杂,基区轻掺杂**——让"注入"方向不对称
3. **三个区、两个 PN 结**:
    - **发射结(BE 结)**:基极-发射极之间
    - **集电结(BC 结)**:基极-集电极之间

PNP 型结构正好相反(P-N-P),工作原理完全对称,电压电流方向相反。先掌握 NPN。

#### 电路符号

```
         C (集电极)
         │
      ┌──┤
      │  │
   B──┤  │         NPN:箭头向外(Emitter 的 E 指向外)
      │  │
      └──┤
         │↘
         E (发射极,带箭头)
```

**看箭头记方向**:NPN 箭头**向外**,PNP 箭头**向内**。箭头方向 = 发射结正偏时的电流方向(这是判断管子类型最快的方法)。

### 4.3 BJT 怎么"放大"——物理过程(最关键的部分)

现在进入核心。我们让 NPN 管工作在**放大区**——具体偏置要求是:

- **发射结正偏**(VBE>0VBE​>0,约 0.7V)
- **集电结反偏**(VCB>0VCB​>0,即 VC>VBVC​>VB​)

也就是说:VC>VB>VEVC​>VB​>VE​,而且 VB−VE≈0.7VVB​−VE​≈0.7V。

接下来发生了 4 件事——**这 4 步你必须记住,是理解 BJT 一切行为的基础**:

#### 步骤 1:发射结正偏 → 电子从 E 区大量注入基区

发射结正偏,就像一个正向导通的二极管。由于**发射区重掺杂**,大量电子从 N 型发射区"涌"进 P 型基区。同时也有少量空穴从基区进入发射区,但这部分可以忽略(因为基区掺杂浅)。

#### 步骤 2:电子在基区"穿越"

现在基区里有大量从 E 注入的电子。这些电子本来应该和基区的空穴复合(就像普通 PN 结那样),但是——

**基区太薄了!**

电子还没来得及复合,就扩散到了集电结边缘。只有**很少一部分**(大约 1%)的电子在基区和空穴复合,贡献了**基极电流 IBIB​**。

#### 步骤 3:集电结反偏 → 把电子"拉"进集电区

集电结反偏,意味着 C 极电位高,形成一个从 C 指向 B 的强电场。这个电场对**电子来说是"顺势而下"**——把从基区扩散过来的电子**一股脑全部扫到集电区**,形成**集电极电流 ICIC​**。

> **关键:反偏结对少数载流子是"抽取"状态,对从基区过来的电子不仅不阻碍,反而加速**。

#### 步骤 4:电流关系

设从 E 注入基区的电子总电流为 IEIE​(等于发射极电流),其中:

- 约 99% 被集电极收走 → IC≈0.99 IEIC​≈0.99IE​
- 约 1% 在基区复合 → IB≈0.01 IEIB​≈0.01IE​

**由 KCL**:IE=IB+ICIE​=IB​+IC​(NPN 中电流方向:E 流出,C 流入,B 流入)

---

### 4.4 两个核心参数:αα 和 ββ

**共基直流放大系数**:

α=ICIE(通常 0.98∼0.995)α=IE​IC​​(通常 0.98∼0.995)

**共射直流放大系数**:

β=ICIB(通常 50∼300)β=IB​IC​​(通常 50∼300)

它们的关系可以推导:

β=ICIB=ICIE−IC=α1−αβ=IB​IC​​=IE​−IC​IC​​=1−αα​

**验证一下**:若 α=0.99α=0.99,β=0.99/0.01=99β=0.99/0.01=99。若 α=0.995α=0.995,β=199β=199。

**为什么 ββ 这么大?** 因为 αα 接近 1,1−α1−α 非常小,分子除以小数得到大数。ββ 的大小**完全由基区的薄度和掺杂浓度决定**——这是工艺参数。

#### 这就是"放大"!

**小的 IBIB​ 能"控制"几十倍到几百倍的 ICIC​**。

IC=β⋅IBIC​=β⋅IB​​

你给基极灌 10μA 电流,集电极就流 1mA(若 β=100β=100)——**基极电流像一个"水龙头",集电极电流像水龙头控制的主水流**。

### 4.5 三种工作区——判断条件与状态

BJT 并不总是在放大。根据两个 PN 结的偏置状态,它有三种工作区:

|工作区|发射结|集电结|状态描述|
|---|---|---|---|
|**截止区**|反偏(或未达 0.7V)|反偏|相当于"开关断开",IC≈0IC​≈0|
|**放大区**|正偏(≈0.7V≈0.7V)|反偏|线性放大,IC=βIBIC​=βIB​|
|**饱和区**|正偏|正偏|相当于"开关闭合",VCE≈0.2VVCE​≈0.2V 很小|

**第四种——"倒置放大区"**(发射结反偏、集电结正偏),实际上极少使用,因为 BJT 结构不对称,倒着用参数很差。

#### 放大区判定(NPN)的工程速记

VBE≈0.7VVBE​≈0.7V,且 VCE>VCE,sat≈0.3VVCE​>VCE,sat​≈0.3V,则在放大区。

#### 饱和区的物理含义

当 VCEVCE​ 小到一定程度(典型 0.2V),集电结不再反偏,反而正偏了。这时集电极不再有"抽走电子"的本事,即使你继续增大 IBIB​,ICIC​ 也不再增长——**IC≠βIBIC​=βIB​ 在饱和区不成立**。

```
    I_C ↑                    
        │                      
        │    放大区            
        │   ──────── I_B=60μA
        │  /────────           
        │ /─────── I_B=40μA
        │/─────────            
        │/──────── I_B=20μA
        │/─────────
        │饱和区
    ────┼────────────→ V_CE
        0 0.3V
```

左边陡峭上升的部分是**饱和区**,右边几乎水平的部分是**放大区**,几乎水平意味着 ICIC​ 几乎只由 IBIB​ 决定,和 VCEVCE​ 关系不大——**这就是"电流控制电流源"的直观表现**。

### 4.6 三种组态——共射、共基、共集

BJT 有三个引脚,信号要从一个引脚输入、一个引脚输出——剩下那个是"公共端"。公共端是哪个,就叫"共 X"组态。

```
    共发射极(CE)        共基极(CB)         共集电极(CC,也叫射极跟随器)
    
    输入:B              输入:E              输入:B
    输出:C              输出:C              输出:E
    公共:E(接地)       公共:B(接地)       公共:C(接电源)
```

三种组态各自有不同的"性格":

|组态|电压增益|电流增益|输入阻抗|输出阻抗|典型用途|
|---|---|---|---|---|---|
|**共射 CE**|大(数十~数百,反相)|大(ββ)|中|中|通用电压放大|
|**共基 CB**|大(同相)|约 1|低|高|高频放大|
|**共集 CC**|约 1|大(ββ)|**高**|**低**|**缓冲、阻抗匹配**|

**记忆要点**:

- **共射**是"全能战士",既放大电压又放大电流,**但输出反相**
- **共基**高频好,但电流不放大,用得少
- **共集**电压不放大(约 1)但**阻抗变换能力极强**——**后面学运放时的"输出级"几乎都是共集**

这是模电考试最爱考的内容之一。

### 4.8 这一课你必须带走的东西

1. **BJT = 两个背靠背 PN 结 + 极薄的基区**。基区薄是放大的根本原因。
2. **放大的物理过程 4 步**:发射结正偏→电子注入基区→基区太薄导致绝大多数电子穿越→集电结反偏把电子扫到集电区。
3. **核心关系 IC=βIBIC​=βIB​ 只在放大区成立**。ββ 是工艺参数,典型 50~300。
4. **三种工作区的判断**看两个结的偏置:截止(两结反偏)、放大(发射正偏+集电反偏)、饱和(两结正偏)。
5. **"控制"的本质**:IBIB​ 小,但它决定了多大比例的电子能穿越基区被集电极收走。**BJT 是电流控制电流源(CCCS)**。
6. **三种组态各有专长**,共射最常用做电压放大(记住"反相"),共集用做**阻抗变换**(缓冲器)。
