# 第 5 课:BJT 放大电路的静态工作点(Q 点)

### 5.1 为什么一定要有 Q 点?

先问一个最根本的问题:**为什么不能把交流信号直接送到 BJT 的基极?**

假设有个麦克风输出的音频信号,幅值 ±20mV,你直接把它接到 BJT 基极,发射极接地。那么:

- 正半周:VBE=+20mVVBE​=+20mV——**远小于 0.7V,BJT 截止,没有输出**
- 负半周:VBE=−20mVVBE​=−20mV——**反偏,BJT 更加截止**

结果:**BJT 根本不导通,整个信号都消失了**。

问题出在哪?

**BJT 的放大区要求 VBE≈0.7VVBE​≈0.7V**。而输入信号是围绕 0V 摆动的,根本达不到 0.7V 的门槛。

**解决方案**:给 BJT 先加一个**直流偏置**,让它"先坐到放大区的工作位置上",然后再把交流信号**叠加**上去。

```
     直流偏置(V_BE = 0.7V)
              +
     交流信号(±20mV)
              ↓
         V_BE 实际 = 0.7V ± 20mV → 680mV ~ 720mV
                 始终在放大区内工作 ✓
```

**这个直流偏置下的工作状态,就叫"静态工作点 Q"(Quiescent Point)**。

"静态"的意思是:**没有交流信号输入时,电路里的电压电流值**。它由 (VBEQ,IBQ,VCEQ,ICQ)(VBEQ​,IBQ​,VCEQ​,ICQ​) 四个量描述(常用 ICQICQ​ 和 VCEQVCEQ​ 作主角)。

---

### 5.2 "先直流后交流"——模电最重要的分析方法

这个思想贯穿整门模电,我要用框起来的方式强调:

> **叠加原理**:因为 BJT 在放大区可以**近似线性**,我们可以把信号分成两部分独立分析:
> 
> - **直流分析**:只看直流电源,求 Q 点(**大信号**)
> - **交流分析**:只看交流信号,求增益、阻抗(**小信号**)
> 
> 最终的实际电压电流 = 直流分量 + 交流分量

这就是上一课讲过的**"大信号定工作点,小信号分析增量行为"**。现在要正式用起来了。

#### 工程记号约定(要认清)

|记号|含义|
|---|---|
|VBEVBE​,ICIC​(大写字母+大写下标)|直流量(Q 点值)|
|vbevbe​,icic​(小写字母+小写下标)|交流量(瞬时增量)|
|vBEvBE​,iCiC​(小写字母+大写下标)|总瞬时量 = 直流 + 交流|
|VCCVCC​,VEEVEE​(大写双下标)|电源电压|

这一套记号你必须现在就适应,后面的课全都用。

### 5.3 最简偏置:固定偏置电路(反面教材)

先看一个"教科书反面典型"——**固定偏置电路**。学它是为了理解**为什么它不好**。

```
       V_CC (+12V)
        │
     ┌──┴──┐
     │     │
    [R_B]  [R_C]
     │     │
     ├─────┤─── v_o (输出)
     │     │
     │    ─┤ C
    ───── │
         │ NPN
         E
         │
        ─┴─ 地
        
    v_i ──[C1耦合电容]── 接基极
```

**直流通路分析**(信号电容 C1 对直流开路,暂时忽略 vivi​):

从 VCCVCC​ 到地,经过 RBRB​ 和发射结:

VCC=IBQRB+VBEQVCC​=IBQ​RB​+VBEQ​IBQ=VCC−VBEQRB≈VCC−0.7VRBIBQ​=RB​VCC​−VBEQ​​≈RB​VCC​−0.7V​​

然后:

ICQ=βIBQICQ​=βIBQ​VCEQ=VCC−ICQRCVCEQ​=VCC​−ICQ​RC​

**数值例子**:VCC=12VVCC​=12V,RB=565kΩRB​=565kΩ,RC=3kΩRC​=3kΩ,β=100β=100。

- IBQ=(12−0.7)/565k≈20μAIBQ​=(12−0.7)/565k≈20μA
- ICQ=100×20μ=2mAICQ​=100×20μ=2mA
- VCEQ=12−2m×3k=6VVCEQ​=12−2m×3k=6V ✓(在放大区)

看起来挺好。**但这个电路有一个致命问题:**

#### 为什么"固定偏置"不好?

**问题出在 ββ 上**。ββ 这个参数:

1. **同一型号的管子,不同个体差异很大**(说明书常写 β=100∼300β=100∼300)
2. **温度变化时 ββ 会变**——每升温 1°C,ββ 约增加 0.5%
3. **一旦 ββ 变了,ICQ=βIBQICQ​=βIBQ​ 就跟着变**,Q 点漂移

假设上面电路里换了一个 β=200β=200 的管子:

- IBQIBQ​ 几乎不变(还是由 RBRB​ 决定的),仍约 20μA
- ICQ=200×20μ=4mAICQ​=200×20μ=4mA
- VCEQ=12−4m×3k=0VVCEQ​=12−4m×3k=0V ——**进入饱和区,不能放大了!**

**结论**:固定偏置把 IBIB​ 锁死,但 ICIC​ 完全随 ββ 起舞。**工程上绝不能用**。

### 5.4 王者方案:分压式偏置电路(重点!)

这是实际电路中最常用、最经典的偏置方式。**务必掌握**。

```
       V_CC (+12V)
        │
    ┌───┴───┐
    │       │
   [R_1]   [R_C]
    │       │
    ├───────┤──────── v_o
    │       │
    │      ─┤ C
    ├──────│
    │      │ NPN
   [R_2]   E
    │      │
    │     [R_E]
    │      │
    │      ├──[C_E]── 地 (发射极旁路电容)
    │      │
    └──────┴──── 地
    
    v_i ──[C_1]── 接基极节点
```

两个关键改动:

1. **基极用 R1R1​、R2R2​ 分压产生稳定偏压**(不再靠 RBRB​ 单条路径)
2. **发射极串一个 RERE​**(引入**负反馈**,稳定 Q 点)
3. RERE​ 两端并一个大电容 CECE​,交流短路,让交流信号感觉不到 RERE​ 存在(交流分析时再讲)

#### 直流分析——"戴维南等效"法

看左边的分压网络 R1R1​、R2R2​,如果 IBIB​ 相对于流过它们的电流可以忽略(这是设计要求),那么基极电压近似为:

VB≈VCC⋅R2R1+R2VB​≈VCC​⋅R1​+R2​R2​​

这是一个**由电阻分压、几乎不受 ββ 影响的稳定值**。

#### 求 Q 点

从 VBVB​ 出发,减去 VBEVBE​,得发射极电压 VEVE​:

VE=VB−VBE≈VB−0.7VVE​=VB​−VBE​≈VB​−0.7V

**关键一步**:IE=VE/REIE​=VE​/RE​。

IEQ=VB−0.7VREIEQ​=RE​VB​−0.7V​​

注意这个式子里**完全没有 ββ**!

然后:

ICQ≈IEQ(因为 α≈1)ICQ​≈IEQ​(因为 α≈1)VCEQ=VCC−ICQRC−IEQRE≈VCC−ICQ(RC+RE)VCEQ​=VCC​−ICQ​RC​−IEQ​RE​≈VCC​−ICQ​(RC​+RE​)

#### 数值例子

设 VCC=12VVCC​=12V,R1=40kΩR1​=40kΩ,R2=10kΩR2​=10kΩ,RC=3kΩRC​=3kΩ,RE=1kΩRE​=1kΩ,β=100β=100:

- VB=12×1040+10=2.4VVB​=12×40+1010​=2.4V
- VE=2.4−0.7=1.7VVE​=2.4−0.7=1.7V
- IEQ=1.7/1k=1.7mAIEQ​=1.7/1k=1.7mA
- ICQ≈1.7mAICQ​≈1.7mA
- VCEQ=12−1.7m×(3k+1k)=12−6.8=5.2VVCEQ​=12−1.7m×(3k+1k)=12−6.8=5.2V ✓

**现在换 β=200β=200 的管子**:

- VBVB​ 不变(还是 2.4V)
- IEQIEQ​ 不变(还是 1.7mA)
- ICQICQ​ 不变
- VCEQVCEQ​ 不变

**Q 点几乎不受 ββ 影响!** 这就是分压式偏置的魔力。

### 5.5 为什么分压式偏置这么稳?—— 负反馈的威力

这是一个**极其重要的概念**,提前渗透,后面专门讲反馈时会回来。

想象一下:某种原因(温度升高)导致 ICIC​ 想增大。

```
    T↑ → I_C↑ → I_E↑ → V_E = I_E·R_E ↑
         
    但是 V_B 几乎不变(由分压锁定)
    
    → V_BE = V_B − V_E ↓
    
    → I_B↓ → I_C↓  (压制住原来的增大趋势)
```

这是一个**自动调节机制**——RERE​ 把输出端(IEIE​)的信息"反馈"到了输入端(VBEVBE​),并且是**反方向修正**,所以叫**负反馈**。

**R_E 越大,负反馈越强,Q 点越稳——但代价是后面会看到,交流增益也会被压制**。所以才要并一个 CECE​,让交流信号绕开 RERE​。

### 5.6 Q 点的"图解法"——直流负载线

这是模电教材的经典图,期末必考。

从 KVL:VCE=VCC−IC(RC+RE)VCE​=VCC​−IC​(RC​+RE​),这是一条**直线**,画在输出特性曲线上:

```
    I_C ↑
        │
    V_CC/(R_C+R_E) ┤──╲
                   │   ╲
                   │    ╲  ← 直流负载线
                   │     ╲
                   │   Q╲× ← Q 点(工作点)
                   │     ╲
                   │      ╲___ I_B=I_BQ
                   │       ╲
                   └────────╲──→ V_CE
                            V_CC
```

- 横轴截距:VCE=VCCVCE​=VCC​(当 IC=0IC​=0)
- 纵轴截距:IC=VCC/(RC+RE)IC​=VCC​/(RC​+RE​)(当 VCE=0VCE​=0)
- **Q 点 = 负载线与 IB=IBQIB​=IBQ​ 那条输出特性曲线的交点**

### 5.7 Q 点的位置——**必须在放大区中央!**

这是设计放大器的黄金法则。**Q 点位置选得不好,信号就会失真**。

#### 情况 A:Q 点太靠近饱和区(VCEQVCEQ​ 太小)

输入交流信号的**负半周**会让 ICIC​ 增大,VCEVCE​ 减小——一旦 VCEVCE​ 掉到饱和区(<0.3V<0.3V),VCEVCE​ 就被钉住了,**波形底部被削平**——**饱和失真**。

#### 情况 B:Q 点太靠近截止区(ICQICQ​ 太小)

输入交流信号的正半周让 ICIC​ 继续减小,但 ICIC​ 不能为负——ICIC​ 被钉在 0,**波形顶部被削平**——**截止失真**(NPN 共射接法下,输出相位反相,所以波形"底部"实际对应输入"顶部")。

#### 情况 C:Q 点居中

VCEQVCEQ​ 约为 VCC/2VCC​/2,ICQICQ​ 留有足够摆动空间。**正负半周都能完整输出**——这就是 Q 点居中的设计目标。

```
    V_CE 的波形(共射接法,输出反相)
    
     V_CC ┤
          │
    V_CEQ ┤─ ─∿∿∿∿∿─ ─  ← Q 点位置决定波形的"中心线"
          │
    0.3V  ┤  └── 低于此线就饱和失真
          │
      0V  ┤
```

**经验设计准则**:VCEQ≈VCC/2VCEQ​≈VCC​/2,ICQICQ​ 视信号需要而定(一般几 mA)。

### 5.8 非线性失真——更深的来源

即使 Q 点选在正中央,**BJT 本身不是完全线性的**——ICIC​ 随 VBEVBE​ 按指数变化:

IC=IS⋅eVBE/VTIC​=IS​⋅eVBE​/VT​

当输入信号幅值大时,这种指数非线性会让输出波形**不对称**:正半周和负半周的增益不一样,产生**谐波分量**——这就是**非线性失真(Nonlinear Distortion)**。

**解决办法有两类**:

1. **小信号工作**:让交流摆幅远小于 VT≈26mVVT​≈26mV,使 BJT 只看到线性区
2. **负反馈**(后面专门讲):用反馈改善线性度

### 5.9 这一课你必须带走的东西

1. **Q 点的意义**:先给 BJT 施加直流偏置,让它"坐到放大区的位置",再叠加交流信号。**没有 Q 点,就没有放大**。
2. **"先直流后交流"的分析方法**:叠加原理把一个放大器问题拆成两个更简单的问题——先求 Q 点(这节课),再分析小信号增益(下节课)。
3. **固定偏置电路 Q 点随 β 漂移**,工程上不用。
4. **分压式偏置是标准方案**:
    - VBVB​ 由 R1R1​、R2R2​ 分压锁定,几乎不受 β 影响
    - RERE​ 引入直流负反馈,进一步稳定 Q 点
    - CECE​ 让交流信号绕过 RERE​,不损失增益
5. **Q 点计算套路**(R1R2RER1​R2​RE​ 电路):
    - VB=VCC⋅R2/(R1+R2)VB​=VCC​⋅R2​/(R1​+R2​)
    - ICQ≈IEQ=(VB−0.7)/REICQ​≈IEQ​=(VB​−0.7)/RE​
    - VCEQ=VCC−ICQ(RC+RE)VCEQ​=VCC​−ICQ​(RC​+RE​)
6. **Q 点位置决定波形能否完整输出**:太靠近饱和 → 饱和失真,太靠近截止 → 截止失真,居中最好。
