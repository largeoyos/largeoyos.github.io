# 第 6 课:BJT 小信号模型与交流分析

### 6.1 为什么需要"小信号模型"

Q 点确定后,BJT 已经稳稳坐在放大区。现在输入信号 vivi​ 在 Q 点附近**小幅摆动**,我们想知道:输出 vovo​ 被放大了多少倍?

**直接用肖克利方程?** IC=ISeVBE/VTIC​=IS​eVBE​/VT​——这是**非线性**的,算起来很痛苦。

**但是**——如果摆幅足够小,非线性曲线的一小段**可以近似当成直线**。这是模电最重要的思想之一:

> **把非线性器件在 Q 点附近"线性化"**——这就是**小信号模型**的灵魂。

从数学上讲,就是你学过的**一阶泰勒展开**:

iC(vBE)≈ICQ+diCdvBE∣Q⋅vbeiC​(vBE​)≈ICQ​+dvBE​diC​​​Q​⋅vbe​

diCdvBE∣QdvBE​diC​​​Q​ 是 Q 点处的斜率,叫**跨导 gmgm​**。

---

### 6.2 跨导 gmgm​——BJT 的"放大本领"

对肖克利方程求导:

gm=diCdvBE∣Q=ISeVBE/VTVT=ICQVTgm​=dvBE​diC​​​Q​=VT​IS​eVBE​/VT​​=VT​ICQ​​gm=ICQVT≈ICQ26 mVgm​=VT​ICQ​​≈26mVICQ​​​

**这个公式极其重要,必背**。它的物理意义是:**每单位 vBEvBE​ 变化,引起多大的 iCiC​ 变化**。

**例子**:ICQ=1 mAICQ​=1mA 时,gm=1mA/26mV≈38.5 mSgm​=1mA/26mV≈38.5mS(毫西门子)。

**注意一件美妙的事**:gmgm​ **只由 Q 点电流决定**,和 ββ 无关,和具体的管子型号几乎无关。这是 BJT 的一个内在本质属性。

---

### 6.3 小信号输入电阻 rberbe​(或记作 rπrπ​)

BJT 的输入端(BE 结)可以被小信号"看成"什么?

BE 结是一个正向导通的 PN 结,我们第 3 课讲过它在 Q 点的动态电阻:

re=VTIEQ≈26 mVIEQre​=IEQ​VT​​≈IEQ​26mV​

但这个是**发射极端**看到的电阻。从**基极端**看,因为 ibib​ 只是 ieie​ 的 1/(β+1)1/(β+1),看到的电阻会大 (β+1)(β+1) 倍:

rbe=(β+1)⋅re=(β+1)VTIEQ≈βVTICQrbe​=(β+1)⋅re​=IEQ​(β+1)VT​​≈ICQ​βVT​​​

**例子**:β=100β=100,ICQ=1mAICQ​=1mA,则 rbe=100×26/1=2600 Ω=2.6 kΩrbe​=100×26/1=2600Ω=2.6kΩ。

**教材里另一种记号**:有些书用 rπrπ​ 代替 rberbe​,完全是一回事。

---

### 6.4 两个模型的等价性:gmgm​ 与 ββ

有两种等价的方式描述 BJT 的小信号行为:

**方式 1:电压控制电流源(跨导模型,混合 π 模型)**

- vbevbe​ 是"控制信号"
- ic=gmvbeic​=gm​vbe​

**方式 2:电流控制电流源(β 模型)**

- ibib​ 是"控制信号"
- ic=βibic​=βib​

这两者必须等价,因为是同一个器件。验证:

ic=βib=β⋅vberbe=β⋅vbeβVT/ICQ=ICQVTvbe=gmvbe✓ic​=βib​=β⋅rbe​vbe​​=β⋅βVT​/ICQ​vbe​​=VT​ICQ​​vbe​=gm​vbe​✓

**也就是说:** β=gm⋅rbeβ=gm​⋅rbe​。两个模型在数学上等价,用哪个取决于你更喜欢以"电压"还是"电流"为核心思考。

---

### 6.5 BJT 的小信号等效电路(混合 π 模型)

把上面的元素组合起来,画出 BJT 在 Q 点的**线性等效电路**:

```
   基极 B ●────┬────● 集电极 C
              │    │
              │    │
           [r_be]  ↓ g_m·v_be  (或 β·i_b)
              │    │           受控电流源
              │    │
   发射极 E ●─┴────┴────● 
              
  注:实际还有一个 r_ce(输出电阻,很大,一般忽略)
```

**要点**:

- BE 之间是一个电阻 rberbe​
- CE 之间是一个**受控电流源**,大小为 gmvbegm​vbe​(或 βibβib​)
- **没有** BC 之间的直连——这在小信号下是合理的近似
- 完整模型还有一个 rcerce​ 并联在受控源两端,表示集电极输出电阻(几十 kΩ 到几百 kΩ),**入门分析经常忽略**

**这个等效电路只在 Q 点附近、小信号条件下有效**。一旦信号幅度变大(如 vbevbe​ 接近 VTVT​),它就不准了。

---

### 6.6 怎么画"交流通路"——关键步骤

拿到一个放大器原理图,怎么转换成小信号等效电路?**三步法**:

#### 第 1 步:电容视为交流短路

耦合电容 C1C1​、C2C2​、旁路电容 CECE​——在信号频率下容抗 1/(ωC)1/(ωC) 很小,近似短路。

> **前提是"在信号频率下"**。后面讲频率响应时会回来——低频下电容不能短路,那就是另一回事了。

#### 第 2 步:直流电源视为交流地

VCCVCC​ 是理想直流电源,**对交流信号来说阻抗为零**(因为电压恒定,不变化)——所以**直流电源节点对交流就是"地"**。

这意味着:接到 VCCVCC​ 的电阻,在交流通路里变成"接地"。

#### 第 3 步:用 BJT 的小信号模型替换 BJT

把原理图中的 BJT 抠掉,替换成 rberbe​ + 受控电流源的等效电路。

---

### 6.7 典型例子:共射放大器的交流分析

用上节课的分压式偏置共射放大器做例子:

```
  原电路:
  
       V_CC
        │
     ┌──┴──┐
     │     │
    [R_1] [R_C]
     │     │
     ├─────┤──[C_2]── v_o
     │     │
     │    ─┤ (BJT)
     ├────│
     │    │
    [R_2] E
     │    │
     │   [R_E]──[C_E]── 地
     └────┴───── 地
     
  v_i ──[C_1]── 接基极
```

#### 画交流通路

- C1C1​、C2C2​、CECE​ 短路
- VCCVCC​ 节点接地
- R1R1​ 的上端原来接 VCCVCC​,现在接地 → R1R1​ 和 R2R2​ 并联接在基极和地之间
- RCRC​ 的上端原来接 VCCVCC​,现在接地 → RCRC​ 接在集电极和地之间
- RERE​ 被 CECE​ 短路掉 → 发射极直接接地

简化后:

```
          ┌───────┐        ┌────────┐
  v_i ────┤       ├── b ───┤        ├─── c ──┬── v_o
          │ R₁∥R₂ │        │  BJT   │        │
          │       │        │        │      [R_C]
          │       │        │        │        │
          └───┬───┘        │   e    │        │
              │            │   │    │        │
             地            └───┴────┘       地
                              │
                             地
```

#### 代入 BJT 小信号模型

```
           ┌─────┐     b ●─────┬──────● c ──┬── v_o
  v_i ─────┤     ├──────────────┤            │
           │R₁∥R₂│              │            │
           │  =  │            [r_be]  ↓g_m·v_be  [R_C]
           │ R_B'│              │            │
           └──┬──┘            e │            │
              │                 │            │
              地              地 ●────────── 地
```

RB′=R1∥R2RB′​=R1​∥R2​(两个偏置电阻并联)。

#### 推导电压增益

**第 1 步**:输入端,vbevbe​ = ?

基极到地的路径:vivi​ 通过 C1C1​(短路)到基极,基极到地经过两条并联路径:RB′RB′​ 和 rberbe​。

**如果信号源是理想电压源**(内阻为 0),那么 vbe=vivbe​=vi​。

**第 2 步**:输出端,vovo​ = ?

集电极的电流是 ic=gmvbeic​=gm​vbe​(注意这个电流是**从集电极流出**,由受控源方向决定)。

这个电流流过 RCRC​ 到地。但注意方向:**受控源把电流从 C 端抽出**,这意味着 C 端电位被拉低——vo=−ic⋅RC=−gmvbe⋅RCvo​=−ic​⋅RC​=−gm​vbe​⋅RC​。

更仔细地说:

```
  V_CC ──[R_C]── c 点 ── 到负载
  
  受控电流源 g_m·v_be 的方向:从 c 流向 e
  → 电流从 V_CC 经 R_C 流到 c,再经受控源流向 e(地)
  → 这个电流在 R_C 上产生压降
  → c 点电位 = V_CC − i_c·R_C
  
  交流分量:v_o = −i_c · R_C = −g_m v_be · R_C
```

**第 3 步**:电压增益

Av=vovi=vovbe=−gmRCAv​=vi​vo​​=vbe​vo​​=−gm​RC​​

或者用 β 形式表达:

Av=−βRCrbeAv​=−rbe​βRC​​

两个公式等价(因为 gm=β/rbegm​=β/rbe​)。

#### 负号的意义

**负号表示输入与输出反相**:输入信号为正时,输出为负;输入上摆,输出下摆。

物理理解:vivi​ 上升 → vbevbe​ 上升 → icic​ 上升 → RCRC​ 上压降变大 → **集电极电位 vCvC​ 下降**。所以输出电压减小——反相。

这是共射放大器的标志性特征——"**共射反相**"。

#### 数值验证

用第 5 课的例子:ICQ=1.7mAICQ​=1.7mA,β=100β=100,RC=3kΩRC​=3kΩ。

- gm=1.7/26≈65.4 mSgm​=1.7/26≈65.4mS
- rbe=100×26/1.7≈1529 Ωrbe​=100×26/1.7≈1529Ω
- Av=−gmRC=−65.4×3≈−196Av​=−gm​RC​=−65.4×3≈−196

**电压增益约 −196 倍**——这个数量级是共射放大器的典型值。

### 6.8 输入阻抗与输出阻抗

一个放大器不是一个孤立的东西,它要连接信号源、连接后级。**放大器看起来像什么**——就是它的输入/输出阻抗决定的。

#### 输入阻抗 RiRi​

从输入端往里看的等效电阻:

Ri=RB′∥rbe=(R1∥R2)∥rbeRi​=RB′​∥rbe​=(R1​∥R2​)∥rbe​

通常 rberbe​(几 kΩ)远小于 R1∥R2R1​∥R2​(几十 kΩ),所以:

Ri≈rbeRi​≈rbe​

**对共射放大器,输入阻抗不高**——几 kΩ 的量级。这是它的一个缺点。

#### 输出阻抗 RoRo​

从输出端往里看(**输入端信号源短路、负载端断开**的条件下)。

受控源 gmvbegm​vbe​ 在 vbe=0vbe​=0 时输出为 0,相当于开路。所以从 C 端往里看,只剩下 RCRC​ 到地:

Ro=RCRo​=RC​

(如果考虑 rcerce​,则 Ro=RC∥rceRo​=RC​∥rce​,但通常 rce≫RCrce​≫RC​,可忽略)

---

### 6.9 带负载的情形

实际电路中,输出要接负载 RLRL​。负载的加入会改变增益:

Av=−gm(RC∥RL)Av​=−gm​(RC​∥RL​)

因为受控电流源产生的电流会同时"看到" RCRC​ 和 RLRL​ 并联后的等效电阻。

**启示**:输出阻抗 RoRo​ 相对负载越小,带负载后增益损失越小——"**低输出阻抗好**"。

### 6.10 RERE​ 不旁路的情形——交流负反馈

如果**去掉 CECE​**,RERE​ 对交流信号也起作用。这时交流分析变为:

发射极不再接地,发射极电压 ve=ieRE≈icREve​=ie​RE​≈ic​RE​。

而 vbevbe​(BJT 两端)实际等于 vi−vevi​−ve​:

vbe=vi−icRE=vi−gmvbeREvbe​=vi​−ic​RE​=vi​−gm​vbe​RE​

解出 vbevbe​:

vbe=vi1+gmREvbe​=1+gm​RE​vi​​

所以:

Av=vovi=−gmvbeRCvi=−gmRC1+gmREAv​=vi​vo​​=vi​−gm​vbe​RC​​=1+gm​RE​−gm​RC​​

当 gmRE≫1gm​RE​≫1 时:

Av≈−RCREAv​≈−RE​RC​​

**一个神奇的结果**:增益只由外部电阻 RC/RERC​/RE​ 决定,**与 BJT 的参数(β、Q 点)无关**!

这就是**负反馈**的神奇之处:用"损失增益"换"稳定性、线性度、参数无关性"。后面专门讲反馈时你会深刻理解这件事。

**对比**:

- 旁路(有 CECE​):Av≈−200Av​≈−200,但高度依赖 ββ
- 不旁路:Av≈−3Av​≈−3(若 RC=3kRC​=3k, RE=1kRE​=1k),稳定且与 ββ 无关

### 6.11 h 参数模型(中国教材特色)

中国教材常用另一种等价模型:**h 参数模型**。本质上和混合 π 模型是同一回事,只是换了符号:

- hie↔rbehie​↔rbe​(输入电阻)
- hfe↔βhfe​↔β(电流放大系数)
- hrehre​:反向电压反馈系数(通常很小,忽略)
- hoehoe​:输出电导(通常很小,忽略)

简化后的 h 参数模型和混合 π 模型完全一样。**不用专门学,混合 π 模型已经够用**。

### 6.12 这一课你必须带走的东西

1. **小信号模型的本质**:把 BJT 在 Q 点处一阶泰勒展开,获得**线性等效电路**,只在小信号下有效。
2. **三个核心参数**:
    - gm=ICQ/VTgm​=ICQ​/VT​(跨导,只看 Q 点电流)
    - rbe=βVT/ICQrbe​=βVT​/ICQ​(输入电阻)
    - β=gm⋅rbeβ=gm​⋅rbe​(三者的关系)
3. **画交流通路三步**:电容短路、VCCVCC​ 接地、BJT 换成小信号模型。
4. **共射放大器的核心公式**:
    - Av=−gmRC=−βRC/rbeAv​=−gm​RC​=−βRC​/rbe​(负号代表反相)
    - Ri≈rbeRi​≈rbe​
    - Ro≈RCRo​≈RC​
5. **RERE​ 不旁路时**:Av≈−RC/REAv​≈−RC​/RE​,**增益由外部电阻决定**,这是负反馈的威力。
6. **"先直流后交流"的完整流程**——你现在已经掌握了:
    - 直流分析 → 确定 Q 点 → 算出 ICQICQ​
    - 根据 Q 点算出 gmgm​、rberbe​
