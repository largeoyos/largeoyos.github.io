# 第 7 课:共集、共基组态与多级放大器

上一课我们深挖了共射组态(CE)。但 BJT 有三个引脚,有三种接法。这一课我们补齐另两种,然后学怎么把它们**组合**起来——实际电路几乎都是多级的。

### 7.1 共集组态(CC)——射极跟随器

#### 电路结构

**信号从基极输入,从发射极输出,集电极作为公共端(交流接地)**。

```
       V_CC
        │
        │
       ─┤ C
      ──┤
        │ NPN
        E ────── v_o (从这里输出!)
        │
       [R_E]
        │
       ─┴─ 地
       
    v_i ──[C_1]── 接基极 (通过偏置电路)
```

**关键特征**:集电极直接接 VCCVCC​(没有 RCRC​),输出从发射极取。

#### 直流 Q 点

和共射的分压式偏置基本一样——只是 RCRC​ 变成了 0(C 直接接 VCCVCC​)。Q 点算法:

VB=VCC⋅R2R1+R2,IEQ=(VB−0.7)/REVB​=VCC​⋅R1​+R2​R2​​,IEQ​=(VB​−0.7)/RE​VCEQ=VCC−IEQREVCEQ​=VCC​−IEQ​RE​

#### 小信号分析

画交流通路(电容短路、VCCVCC​ 接地、BJT 换成等效模型):

```
       b ●────┬──────● c
              │      │
              │      │
           [r_be]   ↓ g_m·v_be
              │      │
              │      │
              e ●────┴── 
              │      
             [R_E]    
              │       
              地      
   (输出 v_o 从 e 端取)
```

注意:集电极 c 在交流通路里**接地**(因为 VCCVCC​ 对交流是地)。

#### 推导电压增益

设发射极电位为 vovo​(相对地)。KVL:vi=vbe+vovi​=vbe​+vo​ (因为 b 到 e 跨 rberbe​,e 到地就是 vovo​)。

**发射极电流 ieie​**:ie=ib+ic=ib(1+β)≈βibie​=ib​+ic​=ib​(1+β)≈βib​

而 ib=vbe/rbeib​=vbe​/rbe​,所以:

vo=ie⋅RE=(1+β)ibRE=(1+β)RErbe⋅vbevo​=ie​⋅RE​=(1+β)ib​RE​=rbe​(1+β)RE​​⋅vbe​

再由 vi=vbe+vovi​=vbe​+vo​:

vi=vbe+(1+β)RErbevbe=vbe⋅rbe+(1+β)RErbevi​=vbe​+rbe​(1+β)RE​​vbe​=vbe​⋅rbe​rbe​+(1+β)RE​​

所以:

Av=vovi=(1+β)RErbe+(1+β)REAv​=vi​vo​​=rbe​+(1+β)RE​(1+β)RE​​​

**这个式子永远小于 1,但非常接近 1**。

**数值例子**:β=100β=100,rbe=1.5kΩrbe​=1.5kΩ,RE=1kΩRE​=1kΩ:

Av=101×1k1.5k+101×1k=101102.5≈0.985Av​=1.5k+101×1k101×1k​=102.5101​≈0.985

几乎等于 1!所以叫"**射极跟随器**"——**输出电压"跟随"输入电压**,同相位、幅度几乎相等。

#### 既然增益是 1,有什么用?

**秘密在阻抗变换**。

##### 输入阻抗 RiRi​

从基极往里看:

Ri′=vbe/ib+vo/ib=rbe+(1+β)RERi′​=vbe​/ib​+vo​/ib​=rbe​+(1+β)RE​

这是一个关键结果:**基极看到的电阻 = rberbe​ + 发射极电阻的 (1+β)(1+β) 倍**。

RERE​ 原来只有 1kΩ,从基极看进去变成了 1500+101×1000≈102.5kΩ1500+101×1000≈102.5kΩ——**放大了 100 倍左右**!

再与偏置电阻 R1∥R2R1​∥R2​ 并联(偏置电阻一般几十 kΩ),总输入阻抗还是比共射高得多。

> **发射极电阻"被基极放大 (1+β)(1+β) 倍看到"**——这个规律叫**阻抗反射**,在 BJT 分析里反复出现,要记住。

##### 输出阻抗 RoRo​

这里推导稍微复杂,结论是:

Ro=RE∥rbe+Rs1+βRo​=RE​​1+βrbe​+Rs​​

其中 RsRs​ 是信号源内阻。如果 RsRs​ 不大,括号里的部分可以估算:

rbe1+β=βVT/ICQ1+β≈VTICQ=re1+βrbe​​=1+ββVT​/ICQ​​≈ICQ​VT​​=re​

**发射极看到的阻抗 = 基极侧电阻除以 (1+β)(1+β)**——和阻抗反射方向相反。

**数值例子**:若 re=26/1.7≈15Ωre​=26/1.7≈15Ω,RE=1kΩRE​=1kΩ,则 Ro≈15ΩRo​≈15Ω(被小的那个支配)。

#### 共集组态的特性总结

|参数|值|特点|
|---|---|---|
|电压增益 AvAv​|≈1≈1|同相、跟随|
|电流增益|≈β≈β|有电流放大|
|输入阻抗 RiRi​|高(几十~几百 kΩ)|**重要优点**|
|输出阻抗 RoRo​|低(几十 Ω)|**重要优点**|

**应用**:**阻抗缓冲器(Buffer)**——不放大电压,但"让强信号源看起来像高阻(对前级友好),让弱负载看起来像低阻(驱动能力强)"。

**经典用法**:放在共射放大器**后面**,用来驱动低阻负载(如扬声器、同轴电缆)。共射输出阻抗高(几 kΩ),直接带低阻负载会损失大量增益;加一级射极跟随器做缓冲,问题就解决了。

### 7.2 共基组态(CB)

#### 电路结构

**信号从发射极输入,集电极输出,基极作为公共端(交流接地)**。

```
       V_CC
        │
       [R_C]
        │
        ├─── v_o
       ─┤ C
      ──┤
        │ NPN
       ─┤ B ───[C_B]── 地 (基极交流接地)
        │
        E ─── v_i(信号从发射极输入)
        │
       [R_E]
        │
       地
```

基极通过一个大电容 CBCB​ **交流接地**(直流上由偏置电阻提供 VBVB​,交流上被电容短路到地)。

#### 小信号分析(直接给结论,推导方法同共射)

**电压增益**:

Av=gmRCAv​=gm​RC​

**注意:没有负号!**——共基是**同相**放大。

数值上和共射差不多大(gmRCgm​RC​),量级数十到数百。

**输入阻抗**:

Ri≈re=VTIEQRi​≈re​=IEQ​VT​​

**非常低**——只有十几到几十欧姆。

**输出阻抗**:

Ro≈RCRo​≈RC​

和共射差不多。

#### 共基组态的特点总结

|参数|值|特点|
|---|---|---|
|电压增益|大,同相||
|电流增益|≈1≈1(αα)|**不放大电流**|
|输入阻抗|很低|一般是缺点|
|输出阻抗|高||
|**高频特性**|**极好**|**核心优点**|

**为什么高频好?** 共射组态有一个叫**密勒效应(Miller Effect)** 的东西——BJT 内部的一个小电容 CμCμ​(BC 间)在共射下会被"放大"几百倍,严重限制高频响应。共基组态里 CμCμ​ 不被放大,所以带宽高得多。

**实际应用**:

- 高频/射频放大器(手机、电视调谐器)
- 和共射组合成**共射-共基级联(Cascode)** 电路——前级共射保证电压增益,后级共基消除密勒效应,综合性能极好。Cascode 在集成电路里是标配。

### 7.4 多级放大器:为什么要多级?

#### 动机

单级放大器有三个局限:

1. **单级增益有限**:共射约 100~500 倍,要做到 10000 倍就不够
2. **输入/输出阻抗不够理想**:共射 RiRi​ 只有几 kΩ,很多信号源(如高阻传感器)会严重衰减
3. **功能单一**:一级无法同时实现"高输入阻抗 + 高增益 + 低输出阻抗"

**解决方案**:**级联**——把多个单级放大器接起来,每级各司其职。

#### 经典级联结构:"三明治"配方

```
     高阻信号源 ─┤ 共集 ├─→ 共射 ├─→ 共射 ├─→ 共集 ├─ 低阻负载
                ↑           ↑       ↑           ↑
              输入级       中间级   中间级      输出级
           (高输入阻抗)   (大增益) (大增益)  (低输出阻抗)
```

- **输入级**用共集(CC):**匹配高阻信号源**,防止信号被前端电阻分压衰减掉
- **中间级**用共射(CE):**提供主要电压增益**
- **输出级**用共集(CC):**低输出阻抗,驱动负载**

这就是很多实际运放内部的基本结构。

#### 级间耦合方式

放大器级与级之间怎么连?三种方式:

##### A. 阻容耦合(RC 耦合)

用**耦合电容**连接两级,像上节课共射放大器里的 C1C1​、C2C2​:

```
  前级 C ──[C_coupling]── 后级 B
```

- **优点**:直流隔离——前级的 Q 点不影响后级的 Q 点,各级独立偏置、独立设计
- **缺点**:**不能放大直流信号**(电容隔直);电容又大又贵,不适合集成电路

**应用**:分立元件电路、音频放大器等只放大交流信号的场合。

##### B. 直接耦合(DC 耦合)

前级的输出端**直接连到**后级的输入端,没有电容:

```
  前级 C ────直线──── 后级 B
```

- **优点**:能放大直流信号、频率响应从 0 开始、容易集成(不需要大电容)
- **缺点**:
    - **Q 点相互影响**:前级的直流输出电平 = 后级的直流输入电平,设计时必须一起考虑
    - **零点漂移**:温度变化导致前级 Q 点漂移,后级把这个漂移也放大了,最后输出端直流电平乱跳

**应用**:集成电路(芯片里不能用大电容)、运算放大器。

**零点漂移问题的解决方案**:**差分放大器**——下节课专门讲,这是运放输入级的核心。

##### C. 变压器耦合

用变压器连接两级:

- **优点**:阻抗变换、隔离直流
- **缺点**:变压器笨重、昂贵、频率特性受限

**应用**:现在已经不常用,除了功率放大的特定场合。

### 7.5 多级放大器的增益计算

关键公式:

Av,total=Av1⋅Av2⋅Av3⋯Av,total​=Av1​⋅Av2​⋅Av3​⋯

**总增益 = 各级增益的乘积**。

**但是**——**每一级的 AvkAvk​ 必须考虑"被下一级加载"之后的值**。

#### 前后级相互影响(重点!)

单看第 kk 级,它的输出阻抗 RokRok​ 和第 k+1k+1 级的输入阻抗 Ri,k+1Ri,k+1​ 会**形成分压**:

```
     前级                      后级
  ──┬──[R_ok]──┬───────────────┬──
   ┌─┴─┐       │               │
   │ v │       │             [R_i,k+1]
   │ok │       │               │
   └─┬─┘       │               │
     └─────────┴───────────────┘
```

前级**空载输出** vokvok​ 经过 RokRok​ 和 Ri,k+1Ri,k+1​ 分压后,才真正进入后级:

vin,k+1=vok⋅Ri,k+1Rok+Ri,k+1vin,k+1​=vok​⋅Rok​+Ri,k+1​Ri,k+1​​

所以**前级的实际增益**变成:

Avk=Avk空载⋅Ri,k+1Rok+Ri,k+1Avk​=Avk空载​⋅Rok​+Ri,k+1​Ri,k+1​​

**启示**:

- 如果 Ri,k+1≫RokRi,k+1​≫Rok​,几乎没损失——**后级高输入阻抗是好事**
- 如果 Ri,k+1≪RokRi,k+1​≪Rok​,大部分信号被分压吞掉——**增益严重损失**

这就是为什么**共集(高输入阻抗、低输出阻抗)放在中间或末尾做缓冲**效果这么好:它既不"拖累"前级,又能强力驱动后级。

#### 两级共射级联的例子

```
  v_i ─── 共射1 ──── 共射2 ─── v_o
         A_v1       A_v2
         R_o1       R_i2
```

前级空载增益 Av1空载=−gm1RC1Av1空载​=−gm1​RC1​,但实际增益:

Av1=−gm1(RC1∥Ri2)=−gm1(RC1∥rbe2)Av1​=−gm1​(RC1​∥Ri2​)=−gm1​(RC1​∥rbe2​)

RC1RC1​ 通常几 kΩ,rbe2rbe2​ 也是几 kΩ——**两者并联后比 RC1RC1​ 小得多**,前级增益大幅损失。

**两级共射级联的总增益** ≈ 前级受损增益 × 后级增益 ≈ 一级的平方……其实没想象的那么好。这就是为什么要用**共集做缓冲**。

### 7.6 输入/输出阻抗在级联中的计算

**总输入阻抗** = 第一级的输入阻抗(后级不影响)。

**总输出阻抗** = 最后一级的输出阻抗(前级不影响,因为我们从输出端往里看)。

**关键**:中间级的阻抗只影响"前级增益的加载"和"后级输入信号的大小",不影响总 RiRi​、RoRo​。

### 7.7 一个完整的多级放大器例子(概念示意)

```
  ┌───────┐     ┌───────┐     ┌───────┐
  │ 共集   │──→│ 共射  │──→│ 共集   │
  │ 输入级 │     │ 增益级 │     │ 输出级 │
  └───────┘     └───────┘     └───────┘
  R_i 高         A_v 大        R_o 低
  A_v≈1          R_o 中        A_v≈1
  
  级间用耦合电容(分立) or 直接耦合(集成)
```

**总体性能**:

- 输入阻抗:高(来自共集输入级)✓
- 电压增益:大(来自共射增益级)✓
- 输出阻抗:低(来自共集输出级)✓

**这就是运算放大器内部架构的简化版**。下下节课讲运放时你会看到。

### 7.8 这一课你必须带走的东西

1. **共集(射极跟随器)** 的核心价值是**阻抗变换**,不是放大:
    - Av≈1Av​≈1,同相
    - 基极看到的阻抗 = 发射极阻抗 ×(1+β)×(1+β)(**阻抗反射**)
    - 发射极看到的阻抗 = 基极侧阻抗 ÷(1+β)÷(1+β)
    - 用途:**缓冲**,匹配高阻源或驱动低阻载
2. **共基** 的核心价值是**高频**:
    - 电压增益大、同相
    - 输入阻抗极低(≈re≈re​)
    - 没有密勒效应,带宽大
    - 常与共射组成 **Cascode** 结构
3. **三种组态的"分工"**:共射放大电压、共集匹配阻抗、共基应对高频。
4. **多级放大的核心思想**:让每级各司其职,组合出单级做不到的性能。
5. **级联公式**:总增益 = 各级增益乘积,但**每级增益要用"被后级加载后"的值**,因为前后级之间存在**阻抗分压**。
6. **耦合方式的权衡**:
    - 阻容耦合:各级独立,但不能放大直流
    - 直接耦合:能放直流,适合集成,但有零漂问题(引出下节课的差分放大器)

### 下一课预告:差分放大器——运放的前世

我们已经看到直接耦合放大器的一个头疼问题:**零点漂移**。温度一变,第一级 Q 点飘了,后面级层层放大,输出端电压"自己在那里乱跑"。

1948 年人们发明了一个绝妙的结构解决这个问题——**差分放大器(Differential Amplifier)**。它是:

- 几乎所有运算放大器的**输入级**
- 所有模拟集成电路的基石之一
