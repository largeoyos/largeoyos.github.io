# 第 8 课:差分放大器——运放的心脏

### 8.1 先回到那个头疼的问题:零点漂移

上一课结尾我提到了**零点漂移**。我们先把它掰开看清楚,你才能体会差分放大器有多漂亮。

想象一个直接耦合的三级共射放大器,每级增益 100,总增益 106106。

**理论上**:输入 0V,输出 0V。完美。

**现实**:温度从 20°C 升到 30°C,第一级 BJT 的 VBEVBE​ 下降了约 20mV(PN 结温度特性:温度每升高 1°C,VBEVBE​ 下降约 2mV)。这 20mV 被三级放大 106106 倍——

Δvo=20mV×106=20000VΔvo​=20mV×106=20000V

当然实际上会被电源限制住,但这意味着**输出早就"撞到天花板"**,毫无意义。输入信号完全淹没在漂移里。

**零点漂移的根源**:温度变化引起 VBEVBE​、ICBOICBO​、ββ 等参数变化,这些变化经每级逐级放大,最终使输出偏离零点。

**本质问题**:温度漂移看起来和真实信号一样,放大器没法区分。

---

### 8.2 差分放大器的核心思想:对称消去

1948 年,一个绝妙的想法诞生了:

> **如果我用两个完全一样的放大器,让温度漂移在两个放大器上同时发生(因为它们在同一块硅片上,温度一致),然后取它们的输出之差——漂移就相互抵消了!**

这就是差分放大器(Differential Amplifier,简称**差放**)。

---

### 8.3 差放的基本结构

```
                V_CC
                 │
          ┌──────┼──────┐
          │             │
         [R_C1]        [R_C2]
          │             │
   v_o1 ──┤             ├── v_o2
          │             │
         ─┤ T1       T2 ├─
        ──┤             ├──
          │             │
    v_i1──┤B         B ├──v_i2
          │             │
          E             E
          │             │
          └──────┬──────┘
                 │
                 │      ←── 两发射极连在一起
              [I_EE]       (这是关键!)
                 │         
                 │       尾电流源(Tail Current Source)
                 │
                V_EE (负电源)
```

**结构要点**:

1. **两个完全相同的 BJT(T1 和 T2)**,参数对称,温度一致(集成电路中就在相邻位置)
2. **两个相同的集电极电阻 RC1=RC2=RCRC1​=RC2​=RC​**
3. **两个发射极连在一起**,接一个**恒流源 IEEIEE​**(尾电流源)
4. **双端输入**:vi1vi1​ 进 T1 基极,vi2vi2​ 进 T2 基极
5. **双端输出**:vo1vo1​ 从 T1 集电极取,vo2vo2​ 从 T2 集电极取

这个对称结构是差放所有魔力的来源。

---

### 8.4 为什么要用恒流源 IEEIEE​?

这个"尾电流源"不是装饰品,是差放的灵魂之一。

**它的物理作用**:把 T1 和 T2 的**发射极电流之和固定为 IEEIEE​**:

IE1+IE2=IEE(恒定)IE1​+IE2​=IEE​(恒定)

这意味着:**两管之间在"争夺"这固定的 IEEIEE​**。T1 多吸一点,T2 就少一点;两者此消彼长,总和不变。

这是差分工作的核心——"**电流翻跷跷板**"。

**为什么不能用普通电阻代替?** 如果发射极直接接一个 RERE​ 到 VEEVEE​,IE1+IE2IE1​+IE2​ 就不是恒定的,温度影响会串进来。恒流源的特性——"无论两端电压如何变化,电流恒定"——是差放稳定的关键。

**恒流源怎么做?** 通常用一个额外的 BJT 做成"电流镜"或类似结构,具体实现我们下一课讲集成工艺时再展开。现在就把它当成一个理想的恒流元件。

---

### 8.5 差模信号与共模信号——重要概念跃迁

这是整个差放分析里最重要的一对概念。请把它当成一种"正交分解"。

**任意两个输入 vi1vi1​、vi2vi2​ 可以分解为两部分**:

**共模分量(Common Mode, cm)**——两个输入的**平均值**:

vic=vi1+vi22vic​=2vi1​+vi2​​

**差模分量(Differential Mode, d)**——两个输入的**差值**(常定义为 vi1−vi2vi1​−vi2​):

vid=vi1−vi2vid​=vi1​−vi2​

**反解**:

vi1=vic+vid2,vi2=vic−vid2vi1​=vic​+2vid​​,vi2​=vic​−2vid​​

#### 物理意义

- **差模信号**:两个输入"**反向摆动**"的部分——一个上摆、一个下摆,幅度相等方向相反。这是**我们想放大的"真信号"**(有用信息通常被编码在两路的"差异"上)。
- **共模信号**:两个输入"**同向摆动**"的部分——两个一起升、一起降,幅度相等方向相同。**这通常就是干扰**(温度漂移、电源噪声、电磁耦合等对两路信号的影响基本一样)。

#### 举例:三种典型输入

|情况|vi1vi1​|vi2vi2​|vicvic​|vidvid​|
|---|---|---|---|---|
|纯差模|+5mV|−5mV|0|10mV|
|纯共模|+5mV|+5mV|5mV|0|
|混合|+6mV|−4mV|1mV|10mV|

**关键洞察**:**如果差放对差模有大增益、对共模有零增益,就能完美"抽取信号、压制干扰"**。

这就是差放的设计目标。

### 8.6 差模信号如何被放大——"翘跷板"分析

**假设输入是纯差模**:vi1=+vid/2vi1​=+vid​/2,vi2=−vid/2vi2​=−vid​/2。

T1 的 VBEVBE​ 增加一点,T1 多导通一些,IE1IE1​ 想增大;同时 T2 的 VBEVBE​ 减少一点,IE2IE2​ 想减小。

由于 IE1+IE2=IEEIE1​+IE2​=IEE​(恒定),T1 增加多少,T2 就减少多少。**完美的翘跷板**:

ΔIE1=−ΔIE2=ΔIΔIE1​=−ΔIE2​=ΔI

由此:

vo1=−ΔI⋅RC,vo2=+ΔI⋅RCvo1​=−ΔI⋅RC​,vo2​=+ΔI⋅RC​

**双端输出**(从 T1 集电极到 T2 集电极):

vod=vo1−vo2=−2ΔI⋅RCvod​=vo1​−vo2​=−2ΔI⋅RC​

两边发射极"共同点"(vEvE​)的电压在纯差模下**不动**——这是关键性质,后面要反复用到。为什么?因为 T1 吸下去多少,T2 就从上面"补"多少,两者对 vEvE​ 的影响正好抵消。

**既然 vEvE​ 不动,就相当于交流接地**——这时分析一侧(比如 T1)就是一个**发射极接地的共射放大器**!

**差模增益(单端输出)**:

Ad1=vo1vid=−12gmvidRCvid=−12gmRCAd1​=vid​vo1​​=vid​−21​gm​vid​RC​​=−21​gm​RC​

(因为 T1 的 vbe1=vid/2vbe1​=vid​/2,所以电流变化 =gm⋅vid/2=gm​⋅vid​/2)

**差模增益(双端输出)**:

Ad=vodvid=vo1−vo2vid=−gmRCAd​=vid​vod​​=vid​vo1​−vo2​​=−gm​RC​​

**这和共射放大器的增益一样**!差放没有在数值上"输给"共射——而且还多了一份对抗干扰的能力。

### 8.7 共模信号如何被"抑制"——对称性的胜利

**假设输入是纯共模**:vi1=vi2=vicvi1​=vi2​=vic​(同升同降)。

两管基极同时升高 vicvic​——**两管想同时吸更多电流**。

**但是**:IEEIEE​ 恒定!两管没法同时多吸电流。会发生什么?

答案是:**发射极共同点 vEvE​ 会被"顶起来"**,使得 VBE1=VBE2VBE1​=VBE2​ 不变,IE1=IE2=IEE/2IE1​=IE2​=IEE​/2 保持不变。

换句话说:**共模信号下,两管的集电极电流完全不变**——vo1vo1​ 和 vo2vo2​ 都不动!

**理想情况下,共模增益 = 0**:

Ac=vodvic=0(理想对称、理想恒流源)Ac​=vic​vod​​=0(理想对称、理想恒流源)

**这就是差放的神奇**:共模信号被"看作是没信号",直接忽略。

#### 单端输出下的共模增益

如果只从一端取输出(比如 vo1vo1​),情况稍微复杂。设恒流源的实际输出电阻是 REEREE​(理想为 ∞∞,实际几十 kΩ 到几百 kΩ)。推导省略,结果是:

Ac1=vo1vic≈−RC2REEAc1​=vic​vo1​​≈−2REE​RC​​

REEREE​ 越大,Ac1Ac1​ 越小(越接近理想)。**恒流源输出阻抗越高,共模抑制越好**——这就是为什么差放对尾电流源的质量要求高。

**双端输出下**:由于完美对称,vo1=vo2vo1​=vo2​,两者相减 = 0。**双端输出从根本上消除了共模增益**。

### 8.8 共模抑制比 CMRR——差放的核心指标

一个评价差放好坏的关键指标:

CMRR=∣AdAc∣CMRR=​Ac​Ad​​​​

常用 dB 表示:

KCMR=20log⁡10∣AdAc∣ dBKCMR​=20log10​​Ac​Ad​​​ dB

**CMRR 越大越好**,表示差放"重视差模、忽视共模"的能力越强。

**典型值**:

- 分立元件差放:60 ~ 80 dB
- 集成运放:80 ~ 120 dB
- 精密仪器放大器:>140 dB

**理想运放 CMRR = ∞**——这是运放"抗干扰"能力的根源。

### 8.9 零点漂移为什么被消除?

现在回到开头那个问题。

**温度上升时**,T1、T2 的 VBEVBE​ 都下降约 2mV/°C——**这对两管来说是完全相同的变化,即一个"共模"性质的扰动**。

差放对共模几乎无响应,所以:

**温度漂移不会传到输出**。

——这就是差放解决零漂的数学本质:**把时间上的漂移"伪装"成空间上的共模信号,然后靠对称性消去**。

**一个重要前提**:**两管必须高度对称**。这在分立元件里很难(两个物理上独立的 BJT 参数难以完全匹配),但在集成电路里天然容易——**两管就做在同一块硅片的相邻位置,工艺完全一样、温度完全一样,匹配度极高**。所以差放是集成电路的"亲儿子",是集成运放的天然起点。

### 8.10 差放的四种接法

根据输入输出是"双端"还是"单端",差放有 4 种接法:

|接法|输入|输出|差模增益|特点|
|---|---|---|---|---|
|双入双出|vi1,vi2vi1​,vi2​|vo1−vo2vo1​−vo2​|−gmRC−gm​RC​|最完美,共模抑制最好|
|双入单出|vi1,vi2vi1​,vi2​|vo1vo1​(或 vo2vo2​)|−12gmRC−21​gm​RC​(一半)|增益减半,共模抑制取决于尾源|
|单入双出|vi1vi1​,vi2=0vi2​=0|vo1−vo2vo1​−vo2​|−gmRC−gm​RC​|单入也能分解成共模+差模,增益不变|
|单入单出|vi1vi1​,vi2=0vi2​=0|vo1vo1​|−12gmRC−21​gm​RC​|最常见的"单端应用"接法|

#### 单入怎么分解?(重要)

vi1=vvi1​=v,vi2=0vi2​=0。分解:

vic=v/2,vid=vvic​=v/2,vid​=v

**单端输入相当于同时施加共模 v/2v/2 和差模 vv**。对差放来说,共模被压制,只剩差模被放大。

这说明:**单入双出时,输入"看起来是单端的",实际上差放内部把它当成"差模 vv"处理**,增益还是 −gmRC−gm​RC​,和双入双出一样。

#### 哪种接法最常见?

**集成运放 = 双入单出**。两个输入端(同相输入 ++、反相输入 −−),一个输出端。下一课讲运放时你就会看到。

### 8.11 差放的输入/输出阻抗

**差模输入阻抗**:从两个输入端之间看,等于 2rbe2rbe​(因为是两个 rberbe​ 串联:vi1vi1​→b1→e1→e2→b2→vi2vi2​)。

**共模输入阻抗**:因为共模下两管在尾电流源上方"并联",而且有 REEREE​ 反射到基极,共模输入阻抗**极高**,通常达到 MΩ 量级。

Ric≈1+β2⋅2REE=(1+β)REERic​≈21+β​⋅2REE​=(1+β)REE​

(来自阻抗反射,REEREE​ 被 1+β1+β 倍看到)

**输出阻抗**:

- 单端输出:Ro=RCRo​=RC​
- 双端输出:Ro=2RCRo​=2RC​

### 8.12 差放的 Q 点(简要)

对称结构下,**每管分到一半尾电流**:

ICQ1=ICQ2≈IEE2ICQ1​=ICQ2​≈2IEE​​VCEQ=VCC−ICQRC−VEVCEQ​=VCC​−ICQ​RC​−VE​

其中 VE≈−VBE≈−0.7VVE​≈−VBE​≈−0.7V(如果两基极接地的话,发射极比基极低 0.7V)。

**一个实用推论**:知道尾电流 IEEIEE​,就知道每管的 gm=ICQ/VT=IEE/(2VT)gm​=ICQ​/VT​=IEE​/(2VT​)。

### 8.13 改进:有源负载

前面用 RCRC​ 做集电极负载。在集成电路里,**电阻既笨又不精确**,实际上会用**电流源做负载**——叫**有源负载**。

用电流源做负载的好处:**负载的等效电阻极大**(几百 kΩ 到 MΩ),所以增益极大:

Ad∝gm⋅Ro′Ad​∝gm​⋅Ro′​

Ro′Ro′​ 由电流源输出阻抗决定——几 MΩ 量级。总增益可达 104104 甚至更高!**这就是集成运放能做到 10 万倍、100 万倍开环增益的秘密**。

细节下一课讲集成电路时再展开。

### 8.14 这一课你必须带走的东西

1. **差放诞生的动机**:解决直接耦合放大器的零点漂移。核心思想:把温漂看作"共模扰动",用对称消除。
2. **结构要点**:
    - 两个对称管子
    - 尾电流源 IEEIEE​(锁定 IE1+IE2IE1​+IE2​,翻跷跷板)
    - 对称的 RCRC​
3. **信号分解**:任意输入 == 共模 + 差模
    - 共模:(vi1+vi2)/2(vi1​+vi2​)/2,通常是干扰
    - 差模:vi1−vi2vi1​−vi2​,通常是真信号
4. **差模被完整放大**(约 gmRCgm​RC​,和共射一样),**共模被压制**(理想为 0)。
5. **CMRR** 是差放的核心指标,衡量"重视差模、忽视共模"的能力。
6. **双端输出天然消除共模**(靠对称),**单端输出靠尾源高输出阻抗压制共模**——所以尾源质量至关重要。
7. **差放是集成电路的天然形式**:对称要求在同一块硅片上极易满足,分立元件难。
8. **四种接法**:双入双出增益最大,双入单出(最常见,对应运放)增益减半,但还是有用。
9. **有源负载**可以把增益推到极高,是集成运放做出大开环增益的关键。

### 下一课预告:集成运算放大器(Op-Amp)——模电的"第二次革命"

你前面花了这么多力气学 BJT、学 Q 点、学小信号、学差放——**从这一课开始,这一切将被"封装"成一个新的元件**。

**运算放大器(Op-Amp)** 是一个黑盒,有两个输入端、一个输出端,内部就是我们前面讲的这些东西的集大成:

- 输入级:**差放**(所以有两个输入端、高 CMRR、低零漂)

当你学会用运放后,**你几乎不再需要从 BJT 级别设计电路了**——拿一个运放 + 几个电阻,就能做出各种功能:放大、求和、求差、积分、微分、滤波、比较、振荡……
