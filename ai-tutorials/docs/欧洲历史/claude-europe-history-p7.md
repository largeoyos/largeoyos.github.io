# 4.3 宗教改革——从德国说起

### 一、先理解"教会"是什么

要理解宗教改革，**先要理解中世纪天主教会到底有多强大**。

我们前面提过教会是中世纪欧洲的"灵魂"，但你可能没意识到它具体强大到什么程度。看一下这几个事实：

**1. 教会是欧洲最大的地主**

中世纪后期，**教会拥有欧洲约1/3的土地**。教会有自己的农奴、自己的军队、自己的法庭、自己的税收（什一税：每个基督徒收入的10%要交给教会）。

**2. 教会是唯一的"跨国机构"**

法国国王管不了英国，神圣罗马帝国皇帝管不了西班牙——但**教皇可以管所有人**。一个法国农民、一个英国贵族、一个西班牙商人，**都听同一个教皇的话**。

**3. 教会垄断"通往天堂的钥匙"**

这一点是关键。中世纪人极度恐惧死后下地狱（这种恐惧今天的我们很难体会）。**而通往天堂的所有手段，都被教会垄断**：

- 你必须通过神父才能向上帝忏悔
- 你必须接受神父主持的圣事（洗礼、婚礼、临终涂油……）才能得救
- 教皇可以宣布把谁"开除教籍"——一个被开除教籍的人，即使是国王，也会被认为死后必下地狱

**4. 拉丁语的垄断**

《圣经》是拉丁文的，**普通人看不懂**。神职人员是唯一能读《圣经》的人，所以"上帝说什么"完全由教会解释。

**这就是1500年时教会的样子——一个庞大、富有、不可挑战的精神帝国。**
