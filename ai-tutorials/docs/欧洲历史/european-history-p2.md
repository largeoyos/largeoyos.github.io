# 第二节：罗马——从小村庄到地中海霸主

### 一、一个不起眼的开始

公元前8世纪，当希腊人在爱琴海边建立辉煌城邦的时候，意大利中部有一个**小到不能再小的村落**，建在七座小山丘上，居民是一群放羊的拉丁人。这就是**罗马**。

按照传说，公元前753年，一对被母狼养大的双胞胎兄弟罗慕路斯和雷穆斯建立了罗马（罗慕路斯后来杀了弟弟，所以城市叫"Roma"）。这个传说不重要，重要的是它透露的气质：**罗马从一开始就和暴力、兄弟相残纠缠在一起**。

地理上，罗马的位置比希腊好得多：

- 意大利半岛**有大片平原**（不像希腊全是山），适合农业
- 位于地中海**正中间**，向四面扩张都方便
- 周围的邻居（伊特鲁里亚人、希腊殖民者、迦太基人）**都不太强**

### 二、罗马的三个阶段

罗马历史很长（约1200年），但你只要记住三个阶段：

|阶段|时间|特点|
|---|---|---|
|**王政时代**|前753–前509|国王统治，被推翻|
|**共和国时代**|前509–前27|元老院+公民大会，扩张到整个地中海|
|**帝国时代**|前27–公元476（西部）|皇帝独裁，最终崩溃|

我们重点讲后两个阶段。

### 三、共和国：一场关于"权力"的伟大实验

公元前509年，罗马人**赶走了暴虐的国王**，建立共和国。注意"共和"（res publica）这个词的字面意思就是"公共事务"——国家是大家的，不是国王的私产。

罗马共和国的设计非常精巧，有点像今天美国宪法的前身：

- **执政官**（2人）：每年选举，相当于总统，但有两个互相牵制
- **元老院**：由贵族组成，掌握实际权力
- **公民大会**：所有公民投票决定重大事务

**关键设计：权力分散，互相制衡，没人能独裁。**

罗马人对"国王"这个词怀有深深的恐惧——这种恐惧持续了500年。后来你会看到，凯撒之所以被刺杀，就是因为他被怀疑想当国王。

### 四、罗马 vs 迦太基：地中海的"二战"

共和国时期，罗马一边完善制度，一边疯狂扩张。它先吞并了整个意大利半岛，然后撞上了一个真正的对手——**迦太基**。

迦太基是北非（今天突尼斯一带）的海上强国，腓尼基人的后裔，商业极度发达。它和罗马打了三场惨烈的战争，叫**布匿战争**（前264–前146，断断续续打了120年）。

第二次布匿战争最戏剧性：迦太基名将**汉尼拔**做了一件军事史上最疯狂的事——他**带着大象翻越阿尔卑斯山**，从北边突袭意大利，在罗马本土横冲直撞了15年，多次几乎打到罗马城下。

但罗马的可怕之处在于：**它不投降**。即使军队全军覆没，元老院也照样组建新军。最终罗马反攻北非，第三次战争后**把迦太基彻底夷为平地**——城市烧毁，土地撒盐，居民全部卖为奴隶。这是古代世界最彻底的一次种族灭绝。

打赢迦太基之后，罗马成为**地中海唯一的超级大国**。

### 五、共和国之死：凯撒和他的继承人

但是，**胜利反而毁了共和国**。

征服带来巨额财富，但分配极不公平：贵族越来越富，普通公民越来越穷，社会撕裂。同时，将军们手握大军，开始不听元老院指挥。

最后这个矛盾爆发为**内战**。最终胜出的是一个名字你一定听过的人——**尤里乌斯·凯撒**（Julius Caesar）。

凯撒是个全才：天才军事家（征服了高卢，也就是今天的法国）、出色的政治家、还是个不错的作家（《高卢战记》是文学经典）。公元前49年，他违抗元老院命令，**带兵渡过卢比孔河**进军罗马（这就是英语成语 "crossing the Rubicon" 的来源，意思是做出不可逆转的决定）。

凯撒成了独裁者。但他权力越大，元老院越恐惧。

**公元前44年3月15日**，凯撒在元老院开会时被60多名元老围攻刺杀，身中23刀。其中一个刺客是他的好友布鲁图斯，凯撒倒下前的最后一句话据说是："**你也有份吗，布鲁图斯？**"（Et tu, Brute?）

凯撒死了，但共和国也救不回来了。经过又一轮内战，凯撒的养子**屋大维**（Octavian）胜出，公元前27年，他被元老院授予"奥古斯都"（Augustus，意为"神圣者"）的称号，成为罗马第一位**皇帝**。

**共和国结束，帝国开始。**

### 六、罗马帝国：人类历史上最成功的帝国之一

奥古斯都之后的两百年，被称为**"罗马和平"**（Pax Romana，公元前27–公元180年）。这是古代世界最稳定、最繁荣的时期：

- 疆域：从英国到伊拉克，从德国到埃及，**地中海成了"罗马的内湖"**
- 人口：约5000万到7000万（占当时全世界人口的1/4）
- 道路：修了8万公里的硬化道路，"条条大路通罗马"不是比喻
- 法律：罗马法成为后世欧洲所有法律的基础
- 工程：引水渠、公共浴场、斗兽场、混凝土建筑

**但是——你可能注意到了一个矛盾：罗马最辉煌的时期，恰恰是它放弃了共和国理想之后。** 这是政治学上一个永恒的难题：**自由和秩序，效率和参与，能不能兼得？** 罗马人的答案是：到了帝国规模，鱼和熊掌不能兼得，他们选择了秩序和效率。

### 七、一个改变世界的事件：基督教的兴起

帝国早期，发生了一件当时没人觉得重要、但**回头看改变了整个欧洲历史**的事。

公元1世纪初，罗马帝国东部一个偏远省份**犹太地区**（今天的以色列/巴勒斯坦），有一个叫**耶稣**的犹太木匠之子开始传教。他宣扬：所有人在上帝面前平等、爱你的敌人、富人难进天堂、来世会有审判……

罗马总督彼拉多在公元约30年把他钉死在十字架上。按理说故事该结束了。

但耶稣的门徒坚信他**复活升天**了，开始在帝国各地传教。基督教有几个特点让它在罗马极有竞争力：

- **平等**：在罗马残酷的等级社会里，基督教告诉奴隶和穷人"你和皇帝在上帝面前一样"
- **来世**：罗马传统宗教不太关心死后，基督教承诺**永生**
- **社群**：基督徒互相帮助，像一个大家庭
- **道德要求**：和希腊罗马那些"乱搞"的神不同，基督教的上帝有严格的道德标准

罗马帝国曾经**残酷迫害**基督徒——把他们扔给狮子、钉十字架、火刑——但越迫害，信徒越多。

到公元4世纪，发生了一个戏剧性大转折：**皇帝君士坦丁本人皈依了基督教**（公元313年颁布《米兰敕令》，承认基督教合法）。到公元380年，基督教成为罗马**国教**。

**这件事的意义怎么强调都不过分**：从此，"欧洲"约等于"基督教世界"，这个等式持续了一千多年。

### 八、罗马的衰落

罗马帝国从3世纪开始走下坡路。原因复杂得历史学家吵到今天：

- **疆域太大**：边境太长，养军队成本极高
- **经济问题**：通货膨胀、税收过重
- **政治混乱**：3世纪50年里换了20多个皇帝
- **蛮族压力**：北方日耳曼部落（哥特人、汪达尔人、法兰克人……）不断入侵
- **东方威胁**：波斯帝国卷土重来

公元395年，帝国分裂为**东罗马**（首都君士坦丁堡，今天的伊斯坦布尔）和**西罗马**（首都罗马）。

**公元476年**，最后一位西罗马皇帝被一个日耳曼蛮族首领废黜——历史教科书把这一年定为**西罗马灭亡**，也是**古典时代的结束**。

但请注意一个常见误解：**东罗马没有灭亡**！它继续存在了1000年，被后世称为**拜占庭帝国**，直到1453年被奥斯曼土耳其攻破君士坦丁堡。这个我们讲中世纪时还会回来。

### 自问自答时间

**Q1：罗马为什么能成功？希腊为什么不能？**

希腊人天才但分裂，罗马人没那么有创造力但**极其善于组织和整合**。罗马有一种独特的能力：把征服来的民族**消化吸收**而不是奴役。一个高卢人或西班牙人，几代人之后就能成为罗马公民，甚至当皇帝。这种"开放性"是罗马维持几百年帝国的秘密。希腊人则始终把外人当"野蛮人"。

**Q2：罗马灭亡是因为基督教吗？**

这是18世纪历史学家**爱德华·吉本**在《罗马帝国衰亡史》里的著名论点：基督教让罗马人不再尚武，转而追求来世。这个观点今天大部分历史学家**不太同意**了，原因是东罗马也是基督教国家，反而又延续了一千年。更主流的看法是：罗马的衰落是经济、军事、政治多重因素叠加，基督教只是变化中的一部分。

**Q3：罗马留给欧洲什么？**

四样东西，每一样都是欧洲文明的支柱：

1. **法律**：今天大陆法系直接继承自罗马法
2. **拉丁语**：演化出意大利语、法语、西班牙语、葡萄牙语、罗马尼亚语；英语词汇也大量来自拉丁语
3. **基督教**：罗马把它从一个犹太小教派变成世界宗教
4. **"帝国"的梦想**：后来的查理曼、拿破仑、希特勒，每一个想统一欧洲的人都在模仿罗马

**Q4：那时候中国在干什么？**

罗马帝国（前27–公元476）和**汉朝**（前202–公元220）几乎同时代，是欧亚大陆东西两端的两个超级大国。两者之间有微弱的接触（丝绸之路），但互相只是模糊的传说。**两国甚至几乎同时崩溃**：汉朝220年灭亡，罗马帝国3世纪也开始大乱——这背后可能有共同的原因，比如气候变化、瘟疫、北方游牧民族压力。这是个很有意思的历史巧合。
