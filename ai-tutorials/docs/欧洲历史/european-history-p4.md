# 第四节：近代早期（约1500–1789）总览

这300年里，欧洲发生了**五场革命**，每一场都改变了世界：

1. **文艺复兴**（意大利领跑）——艺术与人的觉醒
2. **大航海**（葡萄牙、西班牙领跑）——欧洲走向全球
3. **宗教改革**（德国领跑）——基督教世界的大分裂
4. **科学革命**（英国领跑）——人类认识世界方式的重塑
5. **启蒙运动**（法国领跑）——为现代政治奠基

注意一个有趣的现象：**领跑的国家在不断变化**。意大利先繁荣，然后是西班牙，然后是荷兰，然后是法国和英国——欧洲的"重心"在不断转移。这本身就是一个值得思考的现象。
