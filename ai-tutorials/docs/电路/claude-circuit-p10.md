# 3.5 互易定理（Reciprocity Theorem）

### 先看一个神奇的现象

```
           电路 N₁（原电路）                     电路 N₂（互易后）
   ┌──[R1]──┬──[R3]──┐                 ┌──[R1]──┬──[R3]──┐
   │        │        │                 │        │        │
  (+)      [R2]     (I₂?)              │      [R2]      (+)
   U        │        │                (I₁?)     │        U
  (-)       │        │                 │        │        │
   │        │        │                 │        │        │
   └────────┴────────┘                 └────────┴────────┘
     ①左端放电源               ②把电源搬到右端（同样的U），
     测右端电流 I₂                测左端电流 I₁
```

**结论**：对于**线性无独立源、无受控源**的电路，I1=I2I_1 = I_2 I1​=I2​。

**激励（电源）和响应（电流）位置互换，数值不变**！这就是互易定理。

### 三种形式

**形式一：电压源激励 → 电流响应**

在 1-1' 端加电压源 UsU_s Us​，在 2-2' 端**短路**测电流 I2I_2 I2​。

把电源移到 2-2' 端（同样 UsU_s Us​），在 1-1' 端**短路**测电流 I1I_1 I1​。

结论：I1=I2\boxed{I_1 = I_2} I1​=I2​​

**形式二：电流源激励 → 电压响应**

在 1-1' 端加电流源 IsI_s Is​，在 2-2' 端**开路**测电压 U2U_2 U2​。

把电流源移到 2-2' 端（同样 IsI_s Is​），在 1-1' 端**开路**测电压 U1U_1 U1​。

结论：U1=U2\boxed{U_1 = U_2} U1​=U2​​

**形式三：电压源激励 → 电压响应（注意！要求比值）**

在 1-1' 端加电压源 UsU_s Us​，在 2-2' 端**开路**测电压 U2U_2 U2​。

把电压源**换成电流源** IsI_s Is​，移到 2-2' 端，在 1-1' 端**开路**测电压 U1U_1 U1​。

结论：U1Is=U2Us\boxed{\dfrac{U_1}{I_s} = \dfrac{U_2}{U_s}} Is​U1​​=Us​U2​​​

### 互易定理的前提（⚠️ 非常重要）

互易定理**不是对所有线性电路都成立**！必须满足：

1. **线性** ✓
2. **只含电阻**（或更一般地：**互易元件**，电容、电感、变压器也行）
3. **不含受控源**⚠️（受控源会破坏互易性）
4. **不含独立源**（激励源单独考虑，内部不能有其他独立源）

### 一个例子

```
   ┌──[2Ω]──┬──[3Ω]──┐
   │        │        │
  (+)      [6Ω]      短路
  10V       │        │
  (-)       │        │
   │        │        │
   └────────┴────────┘
```

左端 10V 激励，右端短路。求短路电流 II I。

**直接算**：

- 从 a-b 端口看进去（把 10V 短路）：2Ω 和 6Ω 并联，再和 3Ω 串联

等等，让我直接求：节点 a（上方中间）对地电压 UaU_a Ua​：

节点方程：10−Ua2=Ua6+Ua3\dfrac{10 - U_a}{2} = \dfrac{U_a}{6} + \dfrac{U_a}{3} 210−Ua​​=6Ua​​+3Ua​​

解得 Ua=5U_a = 5 Ua​=5 V。右端短路电流 = Ua/3=5/3U_a / 3 = 5/3 Ua​/3=5/3 A。

**根据互易定理**：若把 10V 搬到右端（短路到地），2Ω 左端短路测电流，结果也应该是 5/3 A。（你有兴趣可以验算，会发现确实如此。）
