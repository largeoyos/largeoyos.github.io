# 4.1 电容元件（大学升级版）

高中你学的电容，核心是 **"电荷储存器"**——Q=CUQ = CU Q=CU，静电学里的元件。

大学电路里，电容的"身份"升级了：**它是一个有时间动态的元件**。我们关心的不再是"储了多少电"，而是"电压电流随时间怎么变"。

---

### 一、电容的 VCR（伏安关系）—— 核心公式

从高中的 Q=CUQ = CU Q=CU 出发，我们做一件简单的事：**对时间求导**。

Q=CuQ = CuQ=Cu

两边对 t 求导（C 是常数）：

dQdt=Cdudt\frac{dQ}{dt} = C \frac{du}{dt}dtdQ​=Cdtdu​

而 dQdt\dfrac{dQ}{dt} dtdQ​ 就是电流 ii i（单位时间流过的电荷量）：

i=Cdudt\boxed{i = C \frac{du}{dt}}i=Cdtdu​​

**这就是电容的 VCR**。它是本章最重要的公式，你必须刻在脑子里。

### 二、这个公式揭示的五个惊人事实

#### 事实 ①：电容电流取决于电压的"变化率"，不是电压本身

这和电阻截然不同：

- 电阻：i=u/Ri = u/R i=u/R → 电压越大，电流越大
- 电容：i=C du/dti = C \, du/dt i=Cdu/dt → 电压变化**越快**，电流越大

**一个极端例子**：电容两端加一个 1000V 的**恒定**直流电压，电流是多少？

du/dt=0du/dt = 0 du/dt=0（不变）→ i=0i = 0 i=0

**电容在直流稳态下等效于开路！** 这是重要结论。

#### 事实 ②：电容电压不能突变

假设电压突变（比如 0V 瞬间跳到 5V），那 du/dt=∞du/dt = \infty du/dt=∞，需要无穷大电流——物理上不可能。

电容电压不能突变：uC(0+)=uC(0−)\boxed{\text{电容电压不能突变：} u_C(0^+) = u_C(0^-)}电容电压不能突变：uC​(0+)=uC​(0−)​

这条在第九章分析动态电路初始条件时是**金科玉律**。

#### 事实 ③：电容有"记忆"

把 VCR 反过来积分：

u(t)=1C∫−∞ti(τ)dτu(t) = \frac{1}{C}\int_{-\infty}^{t} i(\tau) d\tauu(t)=C1​∫−∞t​i(τ)dτ

或者从某个初始时刻 t0t_0 t0​ 开始：

u(t)=u(t0)+1C∫t0ti(τ)dτu(t) = u(t_0) + \frac{1}{C}\int_{t_0}^{t} i(\tau) d\tauu(t)=u(t0​)+C1​∫t0​t​i(τ)dτ

看到了吗？**t 时刻的电压，取决于过去所有时刻的电流**。这就是"记忆"——电容记得它经历过什么。

对比电阻 u=Riu = Ri u=Ri：电阻是"无记忆"的，此刻电压只看此刻电流。

#### 事实 ④：电容储能

从功率入手，关联参考方向下：

p=ui=u⋅Cdudtp = ui = u \cdot C\frac{du}{dt}p=ui=u⋅Cdtdu​

从 −∞-\infty −∞（电压为 0）积分到当前时刻：

WC=∫p dt=∫0uCu′ du′=12Cu2W_C = \int p \, dt = \int_0^u Cu' \, du' = \frac{1}{2}Cu^2WC​=∫pdt=∫0u​Cu′du′=21​Cu2 WC=12Cu2\boxed{W_C = \frac{1}{2}Cu^2}WC​=21​Cu2​

**电容储存的是电场能量**（电介质中的电场）。注意：它**储能不耗能**——能存进去、能放出来，但不消耗（理想情况）。

#### 事实 ⑤：电容是"无源"但"动态"的元件

- **无源**：长期来看能量只能"过一下手"，不能无中生有
- **动态**：VCR 里有导数/积分，所以它是"动态元件"

对比总结：

|元件|VCR|动态/静态|耗能/储能|
|---|---|---|---|
|电阻 R|u=Riu = Ri u=Ri|静态（代数关系）|耗能|
|电容 C|i=C du/dti = C \, du/dt i=Cdu/dt|**动态**（微分关系）|**储能**（电场）|

### 三、电容的串并联

这部分规律和电阻**刚好相反**，记的时候要小心：

**并联**（电压相同，电流相加）：

C并=C1+C2+⋯C_{并} = C_1 + C_2 + \cdotsC并​=C1​+C2​+⋯

（像电阻串联的公式）

**串联**（电流相同，电压相加）：

1C串=1C1+1C2+⋯\frac{1}{C_{串}} = \frac{1}{C_1} + \frac{1}{C_2} + \cdotsC串​1​=C1​1​+C2​1​+⋯

（像电阻并联的公式）
