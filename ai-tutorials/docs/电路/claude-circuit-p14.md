# 4.2 电感元件（全新内容，从零开始）

### 一、电感是什么？物理直觉先行

电感最常见的形式就是一个**线圈**：

```
          ┌─────────┐
     ─────┤ ∿∿∿∿∿∿ ├─────
          └─────────┘
```

**物理现象**：通电线圈会产生磁场（奥斯特实验，高中学过）。反过来，变化的磁场会在线圈里感应出电压（法拉第电磁感应定律）。

**关键一步——"自感应"**：

假设你在线圈里通电流 ii i，它产生磁场 Φ\Phi Φ。现在让 ii i 变化 → 磁场 Φ\Phi Φ 变化 → 线圈**自己**就会感应出一个电压来"反抗"这个变化。

u=dΦdtu = \frac{d\Phi}{dt}u=dtdΦ​

而磁通 Φ\Phi Φ 与电流成正比：Φ=Li\Phi = Li Φ=Li（L 是电感系数，单位亨利 H）

两式合并：

u=Ldidt\boxed{u = L \frac{di}{dt}}u=Ldtdi​​

**这就是电感的 VCR**。

### 二、电感 vs 电容——完美的"对偶"

把两个 VCR 对比着看：

||电容|电感|
|---|---|---|
|VCR|i=Cdudti = C \dfrac{du}{dt} i=Cdtdu​|u=Ldidtu = L \dfrac{di}{dt} u=Ldtdi​|
|关心的变量|**电压**不能突变|**电流**不能突变|
|直流稳态|**开路**（电流=0）|**短路**（电压=0）|
|储能|W=12Cu2W = \dfrac{1}{2}Cu^2 W=21​Cu2（电场）|W=12Li2W = \dfrac{1}{2}Li^2 W=21​Li2（磁场）|
|串并联|和电阻**相反**|和电阻**相同**|

看出来了吗？**电感和电容在公式形式上完全对偶**——只要把"u"和"i"、"C"和"L"对换，两个元件的所有性质都能互换。
