# 4.2 耦合电感（互感）

前面讲的电感是**单独一个线圈**的"自感应"。现在我们让两个线圈**靠近**，看会发生什么。

---

### 一、先用物理直觉打地基

#### 场景想象

两个线圈 L₁ 和 L₂ 靠得很近（比如绕在同一根铁芯上）：

```
      i₁ →                    i₂ →
   ┌──[L₁]──┐             ┌──[L₂]──┐
   │  ∿∿∿∿  │             │  ∿∿∿∿  │
   + u₁  -                  + u₂  -
               ╲ 磁场耦合 ╱
                ─────────
```

**Q（请你思考）**：如果我只在 L₁ 里通电流 i1i_1 i1​（L₂ 断开不通电），L₂ 两端会有电压吗？

---

**答**：**会有！**

为什么？i1i_1 i1​ 在 L₁ 中产生磁场，这个磁场**会穿过 L₂**（因为两个线圈靠得近）。如果 i1i_1 i1​ 是变化的，穿过 L₂ 的磁通就变化，根据法拉第电磁感应定律，L₂ 里就感应出电压。

**这就是"互感"**：一个线圈的电流变化在另一个线圈中感应电压的现象。

---

### 二、数学描述

#### 自感磁通 vs 互感磁通

现在 L₁ 里的**总磁通**来自两个源：

- **自己**的电流 i1i_1 i1​ 产生的磁通：Φ11=L1i1\Phi_{11} = L_1 i_1 Φ11​=L1​i1​（自感磁通）
- **L₂** 的电流 i2i_2 i2​ 穿过来的磁通：Φ12=Mi2\Phi_{12} = M i_2 Φ12​=Mi2​（互感磁通，M 是互感系数）

同样 L₂ 的总磁通：

- 自感磁通：Φ22=L2i2\Phi_{22} = L_2 i_2 Φ22​=L2​i2​
- 互感磁通：Φ21=Mi1\Phi_{21} = M i_1 Φ21​=Mi1​

（理论上两个互感系数叫 M12M_{12} M12​ 和 M21M_{21} M21​，但可以证明 M12=M21=MM_{12} = M_{21} = M M12​=M21​=M，简化成一个常数。）

#### 端口电压

每个线圈两端的电压 = 总磁通对时间的导数：

u1=L1di1dt±Mdi2dt\boxed{u_1 = L_1 \frac{di_1}{dt} \pm M \frac{di_2}{dt}}u1​=L1​dtdi1​​±Mdtdi2​​​ u2=L2di2dt±Mdi1dt\boxed{u_2 = L_2 \frac{di_2}{dt} \pm M \frac{di_1}{dt}}u2​=L2​dtdi2​​±Mdtdi1​​​

注意那个 **"±"** —— 这是耦合电感的**最大难点**。正负号取决于两个线圈的**绕向**。

---

### 三、同名端——解决"±"问题的关键

#### 问题的由来

两个线圈的磁场有可能**相互加强**（磁通方向一致），也可能**相互削弱**（磁通方向相反）——取决于它们的**绕线方向**和**电流方向**。

#### 工程上的约定：用"同名端"标记

电路图上画两个线圈时，根本看不出绕线方向（就是两个曲线而已）。工程上在每个线圈的其中一端标一个"**·**"（黑点）：

```
     ·               ·
   ┌──[L₁]──┐     ┌──[L₂]──┐
   │  ∿∿∿   │     │  ∿∿∿   │
```

**同名端的物理意义**：

> **两个黑点端**：若两个电流都**从黑点端流入**，则它们产生的磁通**方向相同**（相互加强）。

#### 互感电压的正负号规则

**一句话口诀**：

> **"入同出同取正，入异出异取负"** —— 看两个电流相对于同名端的"进出"关系。

**更具体地**：对互感电压 Mdi2dtM \dfrac{di_2}{dt} Mdtdi2​​ 在 u1u_1 u1​ 方程中的正负号：

- 若 i2i_2 i2​ 从 L₂ 的同名端**流入**，并且 u1u_1 u1​ 的**"+"端在 L₁ 的同名端**：取 **+**
- 反之取 **−**

#### 举个实例

```
     ·  i₁→          ·  i₂→
   ┌──[L₁]──┐      ┌──[L₂]──┐
   │        │      │        │
   + u₁ -           + u₂ -
   
   黑点都在左上方，i₁、i₂都从黑点流入
   u₁、u₂的"+"端都在黑点端
```

此时：

u1=L1di1dt+Mdi2dtu_1 = L_1 \frac{di_1}{dt} + M \frac{di_2}{dt}u1​=L1​dtdi1​​+Mdtdi2​​ u2=L2di2dt+Mdi1dtu_2 = L_2 \frac{di_2}{dt} + M \frac{di_1}{dt}u2​=L2​dtdi2​​+Mdtdi1​​

（两个互感项都取 **+**，因为电流和黑点端的关系"入同出同"）

### 四、耦合系数 k

问题：互感 M 能不能无穷大？

**不能**。物理限制：**一个线圈的磁场，不可能全部都穿过另一个线圈**（总有漏磁）。

定义**耦合系数**：

k=ML1L2,0≤k≤1k = \frac{M}{\sqrt{L_1 L_2}}, \quad 0 \leq k \leq 1k=L1​L2​​M​,0≤k≤1

- k=0k = 0 k=0：完全不耦合（两线圈离得远，磁场没交集）
- k=1k = 1 k=1：**全耦合**（所有磁通都共享，没有漏磁）—— 这是**理想变压器**的前提

所以 M 有一个上界：M≤L1L2M \leq \sqrt{L_1 L_2} M≤L1​L2​​
