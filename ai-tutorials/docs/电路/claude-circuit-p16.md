# 4.3 理想变压器

### 一、从耦合电感到理想变压器

变压器本质上就是一个**耦合电感**，只不过做到了**三个极端**：

1. **全耦合**：k=1k = 1 k=1（没有漏磁）
2. **无损耗**：线圈电阻 = 0，铁芯无涡流损耗
3. **电感趋于无穷大**：L1,L2,M→∞L_1, L_2, M \to \infty L1​,L2​,M→∞，但匝数比 N1:N2N_1 : N_2 N1​:N2​ 保持有限

把这三个条件"极端化"，耦合电感的方程就退化成一个超级简洁的形式——**理想变压器**。

---

### 二、理想变压器的两个黄金方程

设一次侧（primary）匝数 N1N_1 N1​，二次侧（secondary）匝数 N2N_2 N2​，定义**匝数比** n=N1/N2n = N_1 / N_2 n=N1​/N2​。

#### 电压变换

u1u2=N1N2=n\boxed{\frac{u_1}{u_2} = \frac{N_1}{N_2} = n}u2​u1​​=N2​N1​​=n​

#### 电流变换

i1i2=N2N1=1n\boxed{\frac{i_1}{i_2} = \frac{N_2}{N_1} = \frac{1}{n}}i2​i1​​=N1​N2​​=n1​​

**注意方向约定**：标准电路图里 i1i_1 i1​ 从 1' 端流入一次侧，i2i_2 i2​ 从 2' 端流出二次侧（或反过来，取决于同名端位置）。这种约定下上面两个公式直接成立。

```
      i₁ →              ← i₂
   ┌──[ ]──┐         ┌──[ ]──┐
   │  ∿∿∿  │ ·       · │  ∿∿∿ │
   + u₁ -             + u₂ -
      N₁                N₂
```

### 三、理想变压器的三个关键性质

#### 性质 ①：输入功率 = 输出功率（无损耗）

p1=u1i1=(nu2)(i2/n)=u2i2=p2p_1 = u_1 i_1 = (n u_2)(i_2 / n) = u_2 i_2 = p_2p1​=u1​i1​=(nu2​)(i2​/n)=u2​i2​=p2​

**能量从一次侧瞬时、完整地传到二次侧**。这就是"理想"的真正含义。

#### 性质 ②：阻抗变换

这是理想变压器**最有用**的性质！

假设二次侧接一个负载 ZLZ_L ZL​，则：u2=ZLi2u_2 = Z_L i_2 u2​=ZL​i2​

从一次侧看进去的等效阻抗：

Zin=u1i1=nu2i2/n=n2⋅u2i2=n2ZLZ_{in} = \frac{u_1}{i_1} = \frac{n u_2}{i_2/n} = n^2 \cdot \frac{u_2}{i_2} = n^2 Z_LZin​=i1​u1​​=i2​/nnu2​​=n2⋅i2​u2​​=n2ZL​ Zin=n2ZL\boxed{Z_{in} = n^2 Z_L}Zin​=n2ZL​​

**一次侧看到的负载 = 实际负载 × 匝数比的平方**。

**工程应用**：这正是**阻抗匹配**的关键技术（第五章最大功率传输会再提）。

#### 性质 ③：注意它**不是**普通电感的极限

理想变压器**不储能**！它只是"瞬时"地把能量从一侧传到另一侧，不像电感那样储存磁场能。这是一个重要的概念区别。

### 四、举个简单例子

```
   ─[10V电源]──┤理想变压器 n=2:1├──[4Ω 负载]─
```

一次侧 10V，匝数比 2:1（即 n = 2）。

**二次侧电压**：u2=u1/n=10/2=5u_2 = u_1 / n = 10/2 = 5 u2​=u1​/n=10/2=5 V

**二次侧电流**：i2=u2/4=5/4=1.25i_2 = u_2 / 4 = 5/4 = 1.25 i2​=u2​/4=5/4=1.25 A

**一次侧电流**：i1=i2/n=1.25/2=0.625i_1 = i_2 / n = 1.25/2 = 0.625 i1​=i2​/n=1.25/2=0.625 A

**一次侧看到的等效阻抗**：Zin=n2⋅4=4⋅4=16Z_{in} = n^2 \cdot 4 = 4 \cdot 4 = 16 Zin​=n2⋅4=4⋅4=16 Ω

**验证**：一次侧 10V / 16Ω = 0.625 A ✓

**功率**：

- 一次侧：10×0.625=6.2510 \times 0.625 = 6.25 10×0.625=6.25 W
- 二次侧：5×1.25=6.255 \times 1.25 = 6.25 5×1.25=6.25 W ✓（相等，无损耗）
