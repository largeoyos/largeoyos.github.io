# 5.1 正弦量

### 一、为什么全世界的电都是正弦？

先聊个开放问题，你凭直觉回答：

**你家墙上的插座，电压是 220V（中国）或 110V（美国）。这个电压是恒定不变的吗？还是变化的？如果变化，怎么变？**

---

答案你可能知道：**是正弦变化的**，50Hz（中国）或 60Hz（美国）。

但为什么是正弦？不能是方波、三角波？原因很多（发电机原理、能量传输效率、变压器只对变化的量工作……），但最根本的一个数学原因是：

> **正弦是线性系统的"特征函数"**——正弦波经过任何线性电路（R、L、C 组合），出来还是**同频率**的正弦波，只是幅值和相位可能变了。

这是后面所有分析的基石。

---

### 三、有效值（RMS）—— 一个重要概念

**思考题**：你家插座说"220V"，这个 220 是正弦的**峰值** ImI_m Im​ 吗？

答案：**不是！** 220V 是**有效值**。

#### 有效值的定义（物理意义）

> **有效值 = 能在电阻上产生相同平均发热功率的直流电压/电流值。**

比如说，你家 220V 正弦电压给一个 100Ω 电阻供电，它平均发热功率是多少？

**用有效值算**：P=U有效2/R=2202/100=484P = U_{有效}^2 / R = 220^2 / 100 = 484 P=U有效2​/R=2202/100=484 W

这就等于**用 220V 直流电压接同一个电阻的发热功率**。有效值的好处就是：**算平均功率时可以直接当直流用**。

#### 有效值与峰值的关系（对正弦）

经过数学推导（对 cos⁡2\cos^2 cos2 在一个周期内求平均，取平方根）：

U=Um2≈0.707 UmU = \frac{U_m}{\sqrt{2}} \approx 0.707 \, U_mU=2​Um​​≈0.707Um​

所以 220V 的实际峰值是：Um=220×2≈311U_m = 220 \times \sqrt{2} \approx 311 Um​=220×2​≈311 V

**电气工程的默认约定**：说电压/电流时，**默认是有效值**。只有用"峰值"、"幅值"这些词时才指 UmU_m Um​。

### 四、相位差

两个**同频率**的正弦波之间可以比较"相位差"：

u1=Umcos⁡(ωt+φ1),u2=Umcos⁡(ωt+φ2)u_1 = U_m \cos(\omega t + \varphi_1), \quad u_2 = U_m \cos(\omega t + \varphi_2)u1​=Um​cos(ωt+φ1​),u2​=Um​cos(ωt+φ2​)

**相位差**：φ=φ1−φ2\varphi = \varphi_1 - \varphi_2 φ=φ1​−φ2​

- φ>0\varphi > 0 φ>0：u1u_1 u1​ **超前** u2u_2 u2​（u1u_1 u1​ 先达到峰值）
- φ<0\varphi < 0 φ<0：u1u_1 u1​ **滞后** u2u_2 u2​
- φ=0\varphi = 0 φ=0：**同相**
- φ=±π\varphi = \pm\pi φ=±π：**反相**

> ⚠️ **前提**：只有同频率的正弦波才能比相位差！不同频率比相位差是无意义的。
