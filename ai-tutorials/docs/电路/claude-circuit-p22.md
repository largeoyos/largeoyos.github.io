# 5.4 阻抗与导纳

### 一、阻抗的定义

> **阻抗**（Impedance）ZZ Z：在相量域，一个二端元件（或网络）的电压相量与电流相量之比。
> 
> Z=U˙I˙Z = \frac{\dot{U}}{\dot{I}}Z=I˙U˙​

单位：**欧姆 Ω**（和电阻相同）。

**注意**：阻抗是一个**复数**，不是时域的物理量（不要试图写 Z(t)Z(t) Z(t)）。

---

### 二、阻抗的几种形式

**代数形式**：

Z=R+jXZ = R + jXZ=R+jX

其中：

- RR R：**电阻**分量（实部）
- XX X：**电抗**分量（虚部），单位也是欧姆
    - X>0X > 0 X>0：**感性**（电感为主）
    - X<0X < 0 X<0：**容性**（电容为主）

**极坐标形式**：

Z=∣Z∣∠φZZ = |Z| \angle \varphi_ZZ=∣Z∣∠φZ​

- ∣Z∣=R2+X2|Z| = \sqrt{R^2 + X^2} ∣Z∣=R2+X2​：**阻抗模**
- φZ=arctan⁡(X/R)\varphi_Z = \arctan(X/R) φZ​=arctan(X/R)：**阻抗角**

**阻抗角的物理意义**：电压相位超前电流相位的角度。

φZ=φu−φi\varphi_Z = \varphi_u - \varphi_iφZ​=φu​−φi​

---

### 三、三个基本元件的阻抗

**电阻**：ZR=RZ_R = R ZR​=R（纯实数）

**电感**：ZL=jωLZ_L = j\omega L ZL​=jωL（纯虚数，正虚部）

- 电感的电抗 XL=ωLX_L = \omega L XL​=ωL，称为**感抗**
- **频率越高，感抗越大**（电感对高频的阻碍越强）

**电容**：ZC=1jωC=−jωCZ_C = \dfrac{1}{j\omega C} = -\dfrac{j}{\omega C} ZC​=jωC1​=−ωCj​（纯虚数，负虚部）

- 电容的电抗 XC=−1ωCX_C = -\dfrac{1}{\omega C} XC​=−ωC1​，称为**容抗**
- **频率越高，容抗越小**（电容对高频的阻碍越弱）

---

### 四、阻抗的直观理解

**感抗 XL=ωLX_L = \omega L XL​=ωL 随频率变化**：

```
  X_L
   ↑
   │      ╱
   │    ╱
   │  ╱
   │╱
 0 ┼────────→ ω
```

**容抗 ∣XC∣=1/(ωC)|X_C| = 1/(\omega C) ∣XC​∣=1/(ωC) 随频率变化**：

```
  |X_C|
   ↑
   │╲
   │ ╲
   │  ╲____
   │       ─────
 0 ┼────────→ ω
```

**两个重要极限**：

- ω→0\omega \to 0 ω→0（直流）：ZL→0Z_L \to 0 ZL​→0（短路），ZC→∞Z_C \to \infty ZC​→∞（开路）—— 这和第四章结论一致 ✓
- ω→∞\omega \to \infty ω→∞（高频）：ZL→∞Z_L \to \infty ZL​→∞（开路），ZC→0Z_C \to 0 ZC​→0（短路）

**工程解读**：

- 电感"喜欢直流，讨厌高频"——常用来做**扼流圈**（阻止高频干扰）
- 电容"讨厌直流，喜欢高频"——常用来做**耦合电容**（隔直流、通交流）

### 五、阻抗的串并联（和电阻完全一样！）

这是相量法最妙的地方——**阻抗的串并联公式，和电阻的串并联公式形式完全相同**，只不过把实数换成复数：

**串联**：Z总=Z1+Z2+⋯Z_{总} = Z_1 + Z_2 + \cdots Z总​=Z1​+Z2​+⋯

**并联**：1Z总=1Z1+1Z2+⋯\dfrac{1}{Z_{总}} = \dfrac{1}{Z_1} + \dfrac{1}{Z_2} + \cdots Z总​1​=Z1​1​+Z2​1​+⋯

（两个阻抗并联：Z总=Z1Z2Z1+Z2Z_{总} = \dfrac{Z_1 Z_2}{Z_1 + Z_2} Z总​=Z1​+Z2​Z1​Z2​​）

**分压、分流公式也完全一样**，只是用复数运算。

> 💡 这意味着：第二章学的所有分析方法（节点法、回路法）、第三章学的所有定理（戴维南、叠加……），**在正弦稳态下全部适用**，只要把"电阻"换成"阻抗"，把"电压电流"换成"相量"即可。

### 六、导纳 Y

**导纳**（Admittance）是阻抗的倒数：

Y=1Z=I˙U˙Y = \frac{1}{Z} = \frac{\dot{I}}{\dot{U}}Y=Z1​=U˙I˙​

单位：**西门子 S**

**代数形式**：

Y=G+jBY = G + jBY=G+jB

- GG G：**电导**（Conductance）
- BB B：**电纳**（Susceptance）

**重要陷阱**⚠️：G≠1/RG \neq 1/R G=1/R，B≠1/XB \neq 1/X B=1/X！正确的转换是：

Y=1Z=1R+jX=R−jXR2+X2Y = \frac{1}{Z} = \frac{1}{R+jX} = \frac{R-jX}{R^2+X^2}Y=Z1​=R+jX1​=R2+X2R−jX​

所以 G=RR2+X2G = \dfrac{R}{R^2+X^2} G=R2+X2R​，B=−XR2+X2B = \dfrac{-X}{R^2+X^2} B=R2+X2−X​

**什么时候用导纳？**

- 分析**并联**电路时，导纳相加更方便（因为 Y总=Y1+Y2+⋯Y_{总} = Y_1 + Y_2 + \cdots Y总​=Y1​+Y2​+⋯）
- 用**节点法**分析正弦电路时，导纳是主角
