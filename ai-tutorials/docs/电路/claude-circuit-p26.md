# 5.7 最大功率传输定理（⭐ 重点）

你的教学大纲里把这列为**重点**之一，我们重点讲。

### 一、问题场景

```
    ┌──────────┐         
    │  U_oc    │ a
```

给定一个等效电源（戴维南形式：U˙ocU˙oc​ 串联 ZthZth​），问：

> **负载阻抗 ZLZL​ 取多少，能使负载上获得的有功功率 PLPL​ 最大？**

这是工程上非常常见的问题——比如音响放大器如何匹配喇叭、天线如何匹配收发电路、电池如何向外设供电等等，都涉及"最大功率传输"。

### 二、答案（三种情况）

设 Zth=Rth+jXthZth​=Rth​+jXth​，ZL=RL+jXLZL​=RL​+jXL​。

#### 情况一：ZLZL​ 完全可调（实部虚部都能调）

**结论**：

ZL=Zth∗=Rth−jXthZL​=Zth∗​=Rth​−jXth​​

**负载 = 戴维南阻抗的共轭复数**。这叫"**共轭匹配**"。

**为什么？**

电流大小 I=Uoc∣Zth+ZL∣I=∣Zth​+ZL​∣Uoc​​。负载有功功率 PL=I2RLPL​=I2RL​。

想让 II 最大：Zth+ZLZth​+ZL​ 越小越好。如果 ZLZL​ 的虚部可以任意调，选 XL=−XthXL​=−Xth​，两个虚部抵消！于是 Zth+ZL=Rth+RLZth​+ZL​=Rth​+RL​（纯实数，最小）。

然后问题变成：PL=Uoc2RL(Rth+RL)2PL​=(Rth​+RL​)2Uoc2​RL​​，对 RLRL​ 求极值 → RL=RthRL​=Rth​。

**最大功率值**：

PL,max⁡=Uoc24RthPL,max​=4Rth​Uoc2​​​

（这里 UocUoc​ 是**有效值**）

#### 情况二：ZLZL​ 的虚部固定，只能调实部（比如纯电阻负载）

（这时候 PLPL​ 不能达到 Uoc2/(4Rth)Uoc2​/(4Rth​)，比那个小。）

### 三、物理理解

**为什么是"共轭匹配"？**

共轭匹配的本质：**让电源和负载的"虚部"互相抵消**。

- 电源虚部 +jXth+jXth​ 代表电源内部的储能元件
- 负载虚部 −jXth−jXth​ 代表负载的储能元件
- 两者**互为对偶**——电源释放无功时负载正好吸收，反之亦然
- **无功功率全部在电源和负载之间闭环循环，不占用"实功率通道"**
- 剩下的就是一个纯电阻 RthRth​ 和 RLRL​ 的匹配，RL=RthRL​=Rth​ 时实功率最大（这一步你用直流电路的最大功率定理就熟悉了）
