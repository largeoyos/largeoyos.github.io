# 第六章 频率响应与谐振

第五章我们学了相量法，但所有分析都假定"**一个固定频率**"。现在我们要问一个新问题：

> **如果频率 ω 变化，电路的"响应"会怎么变？**

这就是**频率响应**（frequency response）——电路的"滤波特性"。收音机调台、音响的高低音调节、手机天线、5G 通信……所有这些都离不开频率响应。
