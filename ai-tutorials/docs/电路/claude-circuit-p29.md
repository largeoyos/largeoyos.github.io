# 6.1 网络函数和频率特性

### 一、先从一个问题开始

**思考**：我们以前讲电路时，总说"给定一个激励，求响应"。比如：

- **激励**：电压源 U˙sU˙s​
- **响应**：某支路电流 I˙I˙，或某节点电压 U˙U˙

**问题**：如果我固定激励的幅值和初相（比如 U˙s=1∠0°U˙s​=1∠0°），但改变它的**频率** ωω，响应 I˙I˙ 或 U˙U˙ 会怎么变？

---

**答案**：响应的幅值和相位都会变，因为**阻抗 Z 是 ω 的函数**（ZL=jωLZL​=jωL、ZC=1/(jωC)ZC​=1/(jωC)）！

这就是频率响应的起点。

---

### 二、网络函数 H(jω) 的定义

我们把"响应/激励"的比值定义为**网络函数**：

H(jω)=响应相量激励相量H(jω)=激励相量响应相量​​

**注意几个关键点**：

1. H(jω)H(jω) 是 **ω 的复函数**——不同频率下它是不同的复数
2. 它**只取决于电路本身**（元件值和结构），与激励大小无关
3. "响应"和"激励"可以是同一端口也可以是不同端口

#### 四种常见的网络函数

|类型|激励类型|响应类型|记号|物理意义|
|---|---|---|---|---|
|**驱动点阻抗**|电流|同一端口的电压|Z(jω)Z(jω)|一个端口看进去的阻抗|
|**驱动点导纳**|电压|同一端口的电流|Y(jω)Y(jω)|阻抗的倒数|
|**电压传递函数**|电压|另一端口电压|Hu(jω)Hu​(jω)|电压放大/衰减比|
|**电流传递函数**|电流|另一端口电流|Hi(jω)Hi​(jω)|电流放大/衰减比|

（还有**转移阻抗、转移导纳**等，略。）

---

### 三、H(jω) 的两个面

作为复函数，H(jω)H(jω) 可以分解成**模**和**辐角**：

H(jω)=∣H(jω)∣⋅ejφ(ω)H(jω)=∣H(jω)∣⋅ejφ(ω)

其中：

- **幅频特性** ∣H(jω)∣∣H(jω)∣：幅值如何随频率变化
- **相频特性** φ(ω)φ(ω)：相位如何随频率变化

**画图表示**：把两者分别作为 ω 的函数画出来，就得到"**幅频特性曲线**"和"**相频特性曲线**"。这两条曲线**完全描述了电路的频率特性**。

### 四、一个极简例子：RC 低通滤波器

```
   ┌──[R]──┬────── u_2（输出）
   │       │
   │       │
   u_1    [C]
   │       │
   │       │
   └───────┴──────
```

**输入** U˙1U˙1​（跨在整个电路上），**输出** U˙2U˙2​（从电容两端取）。

**电压传递函数**（分压器原理）：

H(jω)=U˙2U˙1=ZCR+ZC=1/(jωC)R+1/(jωC)=11+jωRCH(jω)=U˙1​U˙2​​=R+ZC​ZC​​=R+1/(jωC)1/(jωC)​=1+jωRC1​

这是一个**标准形式**。令 ωc=1/(RC)ωc​=1/(RC)（待会你会看到它的重要性），则：

H(jω)=11+j(ω/ωc)H(jω)=1+j(ω/ωc​)1​

#### 分析幅频特性

∣H(jω)∣=11+(ω/ωc)2∣H(jω)∣=1+(ω/ωc​)2​1​

**三个关键频率点**：

| 频率 | ω/ωcω/ωc​ | ∣H∣∣H∣ | 物理含义 | |---|---|---|---| | ω=0ω=0（直流） | 0 | **1** | 输出 = 输入，无衰减 | | ω=ωcω=ωc​ | 1 | 12≈0.7072​1​≈0.707 | 衰减到 1/21/2​，即 **−3 dB** | | ω→∞ω→∞ | ∞ | **0** | 完全衰减，输出为 0 |

**幅频曲线**（大致形状）：

```
  |H|
   1 ┤━━━━━╮
       │       ╲
   0.707        ╲─── ω_c 处（截止频率）
       │         ╲
       │           ╲_____
       │                ╲━━━━━
     0 ┼──────────────────────→ ω
       0       ω_c
```

**这是个"低通滤波器"**：低频通过（∣H∣∣H∣ 接近 1），高频被压制（∣H∣∣H∣ 接近 0）。

#### 为什么叫"低通"？—— 物理直觉

从第四章我们知道：**电容对低频像开路，对高频像短路**。

- **低频时**：电容像开路，整个电压都压在电容上 → U2=U1U2​=U1​
- **高频时**：电容像短路，电压被短路掉了 → U2→0U2​→0

这就是低通滤波的物理本质。

---

#### 截止频率 ωcωc​（—3dB 频率）

在 ω=ωc=1/(RC)ω=ωc​=1/(RC) 时 ∣H∣=1/2∣H∣=1/2​。这是一个**非常重要的频率点**，叫**截止频率**（cutoff frequency）或 **−3dB 频率**。

**为什么是 −3dB？**

在工程上人们喜欢用**分贝**表示增益：20log⁡10∣H∣20log10​∣H∣ dB

- ∣H∣=1/2∣H∣=1/2​ 对应 20log⁡(1/2)=−3.0120log(1/2​)=−3.01 dB
- 功率上看，∣H∣2=1/2∣H∣2=1/2，功率**减半**

所以截止频率也叫"**半功率频率**"——在这个频率上，输出功率降到最大值的一半。

---

#### 分析相频特性

φ(ω)=−arctan⁡(ω/ωc)φ(ω)=−arctan(ω/ωc​)

|频率|相位|
|---|---|
|ω=0ω=0|0°0°（同相）|
|ω=ωcω=ωc​|−45°−45°|
|ω→∞ω→∞|−90°−90°（输出滞后 90°）|

**相频曲线**：

```
  φ(ω)
    0 ┼━━━━━╮
      │     ╲
   -45°      ╲_ω_c
      │        ╲
      │          ╲____

```

**意义**：随着频率升高，输出电压相位**越来越滞后**。

### 五、类似地：高通滤波器

把 RC 的 R 和 C 位置互换，输出从 R 上取：

```
   ┌──[C]──┬────── u_2
   │       │
   u_1    [R]
   │       │
   └───────┴──────
```

**传递函数**：

H(jω)=RR+1/(jωC)=jωRC1+jωRCH(jω)=R+1/(jωC)R​=1+jωRCjωRC​

**幅频特性**：低频 → 0，高频 → 1。这是**高通滤波器**。

物理直觉：**电容隔直流、通交流**。低频被电容挡住，只剩高频能过。

### 六、波特图（Bode Plot）简介

工程上通常用**波特图**绘制频率特性，特点：

- **横轴**：ω 取**对数刻度**（比如 1, 10, 100, 1000 等比分布）
- **纵轴（幅频）**：用 **分贝**（20log⁡∣H∣20log∣H∣）
- **纵轴（相频）**：用度或弧度，线性刻度

波特图的好处是：**复杂传递函数可以分解成"零点"和"极点"的直线近似叠加**，画起来非常方便。

这部分属于信号与系统课程，电路课程只需知道有这个概念。
