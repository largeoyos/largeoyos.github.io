# 1.2 基尔霍夫定律（KCL 和 KVL）

这是你高中没学过的内容，也是全课程的**两大基石**。后面所有的电路分析方法（支路法、回路法、节点法）本质上都是在反复使用这两条定律。

### 先建立几个基础概念

看一个电路的"骨架"，需要先认识三个词：

```
         a ────[R1]──── b
         │              │
        [R2]          [R3]
         │              │
         c ────[R4]──── d
```

- **支路（Branch）**：每个二端元件就是一条支路。上图有 4 条支路（R1、R2、R3、R4）
- **节点（Node）**：支路的连接点。上图有 4 个节点（a、b、c、d）
- **回路（Loop）**：从某节点出发，沿支路走一圈回到起点，中间不重复经过节点。上图的 a→b→d→c→a 就是一个回路

> ⚠️ **注意**：在电路课里，"节点"有时指**广义节点**——几条导线直接相连的整个连通块算**一个**节点。比如下面这个：
> 
> ```
>   ──┬──┬──   
>     │  │       这三条线连在一起，只算 1 个节点
>   ──┘  │
>        │
> ```

### KCL：基尔霍夫电流定律

#### 物理本质

电荷守恒。电荷不会在节点里凭空产生，也不会凭空消失。

**换句话说**：流进一个节点的电流总量 = 流出该节点的电流总量

#### 数学表达

对任意节点，取"流入为正，流出为负"（或反过来，但必须统一），则：

∑kik=0\sum_{k} i_k = 0k∑​ik​=0

#### 看个具体例子

```
          I1 ↓
           │
    I2 →───┼───→ I3
           │
           ↓ I4
```

节点处 KCL：**流入 = 流出**

I1+I2=I3+I4I_1 + I_2 = I_3 + I_4I1​+I2​=I3​+I4​

或者统一写成"流入为正"的形式：

I1+I2−I3−I4=0I_1 + I_2 - I_3 - I_4 = 0I1​+I2​−I3​−I4​=0

#### KCL 的推广（重要！）

KCL 不仅对单个节点成立，**对任意一个"闭合面"（高斯面）都成立**。

```
        ┌────────────────┐
        │    I1 →  ┌─┐   │  ← I3
        │          │ │   │
        │    I2 →  └─┘   │  ← I4
        └────────────────┘
              虚线闭合面
```

流入这个框的电流 = 流出的电流：I1+I2=I3+I4I_1 + I_2 = I_3 + I_4 I1​+I2​=I3​+I4​

这个推广在分析复杂电路时非常好用——**你可以把一个复杂的子电路当成一个大"节点"来处理**。

### KVL：基尔霍夫电压定律

#### 物理本质

能量守恒。沿任意闭合回路绕一圈，电位的升降必然抵消（否则就意味着电子绕一圈能白赚能量，永动机就成立了）。

#### 数学表达

沿任意闭合回路绕一圈，各段电压的代数和为零：

∑kuk=0\sum_{k} u_k = 0k∑​uk​=0

#### 怎么定代数和的"正负"？

这是 KVL 的操作难点，我给你一个**万能规则**：

> **沿绕行方向前进时，从"−"跨到"+"取正号，从"+"跨到"−"取负号**

（也可以反过来约定，只要全程统一）

#### 看个例子

```
         ┌──→ 绕行方向（顺时针）
         │
      ┌──┴──────────────┐
      │ +           +   │
    U1│             │U2 │
      │ −           −   │
      └────[R]──────────┘
         +    UR   −
```

假设顺时针绕一圈，从左下角出发：

- 跨过 U1：从"−"到"+" → **+U1**
- 跨过 U2：从"+"到"−" → **−U2**
- 跨过 R：从"+"到"−" → **−UR**

KVL 方程：

U1−U2−UR=0U_1 - U_2 - U_R = 0U1​−U2​−UR​=0

即 UR=U1−U2U_R = U_1 - U_2 UR​=U1​−U2​

### 🎯 一个完整的例题（把 KCL+KVL 用起来）

```
        I1      a       I2
    ┌───→────┬───→─────┐
    │        │         │
   (+)       │        (+)
   10V      [R2]      4V
    │       2Ω         │
   (−)       │        (−)
    │        │         │
    └────────┴─────────┘
           b (地)
         
         R1=1Ω     R3=1Ω
    （R1在10V和a之间，R3在a和4V之间）
```

重新用更清楚的图描述：左边 10V 电源通过 R1=1Ω 接到节点 a；节点 a 通过 R2=2Ω 接到地 b；右边 4V 电源通过 R3=1Ω 也接到节点 a；电源另一端都接到地。

**设电流参考方向**：

- I1：从 10V 经 R1 流向 a（向右）
- I2：从 4V 经 R3 流向 a（向左）
- I3：从 a 经 R2 流向地（向下）

**节点 a 的 KCL**：流入 = 流出

I1+I2=I3...①I_1 + I_2 = I_3 \quad \text{...①}I1​+I2​=I3​...①

**左回路 KVL**（10V、R1、R2 组成的回路，顺时针绕）：

10−I1⋅1−I3⋅2=0...②10 - I_1 \cdot 1 - I_3 \cdot 2 = 0 \quad \text{...②}10−I1​⋅1−I3​⋅2=0...②

**右回路 KVL**（4V、R3、R2 组成的回路，逆时针绕）：

4−I2⋅1−I3⋅2=0...③4 - I_2 \cdot 1 - I_3 \cdot 2 = 0 \quad \text{...③}4−I2​⋅1−I3​⋅2=0...③

三个方程三个未知数，解出：

- I1=3.2AI_1 = 3.2\text{A} I1​=3.2A
- I2=−0.4AI_2 = -0.4\text{A} I2​=−0.4A（负号说明实际方向与假设相反，电流其实从 a 流向 4V 电源——4V 被"倒灌"充电了！）
- I3=2.8AI_3 = 2.8\text{A} I3​=2.8A

---

### 🔑 本节你需要记住的几件事

1. **KCL = 电荷守恒**（节点处 ∑i=0\sum i = 0 ∑i=0，可推广到任意闭合面）
2. **KVL = 能量守恒**（回路中 ∑u=0\sum u = 0 ∑u=0）
3. **两个定律都和元件是什么无关**——电阻、电容、电感、电源、二极管都适用
4. **正负号的关键是参考方向**：先设方向，再列方程，负号自己会说话
