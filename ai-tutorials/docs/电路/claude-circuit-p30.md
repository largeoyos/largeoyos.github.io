# 6.2 RLC 串联电路的频率特性

我们从一个简单的 RLC 串联电路开始：

### 一、总阻抗作为 ω 的函数

总阻抗：

Z(jω)=R+jωL+1jωC=R+j(ωL−1ωC)Z(jω)=R+jωL+jωC1​=R+j(ωL−ωC1​)

**实部**：RR（不随频率变化）

**虚部**：X(ω)=ωL−1ωCX(ω)=ωL−ωC1​

虚部这一部分超级重要，它揭示了 **感抗和容抗的竞争**：

- **低频时**：ωLωL 小，1/(ωC)1/(ωC) 大 → X<0X<0，**容性占优**
- **高频时**：ωLωL 大，1/(ωC)1/(ωC) 小 → X>0X>0，**感性占优**
- **中间某个频率**：ωL=1/(ωC)ωL=1/(ωC) → X=0X=0，**感抗容抗正好抵消！**

这个"中间频率"就是**谐振频率**（下一节的主角）。
