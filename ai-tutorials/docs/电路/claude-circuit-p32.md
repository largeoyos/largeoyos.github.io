# 6.4 并联谐振电路

### 一、电路与"对偶"

```

         │ │      │
```

### 二、并联谐振的三个特征

**特征 ①**：谐振时 Y=GY=G（纯电导），即 Z=RZ=R（纯电阻，阻抗达到**最大值**）

**特征 ②**：当激励是**电流源**时，端电压**最大**（因为阻抗最大）

U0=Is⋅RU0​=Is​⋅R

**特征 ③**：电感和电容的**支路电流**可能远大于电源电流！

IL=IC=Q⋅IsIL​=IC​=Q⋅Is​

其中并联电路的品质因数：

Q=Rω0L=ω0RC=RCLQ=ω0​LR​=ω0​RC=RLC​​​

（注意：并联电路的 Q 公式和串联**相反**！R 在分子上，不是分母。）

### 三、串联 vs 并联谐振对比（⭐ 记忆要点）

|性质|串联谐振|并联谐振|
|---|---|---|
|典型激励|电压源|电流源|

### 四、为什么有"对偶性"？

观察两个电路：

- 串联：KVL 绕一圈，∑Uk=0∑Uk​=0
- 并联：KCL 节点处，∑Ik=0∑Ik​=0

KCL 和 KVL 结构完全相同，只是把"电流"和"电压"互换。再加上阻抗和导纳互为倒数——**整个理论结构都是对偶的**。

|对偶关系|
|---|
|电压 ↔ 电流|
|阻抗 ↔ 导纳|
|串联 ↔ 并联|
|KVL ↔ KCL|
|电感 L ↔ 电容 C|
|电阻 R ↔ 电导 G|
|戴维南 ↔ 诺顿|

学会"对偶"思想：**记住一个结论，自动就会对偶结论**，学习效率翻倍。
