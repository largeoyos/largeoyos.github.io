# 7.2 非正弦周期量的平均值、有效值、平均功率

### 一、平均值

直接按定义：

Favg=1T∫0Tf(t) dt=A0Favg​=T1​∫0T​f(t)dt=A0​

**即平均值 = 直流分量**。这个结论很朴素——傅里叶级数里所有正弦项一个周期内平均为 0，只剩下直流项。

---

### 二、有效值（⭐ 重要且反直觉）

我们先回忆：正弦信号的有效值是 Am/2Am​/2​。那对于非正弦的周期信号呢？

**按定义计算**：

F2=1T∫0Tf2(t) dtF2=T1​∫0T​f2(t)dt

把傅里叶级数代入，展开后利用正弦函数的**正交性**（不同频率正弦的乘积在一个周期内积分为 0），得到：

F=A02+∑k=1∞(Akm2)2=A02+A12+A22+⋯F=A02​+k=1∑∞​(2​Akm​​)2​=A02​+A12​+A22​+⋯​​

其中 Ak=Akm/2Ak​=Akm​/2​ 是 k 次谐波的**有效值**。

#### 这个结论非常漂亮——它说了什么？

> **总有效值的平方 = 各谐波有效值的平方之和**

**关键观察**：有效值"可以加"，但要**取平方再加**。这很像勾股定理——**各谐波在"有效值空间"里是正交的**。

---

### 三、平均功率（同样的"正交性"）

对一个元件，瞬时功率 p(t)=u(t)i(t)p(t)=u(t)i(t)。

设：

- u(t)=U0+∑k=1∞Ukmcos⁡(kω1t+φuk)u(t)=U0​+∑k=1∞​Ukm​cos(kω1​t+φuk​)
- i(t)=I0+∑k=1∞Ikmcos⁡(kω1t+φik)i(t)=I0​+∑k=1∞​Ikm​cos(kω1​t+φik​)

**平均功率**（一个周期内积分取平均）：

P=1T∫0Tu(t)i(t) dtP=T1​∫0T​u(t)i(t)dt

**关键数学事实**：**不同频率**的正弦波的乘积在一个周期内平均为 0；**同频率**的正弦波乘积才有贡献。

所以：

P=U0I0+∑k=1∞UkIkcos⁡φkP=U0​I0​+k=1∑∞​Uk​Ik​cosφk​​

其中 Uk,IkUk​,Ik​ 是 k 次谐波的**有效值**，φk=φuk−φikφk​=φuk​−φik​ 是 k 次谐波的电压电流相位差。

#### 解读这个公式

> **总平均功率 = 直流功率 + 各次谐波平均功率之和**

P=P0+P1+P2+P3+⋯P=P0​+P1​+P2​+P3​+⋯

**惊人的结论**：**不同频率的电压电流相乘，对平均功率没贡献**！只有同频率才能产生平均功率。

这点非常重要——它告诉我们：

> **想要一个元件传递能量，电压和电流必须在"同一个频率"上**。

### 四、一个重要陷阱

**⚠️ 叠加原理只能叠加响应，不能叠加功率**（第三章我们强调过）。

但这里我们却说"总功率 = 各次谐波功率之和"，这不是矛盾吗？

**并不矛盾**。理由：

- **不能叠加的**：同一频率下，多个电源产生的功率（因为电流是相加的）
- **能叠加的**：不同频率下，每个频率分量的功率（因为不同频率正交，乘积积分为 0）

"同频率不能叠加功率，不同频率可以叠加功率"——这是关键区别。
