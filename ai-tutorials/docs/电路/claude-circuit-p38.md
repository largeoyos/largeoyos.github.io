# 7.4 傅里叶级数的指数形式与傅里叶变换（简介）

### 一、指数形式（为什么又要一种形式？）

之前的"cos⁡+sin⁡cos+sin"形式是**实数形式**。但电路分析中我们已经习惯了复数（相量），所以有一个**指数形式**更匹配：

利用欧拉公式 cos⁡θ=(ejθ+e−jθ)/2cosθ=(ejθ+e−jθ)/2，把傅里叶级数改写成：

f(t)=∑k=−∞∞Fk ejkω1tf(t)=k=−∞∑∞​Fk​ejkω1​t​

其中 FkFk​ 是一个**复系数**：

Fk=1T∫0Tf(t) e−jkω1t dtFk​=T1​∫0T​f(t)e−jkω1​tdt

**注意**：这里 k 从 **−∞ 到 +∞**（出现了"负频率"）。

负频率的物理解释：F−kF−k​ 和 FkFk​ 互为共轭（对实信号而言），两者合起来代表一个实数谐波。负频率是数学上的"配对物"，物理上没有独立意义。

**好处**：形式更简洁、统一，和相量理论无缝衔接。特别是到第十章拉普拉斯变换时，指数形式是基础。

### 二、从傅里叶级数到傅里叶变换（关键一步）

**傅里叶级数**：处理**周期信号**（离散频谱——只有 kω1kω1​ 这些离散频率）

**但很多信号是非周期的**：比如一次性脉冲、阶跃信号、语音信号……

**思想**：把周期 T→∞T→∞，让周期"无限大"，这样信号就"不重复"了——非周期信号可以看作"周期无限大的周期信号"。

**数学结果**：离散频谱 → 连续频谱；求和 → 积分。

**傅里叶变换对**：

**物理意义**：F(jω)F(jω) 是信号 f(t)f(t) 的"**频谱**"——告诉你信号在每一个频率上的"密度"。

**这就是第十章拉普拉斯变换的"原型"**。拉普拉斯变换是傅里叶变换的推广：把 jωjω 换成 s=σ+jωs=σ+jω，能处理更一般的信号（包括不稳定的发散信号）。
