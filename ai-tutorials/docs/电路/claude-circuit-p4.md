### 一、电阻元件

#### 定义

电阻是一种**耗能元件**，它的伏安特性（VCR，Voltage-Current Relation）由欧姆定律描述。

**在关联参考方向下**：

u=Ri或i=Guu = Ri \quad \text{或} \quad i = Guu=Ri或i=Gu

其中 G = 1/R 是**电导**（单位：西门子 S）。

#### 高中没强调的几个点

**① 非关联参考方向下要加负号**：

```
     I →
   ────────
   −  U  +
   ──[R]──
```

此时：u=−Riu = -Ri u=−Ri

**② 电阻的功率恒为非负**：

P=ui=Ri2=Gu2≥0P = ui = Ri^2 = Gu^2 \geq 0P=ui=Ri2=Gu2≥0

电阻永远在消耗功率，不会释放。这是它作为**无源元件**的本质。

**③ 线性 vs 非线性**：

- **线性电阻**：R 是常数（本课程默认都是这种）
- **非线性电阻**：R 随 u 或 i 变化（如二极管）

**④ 两个极端**：

- R → 0：**短路**（两端电压恒为 0，电流任意）
- R → ∞：**开路**（电流恒为 0，电压任意）

### 二、独立电源

#### 理想电压源

**定义**：两端电压**强制**为某个定值（或定函数）us(t)u_s(t) us​(t)，与流过的电流**无关**。

```
      +
    ┌─┴─┐
    │u_s│         符号：带 + - 的圆圈
    └─┬─┘
      −
```

**关键性质**：

- 电流由**外电路**决定，可以任意大小、任意方向
- **不能短路**！短路会导致 us=0u_s = 0 us​=0，与定义矛盾（物理上意味着无穷大电流）

#### 理想电流源

**定义**：流出的电流**强制**为某个定值（或定函数）is(t)i_s(t) is​(t)，与两端电压**无关**。

```
     ↑
    ┌┴┐
    │ │         符号：带箭头的圆圈
    │i│         箭头方向 = 电流流出方向
    │s│
    └┬┘
```

**关键性质**：

- 电压由**外电路**决定
- **不能开路**！开路意味着 is=0i_s = 0 is​=0，矛盾（物理上意味着无穷大电压）

#### 一个重要的思维点

独立源的**"倔强"**是它的本质：

- 电压源"咬定"电压不放，电流任你外电路摆布
- 电流源"咬定"电流不放，电压任你外电路摆布

> 💡 后面做戴维南/诺顿等效、电路化简时，这两种"倔强"经常需要你灵活转化。

### 三、受控源（重点！）

这是你列出的难点之一，我们慢慢讲。

#### 为什么需要受控源？

先问你一个问题：**三极管（晶体管）怎么用电路模型描述？**

三极管的工作原理：基极小电流 iBi_B iB​ 控制集电极大电流 iC=βiBi_C = \beta i_B iC​=βiB​。

这个关系用电阻能描述吗？❌ 不行，因为 iCi_C iC​ 不依赖于集电极两端的电压，而依赖于**另一条支路**的电流。

用独立源能描述吗？❌ 也不行，因为独立源的值是固定的，而 iCi_C iC​ 会随 iBi_B iB​ 变化。

**所以我们需要一种新元件**：它的输出值受电路中其他某个电压/电流控制——这就是**受控源**。

---

#### 四种受控源

根据"控制量"和"被控量"的组合，共有四种：

|名称|简写|被控量|控制量|方程|系数单位|
|---|---|---|---|---|---|
|压控电压源|**VCVS**|电压 u2u_2 u2​|电压 u1u_1 u1​|u2=μu1u_2 = \mu u_1 u2​=μu1​|无量纲|
|压控电流源|**VCCS**|电流 i2i_2 i2​|电压 u1u_1 u1​|i2=gu1i_2 = g u_1 i2​=gu1​|西门子 S|
|流控电压源|**CCVS**|电压 u2u_2 u2​|电流 i1i_1 i1​|u2=ri1u_2 = r i_1 u2​=ri1​|欧姆 Ω|
|流控电流源|**CCCS**|电流 i2i_2 i2​|电流 i1i_1 i1​|i2=βi1i_2 = \beta i_1 i2​=βi1​|无量纲|

**符号**：受控源用**菱形**表示，和独立源（圆形）区分开。

```
  独立电压源       受控电压源
     ○               ◇
    (+)             (+)
```

---

#### 受控源的本质：它不是电源，是"约束关系"

这是理解受控源最关键的一句话，请你好好咂摸这句话。

**普通电源**（独立源）：主动提供能量，它的值由"自己"决定。

**受控源**：它的值**完全由控制量决定**，控制量为零它就为零。它更像是一种"**依赖关系**"或"**传递函数**"，描述的是电路某一部分如何"反映"另一部分的状态。

#### 一个直观的例子

```
    i1 →                    i2 = 3·i1
  ────────┐              ┌──────────
          │              │
         [1Ω]           [◇]← CCCS
          │              │
          └──────┬───────┘
                 │
                 地
```

左边支路电流 i1i_1 i1​ 控制右边 CCCS 输出 i2=3i1i_2 = 3i_1 i2​=3i1​。

- 如果左边 i1=2Ai_1 = 2\text{A} i1​=2A，那右边 i2=6Ai_2 = 6\text{A} i2​=6A
- 如果左边 i1=0i_1 = 0 i1​=0，那右边 i2=0i_2 = 0 i2​=0——**受控源自己不会"发电"**

---

#### 受控源的 KCL/KVL 处理

**好消息**：KCL、KVL 对受控源**完全适用**，把它当普通电源列方程就行。

**难点在哪里？**：方程里会多出一个"控制量"变量，你需要**再补一个约束方程**把它和电路里的实际量挂钩。

看个小例子：

```
   10V ──[2Ω]──┬──[3Ω]── 地
        I→    │
              ◇  ← CCVS: U = 4·I (电压参考方向 +在上，−在下)
              │
              地
```

设电流 I（流过 2Ω 的电流）向右。列节点 KCL 和 KVL 时：

- 10V 支路、2Ω、3Ω 支路正常处理
- CCVS 作为一个电压为 4I4I 4I 的"电压源"处理
- **额外约束**：CCVS 的电压值 = 4 × (2Ω 支路的电流 I)

这个"额外约束"就是受控源带来的新东西。**两个未知量（I 和 CCVS 电压），两个方程（KCL/KVL + 约束）**，就能解出。
