# 第八章 三相电路

这一章是前面所有正弦稳态知识的**工程应用**。你家墙上所有插座、整个国家电网、几乎所有工业用电——背后都是三相系统。

---

### 🤔 先问你一个问题

在正式讲之前，让我先抛个问题给你思考：

**家里常见的两孔插座**（比如手机充电器用的那种），用的是**单相**电——一根"火线"、一根"零线"，两者之间就是 220V 交流电。

**但如果你见过工厂车间里的大型设备**（比如三相电机、大型空调外机、数据中心机房），它的电源插头通常是**四根线或五根线**——三根火线 + 一根零线 (+ 一根地线)，叫**三相四线制**（或三相五线制）。

**问题**：为什么工业上要用"三相"而不是简单的"单相"？多用两根线，成本不是更高吗？

### 一、三相制的三大优势（答案）

#### 优势 ①：相同功率下用铜量少

**粗略对比**（定量分析略，结论为主）：

传输相同的电功率时，三相系统的导线总截面积**比单相系统少约 25%**。

原因：三相电流在导线中"互相抵消"（下面会解释），零线中的电流在对称情况下**为零**，理论上可以省掉零线。

#### 优势 ②：三相电机性能优异

单相电机启动麻烦（需要启动电容），效率低，功率有上下波动。

**三相电机**：

- **自启动**（三相电流产生旋转磁场）
- 功率输出**非常平稳**（几乎没有 2 倍频脉动）
- 效率高、结构简单

这是三相制最大的工程优势。

#### 优势 ③：发电机天然就是三相的

发电机的三个绕组在空间上相隔 120°，天然产生三相电压——生产三相电**并不比单相贵**，反而更高效。

### 二、对称三相电压的定义

**对称三相电压**是三相制的理想情形：

uA(t)=Umcos⁡(ωt)uA​(t)=Um​cos(ωt)uB(t)=Umcos⁡(ωt−120°)uB​(t)=Um​cos(ωt−120°)uC(t)=Umcos⁡(ωt+120°)uC​(t)=Um​cos(ωt+120°)

**特点**：

1. **幅值相等**（同一个 UmUm​）
2. **频率相同**（同一个 ω）
3. **相位互差 120°**

对应的相量：

U˙A=U∠0°,U˙B=U∠−120°,U˙C=U∠120°U˙A​=U∠0°,U˙B​=U∠−120°,U˙C​=U∠120°

**相量图**：三个大小相等的向量，两两相差 120°：

```
                     
         U_C          
           ╲          
            ╲         
             ╲  120°  
              ●─────── U_A
             ╱  120°   
            ╱         
           ╱          
         U_B
```

**一个超级重要的结论**：

U˙A+U˙B+U˙C=0U˙A​+U˙B​+U˙C​=0

**证明**（你可以自己验证）：

U∠0°+U∠−120°+U∠120°U∠0°+U∠−120°+U∠120°=U(1)+U(−0.5−j⋅0.866)+U(−0.5+j⋅0.866)=U(1)+U(−0.5−j⋅0.866)+U(−0.5+j⋅0.866)=U(1−0.5−0.5)+jU(0.866−0.866)=0✓=U(1−0.5−0.5)+jU(0.866−0.866)=0✓

**物理意义**：三相电压（或三相电流）**瞬时值之和 = 0**。这是三相制省铜的根本原因——对称情况下零线电流为零。

### 三、相序

**正序（ABC 序）**：A 超前 B 120°，B 超前 C 120° **负序（ACB 序）**：顺序倒过来

工程上默认是**正序**。相序决定了三相电机的**转向**——换相序，电机反转。
