# 8.2 三相电路的连接方式

### 一、三相电源的两种连接

#### 星形连接（Y 形）

```
         A ●─────── 火线 A
           │
           │        
           ●─── Z_A
           │
    N ●────┤       ← N 点（中性点/零线）
           │
           ●─── Z_B
           │
         B ●─────── 火线 B
         
         （C 相同理）
```

每个绕组一端连在一起形成"**中性点 N**"，另一端连火线。

**从中性点 N 引出的线叫零线（中性线）。三根火线 + 一根零线 = 三相四线制。**

#### 三角形连接（Δ 形）

```
         A ●──[Z_A]──● B
                      │
                      │
                     [Z_B]
                      │
                      │
                      ● C
                      │
                     [Z_C]
                      │
                      ●（接回 A）
```

三个绕组首尾相接成三角形。**没有中性点**，只有三根火线 → 三相三线制。

### 二、两种关键电压：相电压 vs 线电压

**相电压**（Phase voltage）UpUp​：

- **每相绕组两端**的电压
- Y 接：$U_p = $ 火线对中性点电压（家用的 220V）
- Δ 接：$U_p = $ 火线对火线电压（同线电压）

**线电压**（Line voltage）UlUl​：

- **两根火线之间**的电压（如 UABUAB​、UBCUBC​、UCAUCA​）
- Y 接：$U_l = $ 两火线间电压（家用的 380V）
- Δ 接：Ul=UpUl​=Up​

### 三、Y 接下线电压和相电压的关系（⭐ 推导）

这是本章**最重要的数学关系**，我们慢慢推。

设相电压：U˙A=Up∠0°U˙A​=Up​∠0°，U˙B=Up∠−120°U˙B​=Up​∠−120°，U˙C=Up∠120°U˙C​=Up​∠120°

**线电压 U˙ABU˙AB​**（A 相对 B 相的电压）：

U˙AB=U˙A−U˙B=Up∠0°−Up∠−120°U˙AB​=U˙A​−U˙B​=Up​∠0°−Up​∠−120°=Up[(1)−(−0.5−j0.866)]=Up[1.5+j0.866]=Up​[(1)−(−0.5−j0.866)]=Up​[1.5+j0.866]=Up⋅1.52+0.8662∠arctan⁡(0.866/1.5)=Up​⋅1.52+0.8662​∠arctan(0.866/1.5)=Up3∠30°=Up​3​∠30°

**结论**：

U˙AB=3 Up∠30°U˙AB​=3​Up​∠30°​

从中提取出**两个核心关系**：

#### Y 接核心关系

1. **线电压幅值 = √3 × 相电压幅值**：Ul=3 UpUl​=3​Up​
2. **线电压超前相应相电压 30°**（UABUAB​ 超前 UAUA​ 30°）

**实例**：中国家用电 220V 相电压（火对零），对应 380V 线电压（火对火）。验证：220×3≈381220×3​≈381 V ✓

（所以你常听说的 "380/220V 三相四线"，指的就是线电压 380V、相电压 220V 的 Y 接系统。）

#### Δ 接核心关系

由对偶性（以及 Δ 接没有中性点）：

- **线电压 = 相电压**（Ul=UpUl​=Up​）
- **线电流 = √3 × 相电流**（Il=3 IpIl​=3​Ip​）

### 四、对称三相电路的分析（关键简化）

如果三相电源对称、三相负载也对称（三个负载阻抗完全相等），那么分析可以**大大简化**。

#### 关键结论："单相等效电路"法

> **对于对称 Y-Y 连接系统，可以只分析 A 相（或任一相），剩余两相直接由对称性得到（相差 ±120°）**。

#### 具体方法

考虑一个 Y-Y 对称系统：

```
  ┌─────○ A ──────[Z]──────● A'
  │     
  │     ○ B ──────[Z]──────● B'
  │           
  │     ○ C ──────[Z]──────● C'
  │             
  │     ○ N ─────────────── ● N'
  │  源端                 负载端
  └──────
```

对称情况下（各相阻抗相同、电源对称）：

**中性线上电流 = 0**！

证明：I˙A+I˙B+I˙C=0I˙A​+I˙B​+I˙C​=0（每相电流 = U˙相/ZU˙相​/Z，而三相电压之和为零）

所以即使**不接中性线**（"三相三线制"），电路也完全可以正常工作！

#### 单相等效电路

既然中性线不载流，我们可以"假装"画一根中性线，只看 A 相：

```
         ┌────[Z]────┐
         │           │
        (+)          ●
        U_A          │（假想中性线，电流 = 0）
        (-)          │
         │           │
         └───────────┘
```

一个单相的 RLC 相量电路——**回到第五章的问题了**！

**流程**：

1. 求 A 相电流 I˙A=U˙A/ZI˙A​=U˙A​/Z
2. B 相电流 I˙B=I˙A∠−120°I˙B​=I˙A​∠−120°（相位滞后 120°）
3. C 相电流 I˙C=I˙A∠+120°I˙C​=I˙A​∠+120°

### 五、对称三相功率

#### 总有功功率

每相的平均功率：P相=UpIpcos⁡φP相​=Up​Ip​cosφ（其中 φ 是相电压超前相电流的相位差，即负载阻抗角）

三相总功率：

P=3P相=3UpIpcos⁡φP=3P相​=3Up​Ip​cosφ

**用线电压和线电流表示**（工程更常用）：

**Y 接**：Up=Ul/3Up​=Ul​/3​，Ip=IlIp​=Il​ **Δ 接**：Up=UlUp​=Ul​，Ip=Il/3Ip​=Il​/3​

两种情况下都得到：

P=3 UlIlcos⁡φP=3​Ul​Il​cosφ​

**注意 φ 是阻抗角，不是 U˙lU˙l​ 和 I˙lI˙l​ 之间的相位差**——这是很多人会搞错的地方。

#### 类似地

Q=3 UlIlsin⁡φQ=3​Ul​Il​sinφS=3 UlIlS=3​Ul​Il​

### 六、瞬时功率的神奇现象

**对比单相和三相的瞬时功率**：

#### 单相电路

p(t)=UIcos⁡φ+UIcos⁡(2ωt+φu+φi)p(t)=UIcosφ+UIcos(2ωt+φu​+φi​)

**有一个 2 倍频的脉动项**——单相电机运转时有明显的扭矩波动。

#### 对称三相电路（总瞬时功率）

p(t)=pA(t)+pB(t)+pC(t)p(t)=pA​(t)+pB​(t)+pC​(t)

代入展开（数学细节略），**2 倍频的三个分量相位互差 240°（即 2×120°），加起来恰好为零**！

p(t)=3UpIpcos⁡φ（恒定值！）p(t)=3Up​Ip​cosφ（恒定值！）​

**这是三相电机平稳运转的根本原因**：总瞬时功率恒定，无任何脉动。

**这就是三相制最强大的优势**——从"电"到"机械"的能量转换非常平滑。

### 七、一道综合例题

**题目**：

一个对称三相 Y-Y 系统，电源线电压 Ul=380Ul​=380 V，负载每相阻抗 Z=6+j8Z=6+j8 Ω。

1. 相电压 UpUp​
2. 相电流 IpIp​（以 U˙AU˙A​ 为参考相量）

4. 总有功功率 P、总无功功率 Q、总视在功率 S
5. 功率因数 cos φ

---

**① 相电压**：Up=Ul/3=380/3≈220Up​=Ul​/3​=380/3​≈220 V

I˙A=U˙A/Z=220∠0°/(10∠53.13°)=22∠−53.13° AI˙A​=U˙A​/Z=220∠0°/(10∠53.13°)=22∠−53.13° A
