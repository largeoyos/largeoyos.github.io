# 9.1 动态电路的方程与初始条件

### 一、什么是动态电路？

**动态电路**：含有电容或电感（或两者都有）的电路。

**为什么叫"动态"？** 因为 L、C 的 VCR 涉及微分/积分，电路方程不再是代数方程，而是**微分方程**。

**暂态**（transient）：电路从一个稳态**切换**到另一个稳态的**过渡过程**。

典型场景：**开关动作、信号突变**。

---

### 二、"换路"与换路定则

**换路**（switching）：电路的**结构或参数突变**——开关闭合、断开，电源接入、撤去，元件值突变等。

设换路发生在 t=0t=0：

- t=0−t=0−：换路**前**的最后一瞬（稳态）
- t=0+t=0+：换路**后**的第一瞬（暂态开始）
- t>0t>0：暂态过程

**换路定则**（第四章的结论，现在正式命名）：

uC(0+)=uC(0−),iL(0+)=iL(0−)uC​(0+)=uC​(0−),iL​(0+)=iL​(0−)​

**物理根源**：储能元件的能量不能突变。

- 电容储能 12CuC221​CuC2​ → uCuC​ 突变需要无穷大电流
- 电感储能 12LiL221​LiL2​ → iLiL​ 突变需要无穷大电压

**注意**：换路定则**只**适用于 uCuC​ 和 iLiL​，对其他量（如电阻上的电压、电感上的电压、电容上的电流）**不成立**——这些量可以突变。

---

### 三、求初始值的三步法

暂态分析的第一步永远是：**求 uC(0+)uC​(0+) 和 iL(0+)iL​(0+)**。

#### 步骤

**Step 1：求 uC(0−)uC​(0−) 和 iL(0−)iL​(0−)（换路前稳态）**

换路前是**直流稳态**（假设），所以：

- 电容**等效开路**
- 电感**等效短路**

用直流电路方法求出电容两端电压、电感流过电流。

**Step 2：用换路定则得到 0+0+ 值**

uC(0+)=uC(0−),iL(0+)=iL(0−)uC​(0+)=uC​(0−),iL​(0+)=iL​(0−)

**Step 3：如果需要其他变量在 0+0+ 的值**

画出 t=0+t=0+ 时刻的等效电路：

- **电容** 等效为 **电压值为 uC(0+)uC​(0+) 的电压源**（因为它电压定了）
- **电感** 等效为 **电流值为 iL(0+)iL​(0+) 的电流源**（因为它电流定了）
- 电阻保持不变，电源按换路后的情况

然后用普通直流电路方法求任何其他量。

### 四、一道例题（感受一下流程）

```
                t=0
          ┌─○/○──[R1=2Ω]──┬────── 
          │                │       
         (+)              [C=1F]   
         12V                │       
         (-)                │       
          │                 │
          └─────[R2=4Ω]─────┘
```

（开关 t=0 时闭合；换路前开关开路很久）

**求**：uC(0+)uC​(0+) 和 iR2(0+)iR2​​(0+)

---

**Step 1：换路前稳态（开关开路，t = 0⁻）**

开关断开，左边回路断开。但电容 C 和 R2 仍构成一个回路：**但 R2 一端在开关处悬空了**。

所以换路前**整个电路没有电流**，电容上**没有充电**：

uC(0−)=0uC​(0−)=0 V

**Step 2：换路定则**

uC(0+)=uC(0−)=0uC​(0+)=uC​(0−)=0 V

**Step 3：求 iR2(0+)iR2​​(0+)**

t=0+t=0+ 时，电容电压 = 0 → 电容此时**等效为短路**（0V 电压源就是导线）。

此时等效电路：

```
          ┌──[R1=2Ω]──┬── 短路 ──
          │           │     ↑(电容换成导线)
         (+)        [R2=4Ω]
         12V          │
         (-)          │
          │           │
          └───────────┘
```

R1 串联后与 R2 并联（R2 和短路线并联）。等等——电容换成短路后，R2 被短路了！

所以电流全部走短路线：iR2(0+)=0iR2​​(0+)=0 A，iR1(0+)=12/2=6iR1​​(0+)=12/2=6 A（全部流入电容）

---

**这道题说明几件事**：

1. 换路瞬间，uCuC​ 连续（= 0），但电流可以突变（R1 上电流从 0 变成 6A）
2. 电容刚"接入"时像短路——这是重要直觉
