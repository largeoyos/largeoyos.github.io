# 9.2 一阶电路

### 一、什么是一阶电路？

**一阶电路** = 只含**一个**独立储能元件（一个 C 或一个 L）的电路（电阻随意多个）。

"一阶"因为描述它的微分方程是**一阶**的。

---

### 二、一阶电路的微分方程长什么样？

看最简单的 RC 电路：

```
      t=0
   ┌─○/○──[R]──┬──
   │           │
  (+)        [C]  uC
   U0          │
  (-)          │
   │           │
   └───────────┘
```

换路后 KVL（绕行方向顺时针）：

U0=Ri+uCU0​=Ri+uC​

用电容 VCR i=CduCdti=CdtduC​​：

U0=RCduCdt+uCU0​=RCdtduC​​+uC​

整理：

RCduCdt+uC=U0RCdtduC​​+uC​=U0​​

这就是一阶线性常微分方程。你说你会解——我们**不解它**，因为电路课里有更巧的方法。

---

### 三、三要素法（⭐ 本节核心）

**一阶电路所有响应都具有同一种形式**：

f(t)=f(∞)+[f(0+)−f(∞)] e−t/τf(t)=f(∞)+[f(0+)−f(∞)]e−t/τ​

**三个要素**：

1. **初始值** f(0+)f(0+)：换路后第一瞬间的值
2. **终值** f(∞)f(∞)：换路后新稳态的值（t→∞t→∞）
3. **时间常数** ττ：反映电路"响应快慢"的参数

只要这三个值求出来，**任何变量（电压、电流）的暂态过程都直接写出来了**，不用解微分方程！

这就是"三要素法"——电路分析里极其实用的技巧。

---

#### 为什么是这个形式？（直觉理解）

观察指数 e−t/τe−t/τ：

- t=0t=0：e0=1e0=1，此时 f=f(0+)f=f(0+) ✓
- t→∞t→∞：e−∞=0e−∞=0，此时 f=f(∞)f=f(∞) ✓
- 中间：从 f(0+)f(0+) 指数衰减（或上升）到 f(∞)f(∞)

**物理图像**：电路从一个稳态"倾斜"到另一个稳态，速度由 τ 控制。

---

#### 时间常数 τ 是什么？

**对 RC 电路**：τ=ReqCτ=Req​C **对 RL 电路**：τ=L/Reqτ=L/Req​

其中 ReqReq​ 是**从储能元件两端往外看的戴维南等效电阻**（把所有独立源置零！）

**单位检查**：

- RC 乘积：Ω · F = Ω · (C/V) = Ω · (A·s/V) = s ✓
- L/R：H/Ω = (V·s/A)/Ω = s ✓

**物理意义**：τ 小 → 电路响应快；τ 大 → 响应慢。

**经验数字**：t=3τt=3τ 后，响应已完成 95%；t=5τt=5τ 后，完成 99.3%——工程上通常认为**过了 5τ 就到稳态**了。

### 四、三要素法实战

回到刚才那道 RC 电路，U0=10U0​=10 V，R = 1 kΩ，C = 1 μF，t=0 时开关闭合，电容初始电压 0。

**要素 ①：uC(0+)uC​(0+)**

换路前电容开路、没有充电：uC(0−)=0uC​(0−)=0 → uC(0+)=0uC​(0+)=0 V

换路后**新稳态**时电容开路，i=0i=0 → 整个 R 上无压降 → 电容电压 = 电源电压：

### 五、两种典型响应（理论分类）

教材通常把一阶响应分成几类，**理解分类有助于理解物理图像**：

#### 零输入响应

**场景**：换路前电容/电感已经充能，换路后**没有外加电源**，储能元件"自己放电"。

uC(∞)=0uC​(∞)=0（最终全部放掉）

uC(t)=uC(0+) e−t/τuC​(t)=uC​(0+)e−t/τ

#### 零状态响应

**场景**：换路前储能元件"没存东西"，换路后**有外加电源激励**。

uC(0+)=0uC​(0+)=0

uC(t)=uC(∞)(1−e−t/τ)uC​(t)=uC​(∞)(1−e−t/τ)

#### 全响应

**场景**：储能元件初始有储能，换路后又有外加激励。

uC(t)=uC(∞)+[uC(0+)−uC(∞)]e−t/τuC​(t)=uC​(∞)+[uC​(0+)−uC​(∞)]e−t/τ

即 **"三要素法"的完整公式**。

**重要性质（叠加）**：

全响应=零输入响应+零状态响应全响应=零输入响应+零状态响应

（因为电路是线性的，分解成"初始条件单独作用"+"输入单独作用"是叠加原理的直接结果。）

### 六、用三要素法做 RL 电路

```
              t=0
        ┌─○/○──[R=5Ω]──┬──────
        │              │    
       (+)           [L=0.5H]   
       20V              │    
       (-)              │    
        │               │
        └───────────────┘
```

换路前开路很久，求 iL(t)iL​(t)（t>0t>0）。

---

**要素 ①：iL(0+)iL​(0+)**

换路前无电流：iL(0−)=0iL​(0−)=0 → iL(0+)=0iL​(0+)=0 A

**要素 ②：iL(∞)iL​(∞)**

换路后新稳态：电感短路 → 电流 = U/R=20/5=4U/R=20/5=4 A

iL(∞)=4iL​(∞)=4 A

**要素 ③：τ=L/R=0.5/5=0.1τ=L/R=0.5/5=0.1 s

**写出解**：

iL(t)=4+(0−4)e−t/0.1=4(1−e−10t) AiL​(t)=4+(0−4)e−t/0.1=4(1−e−10t) A

### 七、三要素法的扩展——求任意变量

上面只求了 uCuC​ 和 iLiL​。如果要求电路中**其他量**（电阻电流、电感电压等），怎么办？

**方法**：任何一阶电路的任何变量都满足同样的指数形式，所以**三要素法对任意变量都适用**：

f(t)=f(∞)+[f(0+)−f(∞)] e−t/τf(t)=f(∞)+[f(0+)−f(∞)]e−t/τ

只需分别求这个具体量的 f(0+)f(0+)、f(∞)f(∞)、τ 即可（τ 对整个电路是统一的）。

**一个常用小技巧**：如果你已经有了 uC(t)uC​(t)，那其他量直接通过 KCL/KVL 代数关系算就行，不用重新用三要素法。

---

### 八、本节核心——记住这些

1. **一阶电路只需求三个要素**：初始值、终值、时间常数
2. **τ = RC 或 L/R**，R 是从储能元件看进去的**戴维南电阻**（所有独立源置零）
3. **三要素法对任何变量都适用**
4. **任何一阶响应 = 零输入响应 + 零状态响应**
