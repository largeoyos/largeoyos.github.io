### 一、为什么要学"阶跃响应"和"冲激响应"？

先抛一个问题给你思考：

**问**：前面我们用三要素法解决了很多具体的暂态问题——换路、开关闭合、电源接入……但每来一个新电路、新激励，我们都要**重新列方程、重新求解**。有没有一种"**一劳永逸**"的方法——**只要知道电路对某种"标准输入"的响应，任何复杂输入的响应都能直接推出来**？

答案：**有**。标准输入有两种经典选择：

1. **阶跃函数 ε(t)**（单位"开关信号"）
2. **冲激函数 δ(t)**（单位"瞬时脉冲"）

对应的电路响应就叫**阶跃响应**和**冲激响应**。

这一节就是围绕它们展开的。

---

### 二、阶跃函数 ε(t)

#### 定义

**单位阶跃函数**（unit step function）：

ε(t)={0,t<01,t>0ε(t)={0,1,​t<0t>0​

（t=0t=0 处定义不严格，有时写 ε(0)=1/2ε(0)=1/2，但工程上一般不在意。）

```
   ε(t)
    ↑
  1 ┤       ━━━━━━━━━━━━━━━
    │
    │
  0 ┼──────┤─────────→ t
           0
```

#### 物理意义

**ε(t) 就是一个 t=0 瞬间"拉开"的开关的信号**：

- t<0t<0：值为 0（开关断开）
- t>0t>0：值恒为 1（开关闭合，信号恒定输入）

**用它表达激励**：

- 在 t = 0 接入 10V 电源：激励 us(t)=10ε(t)us​(t)=10ε(t)
- 在 t = τ₀ 接入 A 电源：激励 us(t)=Aε(t−τ0)us​(t)=Aε(t−τ0​)（延迟阶跃）

#### 阶跃响应

**定义**：电路在**零状态**下（储能元件初始无储能），对**单位阶跃** ε(t)ε(t) 的响应，记作 s(t)s(t)。

"零状态"很重要——它保证响应**纯粹来自外加激励**，不受初始储能干扰。

**例**：RC 电路，输入 us(t)=ε(t)us​(t)=ε(t) V，电容零初始电压。

用三要素法：

- uC(0+)=0uC​(0+)=0
- uC(∞)=1uC​(∞)=1（阶跃稳态值）
- τ=RCτ=RC

s(t)=uC(t)=(1−e−t/RC)ε(t)s(t)=uC​(t)=(1−e−t/RC)ε(t)

（乘一个 ε(t) 表示"从 t=0 开始"，t<0 时响应为 0。）

### 三、冲激函数 δ(t)

**这是本节最"不符合直觉"但也是最核心的概念**。

#### 不严格的直观描述

想象一个"**无限高、无限窄、面积为 1**"的信号：

```
  δ(t)
    ↑
    │
    │       ╷
    │       ║ ← 高度"无穷"
    │       ║
    │       ║
    │       ║
  0 ┼───────╨──────────→ t
            0
             ←→ 宽度"0"
```

**数学化定义**（你说你知道 δ(t)，我就直接写定义）：

δ(t)=0,t≠0δ(t)=0,t=0∫−∞∞δ(t) dt=1∫−∞∞​δ(t)dt=1

**关键性质**（筛选性质）：

∫−∞∞f(t)δ(t−t0) dt=f(t0)∫−∞∞​f(t)δ(t−t0​)dt=f(t0​)

这个性质等一下会变得非常重要——它是卷积的基石。

---

#### δ(t) 和 ε(t) 的关系（关键！）

**ε(t) 对 t 求导 → 得到 δ(t)**：

dε(t)dt=δ(t)dtdε(t)​=δ(t)​

**物理理解**：ε(t) 在 t=0 处"瞬间跳变" → 它的导数在 t=0 处"瞬间爆发"，其他地方为 0。这就是 δ(t)。

**反过来**：

ε(t)=∫−∞tδ(τ) dτε(t)=∫−∞t​δ(τ)dτ​

**这两条公式是阶跃和冲激之间的桥梁，一定要记住。**

### 四、冲激响应 h(t)

**定义**：电路在**零状态**下对**单位冲激** δ(t) 的响应，记作 h(t)h(t)。

#### 怎么求 h(t)？

**方法 1：直接算**（难，一般不用）

**方法 2：利用阶跃和冲激的关系**（⭐ 推荐）

**关键事实**：线性电路是**线性时不变系统（LTI）**，满足"**激励求导 → 响应求导**"。

既然 δ(t)=dε(t)dtδ(t)=dtdε(t)​，那么：

h(t)=ds(t)dth(t)=dtds(t)​​

**"冲激响应 = 阶跃响应的导数"**。这是工程上求 h(t) 的最常用方法。

---

#### 举例：RC 电路的冲激响应

前面我们得到阶跃响应：

s(t)=(1−e−t/RC)ε(t)s(t)=(1−e−t/RC)ε(t)

求导（这里要用**乘积求导法则**，因为既有 ε(t) 又有 1−e−t/RC1−e−t/RC）：

dsdt=1RCe−t/RCε(t)+(1−e−t/RC)δ(t)dtds​=RC1​e−t/RCε(t)+(1−e−t/RC)δ(t)

**第二项在 t=0t=0 时** 1−e0=01−e0=0，所以这一项为 0。于是：

h(t)=1RCe−t/RCε(t)h(t)=RC1​e−t/RCε(t)

**物理图像**：

```
  h(t)
    ↑
 1/RC┤╲
     │ ╲
     │  ╲
     │   ╲___
     │       ╲____
   0 ┼────────────╲━━━━━━→ t
     0
```

**解读**：t=0 时输入一个"瞬间脉冲"→ 电容瞬间被"冲"上一点电荷 → 然后慢慢放电衰减（时间常数 RC）。

---

#### 冲激响应的物理本质（重要直觉）

冲激响应 h(t) 描述的是：

> **电路对"一个瞬间刺激"的自然反应。**

这个"自然反应"只依赖于电路本身的结构（R、L、C 值），完全和输入的具体形状无关——**h(t) 是电路的"身份证"**。

这就是为什么只要知道 h(t)，就能算出电路对任意输入的响应——**任何输入都可以分解成一连串"刺激"的叠加**。

**这就是卷积的思想**。

---

### 五、卷积（⭐ 本章难点）

你之前说卷积是"对一个输入从作用时间开始作用到结束的集合的积分"——这个描述基本对，但我们要把它精确化。

#### 核心想法

任意输入信号 f(t)f(t)，可以看作是**一系列"带权重"的冲激的叠加**：

f(t)=∫−∞∞f(τ)δ(t−τ) dτf(t)=∫−∞∞​f(τ)δ(t−τ)dτ

**这个式子怎么理解？** 它说：把 f(t) 想象成由许多个**位置在 τ、高度为 f(τ)f(τ) 的小冲激**叠加而成。（这个公式就是 δ 的筛选性质反过来写。）

#### 关键一步（线性时不变系统）

- **单个冲激** δ(t)δ(t) → 响应 h(t)h(t)
- **单个延迟冲激** δ(t−τ)δ(t−τ) → 响应 h(t−τ)h(t−τ)（因为时不变）
- **带权重的冲激** f(τ)δ(t−τ)f(τ)δ(t−τ) → 响应 f(τ)h(t−τ)f(τ)h(t−τ)（因为线性）
- **叠加所有这样的冲激** → 叠加所有的响应

所以输入 f(t) 的响应为：

y(t)=∫−∞∞f(τ)h(t−τ) dτy(t)=∫−∞∞​f(τ)h(t−τ)dτ

**这就是卷积**，记作：

y(t)=f(t)∗h(t)=∫−∞∞f(τ)h(t−τ) dτy(t)=f(t)∗h(t)=∫−∞∞​f(τ)h(t−τ)dτ​

#### 对因果信号（从 t=0 开始）的简化

如果：

- 输入 f(t)f(t) 只在 t≥0t≥0 存在（f(t)=0f(t)=0 for t<0t<0）
- 系统是**因果的**（h(t)=0h(t)=0 for t<0t<0）

那么积分上下限可以简化：

y(t)=∫0tf(τ)h(t−τ) dτ,t≥0y(t)=∫0t​f(τ)h(t−τ)dτ,t≥0​

**这就是你之前描述的"从作用开始到现在"的积分**——你的直觉是对的。

---

#### 卷积的几何直觉

```
  f(τ)              h(-τ) —→平移 t —→ h(t-τ)
    ↑                 ↑                ↑
    │  ╱╲              │ ╲             │    ╲
    │ ╱  ╲             │  ╲            │     ╲
    │╱    ╲            │   ╲___        │  ____╲
    └────────→τ        └─────→τ        └────────→τ
                                        t 
                                   
                                   
  把 f(τ) 和 h(t-τ) 逐点相乘再积分 → 得到 y(t) 在 t 时刻的值
```

**操作步骤**（"翻转-平移-相乘-积分"）：

1. **翻转**：把 h(τ)h(τ) 变成 h(−τ)h(−τ)（沿 τ 轴翻转）
2. **平移**：把 h(−τ)h(−τ) 平移 t，得到 h(t−τ)h(t−τ)
3. **相乘**：计算 f(τ)h(t−τ)f(τ)h(t−τ)
4. **积分**：对 τ 积分，得到 y(t)y(t)

每个 t 值都这样做，就得到完整的 y(t)y(t)。

---

#### 卷积的核心地位

> **在线性时不变系统里，只要知道冲激响应 h(t)，就能通过卷积求任意输入的响应。**

这个定理让 h(t) 成为**系统的"基因"**——一旦 h(t) 确定，整个系统的行为就全部确定了。

---

### 六、一个完整的卷积例子

**场景**：RC 电路（R=1Ω，C=1F，所以 τ=1），冲激响应 h(t)=e−tε(t)h(t)=e−tε(t)。

输入：f(t)=ε(t)f(t)=ε(t)（单位阶跃）

**求响应 y(t)**。

---

**用卷积公式**：

y(t)=∫0tf(τ)h(t−τ) dτ=∫0t1⋅e−(t−τ) dτy(t)=∫0t​f(τ)h(t−τ)dτ=∫0t​1⋅e−(t−τ)dτ

（因为 f(τ)=1f(τ)=1 for τ≥0τ≥0）

=e−t∫0teτ dτ=e−t[et−1]=1−e−t=e−t∫0t​eτdτ=e−t[et−1]=1−e−ty(t)=(1−e−t)ε(t)y(t)=(1−e−t)ε(t)

**检验**：这正是我们之前用三要素法得到的 RC 阶跃响应 s(t)！✓

这就验证了一个重要关系：

s(t)=∫0th(τ) dτs(t)=∫0t​h(τ)dτ​

"**阶跃响应 = 冲激响应的积分**"——和前面 "冲激 = 阶跃的导数" 完全对应。
