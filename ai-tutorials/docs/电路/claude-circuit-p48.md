# 9.4 二阶电路的暂态分析

一阶电路只有一个储能元件（C 或 L）。如果电路里**同时**有 L 和 C，情况就完全不一样了。

---

### 一、为什么"二阶"这么特别？

先想一个问题。还记得第六章的 LC 谐振吗？谐振时电感和电容之间会**互相"传递"能量**——电感放能量给电容，电容再还回去，周而复始。

**问**：如果这样一个 LC 电路，再串联一个小电阻，会发生什么？电阻会消耗能量，所以"来回传递"的过程会慢慢衰减——这就是**振荡衰减**。

但如果电阻**很大**呢？能量还没来得及在 L、C 之间传几次，就被电阻全部耗光了——这时候就**不振荡**，只是单调衰减。

**这两种定性完全不同的行为**，正是二阶电路的魅力所在，也是本节的核心。

---

### 二、典型二阶电路：RLC 串联

```
      t=0

   │                      │
  (+)                     
  U_s                     
  (-)                     
   │                      │
   └──────────────────────┘
```

换路后 KVL：

Us=Ri+Ldidt+uCUs​=Ri+Ldtdi​+uC​

用 i=CduCdti=CdtduC​​，代入并整理（以 uCuC​ 为未知量）：

**这是二阶线性常系数微分方程**。两个储能元件 → 方程阶数是 2。

### 四、特征方程和三种情形

你说你会解常微分方程，所以直接上。齐次方程的特征方程：

s2+2αs+ω02=0s2+2αs+ω02​=0

两个根：

s1,2=−α±α2−ω02s1,2​=−α±α2−ω02​​

**根的性质取决于根号里的正负**——这正是决定二阶电路行为的关键：

#### 情形 ①：过阻尼（α>ω0α>ω0​，即 R>2L/CR>2L/C​）

α2−ω02α2−ω02​​ 是**实数** → 两个不同的**负实根** s1,s2s1​,s2​

通解：

uC(t)=A1es1t+A2es2t+uC(∞)uC​(t)=A1​es1​t+A2​es2​t+uC​(∞)

**行为**：**单调衰减**（两个指数衰减的叠加），**无振荡**。

```
   u_C
    ↑
    │╲                  
    │ ╲                 
    │  ╲___             
    │      ╲___         
    │          ╲_______ ← u_C(∞)
    │                   
    └─────────────────→ t
```

**物理解读**：R 太大了，能量在 L、C 之间还没来得及传就被电阻吃掉——没有振荡的机会。

#### 情形 ②：临界阻尼（α=ω0α=ω0​，即 R=2L/CR=2L/C​）

α2−ω02=0α2−ω02​​=0 → 两个相同的实根 s1=s2=−αs1​=s2​=−α

通解（重根情形）：

uC(t)=(A1+A2t)e−αt+uC(∞)uC​(t)=(A1​+A2​t)e−αt+uC​(∞)

**行为**：**最快**的非振荡衰减——刚好不振荡。

**物理解读**：这是**"不振荡"和"振荡"的分界线**。工程上常把临界阻尼作为设计目标（比如汽车减震——既要快速归位，又不能来回颠簸）。

#### 情形 ③：欠阻尼（α<ω0α<ω0​，即 R<2L/CR<2L/C​）⭐

α2−ω02<0α2−ω02​<0 → 根号里是负数 → 两个**共轭复根**：

s1,2=−α±jωd,ωd=ω02−α2s1,2​=−α±jωd​,ωd​=ω02​−α2​

其中 ωdωd​ 叫**衰减振荡角频率**（damped oscillation frequency）。

通解：

uC(t)=e−αt[B1cos⁡ωdt+B2sin⁡ωdt]+uC(∞)uC​(t)=e−αt[B1​cosωd​t+B2​sinωd​t]+uC​(∞)

或统一成：

uC(t)=Be−αtcos⁡(ωdt+φ)+uC(∞)uC​(t)=Be−αtcos(ωd​t+φ)+uC​(∞)

**行为**：**衰减振荡**！幅值按 e−αte−αt 衰减，同时以 ωdωd​ 频率振荡。

```
   u_C
    ↑
    │╲    ╱╲
    │ ╲  ╱  ╲  ╱¯╲
    │  ╲╱    ╲╱   ╲__╱¯╲_____← u_C(∞)
    │                          
    │ 
    └─────────────────────→ t
       包络 e^(-αt) 衰减
```

**物理解读**：R 小，电阻"吃"能量的速度慢于 L、C 互相传能的速度 → 能量在 L、C 之间振荡着"消耗"—— 典型的"衰减振荡"。

### 五、阻尼比 ζ（更紧凑的刻画）

工程上更常用一个无量纲参数：

ζ=αω0=R2CLζ=ω0​α​=2R​LC​​

（读作"zeta"）

三种情况用 ζ 表达：

|情形|条件|特征根|行为|
|---|---|---|---|
|过阻尼|ζ > 1|两个负实根|单调衰减|
|临界阻尼|ζ = 1|重实根|临界衰减（最快非振荡）|
|欠阻尼|0 < ζ < 1|共轭复根|衰减振荡|
|无阻尼|ζ = 0（R = 0）|纯虚根|等幅振荡（LC 谐振）|

**阻尼比 ζ 在控制理论、信号处理、机械振动里都是核心概念**——你以后会反复见。

### 六、一个完整例题

**电路**：R = 2 Ω，L = 1 H，C = 0.25 F 串联，电压源 Us=10Us​=10 V。

开关 t=0 闭合，电容初始电压 0，电感初始电流 0。

**求**：uC(t)uC​(t)

---

**Step 1：判断阻尼类型**

ω0=1/LC=1/0.25=2ω0​=1/LC​=1/0.25​=2 rad/s

α=R/(2L)=2/2=1α=R/(2L)=2/2=1

α<ω0α<ω0​ → **欠阻尼**！

ωd=ω02−α2=4−1=3ωd​=ω02​−α2​=4−1​=3​ rad/s

**Step 2：求稳态（终值）**

t→∞t→∞ 时 L 短路、C 开路 → 电流 = 0 → R、L 无压降 → uC(∞)=10uC​(∞)=10 V

**Step 3：写通解**

uC(t)=e−t[B1cos⁡3t+B2sin⁡3t]+10uC​(t)=e−t[B1​cos3​t+B2​sin3​t]+10

**Step 4：用初始条件定系数**

**条件 1**：uC(0+)=0uC​(0+)=0

0=B1+10  ⟹  B1=−100=B1​+10⟹B1​=−10

**条件 2**：需要另一个方程。用 duCdt(0+)dtduC​​(0+)——回忆电容 VCR：i=CduCdti=CdtduC​​

所以 duCdt(0+)=i(0+)C=iL(0+)C=00.25=0dtduC​​(0+)=Ci(0+)​=CiL​(0+)​=0.250​=0

duCdt=−e−t[B1cos⁡+B2sin⁡]+e−t[−B13sin⁡+B23cos⁡]dtduC​​=−e−t[B1​cos+B2​sin]+e−t[−B1​3​sin+B2​3​cos]

**Step 5：最终答案**

uC(t)=e−t[−10cos⁡3t−103sin⁡3t]+10uC​(t)=e−t[−10cos3​t−3​10​sin3​t]+10

uC(t)=10−203e−tcos⁡(3t−30°) VuC​(t)=10−3​20​e−tcos(3​t−30°) V

- 终态 10V
- 围绕 10V 做衰减振荡
- 时间常数 1 s，振荡角频率 √3 ≈ 1.73 rad/s

### 七、用二阶电路的思维回看控制系统

二阶系统的响应（过/临界/欠阻尼）在控制工程、机械工程里**无处不在**：

- **汽车减震器**：调到临界阻尼附近（快速归位不颠簸）
- **门挡阻尼器**：过阻尼（慢慢关，不"砰"一声）
- **音箱低音单元**：略欠阻尼（更好的瞬态响应）
- **建筑抗震**：调整阻尼比让建筑不共振

**RLC 电路就是所有这些物理系统的"数学孪生"**——电路学到这里，你其实已经掌握了通用二阶系统的分析工具。
