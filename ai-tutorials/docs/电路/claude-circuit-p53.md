# 10.4 运算法（综合应用）

### 一、完整流程

**运算法**是 s 域分析的标准步骤：

```
  Step 1：求初始值 u_C(0⁻)、i_L(0⁻)
          │
          ↓
  Step 2：画 s 域电路图
          - R → R
          - L → sL 串联 L·i_L(0⁻) 电压源
          - C → 1/sC 串联 u_C(0⁻)/s 电压源
          - 独立源 → 拉氏变换后
          │
          ↓
  Step 3：在 s 域中用节点法、回路法、戴维南定理等解出所求量 F(s)
          │
          ↓
  Step 4：反变换得 f(t)
```

**相比时域法的优势**：

- 微分方程 → 代数方程
- 初始条件自动嵌入
- 卷积 → 乘法

### 三、网络函数 H(s) —— 概念升华

s 域的**网络函数**（或传递函数）：

H(s)=响应变换激励变换∣零状态H(s)=激励变换响应变换​​零状态​

**零状态**这个前提很重要——初始条件为 0 时，H(s) 纯粹反映**电路本身的特性**。

#### H(s) 和 h(t) 的关系

H(s)=L[h(t)]H(s)=L[h(t)]​

**网络函数 = 冲激响应的拉氏变换**！

**证明**（直觉版）：输入 δ(t)δ(t) 对应 L[δ(t)]=1L[δ(t)]=1，响应 h(t) 对应 H(s)·1 = H(s)。所以 H(s) 就是 h(t) 的拉氏变换。

---

#### H(s) 和频率响应 H(jω) 的关系

令 s=jωs=jω：

H(jω)=H(s)∣s=jωH(jω)=H(s)​s=jω​

**这就把第六章的频率特性和第十章的运算法连起来了**：

- **s=jωs=jω**：纯虚数 → **正弦稳态响应**（第五、六章）
- **s=σ+jωs=σ+jω**：复数 → **全面的动态响应**（第十章）

**H(jω)H(jω) 是 H(s)H(s) 的一个特例**（在虚轴上的取值）。

---

#### 零极点与电路行为

把 H(s) 写成：

H(s)=K(s−z1)(s−z2)⋯(s−p1)(s−p2)⋯H(s)=K(s−p1​)(s−p2​)⋯(s−z1​)(s−z2​)⋯​

- zizi​ 叫**零点**（使 H(s) = 0 的 s 值）
- pipi​ 叫**极点**（使 H(s) = ∞ 的 s 值）

**极点决定电路的自然响应**：

- 极点在左半平面（Re(s) < 0）：**响应衰减**，电路稳定
- 极点在右半平面（Re(s) > 0）：**响应发散**，电路不稳定
- 极点在虚轴（Re(s) = 0）：**等幅振荡**，临界稳定
- 极点是实数：衰减无振荡
- 极点是共轭复数对：衰减振荡

**这和 9.4 节的过/临界/欠阻尼分类一一对应**！当时我们用了"阻尼比"看特征方程的根——那些根其实就是 H(s) 的极点。
