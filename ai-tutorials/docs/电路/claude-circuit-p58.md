# 11.3 二端口网络的等效与互联

### 一、等效电路

给定了参数矩阵（比如 Z 参数），**是否能画出一个具体电路来实现它？**

能。最常见的两种：

#### T 型等效电路（Z 参数）

```
       I₁ →              ← I₂
      ●──[Z₁₁-Z₁₂]──┬──[Z₂₂-Z₂₁]──●
      │             │              │
      + U₁        [Z₁₂]          U₂ +
      -             │              -
      ●─────────────┴──────────────●
```

（前面 T 型网络的例子，三个阻抗和 Z 参数的对应：Za=Z11−Z12Za​=Z11​−Z12​，Zb=Z22−Z12Zb​=Z22​−Z12​，Zc=Z12Zc​=Z12​）

**对互易网络**（Z12=Z21Z12​=Z21​）可以这样画。**对非互易网络**，需要加一个**受控源**来体现不对称。

#### π 型等效电路（Y 参数）

```
       I₁ →              ← I₂
      ●──────┬───[1/Y₁₂...]───┬──────●
      │     │                  │      │
      + U₁ [Y]               [Y]    U₂ +
      -     │                  │      -
      ●──────┴──────────────────┴──────●
```

（和 Y 参数的对应关系细节略。）

这两种经典结构（T 和 π）在射频、滤波器设计中是**基本单元**。

### 二、二端口网络的互联

当两个二端口网络 NaNa​ 和 NbNb​ 以某种方式连在一起时，**总参数和子网络参数怎么关系？**

#### 串联（Series）：Z 参数直接相加

两个端口都串联起来：

```
     ●──┐    ┌──●
        │ Na │
     ●──┤    ├──●
        │ Nb │
     ●──┘    └──●
```

（端口 1 的两个子端口串联，端口 2 也串联。）

**总 Z 矩阵 = 子 Z 矩阵之和**：

Z=Za+ZbZ=Za​+Zb​

#### 并联（Parallel）：Y 参数直接相加

Y=Ya+YbY=Ya​+Yb​

#### 级联（Cascade）：T 参数相乘 ⭐

T=Ta⋅TbT=Ta​⋅Tb​

**这是 T 参数最强大的用途**——多级系统分析时，各级 T 参数直接矩阵连乘即可。

#### 混联（Series-parallel）：H 参数相加

H=Ha+HbH=Ha​+Hb​

（一个端口串联，另一个端口并联——不常见，略）

---

### 三、互联必须满足"端口条件"

**⚠️ 警告**：上面这些公式并不总是成立——必须满足**"端口条件"**（每对端子流入流出电流相等）。

对于一般电路（尤其是有公共地线的电路），互联后原来的端口条件**可能被破坏**。工程上需要仔细验证；一个常用的做法是**加一个 1:1 变压器**做隔离以保证端口条件。

这部分细节属于进阶内容，你知道"有这个坑"就行。

### 四、典型应用：级联放大器

**场景**：三级放大器级联，每一级用 T 参数描述。

Ta=[AaBaCaDa],Tb,TcTa​=[Aa​Ca​​Ba​Da​​],Tb​,Tc​

**总系统 T 参数**：

T总=TaTbTcT总​=Ta​Tb​Tc​

**只要做矩阵乘法，整个级联系统的特性就出来了**——不需要把三级放大器的所有内部元件放在一起联立求解。

这就是"黑箱化"的威力——**复杂度从"加法"降到了"乘法"**。
