# 第二章 线性直流电路（系统分析方法）

第一章我们学了"基本语言"（KCL、KVL、元件 VCR）。但当电路变复杂时，**节点多、回路多、方程一大堆**，怎么办？

第二章就是解决这个问题：**如何系统化、程式化地列出最少的独立方程，高效求解任意电路。**

---

### 2.1 网络的图（Graph Theory 基础）

#### 为什么要学"图"？

你列出的难点之一是"**独立变量和独立方程**"。这个难点的根源就在这里：

- 一个电路有多少节点？多少回路？
- 能列多少个 KCL？多少个 KVL？
- 其中有多少个是**独立**的（不重复、不冗余）？

要严谨回答这些问题，就需要借助**图论**。

#### 电路 → 图

把电路中每个**二端元件**（不管是电阻、电容还是电源）都**抽象成一条线段**（叫**支路 branch**），连接两个端点（叫**节点 node**），就得到了电路的"图"。

```
   电路：                    图：
   
   ┌──[R1]──┬──[R2]──┐      ┌────┬────┐
   │        │        │      │    │    │
   U        [R3]     I      │    │    │  
   │        │        │      │    │    │
   └────────┴────────┘      └────┴────┘
```

- 图**只保留连接关系**，不关心元件是什么
- 如果支路标了方向（参考方向），就是**有向图**

#### 三个关键术语

**① 支路（branch, b）**：每条线段，对应一个二端元件

**② 节点（node, n）**：支路的端点

**③ 回路（loop）**：从某节点出发，沿支路走一圈回到起点，途中不重复经过节点

#### 树（Tree）—— 这是核心概念

**树的定义**：

> 从图中选出一组支路，满足：
> 
> 1. **连接所有节点**
> 2. **不形成任何回路**
> 
> 这组支路就构成一棵"树"，其中的支路叫**树支（tree branch）**，剩下没选的支路叫**连支（link）**。

#### 一个例子

```
图的结构（4 个节点 a,b,c,d，5 条支路 1,2,3,4,5）：

     a ──1── b
     │       │
     2       3
     │       │
     c ──4── d
         │
         5
         (5 是 a-d 之间的对角线支路)
```

一种可能的树选法：选 {1, 2, 3} 作树支

```
   树支（实线）：  a ──1── b
                  │       │
                  2       3
                  │       │
                  c       d
   
   连支（虚线）：  c ──4── d, a ----5---- d
```

**验证**：树支 {1,2,3} 连接了所有 4 个节点，且不形成回路。✓

#### 关键性质（记住这三条）

对于有 nn n 个节点、bb b 条支路的连通图：

|量|个数|含义|
|---|---|---|
|树支数|n−1n-1 n−1|一棵树恰好有 n−1n-1 n−1 条树支|
|连支数|b−(n−1)=b−n+1b-(n-1) = b-n+1 b−(n−1)=b−n+1|剩下的支路都是连支|
|独立回路数|b−n+1b-n+1 b−n+1|**等于连支数**！|

**为什么独立回路数 = 连支数？**

> 每添加一条连支到树上，就**恰好形成一个新回路**（叫**基本回路**）。有几条连支，就有几个基本回路。这些基本回路相互独立，数目就是 b−n+1b-n+1 b−n+1。

#### 这个公式解决了什么？

前面例子中的电路：n=4n=4 n=4, b=5b=5 b=5，独立回路数 = 5−4+1=25-4+1 = 2 5−4+1=2。

这告诉我们：**最多只能列 2 个独立的 KVL 方程**。如果你写了 3 个 KVL，其中一个一定是另外两个的线性组合（废话）。

---

### 2.2 KCL 和 KVL 的独立方程

现在可以回答"独立方程"的问题了：

#### KCL 独立方程数 = n−1n-1 n−1

**为什么？**

- 每个节点能列 1 个 KCL，共 n 个
- 但所有节点 KCL 相加：每条支路电流在一端"流入"、另一端"流出"，相加=0
- 即 n 个 KCL 加起来恒等于 0，说明它们**线性相关**
- 所以独立的只有 n−1n-1 n−1 个

**实操**：任选 n−1n-1 n−1 个节点列 KCL 即可，剩下那个节点（通常选作"参考节点"或"地"）不用列。

#### KVL 独立方程数 = b−n+1b-n+1 b−n+1

就是上面说的独立回路数。**实操**：找出所有基本回路（每个连支对应一个），每个列一个 KVL。

#### 总结：独立方程总数

(n−1)⏟KCL+(b−n+1)⏟KVL=b\underbrace{(n-1)}_{\text{KCL}} + \underbrace{(b-n+1)}_{\text{KVL}} = bKCL(n−1)​​+KVL(b−n+1)​​=b

**正好等于支路数！** 这就对了：b 条支路有 b 个未知电流（或 b 个未知电压），需要 b 个独立方程才能解出。

### 2.3 支路电流法

#### 基本思路

最直接的方法：**以 b 个支路电流为未知数，列 b 个独立方程。**

步骤：

1. 标出所有支路电流的参考方向（共 b 个未知量）
2. 对 n−1n-1 n−1 个独立节点列 KCL
3. 对 b−n+1b-n+1 b−n+1 个独立回路列 KVL（电压用 RiRi Ri 表示）
4. 联立求解

### 2.4 回路电流法

#### 核心思想

> **不直接求每条支路的电流，而是求每个"独立回路"里的"假想循环电流"。**

想象每个独立回路里有一个**绕圈流动的电流**（叫**回路电流**或**网孔电流**），这样：

- **未知数从 b 个减少到 b−n+1b-n+1 b−n+1 个**（等于独立回路数）
- 每个回路电流**自动满足 KCL**（转一圈流入=流出）
- **只需要列 KVL** 方程

#### 关键点：支路电流 = 相关回路电流的代数和

一条支路可能只属于一个回路，也可能是两个回路的**共有支路**。

```
    回路 I_a          回路 I_b
    ┌─────┐          ┌─────┐
    │     │ 共有 R   │     │
    │     ├──[R]──────┤     │
    │     │←I_a  I_b→│     │
    └─────┘          └─────┘
```

共有电阻 R 上的实际电流 = Ia−IbI_a - I_b Ia​−Ib​（两回路电流方向相反时相减；相同时相加）。

#### 标准步骤

1. 选一棵树，每个连支对应一个独立回路（实操中常选**网孔**——平面电路中最小的回路）
2. 设每个回路电流，标**绕行方向**（通常都取顺时针，便于统一）
3. 对每个回路列 KVL
4. 解方程

#### 标准方程形式（自阻 & 互阻）

对于回路 k：

RkkIk−∑j≠kRkjIj=∑UskR_{kk} I_k - \sum_{j \neq k} R_{kj} I_j = \sum U_{sk}Rkk​Ik​−j=k∑​Rkj​Ij​=∑Usk​

- RkkR_{kk} Rkk​：**自阻** = 回路 k 中所有电阻之和（恒为正）
- RkjR_{kj} Rkj​：**互阻** = 回路 k 和 j 共有的电阻（若两回路电流在共有支路上方向相同则取正，相反则取负；如果所有回路都按同一方向绕，互阻前的符号就都是负）
- ∑Usk\sum U_{sk} ∑Usk​：回路 k 中所有电压源的代数和（与绕行方向相同的电位"升"取正）

#### 💡 回路电流法的"陷阱"

**当电路中含有电流源时**，电流源两端电压未知，KVL 不好直接列。处理方法：

**情况一：电流源只属于一个回路** → 该回路的回路电流直接 = 电流源电流（已知），省一个方程

**情况二：电流源是两个回路的共有支路** → 引入电流源两端的**未知电压 U**，按电压源处理，补一个约束方程（两回路电流差 = 电流源值）。这种处理叫"**超网孔**"（supermesh）。

### 2.5 节点电压法（最常用！）

#### 核心思想

> **选一个节点作"地"（参考节点），其余节点对地的电压就是"节点电压"。以节点电压为未知数。**

- 未知数数量 = n−1n-1 n−1（和 KCL 独立方程数一致）
- 每条支路电流可用**两端节点电压之差**除以电阻表示
- **只需要列 KCL** 方程

#### 为什么节点法用得最多？

- 未知数通常比回路法还少（对**节点少、回路多**的电路尤其高效）
- 所有电路仿真软件（如 SPICE）用的都是**改进节点法**
- 和电路拓扑关系直接，不用找回路

#### 标准方程形式（自导 & 互导）

对节点 k（取地为参考）：

GkkUk−∑j≠kGkjUj=∑IskG_{kk} U_k - \sum_{j \neq k} G_{kj} U_j = \sum I_{sk}Gkk​Uk​−j=k∑​Gkj​Uj​=∑Isk​

- UkU_k Uk​：节点 k 对地的电压
- GkkG_{kk} Gkk​：**自导** = 连接到节点 k 的所有电导之和（恒为正）
- GkjG_{kj} Gkj​：**互导** = 节点 k 和 j 之间支路的电导之和（前面有负号，所以方程里写 −GkjUj-G_{kj}U_j −Gkj​Uj​）
- ∑Isk\sum I_{sk} ∑Isk​：流入节点 k 的电流源电流代数和（流入为正）

```

   │           │           │

  (-)          │          (-)
   │           │           │
   └───────────┴───────────┘
              地
```

选下方为地，只有一个未知节点电压 UaU_a Ua​。

**KCL @ 节点 a**（流入 = 流出）：

- 经 2Ω 从 12V 流入 a 的电流：12−Ua2\dfrac{12 - U_a}{2} 212−Ua​​
- 经 3Ω 从 6V 流入 a 的电流：6−Ua3\dfrac{6 - U_a}{3} 36−Ua​​

#### 节点法"陷阱"：含电压源时的处理

**情况一：电压源一端接地** → 另一端的节点电压直接 = 电压源电压（已知），省一个方程

**情况二：电压源连在两个非地节点间（"悬浮"电压源）** → 引入电压源电流作未知数（或用"**超节点**" supernode 技巧：把电压源两端的节点合并成一个"大节点"列 KCL，再补一个约束：两节点电压差 = 电压源电压）

### 2.6 含运算放大器电路的分析

#### 运放（Op-amp）是什么？

运放是个**五端元件**（实际上看作三端：两个输入 + 一个输出，电源端通常略去不画），符号：

```
      V+ ───▶│＼
              │  ＼___ Vo
      V− ───▶│  ／
              │／
```

**理想运放的两个黄金法则**（这是本节精髓）：

#### 黄金法则一："虚断"

> 两个输入端**电流为零**（输入阻抗无穷大）

i+=i−=0i_+ = i_- = 0i+​=i−​=0

#### 黄金法则二："虚短"

> 两个输入端**电位相等**（开环增益无穷大 + 负反馈）

u+=u−u_+ = u_-u+​=u−​

> ⚠️ **前提**：必须有**负反馈**，即输出通过某种方式回到反相输入端（V−）。没有负反馈时虚短不成立！

#### 经典例题：反相放大器

```
       R_f
    ┌──[Rf]──┐
    │        │
   v_in      │
    │        │       V−  ___
    └──[R1]──┴───────▶│＼
                       │  ＼___ V_out
               地─────▶│  ／
                       │／
                       V+
```

**分析**：

- V+ 接地 → u+=0u_+ = 0 u+​=0
- 虚短：u−=u+=0u_- = u_+ = 0 u−​=u+​=0（叫"虚地"）
- 虚断：反相端无电流流入运放
- 所以 R1 的电流 = Rf 的电流（KCL）： vin−0R1=0−voutRf\frac{v_{in} - 0}{R_1} = \frac{0 - v_{out}}{R_f}R1​vin​−0​=Rf​0−vout​​

vout=−RfR1vin\boxed{v_{out} = -\frac{R_f}{R_1} v_{in}}vout​=−R1​Rf​​vin​​

输出与输入**反相**，放大倍数 Rf/R1R_f/R_1 Rf​/R1​。

#### 虚短 + 虚断的本质

其实运放可以看作一个**特殊的受控源**（VCVS，增益无穷大）。虚短虚断是"增益 → ∞"和"输入阻抗 → ∞"两个极限的数学结果。

|**节点电压法**|n-1|KCL|节点少、支路多时（**最常用**）|

**选择原则**：看电路中**哪个数更少**，就用哪种方法。
