### 3.1 置换定理（Substitution Theorem）

#### 先用一个问题引入

假设我解出了某电路的一条支路：电流 I=2I = 2 I=2 A，电压 U=6U = 6 U=6 V。

现在我把这条支路"挖掉"，换成一个 **2A 的电流源**，电路其他部分会变吗？

---

**稍停一下，请你自己想想**：

换上 2A 电流源后，原先连接这条支路的两个节点之间，电流还是 2A 吗？电压还是 6V 吗？电路其他部分的电流电压会变吗？

（凭直觉答，不用严谨证明）

---

#### 答案：不会变！

**置换定理**的内容：

> 电路中任一支路 k，若已知其电压 UkU_k Uk​ 和电流 IkI_k Ik​，则可以用以下任一"等效元件"置换它，电路其他部分的**所有电压电流保持不变**：
> 
> - 一个电压为 UkU_k Uk​ 的**电压源**
> - 一个电流为 IkI_k Ik​ 的**电流源**
> - 一个电阻 R=Uk/IkR = U_k / I_k R=Uk​/Ik​（只要 Ik≠0I_k \neq 0 Ik​=0）

#### 为什么？（直觉理解）

电路其他部分"看到的"这条支路，只是一个"两端之间有电压 UkU_k Uk​、流过电流 IkI_k Ik​"的黑箱。**黑箱里是什么不重要**，只要端口上的 U,IU, I U,I 一致，外部感受不到任何差别。

这就像你问"隔壁屋子里有人在说话"——你只听到声音，里面是录音机还是真人，你听不出区别。

#### 一个用法示例

```
   ┌──[R1]──┬──[R2]──┐
   │        │        │
  (+)      [R3]     [R4]
   U        │        │
  (-)       │        │
   │        │        │
   └────────┴────────┘
```

假设通过某种方法我算出了 R3 上电流 I3=1I_3 = 1 I3​=1 A。

**接下来想求 R4 上电压？**

- 把 R3 换成 **1A 电流源**——R3 上的电阻值对后续计算毫无影响！
- 电路变简单了：现在是"一个电压源 U，串联 R1，并联一个 1A 电流源（方向固定），再并联一个 R2-R4 支路"

这招在分析时非常省事——**已知量直接"冻结"**，让剩下的部分聚焦在未知量上。

### 3.2 叠加原理（Superposition）

#### 核心思想（一句话）

> **在线性电路中，多个独立源共同作用产生的响应，等于每个独立源单独作用时产生的响应之和。**

注意三个关键词：

1. **线性电路**（所有元件都是线性的）
2. **独立源**（受控源不参与叠加，始终保留）
3. **响应**（可以是某支路电流，也可以是某元件电压）

#### "单独作用" 是什么意思？

**让一个源单独作用** = **把其他独立源"置零"**：

- 电压源置零 = **短路**（因为电压源 U=0 就是一根导线）
- 电流源置零 = **开路**（因为电流源 I=0 就是断开）

> ⚠️ **注意**：只置零**独立源**！**受控源必须保留**（因为它不是"源"，是一种约束关系）。

#### 一个例子

```
   ┌──[2Ω]──── a ───[3Ω]──┐
   │           │           │
  (+)        [6Ω]         ↑
  12V          │          2A
  (-)          │          ↓
   │           │           │
   └───────────┴───────────┘
              地
```

求节点 a 的电压 UaU_a Ua​。

**第一步：12V 单独作用**（2A 电流源开路）

```
   ┌──[2Ω]──── a ───[3Ω]──┐
   │           │           │
  (+)        [6Ω]      开路（悬空）
  12V          │        
  (-)          │           
   │           │           
   └───────────┴──────────
```

3Ω 右端开路，故 3Ω 上无电流、无压降。所以节点 a 的电压就等于 6Ω 上的电压。

这是一个简单的分压：12V 电压源经 2Ω 接到 a，a 经 6Ω 接地。

Ua(1)=12×62+6=9 VU_a^{(1)} = 12 \times \frac{6}{2+6} = 9 \text{ V}Ua(1)​=12×2+66​=9 V

**第二步：2A 电流源单独作用**（12V 电压源短路）

```
   ┌──[2Ω]──── a ───[3Ω]──┐
   │           │           │
  短路      [6Ω]          ↑
   │           │          2A
   │           │          ↓
   └───────────┴───────────┘
```

12V 短路后，2Ω 左端直接接地。所以从 a 往左看是 2Ω，往下看是 6Ω，这两个**并联**：

R左下=2∥6=2×62+6=1.5 ΩR_{左下} = 2 \parallel 6 = \frac{2 \times 6}{2+6} = 1.5 \text{ Ω}R左下​=2∥6=2+62×6​=1.5 Ω

右边 2A 电流源经过 3Ω 到 a。节点 a 对地的等效电阻路径：往右看 3Ω，然后...

等等，让我重新梳理一下。2A 电流源（假设箭头向上）接在 3Ω 右端和地之间。那么 2A 电流从地流出，经电流源向上，再经 3Ω 从右向左流入节点 a，然后从 a 分流到 2Ω 和 6Ω 去。

节点 a 看向地的等效电阻：2Ω ∥ 6Ω = 1.5Ω

流入 a 的电流为 2A（经 3Ω 进来），则：

Ua(2)=2×1.5=3 VU_a^{(2)} = 2 \times 1.5 = 3 \text{ V}Ua(2)​=2×1.5=3 V

**叠加**：

Ua=Ua(1)+Ua(2)=9+3=12 VU_a = U_a^{(1)} + U_a^{(2)} = 9 + 3 = 12 \text{ V}Ua​=Ua(1)​+Ua(2)​=9+3=12 V

#### 齐性定理（顺带提一下）

**齐性**（Homogeneity）：在线性电路中，**所有独立源同时放大 k 倍，所有响应也放大 k 倍**。

叠加 + 齐性 = **线性性质**的两个方面。

#### 叠加原理的典型陷阱

**❌ 功率不能叠加！**

P=UI 是乘积，不是线性关系P = UI \text{ 是乘积，不是线性关系}P=UI 是乘积，不是线性关系

比如：电阻 R 上，源 1 单独作用时 U1,I1U_1, I_1 U1​,I1​，源 2 单独作用时 U2,I2U_2, I_2 U2​,I2​：

P总=(U1+U2)(I1+I2)≠U1I1+U2I2P_{总} = (U_1+U_2)(I_1+I_2) \neq U_1 I_1 + U_2 I_2P总​=(U1​+U2​)(I1​+I2​)=U1​I1​+U2​I2​

**求功率必须先叠加出总电压/总电流，再算功率！**

### 3.3 等效电源定理（本章最重要！⭐）

这是电路分析中**使用频率最高**的定理，你要熟到"看电路就条件反射"的程度。

#### 3.3.1 戴维南定理（Thévenin's Theorem）

##### 定理内容

> 任意一个**线性含源二端网络**（从外部看是两个端子的电路），对外电路而言，都等效于一个**电压源串联电阻**的组合：
> 
> - 电压源值 UocU_{oc} Uoc​ = 该二端网络**开路电压**
> - 串联电阻 RthR_{th} Rth​ = 该二端网络内部所有独立源置零后的**等效电阻**

```
  复杂线性网络        戴维南等效
     ┌───┐               ┌──[R_th]──┐
     │ ? │ a              │          │ a
     │   │    ═══►      (+)         
     │ ? │ b             U_oc        
     └───┘               (-)           
                          │          │ b
                          └──────────┘
```

##### 为什么这么有用？

想象你在设计电路：电源部分可能非常复杂（几十个元件），但你关心的"负载"只要通过**两个端子**连接上去。

→ **把电源部分打包成一个"U_oc 串 R_th"的黑箱**，负载想换什么就换什么，计算超级简单。

##### 求戴维南等效的三步法

**步骤 1：求开路电压 UocU_{oc} Uoc​**

把外电路断开（端口开路），用任何方法（节点法、回路法、叠加法……）求端口电压。

**步骤 2：求等效电阻 RthR_{th} Rth​**

这一步有**三种方法**，按情况选择：

**方法 A：置零法**（电路中**只有独立源**时）

- 所有独立源置零（电压源短路、电流源开路）
- 从端口看进去的电阻即为 RthR_{th} Rth​

**方法 B：外加电源法**（含**受控源**时⚠️）

- 独立源置零，但受控源保留
- 在端口外加一个测试电压 UTU_T UT​（或电流 ITI_T IT​）
- 求端口响应电流 ITI_T IT​（或电压 UTU_T UT​）
- Rth=UT/ITR_{th} = U_T / I_T Rth​=UT​/IT​

**方法 C：开路短路法**

- 求开路电压 UocU_{oc} Uoc​
- 求**端口短路**时的短路电流 IscI_{sc} Isc​
- Rth=Uoc/IscR_{th} = U_{oc} / I_{sc} Rth​=Uoc​/Isc​

**步骤 3：画出等效电路**

```
      ┌──[R_th]──┐
      │          │
     (+)        
     U_oc       端口 a
     (-)           端口 b
      │          │
      └──────────┘
```

#### 3.3.2 诺顿定理（Norton's Theorem）

##### 定理内容

> 任意线性含源二端网络，对外电路而言也等效于一个**电流源并联电阻**的组合：
> 
> - 电流源 IscI_{sc} Isc​ = 该网络**端口短路**时的短路电流
> - 并联电阻 RNR_N RN​ = 和戴维南电阻相同（RN=RthR_N = R_{th} RN​=Rth​）

```
     ┌──────────┐
     │          │
     ↑        [R_N]   
    I_sc        │    端口 a
     ↓          │    
     │          │    端口 b
     └──────────┘
```

##### 戴维南 ↔ 诺顿 互相转换

这两种形式可以**互相转换**（电源等效变换）：

Isc=UocRth,RN=RthI_{sc} = \frac{U_{oc}}{R_{th}}, \quad R_N = R_{th}Isc​=Rth​Uoc​​,RN​=Rth​

```
  戴维南                    诺顿
  ┌──[R]──┐               ┌───────┐
  │       │               │       │
 (+)      │    ═══►       ↑      [R]
  U       │               U/R     │
 (-)      │               ↓       │
  │       │               │       │
  └───────┘               └───────┘
```

**实用技巧**：求等效电路时，哪种好求就先求哪种，需要另一种时转换一下。
