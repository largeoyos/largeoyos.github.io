# 3.4 特勒根定理（Tellegen's Theorem）

这个定理是个"怪才"——它听起来平平无奇，但威力巨大。

### 定理内容

> 对**任意电路**，只要所有支路电压 uku_k uk​ 取"关联参考方向"下的电压，所有支路电流 iki_k ik​ 取同样的参考方向，则：
> 
> ∑k=1bukik=0\sum_{k=1}^{b} u_k i_k = 0k=1∑b​uk​ik​=0

这个等式说：**全电路所有支路上 ukiku_k i_k uk​ik​ 的代数和恒等于零**。

### 它的物理意义是什么？

关联参考方向下 ukiku_k i_k uk​ik​ 就是该支路**吸收**的功率。所以这个公式说的是：

> **整个电路"吸收的总功率" = 0**

换句话说——电源释放的功率 = 所有元件吸收的功率。这就是**能量守恒**！

### 特勒根定理厉害在哪里？

关键是**它对"任意电路"成立**——包括非线性电路、时变电路。只要满足 KCL 和 KVL 就行。

更奇妙的是**特勒根定理的推广形式**：

> 有两个电路 NN N 和 N^\hat{N} N^，**拓扑结构完全相同**（图相同），但元件可以完全不同。用 uku_k uk​ 表示 N 的支路电压，i^k\hat{i}_k i^k​ 表示 N^\hat{N} N^ 的支路电流（参考方向一致），则：
> 
> ∑k=1buki^k=0和∑k=1bu^kik=0\sum_{k=1}^{b} u_k \hat{i}_k = 0 \quad \text{和} \quad \sum_{k=1}^{b} \hat{u}_k i_k = 0k=1∑b​uk​i^k​=0和k=1∑b​u^k​ik​=0

两个**不同电路**的电压和电流"混搭"起来，求和也等于 0！这就神奇了。

### 直觉理解

为什么会这样？

- KCL 只和**电路的图**有关（谁连着谁）
- KVL 也只和**电路的图**有关
- 只要两个电路的图相同，N 的电压和 N^\hat{N} N^ 的电流就"恰好配得上"——像拼图一样能对上

### 用途

特勒根定理主要是**理论证明工具**：

- 证明互易定理（下一节）
- 证明网络的其他对称性
- 电路灵敏度分析

**作为一年级本科生，你只需要记住**：

1. 普通形式 → 能量守恒
2. 推广形式 → 同拓扑电路电压电流的"奇妙乘积关系"
3. 这是一个"理论桥梁"，算具体数值题不常用
