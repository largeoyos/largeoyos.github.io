# 离散数学入门：第 1 节 —— 开课导引与学习地图

你好！很高兴能陪你一起走过离散数学这门课。在正式开始之前，我想先跟你聊几句，而不是立刻抛出一堆定义，因为**离散数学的学习方式**和你之前学过的高中数学很不一样。

### 一、什么是"离散"？

先问你一个小问题来热身：

> 你觉得"连续"和"离散"的区别是什么？能不能各举一个生活中的例子？

在你心里先想一想，我给你我的理解：

- **连续 (continuous)**：像水龙头里流出的水、时间的流逝、一根光滑的曲线。两点之间总能再塞进无数个点。高中学的微积分、极限、实数函数，研究的都是连续对象。
- **离散 (discrete)**：像一粒一粒的米、楼梯的台阶、QQ 好友列表里的人。对象是**一个一个分开的**，可以数、可以列。

离散数学，就是研究这些"一粒一粒"的对象的数学：整数、集合、逻辑命题、图（点和边）、算法步骤……

一个关键直觉：**计算机里的一切本质上都是离散的**（0 和 1，没有"0.5 个比特"）。所以离散数学是计算机科学的数学基础，就像微积分是物理学的数学基础一样。

### 二、这门课你会学到什么

标准的离散数学课程大纲通常包含这几大板块。我画一下我们的"学习路线图"：

|章节|主题|核心回答的问题|
|---|---|---|
|1|数理逻辑|什么是"正确的推理"？|
|2|集合论|如何精确描述"一堆东西"？|
|3|关系与函数|对象之间如何"关联"？|
|4|数论基础|整数有什么神奇性质？（密码学基础）|
|5|组合数学|如何"数"复杂的东西？|
|6|图论|点和连线能建模什么？|
|7|代数结构|群、环、域入门|

我们会**从第 1 章"数理逻辑"开始**，因为后面所有章节的证明和推理，都要用到逻辑这套语言。

### 三、学这门课的心态建议

离散数学和高中数学最大的不同是：

1. **它不是在"算"，而是在"证"**。你做的大部分题不会是"求 x 等于多少"，而是"证明 X 对所有情况都成立"。
2. **定义极其重要**。每个新词都有精确的定义，模糊理解必栽跟头。我会在每次引入新概念时，让你用自己的话复述一下。
3. **符号是朋友，不是敌人**。一开始你会觉得 ∀∃⇒∧∨ 这些符号很烦，但它们其实是为了让思想表达得**更精确、更短**。

### 四、我打算怎么教你

- 每一节聚焦**一个或几个紧密相关的概念**，不贪多。
- 我会**先问你问题**，引导你自己想出来，而不是直接灌答案。
- 每节结束我会给你几道**小练习**，你做完后我们一起讨论。
- 有不懂、不同意、或者想跳过某部分，**随时打断我**。

### 在我们开始第 2 节（命题逻辑）之前，我想先了解你：

为了不讲得太浅或太深，能告诉我下面几点吗？

1. **你的背景**：你是什么专业的？（计算机、数学、其他？）
2. **你学过什么**：高中数学里，你对"集合"、"充分必要条件"、"反证法"这些词有印象吗？听过就行，不用精通。
3. **你的目标**：你是想应付考试，还是想为后面学算法、人工智能、密码学等课打好基础？
4. **节奏偏好**：你希望我讲得**更偏直觉、多举例**，还是**更偏严谨、多符号**？（两者我们都会涉及，只是侧重不同。）

回答完这几个问题，我们就正式进入**第 2 节：命题与逻辑联结词**。🚀
