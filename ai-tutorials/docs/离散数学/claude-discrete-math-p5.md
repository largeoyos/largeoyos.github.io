# 离散数学 第 4 节：逻辑等价与基本等价式

### 一、为什么还需要新工具？

上一节我们学会了真值表 —— 它**万能但笨重**。考虑这个公式：

¬(P→Q)∨(¬P∧Q)\neg(P \to Q) \lor (\neg P \land Q)¬(P→Q)∨(¬P∧Q)

如果我想知道它和另一个公式 ¬P ∨ Q 是不是"一回事"，用真值表要列两张表再对比。如果公式有 5 个命题，就是 2 × 32 = 64 行。

有没有办法**像化简代数式一样直接变形**？比如在代数里我们有：

a(b+c)=ab+aca(b+c) = ab + aca(b+c)=ab+ac

能不能在逻辑里也有 **"¬(P ∧ Q) = ¬P ∨ ¬Q"** 这样的变形规则？

**有**，而且这套规则叫做**逻辑等价式**，是这节的核心。

---

### 二、逻辑等价的定义

先问你一个问题：什么时候我们说两个公式"是一回事"？

你可能会说："真值表一样"，对！这就是定义：

> **定义（逻辑等价）**：设 A、B 是两个命题公式。如果在**所有真值指派下**它们的真值**完全相同**，则称 A 与 B **逻辑等价**，记作：
> 
> A⇔BA \Leftrightarrow BA⇔B

**注意区分两个符号**：

- **↔** 是一个**联结词**，A ↔ B 是一个**公式**。
- **⇔** 是一个**元语言符号**，A ⇔ B 是一个**断言**："A 和 B 这两个公式等价"。

它们的联系很优美：

> **A ⇔ B 当且仅当 A ↔ B 是重言式。**

（想想为什么：A ↔ B 永远为真 ⟺ A 和 B 真值永远相同 ⟺ A ⇔ B。）

#### 小例子验证

**断言**：¬¬P ⇔ P（双重否定）

|P|¬P|¬¬P|
|---|---|---|
|T|F|T|
|F|T|F|

¬¬P 这一列和 P 这一列完全相同，所以 ¬¬P ⇔ P。✅

### 三、基本等价式（"逻辑代数"的基本公式）

下面这个表是**整个命题逻辑的代数工具箱**。**不需要死记**，我会帮你分类理解，做几道题就自然记住了。

我按功能分成 5 组。

#### 3.1 基本律（和代数很像）

|名称|公式|
|---|---|
|**双重否定**|¬¬P ⇔ P|
|**幂等律 (Idempotent)**|P ∧ P ⇔ P； P ∨ P ⇔ P|
|**交换律 (Commutative)**|P ∧ Q ⇔ Q ∧ P； P ∨ Q ⇔ Q ∨ P|
|**结合律 (Associative)**|(P ∧ Q) ∧ R ⇔ P ∧ (Q ∧ R)； ∨ 同理|
|**分配律 (Distributive)**|P ∧ (Q ∨ R) ⇔ (P ∧ Q) ∨ (P ∧ R)  <br>P ∨ (Q ∧ R) ⇔ (P ∨ Q) ∧ (P ∨ R)|

**观察**：这和实数的加法乘法非常像！

- ∧ 类似 ×（乘法）
- ∨ 类似 +（加法）
- 但**逻辑里有一个代数没有的性质**：∨ 对 ∧ 也满足分配律！（代数里 a + (b×c) ≠ (a+b)×(a+c)）

#### 3.2 同一律与零律（单位元与吸收元）

|名称|公式|
|---|---|
|**同一律**|P ∧ T ⇔ P； P ∨ F ⇔ P|
|**零律 (Domination)**|P ∧ F ⇔ F； P ∨ T ⇔ T|

**类比理解**：

- T 对于 ∧ 像 1 对于 ×（不改变对方）
- F 对于 ∨ 像 0 对于 +（不改变对方）
- F 对于 ∧ 像 0 对于 ×（清零）
- T 对于 ∨ 的作用是"吸收"，没有代数对应物

#### 3.3 否定律（排中与矛盾）

|名称|公式|
|---|---|
|**排中律**|P ∨ ¬P ⇔ T|
|**矛盾律**|P ∧ ¬P ⇔ F|

#### 3.4 德摩根律（De Morgan's Laws）⭐

> ¬(P∧Q)⇔¬P∨¬Q\neg(P \land Q) \Leftrightarrow \neg P \lor \neg Q¬(P∧Q)⇔¬P∨¬Q
> 
> > $$\neg(P \lor Q) \Leftrightarrow \neg P \land \neg Q$$

**这是你最常用、最重要的一条规则**。记忆口诀：**"否定进去，且变或，或变且"**。

**直觉理解**：

- "**不是（又下雨又刮风）**" ⟺ "**不下雨 或 不刮风**"（两件事至少有一件没发生）
- "**既不下雨也不刮风**" ⟺ "**不（下雨 或 刮风）**"（两件事都没发生）

**电子工程联系**：这条规则是数字电路化简的核心。你未来学的**卡诺图**本质上就是德摩根律的图形化。而且它告诉我们：**NAND 门（与非）或 NOR 门（或非）单独就能实现所有逻辑功能**，这是 CMOS 工艺的基础。

#### 3.5 吸收律（Absorption）

|名称|公式|
|---|---|
|**吸收律**|P ∧ (P ∨ Q) ⇔ P； P ∨ (P ∧ Q) ⇔ P|

**直觉验证**：P ∧ (P ∨ Q) —— 如果 P 真，那 (P ∨ Q) 也真，整体为真 = P；如果 P 假，整体为假 = P。所以就等于 P，Q 被"吸收"了。

#### 3.6 蕴含与等价的消去（极其重要）⭐

蕴含 → 和等价 ↔ 可以用 ¬、∧、∨ 表达出来：

|名称|公式|
|---|---|
|**蕴含等值式**|P → Q ⇔ ¬P ∨ Q|
|**等价等值式**|P ↔ Q ⇔ (P → Q) ∧ (Q → P) ⇔ (P ∧ Q) ∨ (¬P ∧ ¬Q)|
|**逆否等价**|P → Q ⇔ ¬Q → ¬P|

**蕴含等值式的意义非常深远**：它告诉我们 → 其实是多余的 —— 只要有 ¬ 和 ∨ 就能表达它。化简公式时，**第一步往往就是把所有 → 和 ↔ 消掉**。

**验证蕴含等值式**：

|P|Q|P → Q|¬P|¬P ∨ Q|
|---|---|---|---|---|
|T|T|T|F|T|
|T|F|F|F|F|
|F|T|T|T|T|
|F|F|T|T|T|

"P → Q" 列和 "¬P ∨ Q" 列相同，等价。✅

### 四、怎么用这些等价式？—— 等价变换实战

#### 例 1：化简一个公式

**化简**：¬(P ∨ (¬P ∧ Q))

我们一步步来，**每一步标明用了哪条规则**（养成好习惯，做证明题时老师看的就是这个）：

$$\begin{aligned} \neg(P \lor (\neg P \land Q)) &\Leftrightarrow \neg P \land \neg(\neg P \land Q) \quad \text{（德摩根律）} \ &\Leftrightarrow \neg P \land (\neg\neg P \lor \neg Q) \quad \text{（德摩根律）} \ &\Leftrightarrow \neg P \land (P \lor \neg Q) \quad \text{（双重否定）} \ &\Leftrightarrow (\neg P \land P) \lor (\neg P \land \neg Q) \quad \text{（分配律）} \ &\Leftrightarrow F \lor (\neg P \land \neg Q) \quad \text{（矛盾律）} \ &\Leftrightarrow \neg P \land \neg Q \quad \text{（同一律，F ∨ A ⇔ A）} \end{aligned}$$

**化简结果**：¬P ∧ ¬Q。

我们**用真值表验证一下**（确认没算错）：

|P|Q|原式 ¬(P ∨ (¬P ∧ Q))|¬P ∧ ¬Q|
|---|---|---|---|
|T|T|¬(T∨F) = F|F|
|T|F|¬(T∨F) = F|F|
|F|T|¬(F∨T) = F|F|
|F|F|¬(F∨F) = T|T|

两列一致，化简正确。✅

#### 例 2：证明一个重言式

**证明**：(P → Q) ∧ (P → R) ⇔ P → (Q ∧ R)

这是个有用的等价式："P 同时推出 Q 和 R" 就是 "P 推出 Q 且 R"。

$$\begin{aligned} (P \to Q) \land (P \to R) &\Leftrightarrow (\neg P \lor Q) \land (\neg P \lor R) \quad \text{（蕴含等值）} \ &\Leftrightarrow \neg P \lor (Q \land R) \quad \text{（分配律，反向）} \ &\Leftrightarrow P \to (Q \land R) \quad \text{（蕴含等值，反向）} \end{aligned}$$

**只用了三步**！这比列 8 行真值表快多了。

#### 例 3：证明一个公式是重言式（不列真值表）

**证明**：((P → Q) ∧ P) → Q 是重言式。

$$\begin{aligned} ((P \to Q) \land P) \to Q &\Leftrightarrow \neg((P \to Q) \land P) \lor Q \quad \text{（蕴含等值）} \ &\Leftrightarrow \neg(P \to Q) \lor \neg P \lor Q \quad \text{（德摩根）} \ &\Leftrightarrow \neg(\neg P \lor Q) \lor \neg P \lor Q \quad \text{（蕴含等值）} \ &\Leftrightarrow (P \land \neg Q) \lor \neg P \lor Q \quad \text{（德摩根 + 双重否定）} \ &\Leftrightarrow (P \land \neg Q) \lor (\neg P \lor Q) \quad \text{（结合律）} \end{aligned}$$

观察：¬P ∨ Q 就是 P → Q。而 P ∧ ¬Q 就是 ¬(P → Q)。所以整个式子变成：

¬(P→Q)∨(P→Q)⇔T（排中律）\neg(P \to Q) \lor (P \to Q) \Leftrightarrow T \quad \text{（排中律）}¬(P→Q)∨(P→Q)⇔T（排中律）

**证毕。** 是重言式。

### 五、等价变换的一般策略

做化简题时的**通用流程**（我帮你总结）：

1. **消除 → 和 ↔**，全部用 ¬、∧、∨ 表达（用蕴含等值式）
2. **把 ¬ 往里推**（用德摩根律），直到 ¬ 只作用在原子命题上
3. **用双重否定消除多余的 ¬¬**
4. **用分配律、结合律、交换律整理**
5. **用幂等、吸收、同一、零律、排中、矛盾律消去冗余**

做多了你会形成"代数直觉"，看一眼就知道往哪推。

### 六、代入原理与替换原理（两个容易混淆的规则）

这两个规则初学时容易分不清，我特别强调一下。

#### 代入原理 (Substitution)

> 在**重言式**中，用任意公式统一代换某个命题变元，结果仍是重言式。

**例子**：P ∨ ¬P 是重言式。把所有 P 都换成 (Q → R)：

(Q→R)∨¬(Q→R)(Q \to R) \lor \neg(Q \to R)(Q→R)∨¬(Q→R)

这仍然是重言式（相当于把 P 理解成任何公式都行）。

**关键限制**：**必须"统一代换"**，不能只换其中一个 P。

#### 替换原理 (Replacement)

> 在一个公式中，用**逻辑等价**的公式替换其中某一部分，得到的新公式与原公式**逻辑等价**。

**例子**：在 (P → Q) ∨ R 中，把 "P → Q" 替换成等价的 "¬P ∨ Q"：

(P→Q)∨R⇔(¬P∨Q)∨R(P \to Q) \lor R \Leftrightarrow (\neg P \lor Q) \lor R(P→Q)∨R⇔(¬P∨Q)∨R

**关键区别**：

- 代入：把**变元**换成**公式**，**只能用于重言式**，保持重言性。
- 替换：把**子公式**换成**等价的子公式**，**用于任意公式**，保持等价性。

做等价变换时我们用的全是**替换原理**。
