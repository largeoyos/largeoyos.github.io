# 离散数学入门：第 1 节 —— 开课导引与学习地图


### 一、什么是"离散"？

先问你一个小问题来热身：

> 你觉得"连续"和"离散"的区别是什么？能不能各举一个生活中的例子？

在你心里先想一想，我给你我的理解：

- **连续 (continuous)**：像水龙头里流出的水、时间的流逝、一根光滑的曲线。两点之间总能再塞进无数个点。高中学的微积分、极限、实数函数，研究的都是连续对象。
- **离散 (discrete)**：像一粒一粒的米、楼梯的台阶、QQ 好友列表里的人。对象是**一个一个分开的**，可以数、可以列。

离散数学，就是研究这些"一粒一粒"的对象的数学：整数、集合、逻辑命题、图（点和边）、算法步骤……

一个关键直觉：**计算机里的一切本质上都是离散的**（0 和 1，没有"0.5 个比特"）。所以离散数学是计算机科学的数学基础，就像微积分是物理学的数学基础一样。

### 二、这门课你会学到什么

标准的离散数学课程大纲通常包含这几大板块。我画一下我们的"学习路线图"：

|章节|主题|核心回答的问题|
|---|---|---|
|1|数理逻辑|什么是"正确的推理"？|
|2|集合论|如何精确描述"一堆东西"？|
|3|关系与函数|对象之间如何"关联"？|
|4|数论基础|整数有什么神奇性质？（密码学基础）|
|5|组合数学|如何"数"复杂的东西？|
|6|图论|点和连线能建模什么？|
|7|代数结构|群、环、域入门|

我们会**从第 1 章"数理逻辑"开始**，因为后面所有章节的证明和推理，都要用到逻辑这套语言。

### 三、学这门课的心态建议

离散数学和高中数学最大的不同是：

1. **它不是在"算"，而是在"证"**。你做的大部分题不会是"求 x 等于多少"，而是"证明 X 对所有情况都成立"。
2. **定义极其重要**。每个新词都有精确的定义，模糊理解必栽跟头。我会在每次引入新概念时，让你用自己的话复述一下。
3. **符号是朋友，不是敌人**。一开始你会觉得 ∀∃⇒∧∨ 这些符号很烦，但它们其实是为了让思想表达得**更精确、更短**。
