# 离散数学 第 2 节：命题与逻辑联结词

我们正式进入第一章：**数理逻辑**。今天这一节的目标是彻底搞懂两个东西 —— **命题 (proposition)** 和 **五个基本联结词**。

---

### 一、什么是命题？

我先给你看几个句子，你来判断哪些是"命题"，哪些不是：

1. "北京是中国的首都。"
2. "1 + 1 = 3"
3. "请关门。"
4. "x + 1 = 5"
5. "今天天气真好啊！"
6. "存在最大的质数。"

在继续往下看之前，**你先在心里分一下类**，并试着总结："命题"应该满足什么条件？

---

好，我公布答案并给出严格定义：

> **定义（命题）**：一个**能够判断真假**的**陈述句**，称为命题 (proposition)。它的真假值（非真即假）称为它的**真值 (truth value)**，记作 T (true) 或 F (false)，在电子工程里你也会看到写成 1 和 0。

按这个定义回头看：

|句子|是命题吗？|理由|
|---|---|---|
|北京是中国的首都|✅ 是，真值 T|陈述句，事实为真|
|1 + 1 = 3|✅ 是，真值 F|陈述句，事实为假|
|请关门|❌ 不是|祈使句，谈不上真假|
|x + 1 = 5|❌ 不是|x 未确定，真假不定（这叫**谓词**，下一章讲）|
|今天天气真好啊！|❌ 不是|感叹句 + 主观判断|
|存在最大的质数|✅ 是，真值 F|陈述句，虽然我们需要证明才知道它假|

**两个关键点要抓住**：

- **必须是陈述句**：疑问句、祈使句、感叙句都不是命题。
- **必须有确定真假值**：哪怕你现在不知道它真假，只要客观上有唯一答案，就是命题。比如"哥德巴赫猜想成立"是命题（虽然人类暂时还没证明）。

下面这几个是不是命题？真值是什么？

- (a) "√2 是无理数"
- (b) "你吃饭了吗？"

- (d) "地球上现在活着的人超过 10 亿。"

（c) 比较有意思 —— 你能看出它的问题吗？先自己想想，我们稍后会回到它。

### 二、原子命题与复合命题

看这两个命题：

- P："今天下雨。"
- Q："我带伞。"

现在我把它们**组合**起来：

- "今天下雨**并且**我带伞。"
- "**如果**今天下雨，**那么**我带伞。"
- "今天**不**下雨。"

这些加粗的"并且/如果…那么/不"就是**逻辑联结词 (logical connectives)**。它们像是电路里的逻辑门 —— 把简单的信号组合成复杂的信号。

> **定义**：
> 
> - **原子命题 (atomic proposition)**：不能再拆分的最简命题，通常用 P, Q, R, … 表示。
> - **复合命题 (compound proposition)**：由原子命题通过联结词组合而成。

在电子信息工程里，你可以把**原子命题类比成一根输入导线（0/1 信号）**，联结词就是**逻辑门 (AND/OR/NOT/…)**。这个类比几乎是一一对应的，后面你会越来越感受到。

---

### 三、五个基本联结词

一共就五个，我们一个一个来。

#### ① 否定 ¬ (NOT)

符号：**¬P**（有的教材写作 ~P 或 !P），读作"非 P"。

含义：**P 是真，¬P 就是假；P 是假，¬P 就是真。**

**真值表 (truth table)**：

|P|¬P|
|---|---|
|T|F|
|F|T|

例子：

- P = "今天是星期三"，¬P = "今天不是星期三"。
- 电路：这就是一个**非门 (inverter)**，输入 1 输出 0，输入 0 输出 1。

⚠️ **小陷阱**：自然语言里的"否定"有时很绕。"我不认为他会来"其实否定的是"他会来"，不是"我认为"。写逻辑时一定要先把原命题说清楚再取反。

---

#### ② 合取 ∧ (AND)

符号：**P ∧ Q**，读作"P 且 Q"、"P 与 Q"。

含义：**当且仅当 P 和 Q 都为真时，P ∧ Q 才为真。**

**真值表**：

|P|Q|P ∧ Q|
|---|---|---|
|T|T|**T**|
|T|F|F|
|F|T|F|
|F|F|F|

例子：

- "今天下雨" ∧ "我带伞" = "今天下雨并且我带伞"。两件事都成立才算真。
- 电路：**与门 (AND gate)**，两个输入都是 1 时输出才是 1。

---

#### ③ 析取 ∨ (OR)

符号：**P ∨ Q**，读作"P 或 Q"。

**这里要特别小心！** 数学逻辑里的"或"是**相容或 (inclusive or)**，也就是"至少一个真"，**包括两个都真的情况**。

**真值表**：

|P|Q|P ∨ Q|
|---|---|---|
|T|T|**T** ← 注意！|
|T|F|T|
|F|T|T|
|F|F|F|

例子："这道菜有葱**或**有蒜" —— 在逻辑里，既有葱又有蒜也算真。这和日常生活里"要么 A 要么 B（二选一）"的意思不一样。

⚠️ 日常的"二选一"叫**异或 (exclusive or, XOR)**，记作 **P ⊕ Q**，只有一个真时才真，两个都真反而变假。虽然不属于"五个基本联结词"，但你在数字电路里一定会遇到，我顺便列一下真值表：

|P|Q|P ⊕ Q|
|---|---|---|
|T|T|F|
|T|F|T|
|F|T|T|
|F|F|F|

#### 小检验 ✍️

试着判断下面每句话的真值：

- "1 + 1 = 2 **∧** 2 + 2 = 5"
- "1 + 1 = 2 **∨** 2 + 2 = 5"
- "1 + 1 = 3 **∨** 2 + 2 = 5"

#### ④ 蕴含 → (IF … THEN …)

这是**最难理解也最容易出错的联结词**，我们慢慢来。

符号：**P → Q**，读作"如果 P 那么 Q"、"P 蕴含 Q"、"P 推出 Q"。

- P 叫**前件 (hypothesis / antecedent)**
- Q 叫**后件 (conclusion / consequent)**

**真值表**（请仔细看，尤其是后两行）：

|P|Q|P → Q|
|---|---|---|
|T|T|T|
|T|F|**F** ← 唯一为假的情况！|
|F|T|**T**|
|F|F|**T**|

第一次看这张表的人 99% 都会问："P 是假的时候，为什么 P → Q 还是真？"

我给你举一个生活化的例子来建立直觉：

> 假设我向你承诺：**"如果明天下雨，我就请你吃饭。"** （P = 明天下雨，Q = 我请你吃饭）
> 
> 什么情况下你会说"你撒谎了！违背承诺！"？让我们四种情况都看一遍：
> 
> - **下雨了，请了饭**（T, T）→ 我守约，承诺为真。✅
> - **下雨了，没请饭**（T, F）→ 我违约了，承诺为假。❌ **只有这种情况承诺是假的！**
> - **没下雨，请了饭**（F, T）→ 我没违约（我只承诺了下雨才请，没下雨请你是额外的好意），承诺仍为真。✅
> - **没下雨，也没请饭**（F, F）→ 我也没违约（前提条件没发生），承诺仍为真。✅

关键洞察：**"如果 P 那么 Q"只在"P 成立但 Q 没发生"时算违约。** 当 P 本身就不成立（前件假），无论 Q 如何，这个承诺都没被破坏。

这在数学上叫 **"空虚为真 (vacuously true)"**。比如：

- "如果 2 + 2 = 5，那么我是月亮人。" —— 这句话**是真的**（因为前件永远为假）。
- "如果 x > 10 且 x < 5，那么 x = 100。" —— 也是真的（前件不可能成立）。

听起来很奇怪，但这个定义是**为了让逻辑系统自洽**而设计的。你现在先**死记住这张真值表**，随着做题你会慢慢接受它的合理性。

#### 蕴含的几种中文说法

在数学书里，下面这些说法**都表示 P → Q**，不要被不同措辞骗了：

- 如果 P，那么 Q
- 若 P，则 Q
- P 蕴含 Q
- P 是 Q 的**充分条件** (P 够了就行)
- Q 是 P 的**必要条件** (没 Q 就没 P)
- 只有 Q 才 P
- 除非 ¬Q，否则 ¬P

这里"充分必要"是重点，我举个例子帮你记：

- "下雨 → 地湿"：下雨**足以让**地湿（充分），但地湿不一定是因为下雨（下雨不是必要的，洒水车也行）。

---

#### ⑤ 等价 ↔ (IF AND ONLY IF)

符号：**P ↔ Q**，读作"P 当且仅当 Q"、"P 等价于 Q"。

含义：**P 和 Q 真值相同时为真，不同时为假。**

**真值表**：

|P|Q|P ↔ Q|
|---|---|---|
|T|T|T|
|T|F|F|
|F|T|F|
|F|F|T|

例子：

- "三角形是等边的 **当且仅当** 它的三个角相等" —— 这两件事总是同时成立或同时不成立。
- 电路：这就是**同或门 (XNOR)**，刚好是异或的反。

中文常见说法：

- P 当且仅当 Q
- P 是 Q 的**充分必要条件**（简称**充要条件**）
- P 与 Q 等价
- P iff Q（英文缩写）

### 四、联结词的优先级

当一个式子里出现多个联结词时（就像算术里的 + - × ÷），要有优先级：

¬  >  ∧  >  ∨  >  →  >  ↔\neg \;>\; \land \;>\; \lor \;>\; \to \;>\; \leftrightarrow¬>∧>∨>→>↔

也就是：**¬ 最紧，↔ 最松。** 不确定时就**多加括号**，这是工程师的好习惯。

例子：**¬P ∧ Q → R** 等价于 **((¬P) ∧ Q) → R**。

---

### 五、一个综合例子

把下面这句话翻译成逻辑表达式：

> "如果今天下雨而我没带伞，那么我会迟到或者淋湿。"

先定义原子命题：

- P：今天下雨
- Q：我带伞
- R：我迟到
- S：我淋湿

翻译过程一步一步拆：

- "今天下雨而我没带伞" = P ∧ ¬Q
- "我迟到或者淋湿" = R ∨ S
- "如果……那么……"把前两部分用 → 连起来

最终：**(P ∧ ¬Q) → (R ∨ S)**
