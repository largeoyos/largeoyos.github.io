### 三、指数的核心性质 ⭐⭐⭐

#### 3.1 指数整除欧拉函数

> **定理 1**：ordn(a)∣φ(n)ordn​(a)∣φ(n)。

**证明**：

设 d=ordn(a)d=ordn​(a)。用带余除法：φ(n)=qd+rφ(n)=qd+r，0≤r<d0≤r<d。

由欧拉定理：aφ(n)≡1(modn)aφ(n)≡1(modn)。

aφ(n)=aqd+r=(ad)q⋅ar≡1q⋅ar=ar(modn)aφ(n)=aqd+r=(ad)q⋅ar≡1q⋅ar=ar(modn)

所以 ar≡1(modn)ar≡1(modn)。

但 d 是最小的让 ak≡1ak≡1 的正整数，且 0≤r<d0≤r<d。**所以 r 必为 0**，即 d∣φ(n)d∣φ(n)。∎

**这条定理**就是上节学过的拉格朗日定理推论的特例——Zn∗Zn∗​ 在乘法下是群，元素的阶整除群的阶 φ(n)φ(n)。

**实战意义**：找指数时，**只需要测试 φ(n)φ(n) 的因子**，不用一个个试。

#### 3.2 a^k ≡ 1 的判据

> **定理 2**：ak≡1(modn)ak≡1(modn) ⟺ ordn(a)∣kordn​(a)∣k。

**证明思路**：用同样的带余除法论证。

**应用**：要判断 ak≡1(modn)ak≡1(modn) 是否成立，**只需检查 ordn(a)ordn​(a) 是否整除 k**。

**例**：ord7(2)=3ord7​(2)=3，那么 2300≡1(mod7)2300≡1(mod7)？300/3=100300/3=100，整除，**是的**。

#### 3.3 指数的精确公式

> **定理 3**：ordn(ak)=ordn(a)gcd⁡(k,ordn(a))ordn​(ak)=gcd(k,ordn​(a))ordn​(a)​。

### 四、原根（Primitive Root）⭐⭐⭐

#### 4.1 定义

> **定义**：若 ordn(g)=φ(n)ordn​(g)=φ(n)（**最大可能的指数**），则称 g 是**模 n 的原根**。

**等价说法**：g 的幂 g1,g2,…,gφ(n)g1,g2,…,gφ(n) 跑遍 Zn∗Zn∗​ 中所有元素。

**这意味着**：Zn∗Zn∗​ **是循环群**，gg 是它的生成元！

#### 4.2 例子

回到 Z7Z7​：

- 3 的指数是 6 = φ(7)φ(7)，**3 是原根**
- 5 的指数是 6，**5 是原根**
- 2, 4 的指数是 3，**不是原根**
- 6 的指数是 2，**不是原根**

**验证 3 是原根**：31,32,33,34,35,36(mod7)31,32,33,34,35,36(mod7) = 3, 2, 6, 4, 5, 1。**确实跑遍了 {1,2,3,4,5,6}{1,2,3,4,5,6}**。✅

#### 4.3 关键问题：原根是否总存在？

回到 Z12Z12​ 的例子：所有非零元素的指数都 ≤ 2，**没有指数是 4 = φ(12)φ(12) 的元素**。

**所以 Z12Z12​ 没有原根**！Z12∗Z12∗​ **不是循环群**！

那么**什么样的 n 有原根**呢？

#### 4.4 原根存在性定理 ⭐⭐⭐

> **定理（高斯）**：模 n 有原根 ⟺ n∈{1,2,4,pk,2pk}n∈{1,2,4,pk,2pk}，其中 p 是**奇质数**。

**也就是说**：Zn∗Zn∗​ 是循环群当且仅当 nn 是这种特殊形式。

**特别地**：

- **每个奇质数 p 都有原根** ✅（这是密码学的关键）
- n=2kn=2k（k ≥ 3）**没有原根**
- n=pqn=pq（不同奇质数）**没有原根**

**Z12=Z4⋅3Z12​=Z4⋅3​，不是上述形式**，所以无原根。

#### 4.5 原根的个数

> **定理**：若 n 有原根，则 n 共有 φ(φ(n))φ(φ(n)) 个原根。

**例**：Z7Z7​ 的原根个数 = φ(φ(7))=φ(6)=2φ(φ(7))=φ(6)=2。**确实是 3 和 5**。

**例**：Z11Z11​ 的原根个数 = φ(10)=4φ(10)=4。

**怎么找全部原根**？

如果 g 是原根，则 gkgk 也是原根 ⟺ gcd⁡(k,φ(n))=1gcd(k,φ(n))=1。

**例**：Z11Z11​ 中 g = 2 是原根（验证留作练习）。所有原根：

- g1=2g1=2
- g3=8g3=8
- g7=27=128≡7(mod11)g7=27=128≡7(mod11)
- g9=29=512≡6(mod11)g9=29=512≡6(mod11)

所有原根：**{2, 6, 7, 8}**。

### 五、离散对数（Discrete Logarithm）⭐⭐⭐

#### 5.1 定义

设 g 是模 n 的原根。则对每个 a∈Zn∗a∈Zn∗​，**存在唯一的 k** (0≤k<φ(n)0≤k<φ(n)) 使

gk≡a(modn)gk≡a(modn)

> **离散对数**：这个 kk 称为 a 关于基 g 的**离散对数（指数）**，记作 k=indg(a)k=indg​(a) 或 log⁡galogg​a。

#### 5.2 例子

在 Z7Z7​ 中，原根 g = 3。从前面的表：

- 31=331=3，log⁡33=1log3​3=1
- 32=232=2，log⁡32=2log3​2=2
- 33=633=6，log⁡36=3log3​6=3
- 34=434=4，log⁡34=4log3​4=4
- 35=535=5，log⁡35=5log3​5=5
- 36=136=1，log⁡31=6log3​1=6（或 0）

#### 5.3 离散对数的性质

完全像普通对数：

- log⁡g(ab)≡log⁡ga+log⁡gb(modφ(n))logg​(ab)≡logg​a+logg​b(modφ(n))
- log⁡g(ak)≡klog⁡ga(modφ(n))logg​(ak)≡klogg​a(modφ(n))
- log⁡g1≡0(modφ(n))logg​1≡0(modφ(n))

#### 5.4 离散对数问题（DLP）⭐

正向：给 g,k,ng,k,n，算 gk  ngkmodn —— **快速幂算法 O(log⁡k)O(logk)**，**容易**。

逆向：给 g,a,ng,a,n，求 k 使 gk≡a(modn)gk≡a(modn) —— **离散对数问题**，**困难**！

**目前最好算法**：

- 一般有限域 FpFp​：exp⁡(O((log⁡p)1/3(log⁡log⁡p)2/3))exp(O((logp)1/3(loglogp)2/3)) —— 亚指数
- 椭圆曲线群：O(n)O(n​) —— **指数**！这就是为什么 ECC 比 RSA 高效

**离散对数问题的困难性是 Diffie-Hellman、ElGamal、ECDSA 的安全基础**。

### 六、Diffie-Hellman 密钥交换（DH）⭐⭐⭐

终于到了原根最重要的应用——**人类历史上第一个公钥协议**（1976）。

#### 6.1 问题

Alice 和 Bob 想在公开网络上**协商一个共享密钥**，但他们之前没见过面，**而且整个通信都被 Eve 监听**。

如何在公开通信下产生一个**只有 Alice 和 Bob 知道**的密钥？

**1976 年之前，所有人都认为这不可能**。

#### 6.2 DH 协议

**公开参数**（事先公布）：大质数 p，p 的原根 g。

**协议**：

1. **Alice 私下选 a**，发送 A=gamod  pA=gamodp 给 Bob
2. **Bob 私下选 b**，发送 B=gbmod  pB=gbmodp 给 Alice
3. **Alice 计算**：K=Bamod  p=gbamod  pK=Bamodp=gbamodp
4. **Bob 计算**：K=Abmod  p=gabmod  pK=Abmodp=gabmodp

**两人得到相同的 K=gabmod  pK=gabmodp**！这就是共享密钥。

#### 6.3 安全性

**Eve 看到**：p,g,A=ga,B=gbp,g,A=ga,B=gb。

**Eve 想算**：K=gabK=gab。

**问题**：从 gaga 和 gbgb 算 gabgab（不知道 a 或 b），需要解**离散对数问题**——**困难**！

这就是 **Diffie-Hellman 假设**：从 ga,gbga,gb 推 gabgab 是难题。

#### 6.4 完整例子（玩具大小）

设 p=23p=23，g=5g=5（5 是 23 的原根）。

- Alice 选 a=6a=6，计算 A=56mod  23A=56mod23。52=25≡252=25≡2，54=454=4，56=54⋅52=856=54⋅52=8。**A = 8**。
- Bob 选 b=15b=15，计算 B=515mod  23B=515mod23。算出 B=19B=19。
- Alice 计算 K=196mod  23=2K=196mod23=2。
- Bob 计算 K=815mod  23=2K=815mod23=2。

**共享密钥 K = 2** ✅

实战中 p 是 2048 位以上质数，a, b 是数百位随机数。

#### 6.5 现代应用

- **HTTPS / TLS**：协商对称加密的会话密钥
- **WhatsApp / Signal**：端到端加密的密钥协商
- **WireGuard / IKE**：VPN 协议
- **比特币 / 以太坊**：使用椭圆曲线版的 DH

**你每次访问 https:// 网址，浏览器都在和服务器跑 DH 或 ECDH 协议**。

#### 6.6 中间人攻击（MITM）

DH 不防**冒充**！如果 Eve 能拦截通信：

```
Alice ⇄ Eve ⇄ Bob
```

Eve 可以分别和 Alice、Bob 各跑一次 DH，得到两组密钥，然后中转通信，**Alice 和 Bob 都以为自己在和对方通信**。

**解决**：DH + **数字签名**（验证身份）= **认证 DH**。这就是 TLS 1.3 的设计。

### 七、ElGamal 加密（基于 DH 的公钥加密）

#### 7.1 算法

**密钥生成**：选大质数 p、原根 g，私钥 xx，公钥 h=gxmod  ph=gxmodp。

**加密**（明文 m）：

1. 选随机 k
2. 计算 c1=gkmod  pc1​=gkmodp，c2=m⋅hkmod  pc2​=m⋅hkmodp
3. 密文 = (c1,c2)(c1​,c2​)

**解密**（用私钥 x）：

m=c2⋅(c1x)−1mod  pm=c2​⋅(c1x​)−1modp

**为什么对**？c1x=gkx=hkc1x​=gkx=hk，所以 c2/hk=mc2​/hk=m。✅

#### 7.2 ElGamal 的特点

**1. 概率性加密**：每次加密都用随机 k，**同一明文每次密文不同**。这是 RSA 没有的。

**2. 同态性**：E(m1)⋅E(m2)=E(m1m2)E(m1​)⋅E(m2​)=E(m1​m2​)。即"密文相乘 = 明文相乘的密文"。

**这是同态加密的早期形式**，是隐私计算的基础。

**3. 需要安全随机数**：k 必须真随机且不重复。**Sony PS3 就因 k 重复用，被破解**。

#### 7.3 数字签名版：DSA

DSA（Digital Signature Algorithm）和 ECDSA（椭圆曲线版）是 ElGamal 的变种，专门用于数字签名。

**应用**：

- DSA：早期数字签名标准
- ECDSA：比特币、TLS、SSH 全部使用
- EdDSA / Ed25519：现代推荐（更快、更安全）

### 八、原根与密码学的核心地位

总结一下，**原根这个看似纯数学的概念，撑起了整个公钥密码学**：

|协议|数学难题|
|---|---|
|Diffie-Hellman|离散对数（DLP）|
|ElGamal|DLP|
|DSA / ECDSA|DLP / 椭圆曲线 DLP|
|Schnorr 签名|DLP|
|ZK 证明（zk-SNARK 等）|DLP|

**所有这些**都依赖：

1. **存在原根**（即 Zp∗Zp∗​ 是循环群）→ 高斯定理保证
2. **DLP 是难题**（无快速算法）→ 经验观察 + 部分理论支持
