---

### 第一幕:URL 是什么?

先看你输入的这串字符。它的学名叫 **URL(Uniform Resource Locator,统一资源定位符)**,结构是:

```
https://www.baidu.com:443/index.html
  ↑          ↑         ↑     ↑
  协议      域名      端口   路径
```

📌 **几个关键点**:

- **协议**:告诉浏览器用什么规则去取资源(这里是 HTTPS)
- **域名**:`www.baidu.com` 是一个给人看的名字
- **端口**:HTTPS 默认 443,HTTP 默认 80,通常省略
- **路径**:服务器上的资源位置

#### 🙋 第一个问题给你

计算机网络里,数据包从源头到目的地,依靠的是 **IP 地址**(比如 `110.242.68.66`),而不是 `www.baidu.com` 这个字符串。

**那么问题来了:浏览器拿到 `www.baidu.com` 这个域名后,怎么知道要去哪个 IP 地址?它得先做什么?**
