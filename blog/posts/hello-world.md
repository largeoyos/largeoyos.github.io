# 为什么我要自己写一个博客引擎？

在今天（也就是2026年4月25日），我终于为自己的个人主页 `largeoyos.github.io` 添加了完整的博客功能！

以前我的主页主要是用来导航到各种不同的有趣项目。但一直希望能有一个简单的、轻量级的基于 Markdown 的纯前端动态渲染博客。

## 这个博客有哪些功能？

1. **原生的 Markdown 解析**：你看到的加粗、代码块等都是直接从 `.md` 文件中解析出的。
2. **自动化时间排序**：通过一个 `posts.json` 索引，主页总是显示最新动态。
3. **标签/分类系统**：可以快捷过滤想要阅读的技术文章或碎碎念。

- 把博客扩展出评论系统
- 用 AI 让生成博客更加自动化
